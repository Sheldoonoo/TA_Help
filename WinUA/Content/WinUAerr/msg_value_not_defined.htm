Value Not Defined

 

Value not defined.

 

Problem:

You cannot leave the Value entry field blank. You must assign a value on the Assign Clipboard window or Test Variable window before clicking OK.

 

Solution:

Enter a value in the entry field. The value can be another variable name or a number. Numbers are always stored as floating point numbers. The value can be a specific integer (using Assign Variable) or a value that points to a number in a clipboard list (using Test Variable (Macro Insert Menu)).
