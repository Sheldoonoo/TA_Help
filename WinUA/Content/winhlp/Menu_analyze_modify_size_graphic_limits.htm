Graphic Limits

Use the Graphic Limits option to enter start (100%) and final (0%) sample weights by choosing corresponding X-axis points (temperature or time) using the markers.

 

Follow these steps to perform this analysis:

 

1. Select Graphic Limits from the Analyze/Modify Size menu. Markers will be displayed on the graph.

2. Double click on the desired place on the curve to position the marker at the beginning of the region of the curve you want to analyze.

3. Double click on the desired place on the curve to position the next marker at the end of the region you want to analyze.

4. Right click to display the Analyze pop-up menu, then select Accept Limits (or press Enter). The modified size values are displayed for verification using the Modified Size window.

5. Click the OK button to accept the modified weight or Cancel to exit the analysis. If you accept the results, the program adjusts the percent axis (y-axis) to the modified weight.

 

Any analyses that you perform after modifying the size are based on the new modified size; previous analysis results are unaffected.

 

To restore the original size, select Graph/Data Limits/Full Range. All analyses performed will be lost.
