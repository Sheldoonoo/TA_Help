If

The If...Then...Else statement is used to evaluate whether a condition is True or False. Depending on the result, the statements between the If and the Endif or the Else and the Endif are executed.

Example 1

  If (V1 > 10 )
  DisplayMessage ("Condition was true.")
  [statements]
  Else
  DisplayMessage ("Condition was false.")
  [statements]
  Endif

NOTE: The Else function is optional and can be omitted.

Example 2

  If (V1 > 10 )
  DisplayMessage ("Condition was true.")
  [statements]
  Endif
