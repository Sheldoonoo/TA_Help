Selecting Post Test Actions

 

While an experiment is in progress you can change the process that occurs after the end of a test run.

  1. Right click on the run number in the Sequence Pane to display the run sequence pop-up menu.

  2. Select Post Run Action from the pop-up menu to display the Post Test Action window.

  3. Choose the desired action to be performed after the selected run has completed from the drop-down lists:

    • NONE: No action will be performed

    • IF, IF NOT, and DO: These options enable you to perform the specified analysis at the end of the experiment without operator intervention and decide the next action based on whether the results are successful (IF) or unsuccessful (IF NOT). Pass/Fail results are determined by user-defined verification criteria and calibration analysis results. The DO option is performed irregardless of any pass/fail results.

Available Analyses:

Choose from NONE, Curie Temperature Verification (Q5000 IR), or Curie Temperature Calibration (Q5000 IR), or Humidity Verification (Q5000 SA).

Available Actions: These will be initiated after the chosen criteria (for IF or IF NOT) is met.

GOTO -- goes to the specified run number and continues the sequence

NEXT -- proceeds to the next run in the sequence

STOP -- stops the sequence

GO Next Sequence -- executes the next sequence (e.g., If calibration sequence is completed, goes to an experiment sequence and vice versa.)

    • JUMP TO:  When you choose JUMP TO, another list is displayed to allow you to choose which Run number the sequence will activate after completion of the selected run number.

    • STOP: Stops the experiment.

  1. Select Remove, if you want to delete that item.

  2. Select OK when finished.

You can also SKIP a run in a sequence by selecting this option in the run sequence pop-up menu for the desired run number.
