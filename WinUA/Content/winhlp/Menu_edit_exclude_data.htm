Exclude Data

Use the Exclude Data option to remove the selected range of data points from the data used for your analysis. It does not alter the data file. The Exclude Data functions can be used to:

(1) Provide a way to create very small data breaks between thermal cycles, which allows each cycle to be distinguished with a different symbol using the "Marked as invalid data" option. (See Graph Options.) or (2) Correct explainable data anomalies using the remaining Exclude Data options.

 

To remove data points from a curve, follow these steps:

  1. Select Exclude Data from the Edit menu.

  2. Position the first marker at the beginning of the region you wish to remove.

  3. Position the next marker at the end of the region you want to remove.

  4. Right click to display the pop-up menu and select Accept Limits or press Enter. The Exclude Data window is displayed.

5. Click on the desired option, then select OK. The results will be displayed.

 

NOTE: To restore the graph to its full range of data, use Graph/Data Limits/Full Range.

 

NOTE: The data file will not be changed if you use the "Marked as invalid data" option.

 

To remove data points from the spreadsheet view, follow these steps:

1. Select the cell(s) for the data point(s) to be removed.

2. Select Exclude Data from the Edit menu or from the right-click pop-up menu.

3. Click on the desired option, then select OK. The results will be displayed in the spreadsheet view.
