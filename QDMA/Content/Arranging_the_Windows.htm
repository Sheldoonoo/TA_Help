Arranging the Windows

 

To select or arrange the views of multiple instruments in the application window, choose one of the following menu items from the Windows menu:

Cascade application windows:  Arranges the open Q series instrument windows in an overlapped fashion.

Tile application windows horizontally:  Arranges the open Q series instrument windows in non-overlapping horizontal tiles.

Tile application windows vertically:  Arranges the open Q series instrument windows in non-overlapping vertical tiles.
