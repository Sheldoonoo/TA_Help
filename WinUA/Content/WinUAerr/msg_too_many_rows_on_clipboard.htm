Too Many Rows

 

Too many rows on clipboard.

 

This message is displayed when you attempt to paste too many rows from the clipboard into a smaller number of highlighted rows in the spreadsheet. For example, if you copy four rows of data to the clipboard and attempt to paste it into one highlighted row on the spreadsheet, you will get this message.

 

When more than one cell is highlighted in the spreadsheet, there must be a one-to-one match between the rows on the clipboard and the rows highlighted for pasting in the spreadsheet. If only one cell is highlighted in the spreadsheet, then the columns and rows from the clipboard will be pasted in as required starting with the column and row of the highlighted cell.
