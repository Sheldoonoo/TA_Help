Setting Up a Procedure Page

 

The Procedure Page is used to specify the test conditions and to save the custom test to a procedure file, if desired. The items displayed on this page are dependent upon the clamp and mode selected.

The procedure information displayed is applicable to the run number that is currently highlighted on the Sequence Pane and its selected mode and test. (Make sure that you choose the clamp and instrument mode on the Summary Page before setting up the procedure.)
