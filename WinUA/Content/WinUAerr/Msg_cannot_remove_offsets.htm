Cannot Remove Offsets

Cannot remove offsets.

 

Problem:

The X-axis cursor position from the first curve shift does not provide sufficient information to restore the original curve. The X-axis value of the first point in the curve is equal to that of the last point in the curve, and also to that of the first shift marker.

 

Solution:

Try reading in the original curve again and positioning the shift cross hair marker to a point on the curve with a different X-axis value from eight the first or last points in the curve. If all points in the curve have the same X-axis value, then the offset cannot be removed.
