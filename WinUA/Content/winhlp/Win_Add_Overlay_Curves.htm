Add Overlay Curves Window

Use this window to add curves to your curve overlay configuration. Each curve may be selected from a different data file, with all curves being selected from up to ten different data files. Curves added to the same axis must contain the same units.

 

NOTE: The curve is displayed on the overlay graph using the same conditions that were used in the single file mode. For example, the same data limits, analyses, etc. will be used.

 

Follow the steps below to add curves for overlay configuration:

1. Check the box next to the files that you wish to use from the list displayed in the Select files list. You can also use the Select All or Deselect All buttons to make your file choices.

2. Click on the desired radio button in the Show options. These buttons allow you to display either signals that are compatible between the files you have chosen or all of the signals in the selected files.

3. Select (highlight) the signal, shown in the Select signal list, that you want to display in the curve overlay.

4. Click the Save button, if you want to use these as your new default settings or click the OK button to use these settings for this session only.
