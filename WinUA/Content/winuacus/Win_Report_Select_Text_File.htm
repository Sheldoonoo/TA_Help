Report Select Text File Window

Use the Report Select Text File window to enter or edit the file name and path of the text that will be included in the custom report. Follow the steps below:

 

To enter new text from a file:

  1. Select Insert/File Text from the Custom Report menu.
  2. Enter the file name and path of the text file, or use the Browse button to locate the file.
  3. Click the Font button to open a window allowing you to choose the desired font.
  4. Apply the desired Text alignment or Fill color from the drop-down list of choices.
  5. Click the OK button when finished.

 

To edit existing text from a file:

  1. Highlight the desired text report object on the template.
  2. Select Params from the Edit menu or from the pop-up menu.
  3. Make the desired changes to the file name and path.
  4. Click the Font button to open a window allowing you to choose the desired font.
  5. Apply the desired Text alignment or Fill color from the drop-down list of choices.
  6. Click the OK button when finished.
