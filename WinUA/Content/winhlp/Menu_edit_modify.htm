Modify

Use this function to alter the data point value of a single cell in the spreadsheet view. The data value will be modified in both the spreadsheet and graph view so that you can perform analysis functions, etc. using the modified data.

 

To modify a value:

 

1. Select (highlight) the desired cell.

2. Select Edit/Modify. The Modify Data window is displayed.

3. Enter the desired data value.

4. Click OK. The data point value will be altered accordingly.

 

NOTE: This will not modify the data file.
