  1. Select Begin session from the TTS menu or the TTS toolbar
  2. Click on the Styrene 140C 100Pa-0005c.rsl results file, and set this as the reference temperature.
  3. Select Shift curves from the TTS menu.
  4. When the automatic curve shifting is complete, select Generate master curve from the TTS menu.

NOTE: If you wish, you can study the effect of manually adjusting the shift factors. Send the Creep TTS file to a new graph and tile the two graphs. Now click on curves on the main results graph and shift them. You can see the fit curve changing to match.
