Message 648

LNCS Cool Timeout.

Problem:

While cooling the LNCS must receive a "Cool" command at least once every second or it will revert to its default idle state and this error will be displayed.

Solution:

Try one of more of the following solutions, then retry your operation:

  • Check all cable connections between the instrument and the LNCS to make sure all connections are secure.
  • Reset the LNCS.
  • Reset the DSC.
  • If the error still persists, contact TA Instruments for service.
