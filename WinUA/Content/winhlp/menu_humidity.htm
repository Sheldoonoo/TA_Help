Humidity Analysis

The TA Instruments Humidity Analysis can be used to calculate the humidity composition of a sample based on data obtained from the Q5000 SA.

 

The sample is generally dried, then held at a given temperature and humidity for a period of time. The humidity is gradually increased (the range is from 5 to 95 % relative humidity) until the desired upper range is achieved. Experiments can be quite lengthy, measured in days. Applications include pharmaceutical and food science research.

 

Guidelines for Data Collection

When creating a humidity experiment, you need to take care in creating your method. The sorption isotherm, BET, and GAB plots are all contingent on data markers in the file at the end of each relative humidity isotherm. If you use the Step Humidity method segment, these markers are automatically added. If you use the Relative Humidity method segment, you need to manually add a Mark Data segment to the method. See the example below:

 

  1. Equilibrate at 25 °C

  2. Relative Humidity 0 %

  3. Isothermal 300 min

  4. Mark Data

  5. Step Humidity 5 % for 300 min to 95 %
