Checkbox Allow Multiple Sessions of UA

Check this box to enable a new session of the Universal Analysis program to open each time the icon is selected (multiple copies of the program will be running at the same time). If the box is not checked, the current session of the program will be activated each time the icon is selected.

 

NOTE: You must save this setting to the current initialization file by clicking the Save button on theMain Options  Initialization Page, or it will not take effect.
