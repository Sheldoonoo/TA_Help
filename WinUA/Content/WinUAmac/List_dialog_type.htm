Dialog Type (Macro Dialog)

Select the desired dialog type from the following list:

 

  • Information– When you select this type of message, the word "Information" is displayed in the title bar of the dialog and the icon, is displayed.

  • Confirmation– When you select this type of message, the word "Confirmation" is displayed in the title bar of the dialog and the icon, is displayed.

  • Warning– When you select this type of message, the word "Warning" is displayed in the title bar of the dialog and the icon, is displayed.

  • Error– When you select this type of message, the word "Error" is displayed in the title bar of the dialog and the icon, is displayed.

  • No Type– When you select this type of message, no icon is displayed with your text.
