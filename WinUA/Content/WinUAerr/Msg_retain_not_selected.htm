Retain Not Selected

Retain <result type>? [Result is not selected for plotting.]

 

This message occurs if you perform an analysis that is valid for a certain multiplexed signal (e.g., frequency) on a specific curve, then remove that signal from the displayed graph (using Edit/Frequency List for example), and then choose Edit/Delete Results/Select. When the program comes to the result that is no longer displayed on the window, it will display the message above. Select Yes to keep the result named, No to delete the result, or Cancel to abort this operation.
