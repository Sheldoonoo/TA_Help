Introducing the Q5000 Autosampler

 

The Q5000 Autosampler allows you to place multiple samples on the platform for automatic loading and run sequencing.

Description of the Autosamplers

Q5000 IR Autosampler

Using the standard pan tray for the Q5000 IR instrument (see the figure to the right), up to 25 samples can be accommodated.  Experiments are performed as normal—but samples can be run on a continual basis.

The sample can be sealed in a special aluminum pan and opened (exposed to the environment) immediately before being loaded into the balance via the Autosampler’s built-in punching mechanism. The punching mechanism ensures that only punched pans are loaded onto the balance.  (i.e., If the mechanism determines that a sealed pan has not been punched, a second attempt to punch the pan will occur. If that second punching is not successful, the pan will be skipped.)

An optional ten-pan position tray for use with 180µL metal-coated hemispherical quartz pans is also available.

To calibrate the sample tray and punching mechanism, refer to:

  • Calibrating the Autosampler

  • Calibrating the Punching Mechanism

Q5000 SA Autosampler

The Q5000SA Autosampler (see the figure to the right) allows you to place multiple samples on the platform for automatic loading and run sequencing. Using the quartz pan tray for the humidity chamber, up to ten 180 µL metal-coated hemispherical quartz pans can be accommodated at one time.  Experiments are performed as they normally would be using the SA instrument—but samples can be run on a continual basis. An optional 25-pan position tray for use with platinum, and punched aluminum pans is also available.

Autosampler Pans and Trays

The type of tray that you are using must be specified. Select Instrument Preferences from the Tools menu then select the TGA Page to choose the correct tray for your instrument. The pans listed below are used with the Q5000:  

Standard Pan Tray (25 pans):

  • 100 and 250 µL Ceramic (use up to 1200°C)

  • 50 and 100 µL Platinum (use up to 1000°C)

  • 20 µL Sealed Aluminum (use up to 600°C)

  • 80 µL Aluminum (Bottom of sealed pan) (use up to 600°C)

The sample can be sealed in a special aluminum pan and opened (exposed to the environment) immediately before being loaded into the balance via the Autosampler’s built-in punching mechanism. The punching mechanism ensures that only punched pans are loaded onto the balance.  (i.e., If the mechanism determines that a sealed pan has not been punched, a second attempt to punch the pan will occur. If that second punching is not successful, the pan will not be loaded.)

NOTE:  To calibrate the sample tray and punching mechanism, see Calibrating the TGA Autosampler.

Quartz Pan Tray (10 pans) [standard with the Q5000 SA, optional for the Q5000 IR]:

  • 180 µL Metal-Coated Hemispherical Quartz

 

General Operation

Follow the suggestions below for operating tips:

  • Calibratethe TGA Q5000 IR for weight and temperature at least once a month or when replacing the hang-down wire. See Calibrating the TGA Weight and Calibrating the TGA Temperature for instructions.

  • Calibrate the Autosampler tray if the sample hang-down wire fails to pick up a sample pan properly. See Calibrating the TGA Autosampler to adjust the tray.

  • Calibrate the Autosampler pan punching mechanism if needed. See Calibrating the TGA Autosampler.

  • Select the type of pan. See Choosing a TGA Pan. Once you have selected the type of pan that you wish to use, you must use the same type of pan for all of the samples on the sample platform disk. Thesame type of panthat you use for experiments must also be used as the tare (reference) pan.

  • Tare the sample pans. When you tare the sample pans, the TGA reads the weight of the empty pans in their numbered order and then stores these weights as a set of offsets. These offsets are subtracted from subsequent weight measurements for each numbered sample. Fill the platform with the desired number of empty, clean pans. When you select Calibrate/Tare from the menu or click the button, the Tare Utility window is displayed to allow you to tare any or all of the pans on the tray. (Alternatively you can select TARE TRAY from the Control Menu on the touch screen to tare all of the positions on the current tray.

NOTE: View the Instrument Log after taring to determine which pan, if any, does not tare during the procedure. If a tare error occurs, replace the pan and manually retare it.

NOTE: Do not switch the pans around on the sample platform because they have already been tared in their number position.

  • Load the samples. After you have finished taring the pans, load each sample in the correct sample pan, returning the pan, if removed, to its proper original pan location. Make sure that the wire on the bottom of the sample pans align with the groove in the pan hole, so that the sample can be picked up by the sample hang-down wire.

  • Set up and start the experiment. Use the Wizard, if desired. See also: Basic Steps Needed to Run a Q5000 Experiment

  • If you want to stop an Autosampler run that is already in progress, use either Stop or Reject, found on the Control Menu touch screen or the instrument control menu. If STOP is selected from the instrument control menu, you can choose to either stop the current run and continue with the sequence or stop the sequence. Selecting STOP from the touch screen will automatically stop the sequence.

  • Enter the desired text conditions for each of the samples and its pan number using the instrument control software. Each sample may have a unique experiment.

  • Start the sequence using the instrument control Start function.
