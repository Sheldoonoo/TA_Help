Taring the Pan(s)

 

The Tare function ensures that the weight measured by the balance reflects the weight of the sample only.

When a pan is tared, the instrument reads the weight of the empty sample pan and then stores the weight as an offset, which is subtracted from subsequent weight measurements. For optimum accuracy, the weight reading must be stable before it is accepted as an offset. The TGA determines when the weight reading is sufficiently stable.

You should tare the sample pan before each experiment, even if you use the same pan in consecutive experiments.

Taring is done for the select weight range. See:  Choosing the TGA weight range.

Select from the following topics for more information:

  • Standard TGA Tare Procedure

  • Auto TGA Tare Procedure

Standard TGA Tare Procedure

  1. Place the empty sample pan on the sample platform.

  2. Select the Tare function from one of these places:

    1. Instrument touch screen (Q500)

    2. Instrument keypad (Q50)

    3. Calibrate/Tare from the main menu in the instrument control software

    4. Click on located on the tool bar

Taring is recorded in the Instrument Log. View this log, if desired. The TGA will automatically load and close the furnace when Tare is selected.

Auto TGA Tare Procedure

When an Autosampler is installed, the Tare Utility window is displayed when Tare is selected from the instrument control software (see details below).

Use this window to tare a specific pan number or choose from multiple pan numbers.

  1. Place the empty sample pan(s) on the sample platform.

  2. Select Calibrate/Tare or click on the button. (You could also select TARE TRAY from the touch screen. This option tares all the pans installed on the tray.)

  3. Select the desired tray number from the drop-down list.

  4. Click on the desired pan number to select one pan or hold down the Ctrl key and click to select multiple pan numbers from the list. You can also click the Select All button to choose all pan numbers for taring or click Deselect All to clear your selections.

  5. Click Tare when finished selecting the tray number and pan(s).

  6. Click Close when completed all taring has been completed.

Taring is recorded in the Instrument Log. View this log, if desired. The TGA will automatically load and close the furnace when Tare is selected.
