Curve Symbol Window

Use this window, accessed by clicking in the symbols column of the Curves Page, to select the symbols to be used for the curves.

 

Follow these steps to select symbols:

1. Select the symbol type from the drop-down list. If you select a symbol, it will be displayed at the bottom of the window, then go on to step 2. If you select "No symbols," go on to step 5.

2. Select the symbol to be used for the selected curve from the drop-down list of symbol types.

3. Select the type of differentiation from the drop-down list.

  • If you choose to differentiate by curves, you must select the desired symbol for the selected curve from the symbol drop-down list.

  • If you choose to differentiate by subcurves, the symbols will be used in the order displayed. If you want to edit the list of symbols, click on the Edit List button. Choose the symbols desired from the window displayed.

4. Select the spacing between data points from the drop-down list.

5. Click the OK button when completed.
