ValueWithin

Test whether the value is within a percentage of the required value, returning 1 if it is, 0 if not.

Syntax

ValueWithin (<TestValue>, <Percentage>, <RequiredValue>)

 

Example

v0 = ValueWithin (V1, 5, 10)

If V1 is 10 Â±5% then V0 will be set to 1, otherwise it will be set to 0
