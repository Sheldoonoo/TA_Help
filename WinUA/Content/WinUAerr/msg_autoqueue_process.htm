Process the Autoqueue?

There is a new data file in the autoqueue. Do you want to process the autoqueue now?

 

This is a message that is displayed each time the autoqueue contains records that need to be autoanalyzed.

 

This message has a 60-second time out feature. If no response is made within 60 seconds, the autoqueue will be automatically processed. (This time out can be cancelled by unchecking File/Auto Run on the Autoqueue menu. This will leave the message displayed until you click either Yes or No).

 

NOTE: After each record is processed, it is removed from the autoqueue and the results are gathered in the Autoqueue Log.

 

Choose from the following responses:

 

  • Select Yes to autoanalyze all of the records currently contained in the autoqueue now. This action will take over the Universal Analysis program while the processing occurs.

  • Select No to process the records in the autoqueue at a later time. You can select File/Run from the Autoqueue menu to process the records later.
