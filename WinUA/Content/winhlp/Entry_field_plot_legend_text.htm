Entry Field Plot Legend Text

Type the desired text that will be added to the bottom of the legend. 5,000 characters can be entered. If Multicolor text is checked, the "Multicolors" defined on the Main Options - Color Page will be assigned to each line of text. After 12 lines have been assigned colors, the color selection will be repeated for the remaining lines.

 

NOTE: Even though the text field has a character limit of 5,000 characters, if any line is longer than the plot width it will not appear on the plot. Also, if you enter more lines than the plot area will allow, the extra lines will not appear on the plot.
