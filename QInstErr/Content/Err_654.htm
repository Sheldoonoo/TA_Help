Message 654

Balance purge gas cannot reach specified flow rate.

Problem:

There is a problem with the gas supply. The flow rate specified on the Notes page for the balance cannot be attained for the current run.

Solution:

Check to make sure there is an adequate supply of free flowing gas and that the flow rate has not been set to a value higher than the system will allow. Change the flow rate or replace the purge gas tank if needed, then retry the operation.
