Message 276

GCA too full. Run will start when liquid is reduced to a safe level.

Problem:

This message occurs if you attempt to start a method while the GCA is in the process of boiling off excess liquid nitrogen. See message 275 .

Solution:

No action needed. The GCA will automatically turn on the heaters to boil off the excess liquid nitrogen until the level has receded to a safe level for operation. If a method was already in progress, the method will start running as soon as the liquid has been reduced to the correct level. Wait until the operation has been completed before starting a method.

To avoid this problem, only fill the GCA three-quarters (75 percent) full, or to a total liquid weight of 71 pounds (32 kg).
