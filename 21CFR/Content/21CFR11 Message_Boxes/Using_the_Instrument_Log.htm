Using the Instrument Log

 

The Instrument Log displays the current list of instrument messages along with the date, time, and a brief description of what occurred. The entries displayed in this log include:  instrument errors, run status (e.g., starting stopping, and rejecting runs), calibration (when new calibration results are saved), data file acceptor (path and filename of stored data), Autosampler (loading and unloading of pans), miscellaneous and power supply messages.

  • You can print this list by clicking the Print button.
  • You can clear the current errors by clicking the Clear Log button. This list is cleared automatically when the instrument parameters are reset.
  • You can access online help for the selected error by clicking the Error Help button.
  • You can configure the instrument log's messages to be displayed by clicking on the Configure button to access the Instrument Log Configuration settings.
  • You can save the list to file by clicking on the Save button.

Configuring the Instrument Log

To sort the instrument messages for display in the instrument log, check those that will be displayed and leave those that you do not wish to be displayed blank. You can choose to sort the messages by the categories displayed or by the severity of the message.
