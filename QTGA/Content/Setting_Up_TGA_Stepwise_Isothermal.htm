Setting Up a TGA Stepwise Isothermal Procedure

A technique to increase resolution is raising furnace temperature until the onset of decomposition and then holding temperature isothermally until the decomposition is complete. After which, the temperature is raised again until the next decomposition begins, and so on until the final temperature of interest is reached.

If you want to set up a High-Resolution TGA stepwise isothermal procedure, you can choose one of two techniques—the TGA Wizard or the Experimental View - Procedure Page. See the instructions below for each technique.

When you perform a TGA Stepwise Isothermal experiment, these test parameters need to be specified:

  • Heating Rate: Stepwise isothermal consists of heating the material at a constant rate until a specified rate of weight change is detected and then switching to an isothermal segment to hold constant temperature until the weight change event is completed (as determined by the rate of weight change). This heating-isothermal process is continued until the final temperature is reached. The heating rates used are typically 10 or 20°C/minute.

  • Final Temperature: The final temperature is set above the last weight change of interest or 1000°C (600°C if using aluminum pans). The 1000°C is recommended when evaluating materials for the first time.

  • Enter Isothermal, & Exit Isothermal:Stepwise heating and isothermal holds often improve weight change resolution because weight changes are time dependent as well as temperature dependent. Hence, this approach gives weight changes more time to complete, thereby, reducing overlap with neighboring changes.

The key to obtaining good results with stepwise isothermal is properly choosing the rate of weight changes used to enter and exit the isothermal conditions. The entry rate of weight change should be large enough to respond only when a "real" event is occurring. The exit rate of weight change should be small enough to ensure that the event is actually complete. These values are usually chosen by running a conventional TGA experiment and evaluating the weight change derivative curve. The stepwise isothermal exit rate of weight change is selected to be equal to the baseline rate of weight change encountered during the onset of transitions of interest in the conventional TGA experiment. The stepwise isothermal entry rate of weight change is selected to be 10 times as large as the exit rate but should not be more than 50% of the maximum rate of weight change in the conventional TGA derivative weight change curve. Good default values to use in lieu of a preliminary conventional TGA experiment are 0.05 and 0.5 respectively.

  • Additional Conditions: There are additional experimental conditions that can be changed to optimize specific situations. These are accessed using Advanced Parameters and Post-Test Conditions. For most experiments, the system default values shown are recommended.

Using the TGA Wizard :

Enter the desired parameters, then select Next to proceed or select Back to return to the previous window.

Using the Experimental View Procedure Page:

  1. Display the Procedure Page. Select Stepwise Isothermal from the list of preprogrammed templates.

  2. Enter the desired parameters. When you have finished setting up this procedure, you can enter more information by selecting the Notes Page and Summary Page for this run.

  3. Click the Apply button when finished to save these changes or the Append Run button to save the changes to this run and add another run to the end of the sequence.

NOTE: You can use any test template as the basis for a custom test by selecting the desired test template from the list of tests, then click the Apply button and change the test to Custom. Then you can click on the Procedure Page and then the Editor button to make the changes desired.
