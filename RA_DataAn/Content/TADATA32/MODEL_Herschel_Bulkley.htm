Herschel-Bulkley

This model incorporates the elements of the Newtonian, Power law and Bingham equations.

Newtonian Yield stress = 0        n = 0

Power law  Yield stress = 0       n = power law index

Bingham  Yield stress value n = 0

 

Example material types:  Can often be used as a general-purpose model for shear thinning materials.

A more complex model is the Sisko model.
