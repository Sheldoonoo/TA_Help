Import (Macro)

Use the Import function to import a macro that will be added to the current macro file. This function displays the Open (file) window. After you select the macro file to open, you will see the Macro Import window. Check the desired macro(s) to import to the macro file and select OK.

 

This function is useful when you want to create macros on one computer, and then want to add one or more of them to the macros available in the macro file on your controller.

 

NOTE: Universal Analysis can only open one macro file at a time. A maximum of 30 macros can be contained in any one macro file. Make sure that you do not exceed this limit when importing macros into your current macro file.
