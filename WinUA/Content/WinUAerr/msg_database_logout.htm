Logout from database?

You have unsaved results. Logout from database?

 

Problem:

This message is displayed when you select File/Logout from the main menu during a 21 CFR 11 Universal Analysis session.

 

Solution:

  • Select Cancel to remain logged into the database.

  • Select OK to log out of the system. The Login window is posted. Select Cancel on the Login window to exit the database entirely.

 

NOTE: If you want to leave the 21 CFR 11 Universal Analysis session unattended, but protected, select File/Logout, then reply OK to this message. Upon returning, log in again.
