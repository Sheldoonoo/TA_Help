Saving Data as Result Files

 

All data generated by the software can be saved as a results file. The file extension is the same, no matter what type of procedure was used to generate the data.

Results files have the extension RSL

Select from the following topics for more information:

  • Opening an Earlier Version of a File

Creating a New Data File

To create a new file, you must edit a copy of an existing file.

  1. Open an existing results file.
  2. Display the data in File View .

    Switching to File View:

    1. Right click on the data file name in the Results list.
    2. Click on To File View.
  3. You can the either edit the existing data or delete it all and add new data points.

    Editing Data:

    1. Click on the point to edit.
    2. Select Edit from the menu
    3. Click on Modify Value.

    Delete All

    1. Select all data.
    2. Select Edit from the menu.
    3. Click on Delete.

    Adding Data Points:

    Select Edit from the menu bar.

    Click on Insert Point.

  4. Save the new results file.

Merging/Comparing Data

Merging Data Sets

You may have two or more data sets that contain related data, for example two halves of a long time-scale test. The data software allows you to merge these files into one new results file. You can also use this feature to produced averaged data when performing repeat measurements on samples. For more information click here More.

Comparing Data

The software allows you to place data from more than one results file onto a single graph as follows:

  1. Load two or more results files.
  2. Display them on the same graph.

Using the QC function to compare data:

  1. Click on the reference curve.
  2. Select Analysis from the menu.
  3. Click on QC.
  4. Use the Options item to define your QC specification.
  5. The results (pass of fail) will be displayed on the graph as part of the key.
