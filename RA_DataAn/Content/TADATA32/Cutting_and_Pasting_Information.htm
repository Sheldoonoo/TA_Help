Cutting and Pasting Information

 

The software allows you to transfer information to and from other Microsoft® Windows applications as follows:

File View – To transfer data to another application

  1. Select the data you wish to copy. This can be an individual point, a complete row / column or sections of the table. Drag the mouse key to highlight your selection.
  2. Select Edit from the menu.
  3. Click on Copy.
  4. Switch to the application you wish to paste the data into.
  5. Use the Paste function of the application (usually found in the Edit menu) to insert a copy of the results.

File View – To transfer data from another application

  1. From the other application, select the data you wish to transfer.
  2. Use the applications Copy or cut method.
  3. Switch to the TA Data software.
  4. Click in the table to select the insertion point.
  5. Select Edit on the menu.
  6. Click on Paste.

Graph — To copy and paste a graph into another application

  1. Select Edit from the menu.
  2. Click on Copy.
  3. Switch to the application you wish to paste the graph into.
  4. Use the Paste function of the application (usually found in the Edit menu) to insert a copy of the graph.

NOTE: Depending upon the capabilities of the application, you may be able to edit the inserted graph; please refer to the documentation for that software product.
