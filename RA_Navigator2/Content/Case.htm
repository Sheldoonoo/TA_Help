Case

All the statements immediately following a Case statement are executed (up to the next Case statement) if the integer value specified matches that evaluated in the Select statement.

The parameter given in the Select statement is evaluated and then compared against each of the Case statements contained in the Select / EndSelect block.

Example

Select (VA)
 Case 0
  [statements]
 Case 1
  [statements]
 Case 2
  [statements]
EndSelect

Note: You must test for all expected values. The case functions only work with integer values.
