If you assign the group TA Service to the selected user, all other groups previously assigned to the user will be deleted. Do you want to continue?

In order to be assigned to the TA Service group, the user cannot perform any additional functions within the Advantage Integrity system. Therefore, the user account cannot be assigned to any group other than TA Service group. This is done to protect the integrity of the data. Access rights are granted only to those functions associated with servicing the system.
