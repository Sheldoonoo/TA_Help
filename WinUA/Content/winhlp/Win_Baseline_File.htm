Baseline File Window

Use Baseline File to perform baseline subtraction or addition. Follow the steps below to either subtract or add the baseline.

 

1. Open the baseline reference file and plot the desired signals that will be used to perform operations on the new data file.

2. Open the desired sample file and select the desired signals that will be used to perform operations with the baseline reference file. (Or use Params Same As to copy the parameters.)

3. Make sure that the sample data file window is currently active. Select Tools/Baseline File from the menu. Select the desired reference file to be used as the baseline from the available list. This window is displayed.

4. Check the desired signal(s) from those shown.

5. Click on the desired base: time or temperature.

6. Click on the desired operation: add or subtract the baseline file.

7. Click on the desired signal match: Name & units or Units only.

8. Click the OK button. The resulting data is displayed.

 

NOTE: A baseline signal will be added to or subtracted from each data file signal that has the same units and/or signal name. Those signals that are not compatible between the files are not used.

 

NOTE: The baseline addition/subtraction is only applied toy-axis signals. It is not applied to x-axis signals. If, for example, you wish to subtract the temperature signal, plot temperature on the y-axis instead of the x-axis, then perform the baseline subtraction.
