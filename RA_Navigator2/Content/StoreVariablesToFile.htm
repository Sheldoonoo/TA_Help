StoreVariablesToFile

Stores all currently defined variables (string and numeric) to the file specified. Navigator variable files can be loaded by means of the LoadVariablesFromFile command and provide a means of carrying values over from one run of the script to the next. Variable files can also be read in by any script, not just the one that created them.

Syntax

StoreVariablesToFile(<FileName>)

The default extension for navigator variable files is NAVVAR
