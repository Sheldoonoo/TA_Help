Sensitivity

The Sensitivity option allows you to analyze curve sensitivity using a least-squares averaging technique.

 

The curve data is reread from the data file, smoothed using the analysis region window, and subtracted from the original raw data. The resultant plot is a measure of the signal sensitivity during any portion of the experiment.

 

Before you analyze a curve, you must choose the desired Y-axis to smooth by clicking on the desired Y number displayed on the graph. When you choose the Sensitivity function from the Tools menu, the Analyze Sensitivity Window is displayed to allow you to enter the analysis region width.
