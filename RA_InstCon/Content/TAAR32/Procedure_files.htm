Introduction to Procedure Files

 

Procedure files contain the information required to run tests. Flow, creep, and oscillation procedure files are distinguished by a different extension.

 

Flow procedure files have the extension ARFLOW

Creep procedure files have the extension ARCREEP

Oscillation procedure files have the extension AROSC

Stress Relaxation procedure files have the extension ARSTRESSRELAX

Mixed procedure files have the extension ARPROC
