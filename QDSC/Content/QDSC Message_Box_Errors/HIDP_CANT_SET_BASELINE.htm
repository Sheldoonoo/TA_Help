Cannot Send Baseline Calibration

The baseline calibration cannot be sent to the instrument because the T1 Heat Flow signal is not selected.

This problem occurs because the current heat flow selection is not T1 when you you tried to save the baseline calibration results.

To correct the problem stated in the message, select Tools/Instrument Preferences, then click on the DSC page. Choose the T1 Heat Flow signal from the drop-down Heat Flow Selection list. Now repeat the baseline calibration.
