Params (Custom Report Editor/Edit Menu)

Select this option to edit the parameters of the report object. The window that is displayed when Edit/Params is selected depends on the report object being edited. Click on the window name next to the object to find out more about editing it.

 

  • Results Report Object  displays the Result Options window.

  • Data Table Report Object  displays the Data Table Limits window.

  • Parameter Report Object  displays the Parameters Options window.

  • Method Log Object  editing parameters function is not available.

  • Plot Object  displays the Plot Options window.

  • Image Object  displays the Select Image File window.

  • Text Object  displays the Report Text window.
