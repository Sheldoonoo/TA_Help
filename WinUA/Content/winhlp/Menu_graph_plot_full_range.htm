Plot Full Range Menu

Click on Plot Full Range to check this option. This option is used to have the data that falls outside of the analysis range plotted, but grayed out. Data outside the analysis range is plotted in the "Plot Grid" color. This is the default setting.

 

Leave this option unchecked to plot only data within the analysis range.

 

NOTE: This option applies only to thecurrentdata file. The "Plot data outside of analysis range" checkbox found on theMain OptionsData Pageis the default option that applies to each file as it is first opened.
