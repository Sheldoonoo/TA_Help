Basic Steps Needed to Run a TGA Experiment

 

All of your TGA experiments will have the following general outline. In some cases not all of these steps will be performed.

If you have the QNX/Platinum features installed, many of these steps may be scheduled to run automatically. See Introducing Platinum for more details.

  • Selecting the pan type and material. See Choosing TGA Pans.

  • Specifying the TGA weight range. See Choosing the TGA weight range.

  • Taring the empty sample pan. See Taring the Pan(s)

  • Loading the sample into the pan. See Preparing TGA Samples and Loading/Unloading a Sample.

  • Entering experiment information through the TA controller, this includes both sample and instrument information on the Summary and Notes pages. See Using the Summary Page and Setting Up the Notes Page.

  • Creating or selecting the experimental procedure using the instrument control software.  See Specifying a Test Procedure.

  • Attaching and setting up external accessories as required such as the  purge gas. See Recommended Purge Gases and Flow Rates.

  • Creating a Run Sequence. See Creating Custom Methods or Available TGA Test Templates.

  • Starting the experiment. See Starting a Run and/or Sequence.
