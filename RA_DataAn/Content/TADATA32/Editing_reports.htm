Editing Reports

To edit existing custom reports, use the following steps:

  1. From Rheology Advantage data analysis, select Report on the menu.
  2. Click on Manage.
  3. Select the template that you wish to edit (the one closest to your final design.)
  4. Click on the Edit button. This will start Microsoft ® Word.
  5. Make any changes to the layout of the document. If you wish to add additional ‘place holders’ for graphs, tables of data etc, select them from the Insert menu item in Word.

NOTE: You will often see small text boxes to the left of the tables. These are used by Data analysisand should not be deleted. They will not show up in the final report.

  1. When you have made all the required changes, save this document, overwriting the original.
     

NOTE: If you wish to keep the original, you can save the document with a new name. Once you have done this, you must then click on the Insert menu in Word and select Add to TA template list.

  1. Exit from Microsoft ® Word. See also: Using Microsoft ® Word
