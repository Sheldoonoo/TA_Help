Radio Buttons Match Name/Units

Select the desired radio button as follows:

  • Name & Units: Click this button to permit signal addition or subtraction by signal name and units. This feature permits addition or subtraction of a common baseline signal in one data file from multiple signals that have the same names and the same units in another data file.

  • Units Only: Click this button to permit signal addition or subtraction by units only. This feature permits addition or subtraction of a common baseline signal in one data file from multiple signals that have different names but the same units in another data file.
