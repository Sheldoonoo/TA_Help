Cox-Merz Data Comparison

 

Follow these steps to perform a data comparison:

  1. Carrying on with the results file already loaded, select Transformations from the File menu. Then select Cox-Merz.  A new file will appear on the Results list window called PDMS Extended frequency sweep-0001o cm.rsl.
  2. Send this file to a new (flow) graph and then load the file listed below:
    • PDMS Steady state flow-0001f.rsl
  1. Put this on the graph with the transformed oscillatory data.

Alternatively, you can merge the two data files. Select both PDMS Steady state flow-0001f.rsl and PDMS Extended frequency sweep-0001o cm.rsl in the Results list window. Right click and select Merge from the pop-up window. A new file is created that consists of both sets of data merged into one.
