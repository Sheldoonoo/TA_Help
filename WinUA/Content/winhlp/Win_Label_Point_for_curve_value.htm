Label Point Window

Use this window to label your curve value with one or more of the following items. Check the desired option(s), then click the OK button.

  • Time
  • Temperature
  • Signal
  • Y Axis Value
  • Frequency*
  • Amplitude*
  • Location*
  • Draw arrow to point
  • Draw cross at point

 

* NOTE: This option is available for Curve Value analysis of multiplexed files only.
