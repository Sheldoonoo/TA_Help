Monitoring Real-Time Signals

 

Real-time instrument signals may be monitored from either the instrument touch screen or from the controller Signal Display pane. If you choose to use the touch screen, you can access either the Status or Display options. See also:Display Menu Touch Screen.

Controller Signal Display Pane

The Signal Display pane (shown to the right) is a customizable instrument control window that allows the user to select the real time signals they want displayed during an experiment. These same signals are also available for plotting in the RealTime Plot window.

See also:  Overview of the Instrument Control Window

 

Signal Display Pop-up Menu

The TA Instruments TGA displays a certain set of information on the Signal Display window. Using the pop-up menu (accessed by right clicking), you can customize the signal display by selecting Customize Signal Display.  When the Customize Signal Display window appears, select or deselect the signals then click OK to close the signal selection window and display the new signals. You can reorganize the order of the signals by clicking on a signal name to display a drop-down list of signals to select from.

TGA Instrument Signals

Listed below are all of the possible real-time TGA signals. The available signals are dependent on instrument type and options installed. These signals are available for display within the Signal Display pane, as RealTime plot signals, and they are displayed on the instrument touch screen, if available. A subset of these signals are available as stored signals within the data file.

For a detailed explanation of the signals shown on the window, see the table below:

 

Signal

Description

Average Derivative Weight

The Fourier transformation obtained average over one cycle of the rate of weight change.

Balance Purge Flow, mL\min

The rate of gas being supplied to the balance.

Derivative Weight Amplitude

It is the derivative of the weight amplitude signal and is a measure of the temperature dependence of the rate of weight loss displayed during MTGA experiments. This signal is included in the data file and used in the calculation of the kinetics signals.

Heater Power, Watts

The amount of electrical power being applied to the heater.

Instrument clock

The time set on the current instrument. You can synchronize the instrument with the controller by using the Instrument Setup Wizard.

Method Time, min

The total time that the current method has been running.

Modulated Average Temperature, °C

The temperature signal used in the calculation of the kinetic signals.  

Modulated Average Weight, mg

 

The average sample weight over one modulation period displayed during MTGA experiments. This signal is included in the data file as a weight signal.

Modulated Weight Amplitude, mg

The deconvoluted peak amplitude of the weight signal displayed during MTGA experiments. The amplitude is related to the rate of weight loss as a function of temperature. This signal is not included in the data file.

Modulation Amplitude, °C

The peak-to-peak half amplitude, in degrees Celsius, observed at the sample thermocouple for one complete temperature cycle.

Modulation Period, sec

The time, in seconds, for one complete temperature cycle.

Remaining Run Time, min

The estimated total time remaining until completion of experiment.

Sample Purge Flow, mL\min

The rate of gas being supplied to the sample.

Segment Time, min

The total time that the current segment has been running.

Set Point Temperature, °C

The temperature that the TGA furnace is requested to attain.

Temperature, °C

The current temperature of the sample.

Temperature Amplitude

The deconvoluted peak temperature amplitude displayed during MTGA experiments. This signal is selectable and can be included in the data file.

Weight, mg

The measured weight of the material in the sample pan that is displayed when a method is running. The signal is selectable and can be included in the data file.

Weight Percent, %

The measured weight of the material displayed as a percentage of the starting weight.
