Ignore Breaks in Data Menu

Click on Ignore Breaks in Data to check this function, which is used to have the current data file plotted with a continuous curve, regardless of any breaks in the data points. The data will be interpolated to create an unbroken curve.

 

Leave this box unchecked (which is the default) to have the data file plotted according to existing data points and display any breaks in the curve.

 

NOTE: This option applies only to thecurrentdata file. The "Ignore breaks in data" checkbox found on theMain OptionsData Pageis the default option that applies to each file as it is first opened.
