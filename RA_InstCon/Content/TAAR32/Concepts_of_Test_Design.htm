Concepts of Test Design

 

When using the rheometer to analyze test samples, the most common method used to design tests or experiments is to define various procedures. This help topic describes the basic design of a procedure, the various types of steps involved with a procedure, and provides links to specific types of steps.

  • Designing a Procedure
  • Basic Types of Steps

Designing a Procedure

A procedure consists of one or more steps that are executed in sequence by the instrument. Each step can be made to conditionally end, if required by your experiment. Several types of steps are generally used in procedures as described in the next section.

Follow these steps to design a procedure:

  1. Select Procedure/New from the menu. (To use an existing procedure select Procedure/Open and browse for the desired procedure file.)
  2. Select the type of procedure from the list. The information displayed will depend on the type of test selected. Click on the topic below for more information:
    • Templates

      Provides the ability to step through a procedure creation.

    • Flow
    • Creep
    • Oscillation
    • Mixed

      Mixed Steps

      A mixed procedure allows you to specify a sequence of steps made up of any combination of flow, creep and oscillation procedures. Results generated from this procedure are saved as a sequence of results files each incrementing the run number.

    • Stress Relaxation
  1. Enter or edit the desired Name of the procedure.
  2. Add or delete Steps by use of either the right-click pop-up menu or from the Procedure menu. To include a step in the actual sequence, either check the desired box or select the Perform step option on the General tab.
  3. Edit each step, if desired by selecting the step and then using the dialog on the right to adjust the parameters. The name of the step can be edited from the Step Name entry on the General tab.

NOTE:  You cannot delete the Pre- and Post-Experiment steps.

Basic Types of Steps

Conditioning Step

Use this step type to ensure that the sample is at the correct temperature and any residual normal force (from loading) is at an acceptable level. You can then pre-condition the sample. This allows you to apply a shear to remove any 'history' associated with the sample preparation, storage and loading. If you have a conditioning regime that is more involved than can be carried out in one step, you may add further conditioning steps. These may be added sequentially and can be interspersed with other steps.

Main Procedure Steps

In between the first Conditioning and the final Post-Experiment steps, you can define one or more main procedure steps.

Post-Experiment Step

This step allows you to automatically save the data generated by this procedure. You can also get the rheometer to head towards a new temperature. This can be handy as an aid to cleaning the rheometer. You can only have one Post-Experiment step and it is always placed as the very last item in the sequence.

Conditional Step Termination

The software allows you to define conditions in which a step is halted ahead of its normal termination conditions. You can use this to ensure that, for instance, the instrument does not over speed or apply excessive strains.

Rather than running a step for a certain amount of time, you may wish to run it until stable data is obtained. You can set an equilibrium limit (such as the viscosity value becoming constant when running a single shear with time) that will stop the currently active test.
