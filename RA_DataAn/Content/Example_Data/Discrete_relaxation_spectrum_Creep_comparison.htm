Discrete Relaxation Spectrum - Creep Comparison

 

Follow the steps below:

  1. Switch back to the frequency sweep graph, then select Discrete relaxation spectrum from the model list. Apply this model.

NOTE: This model will take a few minutes to complete. If you wish to watch the progress, turn on the Running Analysis window from the View tab of the Settings menu.

When the model has been calculated, a new file (called PDMS Extended frequency sweep-0001o-spx.rsl) will have been generated. Right click on this file (in the Results list window) and select Creep… from the pop-up window. Accept the default values. A creep file is generated.

  1. Send the creep file (PDMS Extended frequency sweep-0001o-spx-c.rsl) to a new graph, then load the file:
    • PDMS creep-0001c.rsl
  2. Send this to the same graph as the generated creep curve to compare the transformed data to the generated creep curve.
