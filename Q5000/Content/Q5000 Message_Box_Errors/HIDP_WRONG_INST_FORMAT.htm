Wrong Instrument Format

Cannot use the calibration file.  It was written by a ___ not a ___.

The calibration file selected is not applicable to the instrument (and cell type) installed or the filename may be incorrect. The calibration file is for a different instrument family. For example, a DSC calibration file was attempted for analysis by the TGA calibration function.

Check the filename and path and try the operation again. If the error persists, it is invalid and an appropriate file must be chosen.
