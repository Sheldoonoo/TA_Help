Results Spreadsheet Object (Custom Report Editor/Insert Menu)

Use the Results/Spreadsheet function to insert an object into the template that represents the results of your analysis in a Microsoft Excel® spreadsheet.

 

When you select this item from the Custom Report Editor Insert menu, the object block is inserted into the template. It can be moved, resized, and formatted as desired.

 

When the custom report is dynamically created in Microsoft Word®, the spreadsheet can be further edited and modified by double clicking to start Microsoft Excel®.
