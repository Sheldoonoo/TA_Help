Account Disabled

The User Account [NAME] has been disabled for security reasons.

User access (log in) is based upon a User ID (or log-in name) and password. If you fail in your attempts to log into the system too many times your account will become inaccessible (disabled).  If the account becomes inaccessible, the system manager must re-enable the account.

The number of log-in attempts allowed is established using the Maximum # of Log-in Attempts function found on the System Settings Manager/Settings Page.
