Other Groups Will Be Deleted

You are assigning the System Manager Group to the selected user, if you proceed, all other groups previously assigned to the user will be deleted. Do you want to continue?

In order to protect the integrity of the Advantage Integrity system, a user who is a member of the System Manager Group can ONLY perform those functions assigned to a System Manager. Therefore, any projects, modules, or other groups cannot be assigned to a member of the System Manager Group.

  • Select Yes, if you wish to assign this user to the System Manager Group. All other groups previously assigned to this user will be deleted from this person's account.
  • Select No, to cancel the operation.
