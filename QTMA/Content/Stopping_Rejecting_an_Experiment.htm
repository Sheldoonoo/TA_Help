Stopping/Rejecting an Experiment

 

When an experiment is in progress you can stop it and save the data or reject it and discard the data as directed below:

Stop and Save Data

There are several ways to halt an experiment and save the data:  

  • Select Stop on the Control menu.
  • Click on the tool bar.
  • Select the STOP key on the instrument touch screen or the keypad.

Selected Post Test conditions will activate after this action. To open the Post Test window, select the Procedure Page, then click on the Post Test button.

A confirmation message will be displayed when Stop is chosen, unless this option is turned off on the General Page accessed under the Tools/User Preferences.

Reject and Discard Data

Use Reject on the Control menu or on the tool bar to halt an experiment and discard the data. Selected Post Test conditions will activate after this action. To open the Post Test window, select the Procedure Page, then click on the Post Test button.

A confirmation message will be displayed when Reject is chosen, unless this option is turned off on the General Page accessed under the Tools/User Preferences.
