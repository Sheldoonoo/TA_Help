Too Many Records in Autoqueue

Autoqueue file contains too many records. Excess records will be ignored.

 

Problem:

You need to process (autoanalyze) the autoqueue. It has been accumulating records and has now reached more than 90 percent capacity. If the records are not processed and removed from the autoqueue to allow more space, excess or incoming records will be ignored or lost.

 

Solution:

When the message appears to process the autoqueue (autoanalyze), select Yes and allow the records to be processed. Another way to autoanalyze is to select File/Run from the Autoqueue menu. The results will be added to the autoqueue log. Alternatively, you can delete records from the autoqueue.
