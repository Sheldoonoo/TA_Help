Cannot find hit label

 

Problem:

You have executed a macro step that is normally performed on a label (for example, changing color or deleting a label) and the target label cannot be found. This usually results from improper modification of a macro step or execution of the macro when the program is not in a state that will allow the macro to find the label correctly.

 

Solution:

Make sure the program is in the proper state to allow the macro to execute as was intended. Be careful when editing macro steps that you do not make it impossible for the label to be found.
