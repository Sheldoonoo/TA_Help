Using the Punch Force Test

NOTE:  This option is available in Service mode only. Contact TA Instrument service, if you require assistance with the pan punching mechanism.

Q5000 IR Autosampler

Using the standard pan tray for the Q5000 IR instrument (see the figure to the right), up to 25 samples can be accommodated.  Experiments are performed as normal—but samples can be run on a continual basis.

The sample can be sealed in a special aluminum pan and opened (exposed to the environment) immediately before being loaded into the balance via the Autosampler’s built-in punching mechanism. The punching mechanism ensures that only punched pans are loaded onto the balance.  (i.e., If the mechanism determines that a sealed pan has not been punched, a second attempt to punch the pan will occur. If that second punching is not successful, the pan will be skipped.)

To calibrate the sample tray and punching mechanism, refer to:

  • Calibrating the Autosampler

  • Calibrating the Punching Mechanism

Punch Force Testing

Use the Service/Punch Force Test to cycle through loaded pans and test the punching mechanism.

NOTE:  Before placing the sample in the pans and sealing them, inspect the pan lids to make sure there is a complete, circular "score" all the way around the lid. If the scoring is incomplete or nonexistent, discard the pan lid. See the figure to the right to determine the location and quality of the scoring.

  1. Load the desired number of sealed pans on the tray. See Preparing Sealed TGA Pans, if needed.

  2. Click on the pan number positions that you wish to test. Click the Select All button to choose all 25 positions or Deselect All to clear all 25 positions.

  3. Select the Start button to begin cycling the chosen pan positions through the punching mechanism. (You can stop the test by clicking the Abort button.)

  4. View a completed report in Microsoft Excel by selecting the Get File button.

  5. Clear the report when finished by selecting the Reset File button.
