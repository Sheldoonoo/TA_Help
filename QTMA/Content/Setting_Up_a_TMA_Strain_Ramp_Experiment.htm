Setting Up a TMA Strain Ramp Experiment

 

In a Strain Ramp experiment the sample is exposed to a constant strain rate while the temperature is held isothermal.

If you want to set up a TMA Strain Ramp procedure, you can choose one of two techniques—the TMA Wizard or the Experimental View - Procedure Page. See the instructions below for each technique.

When you perform a Strain Ramp experiment, the following parameters need to be chosen:

  • Preload Force: There are two types of forces used when evaluating materials with the TMA.  “Preload force” is the force initially applied either to ensure good contact between the measurement probe (e.g., expansion, penetration) and the sample, or to remove slack in the sample (e.g., film/fiber probe), before measuring its length.  Although the amount of preload force required depends on the material being evaluated, it is recommended that the force used be limited to the smallest amount needed to accurately measure initial sample length.  (0.001N is a good default value.)  Otherwise, the preload force could affect results (e.g., creep) particularly with softer materials.  (This is true even though the experimental “applied force” is automatically adjusted to account for the preload force.) “Applied force” is the force introduced at the initiation of the actual experiment and is the force under which the measurement is made.  For example, in creep measurements, the applied force is used to calculate compliance.  The TMA provides applied forces up to 1 Newton.
  • Isothermal Temperature:  Strain curves are generated isothermally.  Enter the temperature of interest.  The temperature range for TMA is -150 to 1000 °C.  This entire range is available if liquid nitrogen is used in the reservoir surrounding the furnace.  If the MCA is used for cooling, the allowable range is -70 to 400°C.
  • Strain Rate:  This value sets the rate (in %/min) at which the sample length will be changed during the experiment.
  • Final Strain:  The experiment will end when the sample length has been changed by this percentage.
  • Additional Conditions:  There are additional experimental conditions that can be changed to optimize specific situations.  These are accessed using Advanced Parameters and Post-Test Conditions.  For most experiments, the system default values shown are recommended.

Using the TMA Wizard :  

Enter the desired parameters, then select Next to proceed.

Using the Experimental View Procedure Page:

  1. Display the Procedure Page. Select the desired experiment from the list of available preprogrammed templates.
  2. Enter the desired parameters. When you have finished setting up this procedure, you can enter more information by selecting the Notes Page and Summary Page for this run.
  3. Click the Apply button when finished to save these changes or the Append button to save the changes to this run and add another run to the end of the sequence.

NOTE:  You can use any test template as the basis for a custom test by selecting the desired test template from the list of tests, then click the Apply button and change the test to Custom. Then you can click on the Procedure Page and then the Editor button to make the changes desired.
