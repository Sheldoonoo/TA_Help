Analysis Options - Step/Change Page

Use this page to set up the parameters for the program to use when you perform a step transition, glass transition, or signal change analysis.

 

Follow the steps below to set up these options:

 

1. Check the Manual tangent selection option, if you wish to manually select the start and stop limits for the tangent lines. The program will use the limits you select to fit the tangent lines to the curve.

2. Click on the arrow to display the drop-down list and select the step transition Midpoint.

3. Click on the arrows to display each drop-down list and select the two Step change between options to be used in step transition calculations.

4. Check the desired options to be used when labeling the step transition analysis results: Limits, Onset, End, Change, Delta Cp, Signed change, and/or Signed delta Cp.

5. Check the desired options to be used when labeling the signal change analysis results: X-limits, Y-limits, Change, and/or Signed change.

6. Click the Save button, if you want to use these as your new default settings or click the OK button to use these settings for this session only.

 

NOTE: You can choose the default options by clicking on the Default button.
