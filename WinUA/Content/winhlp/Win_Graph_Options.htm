Graph Options Window

Use the Graph Options window to choose the graph settings that will apply to the file currently displayed. When you want to set up the appearance of your graph, follow these steps:

 

  1. Select the radio button for the Y-axis label and curve that you wish to set up.

  2. Check the Show Axis box to show the axis label and tick marks for the selected Y-axis.

  3. Edit the names for the Y-axis and X-axis labels, if desired.

  4. Select the color for the graph and label from the drop-down list.

  5. Select the grid type from the drop-down list.

  6. Select the line type from the drop-down list.

  7. Check the show symbols box to enable symbols on the current axis.

  8. Select the symbol type from the drop-down list. If you select a symbol, it will be displayed at the bottom of the window, then go on to step 9. If you select "No symbols," go on to step 10.

  9. Select the type of differentiation from the drop-down list.

  10. Select the spacing between data points from the drop-down list. If you want to edit the list of symbols, click on the Edit List button.

  11. Repeat steps 1 through 10 for each axis. Grid type and symbol differentiation apply singularly to the entire plot. They cannot be set up uniquely for each axis (except in a curve overlay plot).

  12. Click the Save button, if you want to use these as your new default settings or click the OK button to use these settings for this session only. 

 

NOTE: Select the Default button to use the default settings for this window.
