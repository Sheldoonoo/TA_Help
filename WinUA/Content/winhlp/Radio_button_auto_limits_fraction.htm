Radio Button Derivative Fraction

Click on the Derivation fraction radio button to determine the weight change using the point determined by the derivative fraction entered.

 

NOTE: The default derivative fraction of 0.05 (5 percent) is recommended for both Point 1 and Point 2. If you do not use the default value, it is recommended that you use the same derivative fraction for both points.

 

When doing a succession of weight change analyses on the same curve, the derivative fraction used for the Point 2 of the first step should be the same as the fraction used for Point 1 of the next step.

 

NOTE: If the program does not locate an adjacent peak to use for the midpoint determination, it will use the end of the data curve.
