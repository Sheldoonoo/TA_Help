Finding Linear Region for Solid in Torsion

 

The recommended method for running a temperature ramp test is to use "strain control". Therefore we need to find a strain value in the linear viscoelastic region. An easy way to determine the linear region for a solid sample in torsion is to run a "stress sweep" and sweep through a range of "torque".

  • A logarithmic sweep taking at least 10 points per decade is recommended.
  • For samples below Tg start with a torque range of 100 to 100,000 micro N•m
  • For samples above Tg start with a torque range of 1 to 10,000 micro N•m
