Reset Factory Settings?

All standard settings will be reset to the manufacture default values. Do you want to continue?

Under normal circumstances the mass flow controller (MFC) should not need to be calibrated by you, the customer. Calibration is done at TA Instruments for you. If you need to return the calibration values to their factory settings for any reason, select Calibrate/Mass Flow Control to view the calibration tables. Select the desired table, then click the Restore Values button. This confirmation message is displayed. Select Yes and the original calibration values will be restored.
