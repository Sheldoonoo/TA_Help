Saved Record Read Error

Saved record read error.

 

Problem:

A check sum error was encountered while reading a saved analysis record from a data file. The file is damaged, altered or not compatible with this data analysis program.

 

Solution:

If possible, obtain a fresh copy of this file from the source. Verify that this file can be processed by this program. The data analysis program name and version number that saved this analysis can be determined by opening the data file with the Report Editor and examining the last few lines of the file.
