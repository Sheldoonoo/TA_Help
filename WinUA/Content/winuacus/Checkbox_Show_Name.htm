Checkbox Show Name

Check this box to include the name of the individual parameter on the custom report. First highlight the desired parameter and check or uncheck this box.

 

Leave this box blank to include only the parameter value (if it is available in data file) and not the name on the custom report.

 

NOTE: If the selected parameter is not available in the data file, the parameter name will still appear on the custom report when Show Name is checkedbut there will be no value with it.

 

Examples:

Cell Constant: 1.62140 (shows name)

1.62140 (name of parameter not shown)
