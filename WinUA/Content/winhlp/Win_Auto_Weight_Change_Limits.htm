Auto Weight Change Limits Window

Use this window to determine the way analysis limits are chosen for repeat analyses on materials that you expect to exhibit similar thermal plots. This option is particularly useful when used in combination with the Autoanalysis option. The Auto Weight Change Limits window is available and can be displayed when you perform a TGA weight change analysis.

 

You can set up and use the Auto Weight Change Limits function as follows:

 

1. Open a good representative TGA data file for the material(s) being analyzed.

2. Plot both the weight and derivative signals. Click on the weight curve.

3. Select Analyze/Weight Change from the menu, the cursors (markers) will be displayed.

4. Position the cursors so they include the desired area that you want to analyze for weight change.

5. Right click to display the pop-up analysis menu, and choose Auto Limits from the menu.

6. If you want the program to always use the exact position of the cursors that you will choose, select the Use cursor position radio button for one or both limits. NOTE: Selecting the Use cursor position radio buttons for both limits is the same as selecting Accept Limits from theAnalyze pop-up menu. This will let you select a start or stop limit using the cursors for each successive analysis.

7. If you want to allow the program to determine the weight change using the derivative fraction (which is the recommended option), click on the Derivative fraction radio button and enter your own value or click the Default button to use the program default derivative fraction of 0.05. See also: What is the Derivative Faction?

8. Select one of the buttons that will determine how this information is used:

  • If you select the OK button, these options are used for this session only.

  • If you select the Save button, these options are saved as in the initialization (.ini) file and will be recalled each time the program is started.

  • If you select the Cancel button, the operation will be aborted.

 

NOTE: The default derivative fraction of 0.05 (5 percent) is recommended for both Point 1 and Point 2. If you do not use the default value, it is recommended that you use the same derivative fraction for both points.

 

When doing a succession of weight change analyses on the same curve, the derivative fraction used for the Point 2 of the first step should be the same as the fraction used for Point 1 of the next step.

 

NOTE: If the program does not locate an adjacent peak to use for the midpoint determination, it will use the end of the data curve.

 

After you set up the Auto Limits options, you can open similar data files and perform the analysis quickly by selecting Auto Limits from the pop-up analysis menu and simply clicking the OK button to use the stored limits.
