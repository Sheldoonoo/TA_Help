Macro File Name Window

The Macro File Name window is used to insert a macro step that will identify the specified file (data file, session file, etc.) to open or save when the macro step runs.

 

Follow these steps to insert an File Name identification step into your macro:

 

1. Open the desired macro.

2. Capture the desired open or save function.

3. Select the desired step position for the File Name macro step. The position should be immediately after the "Open File" or "Save Analysis/Session" step.

4. Select File Name from the Insert menu found on the Macro Editor window.

5. Choose one of the radio buttons: Use default file name (inserts the default filename that is displayed when the macro runs) or File (requires that you enter the path and filename for the desired data file or use the Browse button to locate and select a data file).

6. Check the overwrite existing file on save, if you want the system to replace the existing file with the new file that is created if the macro performs a save file function.

7. Select OK.
