Checkbox Auto Launch

Check this box to have the program automatically run (launch) the Windows® application associated with the file extension of the current file. For example, if this box is checked and you open a file with the .pdf extension in Universal Analysiss File Open window, the program will run Acrobat Reader and display the file.

 

NOTE: Universal Analysis will not auto launch files with the extensions .tai, .txt, .sav, or.asc, even if they have an associated Windows® application. In addition, the extension must have at least one alphabetic character in order to be automatically launched from the program.
