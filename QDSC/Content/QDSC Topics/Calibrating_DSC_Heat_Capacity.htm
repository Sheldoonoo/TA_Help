Calibrating DSC Heat Capacity

 

The type of heat capacity calibration that you should perform will depend upon whether you are calibrating the Q2000 or Q1000 in its standard operating mode, or when calibrating the Q2000, Q200, Q1000, or Q100 in the MDSC operating mode.

Select from the following topics for more information:

  • Heat Capacity Calibration for Q2000/Q1000 (standard)
  • Heat Capacity Calibration for MDSC

Heat Capacity (Q2000/Q1000) Calibration

Heat capacity can be obtained in a single experiment with the DSC Q2000/Q1000. Heat capacity calibration is required to measure accurate sample heat capacity values. The calibration uses a standard material (such as sapphire) with a known heat capacity at a specific temperature of interest. Before performing the calibration experiment, the previously used heat capacity calibration values need to be manually set to the default value of 1.0. Calibration is typically done at a heating or cooling rate of 10 to 20 °C/min. Since this calibration (measurement) uses the absolute heat flow signal (Cp = Heat Flow Rate/Heating Rate) to calculate the sample's heat capacity, the first segment of the method should be a "Zero Heat Flow" segment at a temperature in the middle of the temperature range for the experiment. This corrects for any drift in the heat flow signal, since the last calibration.

At the end of the experiment the heat capacity calibration constant is calculated by dividing the expected value of heat capacity by the measured value at a specific temperature. The calculated constant is then manually entered into the instrument control software on the accessed from the Calibrate/Cell/Temperature Table menu, where it is automatically applied to future experiments.

Q2000/Q1000 Heat Capacity Calibration Procedure

This heat capacity calibration is performed after calibration of the DSC for Cell Resistance and Capacitance, Cell Constant and Temperature. (The calibration wizard can be used to perform the aforementioned calibrations. See also: DSC Calibration Wizard)  Heat capacity calibration is only done using the T4P heat flow signal on the Q2000/Q1000, and is only necessary if accurate heat capacity measurement is desired. Follow the instructions below:

Sample Preparation

Prepare and load the calibration sample as directed in the following steps.

  1. Use the type of pan that will be used to run samples of interest. Generally this will be:
  1. For the Q2000: Tzero™ pans (PN 901683.901) or Tzero low mass pans (PN 901670.901).
  2. For the Q1000: Crimped aluminum pans and lids (PN 900786.901 and PN 900779.901) or hermetic pans and lids (PN 900793.901 and 900794.901).
  3. See also: Selecting a Sample Pan
  1. Select a pair of pans and lids that will be used for the sample and reference pans. It is good practice to match the weight of the two pans (with lids) to within 50 µg. However, when utilizing the T4P mode of the Q2000 or Q1000, it is not necessary to match the pan mass.  Make sure that you weigh the pans and lids, and record their weights for use later in the procedure.
  2. Place the sapphire heat capacity calibration sample (PN 915079.901 for all Tzero pans (including standard Tzero pans) or PN 915079.902 for hermetic pans) into the sample pan and crimp the lid in place using the sample encapsulating press. Make sure that you record the mass of the sapphire for use later in the procedure. See also: Encapsulating a DSC Sample  
  3. Place the pans into the DSC cell when ready to perform the calibration experiment.

Instrument Preparation

Set up the instrument parameters as directed below:

  1. Access the Experimental View//Summary Page. Select "Standard" from the Mode drop-down list. Calibration of heat capacity is not performed in the "Calibration" mode.
  2. Select Calibrate/Cell/Temperature Table from the instrument control main menu. Set the Cp Constant (Direct) to 1.0 prior to starting the experiment.
  3. Access the Experimental View/Summary Pageagain.
  4. Enter the requested sample information, including the Sample Name and Sample Size (mass). Enter the weight of the empty sample and reference pans (with lids). The software will correct for any difference in weight when the T4P signal is used.
  5.  Select the Pan Type.
  6. For  Autosamplers only: Select the Pan number and Reference number.
  7. Select a file name for the data to be analyzed later.
  8. Access the Notes Page. Verify that the cell purge gas is ON and that the flow rate is correct. The same purge gas and flow rate should be used as were used for previous calibration experiments and which will also be used for actual sample runs.
  9. If a Refrigerated Cooling System (RCS) is being used, it should be running for at least one hour prior to calibration or performing any sample runs.

Experimental Conditions

Set up and run the following calibration experiment:

Access the Experimental View/Procedure Page. Select a "Custom" test, then click on the Editor button. See Creating Custom Methods for detailed instructions. Program and run a method as defined below where:

a = desired starting temperature

b = desired ending temperature

c = average of temperatures a and b

d = selected heating rate.

Use the definitions above to create the following method:

1  Zero heat flow at c

2  Sampling Interval 1.0 seconds/point

3  Equilibrate at a

4  Isothermal for 5.0 minutes

5  Ramp d °C/min to b

A heating rate of 10 to 20°C/min (d in step 2) provides the best combination of accuracy, precision, and resolution.

Calculation and Input of Calibration Factor

After you have run the heat capacity calibration experiment outlined above, open the Universal Analysis program and follow these steps:

  1. Open the calibration data file and plot the Heat Capacity signal versus Temperature.. Data should appear as seen in the figure below, with the exception of the labeled values.

  1. At approximately the same temperature as that used to zero the heat flow signal, which is the midpoint of the desired temperature range, select Analyze/Curve Value at X from the main menu. Move the cursor to the desired temperature to have the program label the value of the curve as seen in the figure above.
  2. Refer to the table of heat capacity values for sapphire as a function of temperature as found in TA Instruments Technical Note 008. Click on the following link to open an Acrobat Reader pdf file and see Table 1 Open Heat Capacity Tables.  From the table obtain the theoretical value of sapphire from the temperature selected in Step 2.
  3. Calculate the heat capacity calibration factor using the following formula (also shown in the figure above):

KCp (Direct) = Theoretical (Table) Value / Measured Value

  1. Enter the calibration factor into the instrument control software. This is done by selecting Calibrate then Cell/Temperature Table from the main menu and choosing the Cell Calibration tab. Set the Cp Constant (Direct) to the value determined above. Generally, the constant should be in the range of 0.90 to 1.10.

Calibration is now complete and the instrument can be used to measure the heat capacity of actual samples. It can be done with a single experiment simply by saving the signal "Heat Capacity" as one of the stored signals in the file. See also:  Selecting Signals to Store

NOTE: For the most accurate heat capacity values it is recommended to calculate KCp (Direct) at each temperature of interest.  This can be done easily by exporting temperature and heat capacity data to Microsoft® Excel for calculation.

Heat Capacity (MDSC) Calibration

MDSC heat capacity calibration is only required if it is desirable to measure the absolute value of the sample's heat capacity.  The calibration uses a standard material (such as sapphire) with a known heat capacity at a specific temperature of interest. Before performing the calibration experiment, previously used calibration values need to be manually set for both Heat Capacity and Reversing Heat Capacity to the default values of 1.0. Except for heating rate, the calibration experiment should be run under similar conditions (pan type, modulation amplitude, and period) as will be used for subsequent samples. The heating rate can be set to a nominal value of 2 to 3 °C/min or the calibration experiment can be performed isothermally, if only calibrating the Reversing Cp signal.

At the end of the experiment the calibration constants for the Total and Reversing Heat Capacity signals are calculated by dividing the theoretical value of heat capacity, at the desired temperature, by the measured value.  The constants are then manually entered into the instrument control software on the accessed from the Calibrate menu, where they are automatically applied to future experiments.

MDSC Heat Capacity Calibration Procedure

This heat capacity calibration is performed after calibration of the DSC for Cell Resistance and Capacitance, Cell Constant and Temperature. (The calibration wizard can be used to perform the aforementioned calibrations. See also: DSC Calibration Wizard)  MDSC Heat capacity calibration can be done on the Q200/Q100 (requires MDSC option) or Q2000/Q1000 and is not necessary unless it is desired to measure absolute sample heat capacity. Follow the instructions below:

Sample Preparation

Prepare and load the calibration sample as directed in the following steps.

  1. Use the type of pan that will be used to run samples of interest. Generally this will be:
  1. For the Q2000: Tzero pans (PN 901683.901) or Tzero low mass pans (PN 901670.901).
  2. For the Q1000: Crimped aluminum pans and lids (PN 900786.901 and PN 900779.901) or hermetic pans and lids (PN 900793.901 and 900794.901).
  3. See also: Selecting a Sample Pan
  1. Select a pair of pans and lids that will be used for the sample and reference pans. It is good practice to match the weight of the two pans (with lids) to within 50 µg. However, when utilizing the T4P mode of the Q2000 or Q1000, it is not necessary to match the pan mass.  Make sure that you weigh the pans and lids, and record their weights for use later in the procedure.
  2. Place the sapphire heat capacity calibration sample (PN 915079.901 for all Tzero pans (including standard Tzero pans) or PN 915079.902 for hermetic pans) into the sample pan and crimp the lid in place using the sample encapsulating press. Make sure that you record the mass of the sapphire for use later in the procedure. See also: Encapsulating a DSC Sample  

NOTE:  When using hermetic pans with solid samples (such as the sapphire calibration sample), it is recommended that the lid be inverted to provide good thermal conduct between the sample and both the top and bottom of the pan. This will minimize thermal gradients and provide a more accurate measurement of heat capacity. The bottom of the pan should be checked for flatness after crimping. If there appears to be any roundness, it can be flattened with light pressure by using the flat portion of the tweezers and any flat surface.  

  1. Place the pans into the DSC cell when ready to perform the calibration experiment.

Instrument Preparation

Set up the instrument parameters as directed below:

  1. Access the Experimental View/Summary Page. Select "Modulated" from the Mode drop-down list. Calibration of heat capacity is not performed in the "Calibration" mode.
  2. Select Calibrate/Cell/Temperature Table from the instrument control main menu. In the MDSC section of the page set the Cp Constant (Total) and the Cp Constant (Reversing) calibration values to 1.0 prior to starting the experiment.
  3. Access the Experimental View/Summary Page. Enter the sample name and sample size (mass). Enter the weight of the empty sample and reference pans (with lids). The software will correct for any difference in weight if the T4P signal is being used (Q2000/Q1000 only).
  4. Select a file name for the data to be analyzed later.
  5. Access the Notes Page. Verify that the cell purge gas is ON and that the flow rate is correct. The same purge gas and flow rate should be used as were used for previous calibration experiments and which will also be used for actual sample runs.
  6. If you are using a Refrigerated Cooling System (RCS), it should be running for at least one hour prior to calibration or performing any sample runs.
  7. Access the Experimental View/Summary Page. Click the button to open the Signal Selection window. Verify that the Heat Capacity and Reversing Cp signals are being saved to the data file. See Selecting Signals to Store for information.

Experimental Conditions

Set up and run the following calibration experiment:

  1. Since there are no transitions in the sapphire standard, a heating rate of 2°C/min is recommended.
  2. Access the Experimental View/Procedure Page. Select a "Custom" test, then click on the Editor button. See Creating Custom Methods for detailed instructions. Program and run a method as defined below where:

a = desired starting temperature

b = desired ending temperature

c = average of temperatures a and b

Use the definitions above to create the following method:

1  Zero heat flow at c

2  Sampling Interval 1.0 seconds/point

3  Equilibrate at a

4  Modulated +/- 1.0°C every 100 to 120 seconds

5  Isothermal for 5.0 minutes

6  Ramp 2°C/min to b

The modulation period selected for measuring absolute heat capacity is dependent on the pan type selected. A modulation period of 100 seconds is recommended when using either standard aluminum pans or the Tzero low mass aluminum pans. Use a longer period of 120 seconds when using either aluminum hermetic pans or Tzero aluminum pans.  It should be noted that the period of modulation for the specified Tzero pan is the same no matter which lid is chosen.  The longer period provides more time for the heat to flow and thus improves both accuracy and precision.

Calculation and Input of Calibration Factor

After the calibration experiment is complete, open the Universal Analysis program and follow these steps:

  1. Open the calibration data file and plot the Heat Capacity and Reversing Cp signals versus Temperature. Data should appear as seen in the figure below, with the exception of the labeled values.

  1. Select the Heat Capacity signal by clicking on it. At approximately the same temperature as that used to zero the heat flow signal, which is the midpoint of the desired temperature range, select Analyze/Curve Value at X from the main menu. Move the cursor to the desired temperature to have the program label the value of the curve in the figure above.
  2. Select the Reversing Cp by clicking on it. Repeat the measurement as described in the previous step. The plot seen above should be similar to your results.
  3. Refer to the table of heat capacity values for sapphire as a function of temperature as found in TA Instruments Technical Note 008. Click on the following link to open an Acrobat Reader pdf file and see Table 1: Specific Heat Capacity Tables. From the table obtain the theoretical value of sapphire for the temperature selected in steps 2 and 3.
  4. Calculate the heat capacity calibration factor for each signal using the following formulas:

KCp (Total) = Theoretical (Table) Value / Measured Value

KCp (Reversing) = Theoretical (Table) Value / Measured Value

  1. Enter the calibration factor into the instrument control software. This is done by selecting Calibrate then Cell/Temperature Table from the main menu and choosing the Cell Calibration tab. Input each of the measured calibration values. Generally, the calibration values should be in the range of 0.90 to 1.10.

Calibration is now complete and the instrument can be used to measure the heat capacity of actual samples using the Reversing Cp (Rev Cp) signal. See also:  Selecting Signals to Store

NOTE: When using MDSC to calculate the absolute Cp it is ONLY necessary to utilize the Reversing Cp signal.

NOTE: For the most quantitative absolute heat capacity values of a sample it is recommended that you calculate KCp (Direct) at each temperature of interest. This can be done easily by exporting temperature and heat capacity data to Microsoft Excel for calculation. The existing spreadsheet can be altered and then reopened in Universal Analysis.
