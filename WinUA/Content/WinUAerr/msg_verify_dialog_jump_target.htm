Jump Target Incorrect

The jump target for this dialog may have moved. Verify that the correct target step is selected.

 

Problem:

The number of text lines in a multi-line macro dialog containing a "Jump" step has changed because "Extended Text" steps, following a "Dialog" step, have been inserted or deleted. This message alerts you that the target step for your "Jump" step may now be incorrect.

 

Solution:

Check your "Jump" step target and make any necessary changes to allow the macro to work correctly.
