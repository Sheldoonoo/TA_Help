Lists for Step Change Between

Use this list to select how you want the step change (height) to be determined in step transition calculations. The step change is calculated as the change in the Y-axis between the two choices made on the Analysis Options  Step/Change Page. Choose from the following:

 

  • Start  the step change will be computed using the specified start limit on the Y-axis.

  • Onset  the step change will be computed using the step transition onset point on the Y-axis.

  • Midpoint  the step change will be computed using the onset tangent or end tangent Y-axis value evaluated at the midpoint X-axis value.

  • End  the step change will be computed using the step transition end point on the Y-axis.

  • Stop  the step change will be computed using the specified stop limit on the Y-axis.
