Status Message 735

Cooling unit selected based on calibration file.  Verify before use.

Problem:

This message is displayed after you reset the instrument parameters using the Instrument Setup wizard. When the instrument parameters are reset, the type of cooler selected previously is lost. The cooling unit will be selected based on the current calibration file.

(The instrument parameters are reset as follows:  Select Tools/Instrument Setup. Then press Next until the Transfer Program Files to the Instrument window is displayed. The Reset Parameters button is clicked and the reset button on the back of the instrument is pressed.)

Solution:

You will need to verify that the cooler selected for your instrument is correct. Select Tools/Instrument Preferences/DSC Only page. Verify the cooler or select the appropriate cooler, then click OK.
