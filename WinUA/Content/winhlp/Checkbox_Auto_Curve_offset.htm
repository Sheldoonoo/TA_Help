Checkbox Auto Curve Offset

Check the Curve Offset box to automatically apply the Auto Offset settings from the Curve Overlay Options  Auto Page to each axis in the curve overlay plot. To change the offset settings, use the Overlay/Manual Setup function to manually adjust the offset for each curve in the overlay plot.

 

Auto Curve Offset Checked

Checking the Auto Curve Offset box is equivalent to pressing the Auto Offset button on the Curve Overlay Options  Curves Page for each overlay axis.

 

Auto Curve Offset Unchecked

If you do not check the Auto Curve Offset box, then no offset is applied to any curve in the overlay plot.
