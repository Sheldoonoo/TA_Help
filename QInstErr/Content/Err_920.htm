Message 920

Poisson's Ratio set

This is an informational message to let you know that Poisson's ratio has been set at the value displayed. This is the DMA parameter that defines the ratio of transverse contraction per unit dimension to the elongation per unit length, when the sample is subjected to a tensile stress. Typical values are:  0.5 for rubber materials, 0.33 for metals, and 0.44 for most plastics. Consult a chemistry reference book such as the CRC Handbook of Chemistry and Physics for applicable values.
