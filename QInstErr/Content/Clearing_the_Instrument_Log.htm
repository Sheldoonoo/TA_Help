Clearing the Instrument Log

 

You can selectively clear off the instrument messages, if desired, using this window. You can select the Keep the Last button then choose the Month(s) and Week(s) that you wish to leave in the log.

NOTE:  Three months is the maximum amount of time messages that can be saved in the log. If you wish to save messages for a longer period, you will need to save them to a file. See Using the Instrument Log for information.

If desired, you can click the Delete All Entries button to clear off all messages.

For more information about the Instrument Log, click here Using the Instrument Log.
