Introduction to QC and Statistics

The following functions are available in the software:

  • Smoothing Data
  • Quality Control (QC) Pass / Fail
  • Statistics
  • Standard Error

Quality Control (QC) Pass / Fail Determination

  1. Load two or more results files.
  2. Display them on the same graph.
  3. To use the QC function:
  4. Click on the reference curve.
  5. Select Analysis from the menu.
  6. Click on QC.
  7. Use the Options item to define your QC specification. The results (pass or fail) will be displayed on the graph as part of the key.

Statistics

  1. Display the results file in File view.
  2. Select View from the menu.
  3. Click on Statistics. You can select to show the mean, standard deviation and variance.

NOTE: These values are for each column in one results file.
