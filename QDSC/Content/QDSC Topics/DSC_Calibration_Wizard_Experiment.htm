DSC Calibration Wizard Experiment

Experiment in Progress

This window displays the progress of the active calibration experiment in real time. Start Experiment initiates the experimental run. Once the experiment is started, the Start Experiment key changes to Stop Experiment, allowing you to stop the experiment if a problem occurs. Cancel aborts the wizard. When the experiment is complete, click the Next button to advance to the next step in the calibration process.

This window is displayed several different times during the use of the Calibration Wizard. For information see the topics below.

Cell Resistance and Capacitance Calibration Results

This window is displayed after the Baseline Run and Sapphire Run are complete. The software will automatically calculate and display the sample and reference resistance and capacitance values over the calibration temperature range and send this result to the instrument. A typical result should be two sets of parallel, closed spaced curves. The resistance curves should have a negative slope (i.e., resistance decreases with increasing temperature). The capacitance curves should have a positive slope (i.e., capacitance increases with increasing temperature). A copy of the calibration results file is saved in a subfolder "calibration" within the original data file location. The filename structure of these files is {cell number} {cooler} {date/time}. Once you have reviewed these results, press Next to continue with the next phase of calibration. The Calibration Wizard goes on to DSC Calibration Wizard Constant & Temperature. See also: Using the File Utility.

Cell Constant & Temperature Results

An analyzed plot of the melting peak for the metal standard appears at the end of the experiment. If you are satisfied with the analysis, press Next to save the cell constant and temperature calibration results to the instrument. Otherwise, select the Analyze button to display the analysis markers. Reposition these markers at the desired location and select Limits OK. You can also select these options using the pop-up menu.

If you click the Cancel button, the Calibration Wizard is terminated and the cell constant is reset to a value of 1.0.
