Macro Editor Window

Use this window to view a macro as you capture it and to edit a macro after capture.

 

You will see a list of the macro steps already captured, along with a specific macro menu across the top of the window. Select the functions desired from the menu and close the window when you have finished.

 

NOTE: If the Macro Editor window gets covered up by another window, you can bring it back to the front by pressing Ctrl+E two times.
