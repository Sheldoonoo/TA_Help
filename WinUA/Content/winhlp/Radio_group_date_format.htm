Radio Buttons - Date Format

Choose from one of the following ways to set up the date displayed by the program (e.g., displayed on the printed graph and in the parameter block information).

  • DD-MMM-YYYY: Click on this format to have the date displayed with the two-digit day first, followed by the three-letter abbreviation for the month, then the four-digit year. For example: 24-Dec-2003

  • YYYY-MM-DD: Click on this format to have the date displayed with the four-digit year first, followed by the two-digit month representation, then the two-digit day. For example: 2003-12-24.
