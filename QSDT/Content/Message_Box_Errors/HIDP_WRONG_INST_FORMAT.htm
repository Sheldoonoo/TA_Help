Cannot Use Calibration File

Cannot use the calibration file.  It was written by a ___ not a ___.

When using previously created calibration files to calibrate an instrument, you must use a data file that was created using the same type of instrument. If you need to create a SDT Q600 calibration file, for example, you can use the SDT Q600 calibration wizard or manual method to create the file, but it must be made on an SDT Q600.
