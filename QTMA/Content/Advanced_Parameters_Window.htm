Equilibration criteria are used at the beginning of a stress method to allow the sample to mechanically and thermally equilibrate before the sample is displaced.

Choosing the Advanced Parameters  

 

Use the Advanced Parameters window to define more advanced instrument parameters that may not normally be changed for experiments, once they are set.

To enter advanced parameters, follow these steps:

  1. Enter the desired Data Sampling Interval in seconds per point. This instrument parameter defines how often the system reads and stores a data point per second.  For example, if 2.0 is entered, the system stores a data point every two seconds, provided the data compression threshold values are reached.  Range is between 0.1 and 1000.0.
  2. Enter any Additional Weight on Tray. This instrument parameter is used to enter the weight, in grams, of the items placed on the weight tray. Up to 100 grams may be added. The static weight is added to the programmed force in the data file.
  3. Choose the Method Start Conditions, which are functions that can be automatically executed before a method begins. Select the desired Auto Zero Dimension:
    • Selecting Yes automatically sets the signal value to zero microns (µm) before the method begins.
    • Selecting No maintains the current signal value at method start.
  4. Select the radio button for desired Equilibration Criteria, if applicable:
  1. Select the Time button if you wish to have the equilibration criteria expressed in minutes.
  2. Select the Rate button if you wish to have the equilibration criteria expressed in the rate of deformation (µm/min).
  1. Click the OK button when finished.
