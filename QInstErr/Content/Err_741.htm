Message 741

Cannot select another macro filename.  Can only select a maximum of ____ filenames per instrument.

Problem:

You have attempted to select another macro filename, but the system cannot support any more macro files. There is a limit to the number of macro files that can be loaded in a sequence.

Solution:

Use one of the macros already loaded in another run or delete one of the macro filenames used by another run and replace it with the new macro file.
