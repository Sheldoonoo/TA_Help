Window Already Open on Touch Screen

Platform calibration window already opened on instrument touch screen.

There are two ways to access the TGA platform calibration functions, either through the instrument control program (Calibrate/Platform) or by touching the PLATFORM CAL key on the TGA Q500 Control Menu touch screen.

This message occurs when you access the platform calibration functions using the touch screen and, while the instrument is actively calibrating the platform, you attempt to access the same functions using the instrument control program.

We recommend that you choose one of the two methods to perform platform calibration and run the operation all the way from beginning to end using the chosen window. Do not switch between the instrument touch screen and the controller in the middle of platform calibration.
