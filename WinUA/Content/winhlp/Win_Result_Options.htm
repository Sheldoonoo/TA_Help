Result Options Window

Use the Results Options window to individually select the types of results and options you want to display in your results report, whether it is displayed in the regular Universal Analysis report form or in a spreadsheet.

 

You can click the All button to turn on all options and values or click the None button to turn off all options and values.

 

  1. Check or uncheck the desired result values from the list that you want to display in your results report or spreadsheet.

  2. Check or uncheck the desired result options that you want to display in your results report or spreadsheet. See the figure below to help you identify the "Display options" parts of the report (Multiplex value and Signal name are not shown).

  3. Click Save to save the settings to the initialization file or OK to save the settings for this session only.
