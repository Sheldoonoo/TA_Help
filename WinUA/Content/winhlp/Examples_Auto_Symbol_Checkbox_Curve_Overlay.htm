Examples Auto Symbol Checkbox (Curve Overlay)

When using the Auto Symbol checkbox on the Auto Configure window, the following applies:

 

Auto Symbol Checked

Checking Auto Symbol for Curves (see the figure below) is equivalent to pressing the Auto Symbol button on the Curve Overlay Options  Curves Page for each overlay axis.

Checking Auto Symbol for Axis (see the figure below) assigned the first Auto Symbol type to all curves on overlay axis Y1, the second Auto Symbol type is assigned to all curves on axis Y2, etc.

Auto Symbol Unchecked

If the Auto Symbol box is not checked, the symbol type for each curve in the overlay will be the same as that used in the individual file graph.
