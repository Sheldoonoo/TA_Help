Universal Analysis Advantage Integrity Help Topics

 

Within the Advantage Integrity system, much of the operation of the Universal Analysis program is unchanged when compared to its nonIntegrity counterpart. Universal Analysis can open both data stored to file (to view historical data) and data stored in the TA database. However, while logged into the Advantage Integrity database, only database record files can be accessed and audit trail generation begins. These two modes of operation (e.g., file-mode and database-mode) are mutually exclusive.
