Viewing Calibration Parameters

 

Access the Cell/Temperature Table from the Calibration menu to view the currently loaded calibration constants and temperature calibration points for the cell.

Cell Calibration Parameters

Use the Cell Calibration Parameters area to set the cell constant that the instrument will apply to the corrected data.

Cell Constant:

Cell Constant is a calibration factor used to adjust the calorimetric response of a DSC cell or the thermal expansion of a TMA. The cell constant is the ratio of the known value to the measured experimental variable. Typically the expansion coefficients of aluminum or copper (TMA) are used. The calibration is based on a run made in the calibration mode using the experimental conditions to be used in subsequent measurements. See also:  Calibrating TMA Cell Constant.

The Cell Constant parameter is automatically updated in this window after cell constant calibration has been performed, accepted, and saved under Calibrate/Analysis.

  • If this Cell Constant value is correct, select OK.
  • If a change is desired, enter the correct Cell Constant value in the appropriate field and select OK.

Temperature Calibration Table

This area shows the temperature calibration table that the instrument applies to the collected data. Use this window to enter from one to five temperature calibration points (pairs of observed and correct temperature points). The observed and correct temperature corresponds to the experimental and theoretical transition temperature (e.g., melting point) of the calibrant respectively.

NOTE: A multiple-point calibration is more accurate than a one-point calibration. A single-point calibration shifts the sample temperature by a constant amount. A two- or more-point calibration shifts the temperature by a constant amount below the first point, uses a smooth curve through the calibration points and maintains a constant temperature shift after the last point.

Temperature calibration is useful for experiments in which precise transition temperatures are essential.

To temperature calibrate the TMA, you need to analyze a high-purity metal for its melting temperature, and then enter the observed and correct values in the temperature calibration table. See also:  Calibrating the TMA Temperature.

  • If these values are correct, select OK.
  • If new values are to be entered:
    1. Select the Reset Temperature Table button to remove all previously stored values.
    2. Determine the melting temperature points.  
    3. Enter the Observed and Correct temperature points in the table.
    4. Select OK when all points have been entered to save the settings to the instrument.
