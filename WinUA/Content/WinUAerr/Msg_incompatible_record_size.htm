Incompatible Record Size

Incompatible record size.

 

Problem:

One of the saved analysis records stored in this data file is larger than expected.

 

Solution:

The saved analysis records in this file were probably written by a newer version of the data analysis program. Install the newer version program or open the data file without restoring the analysis. The data analysis program name and version number that saved this analysis can be determined by opening the data file with a text editor (such as the Report Editor) and examining the last few lines of the file.
