Instrument Parameters Window - DMA

The DMA Instrument Parameters window is used to enter the desired Poissons ratio.

 

1. Enter value desired for the Poisson's ratio. This is a DMA parameter that defines the ratio of transverse contraction per unit dimension to the elongation per unit length, when the sample is subjected to a tensile stress.

2. Click the Save button, if you want to use these as your new default settings or click the OK button to use these settings for this session only. 

 

  • 0.5 for rubber materials

  • 0.33 for metals

  • 0.44 for most plastics
