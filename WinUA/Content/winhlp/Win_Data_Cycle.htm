Data Cycle Window

Use the Data Cycle window to choose data for segregation using the cycle markers contained in the data file. (See the online manual accompanying Thermal Advantage Instrument Control for information on creating methods that will add cycle markers to your data.)

 

1. Enter the desired cycle number to be used for analysis. NOTE: Multiplexed data (e.g.,multifrequency, multiamplitude data) may also be segregated using this option.

2. Click the OK button.

 

NOTE: This option is only available for data files containing either cycle markers, multiplexed data, or data storage off/on method segments.
