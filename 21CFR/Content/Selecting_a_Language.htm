Selecting a Language

 

You can select the language to be used for the TA System Manager, after logging in, by following these instructions:

  1. Click the icon in the upper left-hand corner of the TA System Manager window to display the drop-down menu.
  2. Choose Select language, then select one of the three available languages, English, Deutsch (German),  (Japanese) or(Chinese). After you select the language, the program will automatically switch and will be displayed in the chosen language.
  3. Repeat the process to change the language again.
