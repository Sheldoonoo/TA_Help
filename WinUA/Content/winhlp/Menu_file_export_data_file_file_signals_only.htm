File Signals Only

Use this option to export selected signals from the data file only. When you select File Signals Only, the Export Data File window is displayed with the signals obtained from the data file displayed and checked.

 

You can limit the output signals by selecting only those signals that you want to save in the exported file. (At least one signal must be selected.)
