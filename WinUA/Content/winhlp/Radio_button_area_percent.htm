Area Percent (Integral Type)

Use the Area Percent radio button to take the end result of the raw integration and show the percent contribution of each peak in the selected region of the curve (assuming the selected region to be 100%). If you want to determine the percent composition of various components in a material, select the area percent option. Then, when the program finishes plotting the integral curve, use one of the Curve Value functions to mark each step of the curve with its percent contribution.
