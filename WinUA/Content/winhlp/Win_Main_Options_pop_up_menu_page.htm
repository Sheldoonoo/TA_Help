Main Options Window  Pop-Up Menu Page

Use the Pop-Up Menu page of the Main Options window to set up the items that you want to display every time the pop-up menu is displayed by clicking the right mouse button.

 

Follow these steps to set up the pop-up menu:

  1. Select the menu items that you want to display by using one of two methods:

    1. Check each desired menu name from the scrollable menu item list.

    2. Click on the button that selects the default menu items that apply to the button label. For example, you can click on the General button to use only the most common menu items.
  1. Click the Save button, if you want to use these as your new default settings or click the OK button to use these settings for this session only.
