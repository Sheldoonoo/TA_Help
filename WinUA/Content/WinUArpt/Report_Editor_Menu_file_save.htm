Save (Report Editor/File Menu)

Use the Save option to save the current text file. The default text file format is Rich Text Format (.rtf). Use Rich Text Format to save the font, text, size, etc. Use ordinary test format (.txt) to save only the text characters and not the format information. Always use ordinary text format (.txt) to save ASCII or spread sheet data files.

 

NOTE: The file extension (.rtf or .txt) determines the save format. Any extension other than .rtf will save as plain text (.txt).
