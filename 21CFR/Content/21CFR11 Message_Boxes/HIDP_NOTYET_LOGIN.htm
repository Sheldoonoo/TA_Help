Not Logged In

The selected user did not log in or a Log-in Session Time Out occurred causing their session to be terminated.

You must log into the database system properly in order to access its features. If you stop using the program, the session will time out and you will have to log in once again. The amount of time before the session times out is established using the Log-in Session Timeout function found on the System Settings Manager/Settings Page.
