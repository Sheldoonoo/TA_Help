Language Selection Successful

Change of language successfully completed. Press the OK button to close this program and then press the reset button on the back of the instrument to complete the process.

  1. Select OKon this message box.  The instrument control software will automatically close.  

  2. Complete this operation by shutting down the instrument:  Select Control\Shutdown from the menu. Click the Shutdown button to begin the shutdown process. (Alternatively, you can perform an instrument shutdown from the instrument’s touch screen – Display options, if available.)

If the language you want is not displayed (English, German, Japanese, and Chinese are the available options), you need to download the language file as described in Updating Software and Downloading a Language.
