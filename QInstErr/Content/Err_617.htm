Message 617

Selected pan is not on current tray. Please remove tray #___ and install tray #___, then press OK to resume run.

Problem:

You have entered a pan number for your experiment that is not contained on the tray currently installed on the Auto TGA. Each tray holds up to 16 numbered samples and there are four available trays. Therefore, you can have samples numbered from 1 through 64 when using the Auto TGA.

Solution:

Remove the current tray and install the tray with the selected pan number on the AutoTGA platform. Press OK to resume the experiment.
