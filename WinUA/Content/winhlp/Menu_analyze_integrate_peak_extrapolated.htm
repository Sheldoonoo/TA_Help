Extrapolated

Use the Extrapolated option to perform a peak integration using an extrapolated baseline.

 

To calculate an extrapolated baseline, four points are selected (in any order). The program automatically sorts the points. Tangent lines will be drawn between two of the points so that the extrapolated baseline intersects the range of data bracketed by the other two points (intersection range). The program automatically determines whether the first two points (as seen in Figure 1) or the last two points (as seen in Figure 2) specify the tangent line. This is accomplished by the program as it tries both potential tangent lines and then uses the tangent line that will intersect the data between the other points.

Figure 1 (left)   Figure 2 (right)

 

For example, in Figure 1, a tangent line drawn between Point 1 and Point 2 will not intersect the data found between Points 3 and 4. This arrangement is discarded as incorrect. The program will then draw a tangent line between Points 3 and 4. This tangent line does intersect the data found between Points 1 and 2, so this becomes the final extrapolated baseline.

 

If two of the four points are placed on top of each other, the baseline is extrapolated to be a horizontal baseline from these two points to intersect the data between the other two points, as seen in Figure 3.

  Figure 3

 

Tip:

You can use this option to integrate a peak where one end of the integration baseline does not intersect the curve as follows:

 

1. Select Analyze/Integrate Peak/Extrapolated. Four markers will be displayed.

2. Place three markers directly on top of each other at the point of baseline start on the curve. (If you use the Manual Limits Window to enter the limit points, be sure to specifically enter or tab through all four entries for the limit points to ensure that the program picks up the exact limit values.)

3. Place the fourth marker anywhere further out past the peak maximum.

4. Right click and select Accept Limits from the pop-up menu. The program will draw a horizontal baseline from the start point to a point that intersects a vertical line at the fourth marker position.

NOTE: This will also work in reverse where the extrapolation is made backwards (i.e., the point in space occurs at a lower x-axis value than the baseline start point.)
