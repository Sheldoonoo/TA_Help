Open Menu

Use the Open menu item to open a file. You can open a TA Instruments data file for analysis using the Universal Analysis program or you can launch applications associated with specific file extensions. For example, you can open Microsoft® Word files containing a ".doc" extension directly from Universal Analysis using this window to launch the Microsoft Word program.

 

You may also open certain non-TA Instruments data files using Universal Analysis. You will be prompted to convert the file to a TA Instruments file, if desired.

 

NOTE: The program must be installed on the computer in order to launch it.
