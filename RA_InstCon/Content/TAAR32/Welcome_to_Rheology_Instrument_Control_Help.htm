The Rheology Advantage Instrument Control software allows you to operate the rheometer manually or by  using procedures (or test methods).  These test methods can be saved as Procedure Files (Flow, Oscillation or Creep). When you analyze test samples, you need to configure the Procedure, Geometry and Notes.

The software also allows you to configure various options to do with the general operation and calibration of the system.
