Line Too Long

Text line exceeds maximum length of 60 characters.

 

Problem:

Each line of text contained in the macro dialog box can be only 60 characters long. You have attempted to enter too many characters in a line.

 

Solution:

When you insert a dialog into your macro, type only 60 characters per line. Use the Enter key to create a line break at the end of the 60-character line and continue typing your message on the next line.
