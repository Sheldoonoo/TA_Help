Opening a File

 

You can open a file on your computer hard disk or on a network drive that you are connected to. You can also open a file on a network, even if you don't have a connection to a network server, as long as your network supports UNC addresses.

Select from the following topics for more information:

  • Opening a File on Your Hard Disk or a Network
  • Opening a File on a Network Using a UNC Address
  • Opening an Earlier Version of a File

Opening a File on Your Hard Disk or a Network

  1. Click Open... in the File menu item.
  2. In the Look in box, click the drive, folder, or Internet location that contains the file.
  3. Click the file type you want in the Files of type box.
  4. In the folder list, double-click folders until you open the folder that contains the file you want.
  5. Double-click the file you want to open.

Opening a File on a Network Using a UNC Address

  1. Click Open... in the File menu item.
  2. Click the file type you want in the Files of type box.
  3. In the File name box, type the UNC address for the network share that has the file you want, and then press ENTER. For example, type \\Computer1\Files to open a file on the server Computer1, on the share Files.
  4. In the folder list, double-click folders until you open the folder that contains the file you want.
  5. Double-click the file you want to open.

Opening an Earlier Version of a File

  1. Click Open... in the File menu item.
  2. In the Look in box, click the drive, folder, or Internet location that contains the file.
  3. In the Files of type box select the Version 1 (V1) option.
  4. In the folder list, double-click folders until you open the folder that contains the file you want.
  5. Double-click the file you want to open.
