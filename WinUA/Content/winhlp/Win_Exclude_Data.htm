Exclude Data Window

Use this window to choose the desired option to be used when the program excludes the selected data region. Each option produces a different result.

 

You can remove the selected range of data points from the data used for your analysis. It does not alter the data file. The Exclude Data functions can be used to:

(1) Provide a way to create very small data breaks between thermal cycles, which allows each cycle to be distinguished with a different symbol using the "Marked as invalid data" option. (See Graph Options.) or (2) Correct explainable data anomalies using the remaining Exclude Data options.

 

1. Click on one of the following options to exclude points:

Marked as invalid data

Fitted to a straight line

Smoothed and blended with adjacent data

Removed and blended with adjacent data

2. Click the OK button to accept this choice or Cancel to abort the operation.

 

NOTE: Use caution when selecting the data exclusion range for all "blended" options. The blending region extends over a range that is approximately three times the width of the exclusion range. Transitions that are in close proximity to the exclusion range may be altered. If necessary, perform multiple smaller exclusions or select the "Straight line fit" and then smooth and blend the region around the straight line end points.

 

NOTE: To restore the graph to its original data, use Graph/Data Limits/Full Range.

 

NOTE: The data file will not be changed if you use the "Marked as invalid data" option.
