Macro Assign Clipboard Window

The Assign Clipboard window is used to insert a macro step that will copy the current value of a macro variable and/or a text string to the clipboard. This is useful for building custom annotations and tables that are placed on a plot or added to a report.

 

  1. Select (highlight) the desired position (step number) for the macro step insertion.
  2. Select Insert/Assign Clipboard from the Macro Editor menu. The Assign Clipboard window is displayed.

 

To use this window:

  1. Enter a Variable name, if desired. A macro variable name may consist of up to 30 characters. The first character must be alphabetic. The remaining characters may be alphabetic, digits, or the underline (underbar) character. Imbedded spaces are not permitted in macro variable names. A macro variable array element is specified by following the variable name with an array index enclosed in square brackets ("[<index>]"). The array index may be an integer constant or a simple variable name (no index).
  2. Enter the number of Significant figures for the copied value.
  3. Check the Round to integer box, if you want to round the value to the nearest integer.
  4. Enter a Text string of up to 30 characters, which will be appended to the end of the value when it is copied. If no variable name is specified, then only the text string is copied.

 

NOTE: Special characters such as trailing spaces can be included in the text portion of a macro step, if the entire text string is enclosed in single quotes (apostrophes). Two sequential apostrophes in a quoted string denote a single character, namely an apostrophe.

For example:

'Quoted text string.' {"Quoted text string."}

'A variable''s units.' {"A variable's units."}

''''   {single apostrophe}

''   {null string}

''   {single space}

'Text'  {"Text" followed by 4 spaces}

 

  1. Check the Line terminator box, if you want to append an optional carriage return and line feed (<crlf>) line terminator to the copied text.
  2. Select Clear clipboard before assigning value to replace any text currently on the clipboard with the entered text. Select Append value to existing clipboard text to append the entered text to any text already on the clipboard.
  3. Click OK when finished. The step will be inserted at the selected position. Click here to see example macros that have an Assign Clipboard step inserted.
