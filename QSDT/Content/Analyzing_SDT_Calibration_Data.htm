Analyzing SDT Calibration Data

 

Before analyzing the DTA Baseline, Temperature, TGA Weight, or DSC Heat Flow calibration data you will need to conduct the necessary calibration experiments.

Once the calibration experiments are complete, as indicated by a check mark next to the run numbers (s) on the sequence list, select the desired analysis option from the Calibrate/Analyze menu options or by clicking the   button on the tool bar. Specify the data filename(s). Two data files are needed to calibrate TGA weight and DSC heat flow.

Calibration results may be analyzed and saved after each calibration experiment or all together (e.g., after all the experiments have completed). Follow the instructions below:

  1. Temperature Calibration Only:  Specify the standard used for calibration. To choose the standard setting, click on the Setup button. See also:  Setting Up Calibration Standards.

  2. Rescale the plot, if desired. This can be done using the mouse to expand part of the curve. You can selectPrev(ious)ScaleorFull Scaleto display previous scaling.

  3. Analyze the curve by clicking on the Analyze button (or choose Analyze from the pop-up menu) to display the limit point markers. The active marker is indicated by a full-length vertical red line. Other markers are indicated by smaller red cross-hairs. Position the markers for the desired analysis limits. See the notes below.

NOTE:  For TGA Weight Calibration and DSC Heat Flow Calibration Only:  When selecting your analysis limits, the default limits of 200 and 1400°C are the optimum limits to use when calibrating the SDT up to 1500°C.  If you are only interested in a particular portion of the curve, such as running methods up to 1000°C, you may decide to choose more appropriate limits.

NOTE:  For DTA Baseline Calibration:  When selecting your analysis limits, position the two analysis markers in the desired location (bracketing the analysis region of interest).

    1. Move the active marker to the desired position on the curve using either the mouse or keyboard (see the directions below).

    2. Select (activate) the next marker by:  (a) Pressing the TAB key or (b) Positioning the pointer over a marker and clicking the left mouse button. (Pressing back TAB (Shift-TAB) will activate the previous marker)

    3. Repeat steps a and b until you have selected all the necessary points for analysis.

  1. Select the Limits OK button (or select Limits OK from the pop-up menu) to accept the  points. When calculated, the calibration results will be displayed.

  2. If the results are acceptable, all you need to do is save the results for each calibration by clicking on the Accept button (or by choosing Analyze/Accept results from the pop-up menu). These results will be automatically sent to the instrument.

  3. Print the results, if desired, by clicking on the Print button (or by choosing Print from the pop-up menu).

  4. Select the Close Analysis from the Analyze menu to close the analysis window.

Moving the Markers Using the Mouse:

  1. Position the pointer at the desired location.

  2. Double-click mouse button 1 to move the active marker to this point. This action causes the next marker to become active.

or

  1. Position the pointer over the active marker.

  2. Press and hold down the left mouse button while you drag the marker to the desired location.

  3. Release the mouse button. At this point you may use the arrow keys to position the marker more closely (see the directions below).

Moving the Markers Using the Keyboard:

Press the arrow keys to move the active marker to the desired position.

The marker will move in the following increments:

  • Arrow key  =  smallest step

  • Alt/arrow key  =  second smallest step

  • Shift/arrow key  =  second largest step

  • Control/arrow key  =  largest step
