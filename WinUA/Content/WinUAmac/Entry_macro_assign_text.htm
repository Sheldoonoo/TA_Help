Entry Text

Enter a Text string of up to 30 characters, which will be appended to the end of the value when it is copied. If no variable name is specified, then only the text string is copied.

 

NOTE: Special characters such as trailing spaces can be included in the text portion of a macro step, if the entire text string is enclosed in single quotes (apostrophes). Two sequential apostrophes in a quoted string denote a single character, namely an apostrophe.

For example:

'Quoted text string.' {"Quoted text string."}

'A variable''s units.' {"A variable's units."}

''''   {single apostrophe}

''   {null string}

''   {single space}

'Text'  {"Text" followed by 4 spaces}
