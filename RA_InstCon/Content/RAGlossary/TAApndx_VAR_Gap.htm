Gap

Description

The distance between the upper and lower stages of the measuring system, with the last gap zero position obtained taken as the reference. The gap is constantly monitored during an experiment, and may vary if the Control normal force option is active. It can be corrected for the system thermal expansion if the Gap temperature compensation option is selected.
