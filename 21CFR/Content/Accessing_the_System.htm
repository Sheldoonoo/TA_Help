Accessing the System

Logging In

When you open any software component of the database system, either the TA System Manager Program, an Instrument Control program, or the Universal Analysis program, the Log In window is displayed.  User access (log in) is based upon a user ID (or log-in name) and password.

  • The first time that you use the program you will be required to enter your User ID and default password, as defined by the TA System Manager.  This process requires the first-time user to immediately change their password upon initial log in. See also:  Changing your Password
  • User ID and/or password may be case-sensitive if this option is selected by your TA System Manager. See also:  Introducing the TA System Manager
  • After a period of inactivity the system will automatically log you out.  This time period (referred to as the "log-in session timeout") is defined by the TA System Manager.  However, it is recommended that you log out of  the system when you leave the system. See also:  Logging Out

The first time that you use the program you will be entering a default User ID and password.  This must be changed immediately.  Enter the case-sensitive default User ID and Password, then click the New Password button and follow the instructions in the next section.

Changing Your Password

The database system relies on password protection to maintain the integrity of the system and the audit trail.  Your log-in password may be changed at any time and must be changed when it expires.  The expiration date is set by your TA System Manager.  Shortly before your password expires, you will begin to receive “password expiration” notices each time you log in.

To change your password, follow these instructions:

  1. Open the desired program. Once the Login window is displayed, type in your User ID and old password, then select New Password.  
  2. Type in the New Password. All passwords must follow these rules:
    • The password must contain the minimum number of  characters defined by the TA System Manager. Any combination of numbers, symbols, or letters is allowed.  The window shown reflects the allowable number of characters. To change that number, see Setting Up the System.
    • Passwords cannot contain spaces.  
    • Passwords can not be reused for a specific user account
  3. Type in the new password again to confirm previous entry.
  4. Click the OK button.

Logging Out

When you are finished accessing these programs, you should log out of the system.  

  • If you are in Instrument Control, simply close the instrument window to log out.  This will not affect data collection.  
  • If you are in Universal Analysis, you can close the software (causing you to close all open records) or select File/Logout to temporarily log out of the system, maintaining the state of your work.
  • If you forget to log out, after a specific period of inactivity (interaction within these modules) as specified by your TA System Manager, the log-in window will automatically be posted and requiring log in to access the module again.
