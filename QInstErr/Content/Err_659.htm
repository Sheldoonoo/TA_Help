Message 659

Cannot select gas as "none" when "outside purge" is other selected gas.

Problem:

You have set one gas to be an "outside purge," this means that the flow cannot be regulated through the instrument using the Mass Flow Controller (MFC). An external flow meter is required. Therefore you cannot select the other gas to be "none" for the TGA because the gas flow rate to the balance and sample area must be individually regulated.

Solution:

  • Select another gas for Gas #2.

or

  • Select a gas other than "outside purge" for Gas #1, which can be controlled via the MFC individually for both the furnace and sample areas.
