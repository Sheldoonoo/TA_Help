Printing

 

Selecting the Print… item from the File menu will enable you to output the currently active screen.

Printing File View

You can output the table of results to a printer, a file or the clipboard. The software allows you to configure the output options.

  • Title – Allows you to specify a title (header) for the print out.
  • Details – You can specify how much information you wish to appear on the printout.
    • Full - All information about the procedure and instrument settings will be printed.
    • Full, no options - All information about the procedure will be printed.
    • Headings - Sample, procedure and geometry names will be printed.
    • Headings and notes - Sample, procedure and geometry names and notes will be printed.
    • In addition, you can specify if you wish one or two columns to be used.
  • Data – In some cases, you may not wish to print out all of the data, for instance if you have a long time-scale experiment. The software allows you to print out the start, end or a spread of data.
  • Output device – You can send the output to a printer, a file or the clipboard.

Printing a Graph

There are no printing options to configure for the graph. You can use the printer settings to adjust such items as orientation, print quality etc.
