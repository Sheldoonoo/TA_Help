Procedure for Setting the Gap

The actual gap used by a measuring geometry is not the same as the zero datum point, for instance a cone is backed away from the datum point by the truncation value of the cone.

  1. Select Geometry on the menu.
  2. Click on View.
  3. Click on the Dimensions tab.
  4. Enter the gap set value as follows:
    •  Cones: Enter the truncation height (from the geometry certificate.)
    •  Plates: Enter the required gap.
    •  Cylinders: Enter the gap (as specified on the geometry certificate.)
