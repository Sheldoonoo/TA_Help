Action

Select the action that you want to be performed after your message dialog response has occurred from the following list:

 

Continue macro executionWhen you select this action, after the appropriate response has been selected, the macro will continue.

Stop macro, wait for runWhen you select this action, after the appropriate response has been selected, the macro will stop and you must select Run from the Macro menu (or type Ctrl+R) to continue the macro.

Stop macro, wait for Accept LimitsWhen you select this action, after the appropriate response has been selected, the macro will stop and you must select the cursor limits on the curve and accept the limits. (In this case, the dialog replaces the standard Accept Limits function that can be captured during an analysis.)

Stop macro, wait for next OKWhen you select this action the macro will stop the running of the macro to wait for the operator to press the OK button on a window. Once the OK button has been pressed, the macro will continue.

Jump to macro startWhen you select this action, after the appropriate response has been selected, the macro will jump to its first step and begin running again.

Jump to macro endWhen you select this action, after the appropriate response has been selected, the macro will jump to its last step and end.

Jump to macro stepWhen you select this action, after the appropriate response has been selected, the macro will jump to a step number that you have entered in the Jump Step field.

 

NOTE: If the person responding to your macro dialog chooses the button that is opposite of the response you are asking for (the expected response is Yes and they choose No), the macro will keep on going and will not perform the action you requested for your dialog. If Cancel is chosen on the macro dialog, the macro will stop running at that point.
