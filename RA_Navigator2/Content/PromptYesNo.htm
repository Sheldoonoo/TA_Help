PromptYesNo

Prompts the user with a message requiring a yes or no answer. If the user selects "Yes," the return value is 1; if the user selects "No," the return value is 0.

Syntax

v0=PromptYesNo(<Message>, <CancelRequired>)

CancelRequired
 no cancel button = 0
 show cancel button = 1
