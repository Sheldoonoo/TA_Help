TAMA Not Installed

The wizard did not find the TA Message Agent (TAMA) installed on this computer. In order to import all Integrity-enabled instruments into the system, you need to install the TAMA or run the wizard on the computer with TAMA installed. If you continue, the import function will not be available and you need to use TAISM to add instruments later. Do you want to continue?

The TA Message Agent (TAMA) functions in the background to gather all online instrument information and transfer data to the database, therefore it must be present in order for the system to function correctly.

The database wizard looks for the TAMA portion of the Advantage software (TA Explorer) in order to include the ability to import Integrity-licensed instruments into the system. You can continue with the wizard, but the function used to import the instruments will not be available. However, instruments can be added later using TAISM.

  • Select Yes to continue.
  • Select No to abort the wizard, then go back and run this program from a computer which has the Q Series Advantage installed on the computer. If you continue you will need to manually import the instruments into the TA System Manager.
