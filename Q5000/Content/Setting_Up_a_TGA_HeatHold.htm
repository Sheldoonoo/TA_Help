Setting Up a TGA Heat and Hold Procedure

 

Heat and Hold experiments are used to assess the thermal stability of materials over time at elevated temperatures. They are also used to assess the rate of volatiles (water, solvent) evolution/drying at a specific temperature, as well as the adsorption/desorption of gases in applications such as catalyst reactivity, porosity of ceramics and oxidative stability.

If you want to set up a TGA Heat and Hold procedure, you can choose one of two techniques—the TGA Wizard or the Experimental View - Procedure Page. See the instructions below for each technique.

When you perform a TGA Heat and Hold experiment, these test parameters need to be specified:

  • Heating Rate: The acceptable range for the Heat and Hold temperature is 25 to 1000°C.

  • IsothermalTemperature: The temperature of choice is selected by running a preliminary survey (TGA Ramp) experiment on the material.

  • Isothermal Time: The isothermal time should be set to a value which is greater than that required to measure the event of interest. If the time required is unknown, a large value should be entered. No analysis time will be wasted, provided proper "End Experiments" conditions are set.

  • End Experiment When: There are two alternatives for "End Experiment" conditions that can be chosen, if you check the box:

    One is based on stopping the experiment when the total weight change exceeds a specific value. This alternative is useful for situations where the amount of total weight change is known (usually from previous TGA ramp experiments) and the Heat and Hold experiment is being run to provide the rate of weight change. For example, when studying drying or volatilization processes. Enter the value for the maximum total weight change percentage.

    The other alternative stops the experiment when a specific rate of weight change is reached. For example, when measuring oxidative stability time to the onset of thermal degradation. In these latter experiments, the final amount of total weight change is not important. Enter the value for the rate of weight change in %/min.

  • Additional Conditions: There are additional experimental conditions that can be changed to optimize specific situations. These are accessed using Advanced Parameters and Post-Test Conditions. For most experiments, the system default values shown are recommended.

Using the TGA Wizard :

Enter the desired parameters, then select Next to proceed or select Back to return to the previous window.

Using the Experimental View Procedure Page:

  1. Display the Procedure Page. Select Heat and Hold from the list of preprogrammed templates.

  2. Enter the desired parameters. When you have finished setting up this procedure, you can enter more information by selecting the Notes Page and Summary Page for this run.

  3. Click the Apply button when finished to save these changes or the Append Run button to save the changes to this run and add another run to the end of the sequence.

NOTE: You can use any test template as the basis for a custom test by selecting the desired test template from the list of tests, then click the Apply button and change the test to Custom. Then you can click on the Procedure Page and then the Editor button to make the changes desired.
