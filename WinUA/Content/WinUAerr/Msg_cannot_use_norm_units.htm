Cannot Use Norm Units

Cannot use normalized units with zero size.

 

Problem:

Normalized signal units have been selected for plotting but the sample size is zero.

 

Solution:

If you are trying to read a data file in which sample size is not important (e.g., an empty pan was run in a method), then select the signal units that do not contain sample size. If the sample size was not entered when the data file was created, then determine the correct sample size and enter it on the Data File Information window when the file is opened.
