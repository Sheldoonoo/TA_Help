Curve Value At Y

Use the Curve Value At Y analysis option to determine the axis value(s) at the selected Y-axis point on the curve. If more than one point is encountered by the marker, the first point is used. Marking the curve with this feature allows you to label a curve at any point.

 

Follow these steps to perform this analysis:

 

1. Select Curve Value at Y from the Analyze menu. A horizontal marker is displayed on the graph.

2. Position the marker at the point on the curve you want to label.

3. Right click to display the Analyze pop-up menu, then select Accept Limits (or press Enter). The Label Point Window is displayed.

4. Check the box for the item(s) that you wish to use for the analysis. Click the OK button. The point you selected is labeled with the selected items on the curve.
