Defining Columns and Variables

The table of data allows you to display various pre-defined variables. Often, however, you may wish to display and work with your own defined variables.

Once you have created a new column, the variable specified in that column is also available for use on the graph.

Click on the topic below for more information:

  • Exporting Importing Columns

Creating a New Variable

To create a new column, first load the required data file and switch to table view:

  1. Select User Columns on the menu.
  2. Click on New column.
  3. The new column editor is shown.
  4. Enter a Name for this new column variable.

NOTE: The name can only consist of the letters A to Z and cannot contain spaces.

  1. Create the required variable definition in the Data Function box. Add existing variables to the function by selecting them in the right hand box and either double click or use the << button.

NOTE: You can type the variables directly, but it is generally better to select from the list to ensure the correct spelling.

  1. The function can contain any numerical operators ( + - * / ) as well as any of the mathematical functions available to Microsoft® Visual Basic® Scripting Edition (VBScript). In addition, you can write simple VBScripts that generate a numerical output
  2. You can specify the number of Decimal Places used and the label that will be displayed for the Units.
  3. The Update if source column(s) change item is selected by default. This ensures that the new variable will be updated if any of the variables used to generate change. Turn this off if you want to lock the new variable.
  4. Click on the OK button.

Once you have created a new column, the variable specified in that column is also available for use on the graph.

Exporting/Importing Columns

Once you have created a user defined column/variable, you can save it to a file by means of the Export function. This file can then be imported into other copies of the Rheology Advantage data analysis program.

To Export (save) a user defined column:

  1. Select User Columns on the menu.
  2. Click on Export Column function.
  3. Click on the column function name you wish to export.
  4. Click on the Export button.
  5. Select a name and folder location for the file.
  6. Click on the Save button.

Back to top

To Import (open) a user defined column:

  1. Select User Columns on the menu.
  2. Click on Import Column function.
  3. Locate the file you wish to open.
  4. Click on the Open button.
