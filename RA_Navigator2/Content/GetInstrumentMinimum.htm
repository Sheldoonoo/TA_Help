GetInstrumentMinimum

Returns the minimum permitted limit for the specified variable

Syntax

v0 = GetInstrumentMinimum(<variable>)

 

variable
torque = 0
shear stress = 1
velocity = 2
shear rate = 3
temperature = 4
frequency = 5
ang. frequency = 6

displacement = 7
% strain = 8
strain = 9
