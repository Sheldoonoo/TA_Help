Using the Frequency/Amplitude/Stress/Force Tables

 

Frequency tables are needed for Multifrequency Controlled Strain and Multifrequency Controlled Stress experiments, amplitude or stress tables are needed for Multistrain mode experiments, and stress or force tables are needed for Multistress mode experiments. These contain a listing of one or more entries (up to 50) over the allowable range.

NOTE:  When you create a table with more than one frequency, you can begin with the highest frequency and end with the lowest to shorten the sweep time. When you create an amplitude table with more than one amplitude, you can begin with the lowest amplitude and sweep to the highest value.  When you create a stress table with more than one entry, you can begin with the lowest value and end with the highest value.

The allowable ranges are:

  • Frequency  =  0.01 to 200 Hz

  • Amplitude  =  0.5 to 10,000.0 µm

  • Force  =  0 to 18 N

  • Stress =  0 to 1E6 MPa

To create a table, follow the guidelines below for the desired type of table entries:

  • Tables with Single Entries

  • Tables with Multiple Entries

    • Tables with Log Entries

    • Tables with Linear Entries

  • Tables with Discrete Entries

  • Saving a Table

Tables with Single Entries

  1. Click the Single radio button.

Single entry is used for screening and comparing materials at a reference value or at an end-use value. 1 or 10 Hz is preferred for single frequency experiments because more complete oscillations are obtained in a shorter time.

  1. Enter the desired value, then click Refresh Table to update the displayed table.

For Multifrequency mode temperature ramp tests, one frequency is normally used. You can use more than one frequency, but they should be higher frequencies (above 1 Hz). To measure the time required to sweep through all of the frequencies in a table, observe the sweeping process using the Signal Control Pane, when the drive motor is turned on. Set the heating rate on the Procedure Page so that the sweep time occurs in about 2 to 3 degrees.

Tables with Multiple Entries

  • Multifrequency studies:  (useful for evaluating changes in material response with frequency) should be set up to cover multiple decades of frequency (usually 3-4 decades) and should begin with the highest frequency and end with the lowest.

  • Multiamplitude studies:  Up to 50 amplitudes over the range 0.5 to 10,000 µm are possible.  In practice, however, the range of acceptable amplitudes is determined by the stiffness of the sample and the maximum drive force of the DMA (18N).  When sweeping amplitudes, sweep from the lowest to the highest value.

  • Multistress studies:  Up to 50 stress or force amplitudes are possible. In most cases it is desirable to sweep from lowest to highest stress or force to minimize recovery effects as the stress or force level is decreased.

Tables with Log Entries

Click the Log radio button to space the entires “logarithmically” (e.g., 10, 5, 2, ). Enter the initial and the final entries, then enter the number of Points Per Decade. The entries will be calculated automatically. Click the Refresh Table button to update the table values displayed.

Tables with Linear Entries

Select the Linear radio button to space the entries linearly (e.g., 11, 9, 7, 5, 3, 1). Enter the initial and the final entires, then enter the Number of Points desired between the range entered. The entries will be calculated automatically. Click the Refresh Table button to update the table values displayed.

Tables with Discrete Frequency/Amplitude/Stress Entries

If you want to edit the table directly, select the Discrete radio button. The buttons become available for use in editing the table. Place the cursor in the table at the desired point and make your editing changes. The entries will be used in order until the first 0.00 entry is encountered. An entry of 0.00 is recognized as the end of the table.

  • Use the Insert button to insert a new value at the selected location, shifting all of the other table entries down one position.

  • Use the Delete button to remove the selected value, shifting all of the table values below the deleted entry up one position.

  • Use the Reset button to clear all of the table values, leaving only one entry.

Saving a Table

To save your table for future recall, save the entire procedure to a file using the Save Button located at the top of the Procedure Page.
