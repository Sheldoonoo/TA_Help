Bitmap Pixel Format List

Select the format to be used for the exported bitmap file. Choose from the following:

 

Device pixels = Exports the file using the default number of pixels for the selected device. If the device has no default number, then the current screen color resolution setting is used (see the following bit-to-color conversions).

1-bit/pixel = black and white file only (smallest bitmap file)

4-bits/pixel = 16 colors

8-bits/pixel = 256 colors

15-bits/pixel = 32767 colors

16 bits/pixel = 65535 colors

24-bits/pixel = 65535 colors plus custom colors (3-byte storage)

32-bits/pixel = 65536 colors plus custom colors (4-byte storage)

 

Tip:

For the best results when exporting Universal Analysis plots, select 4-bits/pixel. This results in the smallest bitmap file without the loss of color.

 

NOTE: The smaller the number of bits selected, the smaller the bitmap file size.
