Checkbox No Parameter Block

Check this box to export your data file without any parameter block information. Note that if you check this option, the exported file will not be able to be opened in Universal Analysis as a data file, because there will no longer be any parameter block information available.

 

NOTE: When this option is checked, the name of each signal is inserted at the top of the file in a standard spreadsheet format so that the names can be used to identify each column of data.

 

Leave this box unchecked (default condition) to export your data file with the parameter block intact.
