Paste (Macro Edit Menu)

Use the Paste menu item to insert the macro step, which was previously copied to the clipboard (using either Cut or Copy), into the macro at the selected place.

 

Shortcut Keys

Ctrl+V

 

  1. Ensure that the macro you want to edit is active by opening the desired macro.
  2. Select Macro Editor from the main Macros menu to display the Macro Editor window.

  3. Cut or copy the desired macro step(s).

  4. Click on the place where you want to insert the macro step.
  5. Select Edit/Paste from the menu (or press Ctrl+V). The step will be copied from the clipboard at the numbered position selected.
