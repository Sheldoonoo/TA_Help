Message 80

Data input buffer overrun. Data point lost. Software error.

Problem:

A problem exists in the software. It is not user caused.  

Solution:

Try one or more of the following operations to solve the problem, then restart your experiment.

  1. Close the program, then restart it again.
  2. Reset the instrument that was in operation when the error occurred.
  3. Reboot the system.
  4. Increase the amount of available memory and retry the operation.
  5. Call TA Instruments for service, if the error persists.
