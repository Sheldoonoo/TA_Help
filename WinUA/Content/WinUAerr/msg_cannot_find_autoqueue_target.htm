Cannot Find Autoqueue Target

Cannot find autoqueue data file.

Cannot find autoqueue macro file.

Cannot find autoqueue macro.

 

Problem:

Each record in the autoqueue contains a specific data file, macro file, and a macro. When the autoqueue is processed (autoanalyzed), the program must be able to find each one of these components. If this error occurs, the program cannot locate one of the components of a record contained in the autoqueue.

 

Solution:

Select Macros/Autoqueue to open the autoqueue file. Check the records displayed on the Autoqueue window. You may be able to recognize the data file, macro file, or macro that is causing the problem. Once you have determined which record contains an erroneous component, select Edit/Modify from the Autoqueue menu to display the Autoqueue Parameters window. Use the Browse button to locate a suitable data file or macro file, or use the drop-down list to locate a desirable macro to use. Then try the autoanalysis again by selecting File/Run from the Autoqueue menu.
