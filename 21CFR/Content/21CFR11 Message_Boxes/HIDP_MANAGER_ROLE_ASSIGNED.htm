Cannot Assign Another Group

The selected user is already a member of the System Manager Group. Members of the System Manager Group cannot be assigned to any other groups.

In order to protect the integrity of the system, a user who is a member of the System Manager Group can ONLY perform those functions assigned to a System Manager. Therefore, any projects, modules, or other groups cannot be assigned to a member of the System Manager Group.

If you wish to assign this user to projects, modules, or other groups, you must remove the System Manager Group from this person's account before you try again.
