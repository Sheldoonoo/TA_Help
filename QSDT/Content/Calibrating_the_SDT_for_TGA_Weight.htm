Calibrating the SDT for TGA Weight

 

Weight calibration should be performed on the SDT at least once a month. TGA weight calibration is based on two runs conducted in the calibration mode—one using calibration weights and one using no weights (empty beams). The data from both runs is analyzed and beam growth and weight corrections factors are calibrated. (This experiment usually utilizes the same baseline data as that obtained for DTA baseline calibration for the first experiment, if applicable.)

NOTE:  The calibration data must be obtained with the instrument in the calibration mode.

Use the following instructions to gather and analyze the two data files needed for TGA weight calibration. This process uses the "TGA weight" test template.

 

Gathering TGA Weight Calibration Data

Setting Up the First Calibration Run (Empty Beams):

  1. Check that your purge gas is connected.

  2. Open the furnace. Remove all SDT cups, sample, and weights from the sample and reference beams.

  3. Close the furnace.

  4. Select Experiment View and access the Summary Page, then set the instrument to "calibration" mode using the Mode drop-down list.

  5. Select the "TGA Weight" test from the Test list.

  6. Enter the requested sample information, entering zero for the Sample Weight and Reference Weight.

  7. Specify a data file name for the first run (without weights) using the Browse button.

  8. Click on the Procedure tab.

  9. Check Use Current as the starting temperature or enter the desired Start Temperature (e.g., 50.00°C).

  10. Enter your upper temperature (e.g., 1500°C) as the Final Temperature.  

  11. Enter the Heating Rate that will be used in subsequent experiments (e.g., 20°C/min).

  12. Select the Advance button to access and select the Advanced Parameters.

  13. Click the Post Test button to access the Post-Test condition parameters that allow you to specify the furnace and air cool options to use once the experiment is complete. (Select "Furnace Closed" and "Air Cool to 50°C.")

  14. Click on the Notes Page.  Enter the requested information, including the desired flow rates (typically 100 mL/min).

  15. Click on Apply to store the changes made to these pages.

  16. Schedule this run, if more than one run is in the sequence list.

  17. Select Start to begin the calibration run.  Wait for the experiment to finish and allow the SDT to cool (<50°C).

Setting Up the Second Calibration Run (with weights).

  1. From the sequence list, double click on the complete icon (check mark) next to reschedule the run.  This will store the existing information from the previous run and automatically increment the filename. Alternatively, you could just change the filename or append a new run.

  2. Press the FURNACE key on the SDT touch screen to open the furnace and place the calibration weights on the beams as follows:  the weight marked with an "S" should be placed on the sample (front) platform and the unmarked weight should be placed on the reference (back) platform.  Close the furnace. NOTE:  Do not the touch the calibration weights with your fingers; use tweezers.

  3. Select Experiment View and access the Summary Page. Set the instrument to calibration mode using the Mode drop-down list.

  4. Select the "TGA Weight" test from the Test list.

  5. Enter the requested sample information, entering the weights of the sample and reference calibration weights found on the bottom on the box. The filename will automatically increment.  

  6. If desired, change the data file name for the second run (with weights) using the Browse button.

  7. Click on the Procedure tab. The procedure information should be the same as the first run without weights. (e.g., steps 9 through 13)

  8. Click on the Notes Page.  Enter the requested information, including flow rate. (The flow rate found in step 14 should be used here also.)

  9. Click on Apply to store the changes made to these pages.

  10. Schedule this run, if more than one run is in the sequence list.

  11. Select Start to begin the calibration run.  Wait for the experiment to finish and allow the SDT to cool (<50°C).

  12. Analyze the calibration data as directed in the next section.

Analyzing the TGA Weight Calibration Data

  1. Select Calibrate/TGA Weight from the menu. The Weight Calibration window is displayed.

  2. Select the file for the first experiment (empty beams) by clicking on the Browse button.  Select the file for the second experiment (with weights) by clicking on the Browse button.  

  3. Select OK when both the calibration files have been entered. The Calibration program will open and display a series of four curves.  

  4. Select Analyze from the Analysis menu.

  5. Position the two analysis markers in the desired location (bracketing the analysis region of interest) then select Limits OK from the pop-up or Analyze menu.  

  6. When selecting your TGA limits, the default limits of 200 and 1400°C are the optimum limits to use when calibrating the SDT up to 1500°C.  If you are only interested in a particular portion of the curve, such as running methods up to 1000°C, you may decide to choose more appropriate limits.

  7. Save the calibration results using Analyze/Accept results.  (TGA weight calibration values are not displayed, only the resultant curve.) The weight signals should track each other in a smooth continuous upwards sweep.  Likewise, the corrected baseline signal should be flat across the screen using the default scaling.  If any of these curves are not as described, rerun your baseline calibration.  Pay careful attention to make sure your instrument is not physically jarred, that the calibration weights are seated properly on the platform, and that the purge flow is steady.  If the curves still do not look proper after rerunning the calibration, contact your service representative.

  8. Close the Calibration window.

The Calibration Report can be used to view the date this calibration was last conducted.
