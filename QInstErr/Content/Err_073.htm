Message 73

No heater power at method start. Cannot start run.

Problem:

The instrument cannot start the run because no heater power is available.

Solution:

Try one of the following procedures to solve the problem, depending on the instrument currently displaying the message. Then restart the experiment.

  1. Check to see if the heater windings are faulty by switching the current cell (DSC) or furnace (TGA, TMA) with a different one, and trying the operation again.
  2. Call TA Instruments for service, if the problem persists.
