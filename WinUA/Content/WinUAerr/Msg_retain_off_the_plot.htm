Retain Off the Plot

Retain <result type>? Result is off the plot.

 

This message occurs when you perform one or more analyses, then you rescale the curve so that a result is no longer displayed, and chooseEdit/Delete Results/Select. When the program comes to the result that is no longer displayed on the window, it will display the message above. Select Yes to keep the result named, No to delete the result, or Cancel to abort this operation.
