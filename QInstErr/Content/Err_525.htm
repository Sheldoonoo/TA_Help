Message 525

The Autosampler failed to load a reference pan. Pan number xx.

Problem:

The Autosampler has been commanded to load a pan to the cell reference position and did not perform that task.  This could occur if the reference position has no pan; i.e., it was manually removed from the tray, or if the Autosampler arm is not working correctly.

Solution:

  1. Stop the Autosampler sequence, if necessary and reset the instrument.
  2. Then place a pan containing the reference material in the tray. Once the Autosampler has reset after reboot, try the operation again.
  3. Call TA Instruments for service, if the error persists.
