Smoothing Data

 

The software allows you to apply a mathematical filtering of noise from your data. It can be used, for example, when editing of discrete points would be impractical.

Original Data

After Repeated Smoothing

The smoothing function is accessed from the Analysis menu.

NOTE:  If you wish to remove one or more bad data points, simply click on them on the graph view and then press Ctrl D (or select Del from the Edit menu.)
