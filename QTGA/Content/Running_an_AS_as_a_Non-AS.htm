Running an Autosampler as a Non-Autosampler

 

To use the Autosampler as a single sample instrument, you can disable the Autosampler one of two ways:

(1) Using the Control Menu

Select Control/Autosampler from the main menu, the uncheck the Enable Auto Sampler menu item. This converts the Autosampler to a single sample instrument. If checked (as shown in the figure to the right) the Autosampler is enabled.

 

(2) Changing Instrument Preferences

Select Tools/Instument Preferences, then click on the Autosampler Page. At the bottom of the window is a checkbox entitled, "Enable autosampler."

Uncheck this box, if you want the Autosampler to work as a single sample TGA. The instrument will not automatically advance to the next run. If this option is checked (default condition), the Autosampler will operate as usual.

Select Apply or OK to save the settings.

NOTE: This option is unavailable (grayed) when an Autosampler sequence is active and cannot be changed until the sequence is completed or stopped. It is also unavailable when the Autosampler is in “manual” mode rather than “automatic” mode.
