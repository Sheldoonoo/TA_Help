TTS Signals

Use this option to export selected DMA or DEA Time-Temperature Superposition (TTS) signals from the data file. When you select TTS Signals, the Export Data File window is displayed with the signals obtained from the data file displayed and checked, and the other proper export options selected. You can not change this information. Press Finish to specify the exported filename and location.

 

To select different export options, access the File/Data File Export/File and Plot Signals option.
