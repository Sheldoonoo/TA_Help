Storage Modulus (G')

Description

Sometimes called the elastic modulus. The real part of the complex modulus, which may be represented as a complex function under sinusoidal conditions.

Symbol 

G'

Units

 

Description

Abbreviation

SI

newtons per square meter (pascals)

Pa

CGS

dynes per square centimeter

dyne / cm2

Related Variables

G* = G' + iG"   (i is the square root of -1)

G' = |G*| cos(d)
