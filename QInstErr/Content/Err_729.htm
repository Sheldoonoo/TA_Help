Message 729

T1 heat flow selected. Cannot start modulated run.

Problem:

The DSC cannot run Modulated DSC (MDSC) experiments when the selected heat flow is T1.

Solution:

Perform one of the following actions:

  • Run a non-MDSC experiment.
  • Select Tools/Instrument Preferences/DSC Only from the menu, then choose a signal other than "Heat Flow T1" from the drop-down list of available Heat Flow signals. You must then recalibrate the instrument for cell constant and cell temperature before starting the run.
