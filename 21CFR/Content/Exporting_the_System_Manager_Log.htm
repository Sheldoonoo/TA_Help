Exporting the System Manager Log

 

The System Manager Log can be exported as a text file that can be read and printed by other applications (e.g. Microsoft Excel®) by following the instructions below.  

  1. Log into the TA System Manager. See also: Accessing the System.
  2. Click the System Settings icon in the left-hand pane. Then select the System Log tab.
  3. Click the arrow to the right of the drop-down date menu located at the bottom of the window to select the starting date from the calendar. See the figure below.

Back to top

  1. Click the Export button to save the System Log to a file. The window shown below is displayed.

  1. Enter or browse for the file name and location that will be used for the exported file.
  2. Set the File type to Unicode from the drop-down list.
  3. Click the OK button. The System Manager Log is exported and saved in the selected location.
