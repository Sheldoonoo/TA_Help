Edit Parameter Block Information

This function makes an exact copy of the current data file including a new user-modified parameter block. It is best to edit only those parameters that you are familiar with. Incorrect changes may affect the integrity of the data. We recommend that you do not overwrite the original data file.

 

If you want to save the edited parameter block with the current data, give the file a new name so that you do not overwrite your original data file. This may help prevent errors that can occur with incorrect parameter block changesyou can still retrieve your original data file.
