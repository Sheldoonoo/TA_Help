Login/Logout

 

The Advantage Integrity Instrument Control and Universal Analysis programs are part of the closed system for creation and storage of electronic records. As such, each person must log in for identification and permission to use the system.

Logging in

When you open each Integrity-licensed program you will need to log in on the window displayed. (If the Login window was not displayed, then the Database Setup was not performed to allow database access. Consult the Advantage Integrity Installation instructions.)

 

Enter your "User ID" and "Password" to gain access to the system modules.

 

  • If you select Cancel on this window you will not gain access to the TA database records, but only to data files created outside the Advantage Integrity system.

  • If Universal Analysis was already opened, but you were not logged in yet select File/Login to log into the system. Any open data files must be closed. ("File-mode" and "database-mode" are mutually exclusive.)

Logging out

When you are finished accessing these programs, you should log out of the system.

 

  • If you are in Instrument Control, simply close the instrument window to log out. This will not affect data collection.

  • If you are in Universal Analysis, you can close the software (causing you to close all open records) or select File/Logout to temporarily log out of the system, maintaining the state of your work.

  • If you forget to log out, after a specific period of inactivity (interaction within these modules) as specified by your TA System Manager, the log-in window will automatically be posted and requiring login to access the module again.

 

NOTE: If you want to leave the Advantage Integrity Universal Analysis session unattended, but protected immediately, selectFile/Logout, thenOKto the confirmation message displayed. The Login window is displayed. Upon returning, log in again.
