Cursor (Data Limits)

Use the Cursor option to display markers that allow you to manually select the data points that will be used for analysis.

 

1. Position the first marker at the beginning of the region you wish to select and press TAB to activate the next marker (or just click on the next marker).

2. Position the next marker at the end of the region you want to select.

3. Right click to display the pop-up menu and select Accept Limits (or press the Enter key). The selected data range only is read from the file and displayed.
