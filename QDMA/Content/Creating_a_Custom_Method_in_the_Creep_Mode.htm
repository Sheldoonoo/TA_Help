Creating a Custom Method in the Creep Mode

 

The steps needed to create a "custom" method of your own for the DMA are slightly different depending upon the mode selected.

Follow these steps to create a custom method for the Creep Mode:

  1. Access the Experimental View/Summary Page.

  2. Select the "Creep" Mode from the drop-down list available. See also:  Selecting a DMA Operating Mode.

  3. Select the signals that you want the software to store in the data file by clicking the button. This displays the Select Signals to Store dialog. Check the desired signals. (Compliance, Strain, Stress, Recoverable Strain, and Recoverable Compliance are the most essential signals to store for creep experiments.)

  4. Select the "Custom" type of Test from the drop-down list available.

  5. Select the name of the installed Clamp from the drop-down list. See Available DMA Clamps for more information.

  6. Select the type of Sample Shape that you will be using for your sample from the list shown.

  7. Enter the Dimensions appropriate to the sample shape selected above.

  8. Enter the Sample Name. Enter up to 32 alphanumeric characters to identify the material.

  9. Enter any Comments.

  10. Enter a Data File Name by clicking on the button, then select the desired storage location and enter in a filename. Click OK when finished.

  11. Check the Network Drive checkbox if you want to send your instrument data to another location on your system (e.g., network drive). Click the Browse button to the right, then select the desired archive location. Click Select when finished. The data will be transferred at the end of the experiment.

  12. If you want to automatically analyze the resulting data file using a macro (created with the Universal Analysis Autoanalysis software option), check the Autoanalyze box. Then locate the desired macro filename using the Browse button. Enter the drive, directory, file location, and name for file.

    1. If you want to automatically have the results of the autoanalysis (checked above) sent to a remote computer, which is one  other than the controller computer attached to the instrument (also called the local computer), check the Start Remotely box. Then enter the same path and data file name chosen in step 9 above. When the data file has been collected, Universal Analysis will automatically open and begin the autoanalysis of the file on both the local and remote computers.

    2. Click on the Analysis Macro list and select a macro from those displayed. This macro will be used to automatically analyze the data file that will result from this experiment.

     

NOTE: See the Autoanalysis documentation accompanying the Universal Analysis software for further information on creating macros.

  1. Click on the Procedure Page tab.

  2. Type any Notes that you want to write about the test you will be running.

  3. Enter the desired Preload Force.

    Preload force: This is the force applied to the sample prior to initiating the experiment.  

    • A preliminary preload force is available for creep measurements using the tension (film) and tension (fiber) clamps.  This preload force ensures that the sample material is fully elongated (has no slack).  A preliminary preload force is also available when using the compression and 3-point bend clamps for creep measurements.  In those clamps the preload force ensures that the movable part of the clamp is in contact with the material prior to starting the experiment.  In all cases, the preload force should be limited to the minimum force required to achieve the desired starting condition.  Otherwise, it may affect the creep results.  (NOTE:  Preload force will also affect the creep recovery results.  Since creep recovery assumes “zero force,” the impact of improper preload force is greater during that portion of the experiment.)  0.001N is the lowest force which can be applied.  

    • Preload force is not used when you select a nontensioning clamp (i.e., single/dual cantilever, shear sandwich).  

     

  4. Enter the desired Stress.

    Stress is the force applied to the sample that is normalized for the contact area during the displacement step of the creep experiment. When using tension/compression clamps, stress is the force divided by the sample cross sectional area. See also:  Calculations Based on Clamp Type.

  5. If you want to edit the current method (the name and segment is displayed), click the Editor button to display the Method Editor window. This allows you to change the Displacement time (for application of creep stress) and Recovery time (for monitoring viscoelastic recovery of stain).

  6. To change the parameters that the instrument will use to perform the experimental run, click the Advanced button  to display the Advanced Parameters window applicable to the current instrument.

  7. To change the conditions that will apply at the end of the run (method-end conditions), click the Post Test button to display the current Post Test window. See also:  Guidelines for Running Subambient Experiments.

  8. When you have finished setting up this procedure you may add notes or verify the information on the Notes Page.

  9. Click the Apply button when finished to save these changes or click the Append Run button to add another run to the end of the sequence, if needed.

NOTE:  After a run is appended, the DMA clamp type cannot be changed.

 

To see a summary of the current procedure for this run, open the Procedure Summary window using the  button located on the Summary Page. To view a summary of a saved procedure file, open the File Utility window using Tools/File Utility.

NOTE:  You can use any test template as the basis for a custom template by selecting the desired test template from the list of tests, then click the Apply button and change the test to Custom. Then you can click on the Procedure Page and then the Editor button to make the changes desired.
