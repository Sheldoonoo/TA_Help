Auto Oxidation Limits Window

Use this window to determine the way analysis limits are chosen for repeat analyses on materials that you expect to exhibit similar thermal plots. This option is particularly useful when used in combination with the Autoanalysis option. The Auto Oxidation Limits window is available and can be displayed when you perform an oxidation time or temperature analysis.

 

You can set up and use the Auto Oxidation Limits function as follows:

 

1. Open a good representative data file for the material(s) being analyzed.

2. Plot the desired signals.

3. Select Analyze/Oxidation Time or Analyze/Oxidation Temp from the menu, the Oxidation Time or Oxidation Temp window is displayed. Select the desired value and click OK, cursors (markers) will be displayed.

4. Position the cursors so they encompass the desired data and allow enough room on either side of the analysis range for a small amount of shifting that can occur on different data files.

5. Right click to display the pop-up analysis menu, and choose Auto Limits from the menu.

6. If you want the program to store the position of the cursors that you have chosen, select the Auto Set button. The program will automatically select the Inflection offset and fill in the offset value from the position of the cursors relative to the current transition.

7. If you do not want to use the inflection point offset for both the start and stop limits, you can choose the Use cursor position option for one or both limits. This will let you set a specific start or stop limit using the cursors (markers). (i.e., The cursor will be positioned in the exact location of the marker.)

8. Select one of the buttons that will determine how this information is used:

  • If you select the OK button, these options are used for this session only.

  • If you select the Save button, these options are saved as in the initialization (.ini) file and will be recalled each time the program is started.

  • If you select the Cancel button, the operation will be aborted.

 

NOTE: You do not have to set up both the Start and Stop limits so that they use the same type of option. For example, you can set it up so that the start limit can be selected each time a new file is analyzed, but the program will use the offset that was preselected as the stop limit. This would be accomplished by choosing Use cursor position for the Start limit and Inflection offset for theStop limit.

 

After you set up the Auto Limits options, you can open similar data files and perform the analysis quickly by selecting Auto Limits from the pop-up analysis menu and simply clicking the OK button to use the stored limits.
