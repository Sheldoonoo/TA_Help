Run (Autoqueue)

Use the Run menu item to process (autoanalyze) the records contained in the autoqueue, beginning with the first record.

 

NOTE: To stop autoanalysis of the autoqueue, press the Escape key.

 

NOTE: For more details on autoanalysis and the autoqueue, see the Universal Analysis online manual (Help/Manual) or use Acrobat Reader to access the Uamanual.pdf file contained on the CDROM.
