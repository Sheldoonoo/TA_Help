Analyze Humidity GAB Analysis

Select this item to fit the sorption isotherm data from the Q5000SA to the GAB model.

 

To use this function:

  1. Collect your data following the guidelines suggested. See Humidity Analysis.
  2. Select File/Open, then locate and open the desired humidity data file.
  3. Select Analyze/Humidity/Options and set up the desired analysis options.
  4. Plot Weight Change versus Time for the desired Q5000SA data.
  5. If sample drying was conducted during this experiment, then the dry weight needs to be determined before generating this plot. Use the Analyze/Reset Size function.
  6. Select Graph/Data Limits/Time, and select the Start time as the time after the drying step, and end time as the experiment end time.
  7. Manually mark the adsorption/desorption isotherms, if desired. Select Analyze/Humidity/Mark Adsorption/Desorption then select the desired position on the curve.
  8. Select Analyze/Humidity/GAB Analysis. A new window will be displayed with the resultant data, based on the chosen humidity options (see Humidity Analysis Options - GAB Analysis. The sorption isothermal data from the selected file is processed using the GAB equation and displayed.

This analysis results in a plot of the adsorption and/or desorption data values. No additional analysis of the data is performed.

 

A data table can be obtained for this plot by selecting View/Data Table.

 

NOTE: If sample drying was conducted during this experiment, then the dry weight needs to be determined before generating this plot. Use theAnalyze/Reset Sizefunction.

 

Background

In 1946 Anderson[1] proposed a modification to the BET model (see Humidity BET Analysis), which was later adapted by Guggenheim[2] and de Boer[3], resulting in a model that came to be called the GAB (Guggenheim, Anderson and de Boer) equation.

 

Anderson1 mentions in his paper that, "while the simple B.E.T. equation provides an excellent method for estimating surface areas, it usually holds only for relative pressure of 0.05 to 0.40. In almost every case the amount adsorbed at relative pressure higher than 0.40 is less than predicted by the simple B.E.T. equation." He proposed that the BET equation could be used to fit the isotherms data at the relative pressures greater than 0.40, if the relative pressure is multiplied by a constant less than unity.
