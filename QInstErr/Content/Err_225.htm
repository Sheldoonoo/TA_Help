Message 225

Reference weight out of range. Cannot start run or tare.

Problem:

This message was caused by one or more of the following:

  • There is no SDT cup loaded on the reference beam.
  • The reference material weight exceeds the SDT's weighing capacity.
  • The reference balance beam is not operating correctly.

Solution:

Try one or more of the following operations to solve the problem, then restart your experiment.

  1. Load a SDT cup onto the reference beam.
  2. Use a lighter reference material or mechanically tare your reference using weights on the reference balance beam.
  3. Call TA Instruments for service.
