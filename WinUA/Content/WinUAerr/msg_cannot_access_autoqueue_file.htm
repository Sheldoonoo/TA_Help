Cannot Access Autoqueue

Cannot access system autoqueue file.

 

Problem:

The program looks for the autoqueue file in a certain directory. If the program cannot find the file, or if the file has been renamed, then the program cannot access the autoqueue.

 

Solution:

Try one of the following:

  1. Place the autoqueue file back into the TA\Data folder.
  2. Rename the autoqueue file back to AutoQ.txt.
  3. Create a new autoqueue file from Instrument Control. See the Instrument Control online documentation or the User Reference Guide for instructions.
