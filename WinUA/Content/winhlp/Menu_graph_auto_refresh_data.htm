Auto Refresh Data

Select this function to automatically refresh the plotted data from an active data file (real time plot), without losing completed results. The minimum amount of time between refreshes is set on the Main Options - Data Page. The default is five seconds.

 

  • Auto refresh is automatically suspended when activity is detected (e.g., mouse movements, keyboard input, active dialogs, etc.).

  • Auto refresh will continue in the background if Universal Analysis is not the active application.

  • Auto refresh can be toggled on and off using this menu item.

  • Auto refresh can be turned on as a default condition by checking the Auto refresh checkbox on the Main Options  Data Page.
