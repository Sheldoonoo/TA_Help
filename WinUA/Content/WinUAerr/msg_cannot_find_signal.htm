Cannot Find Signal

 

Cannot find signal.

 

Problem:

A required signal is missing from the primary data file or from a temporary work file. The selected analysis cannot be performed.

 

Solution:

Try the following:

 

  • If the signal is available from the instrument, but not selected, select it for output to the data file.

  • The message can also result from an error in the data analysis software. Contact TA Instruments for help.
