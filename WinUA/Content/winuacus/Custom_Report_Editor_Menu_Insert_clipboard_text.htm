Clipboard Text Object

Use the Insert/Clipboard Text function to create a report object that will extract text from the clipboard as the report is being generated, and then print that text on your custom report. The text to be reported must be stored on the clipboard prior to launching the custom report. The text can be generated by any mechanism, including a Copy operation or a macro that builds custom report text on the clipboard.

 

When you select this item from the Custom Report Editor Insert menu, the Clipboard Text window is displayed to allow you to specify text justification, background color and font. When you click the OK button, the object block is inserted into the template. It can be moved, resized, and formatted as desired.
