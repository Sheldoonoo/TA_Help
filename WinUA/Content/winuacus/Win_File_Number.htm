File Number Window

Use the File Number option to enter or change the file number that is used to generate the selected (highlighted) report object. The files are numbered in the order that they are opened.

 

Follow these steps:

  1. Highlight the desired report object on the template.
  2. Select File Number from the Edit menu or from the pop-up menu.
  3. Enter the desired file number.
  4. Click the OK button when finished.

When you export a file that contains multiple file numbers to Microsoft Word (select File/Export to Word or click the button on the tool bar), the Select File window is displayed. This window enables you to direct the program to the filename to be used for each file number in the template. You do not have to numerically select the files according to the order in which they were opened.
