Data Multiplex Window

Use the Data Multiplex (Frequency or Amplitude) window to separate the data according to a chosen frequency or amplitude. This option is very useful when performing a function on an individual mulitplexed value.

 

1. Click on the button to display the drop-down list of values available.

2. Click on the desired value, then click the OK button. The data in that value range will be available for analysis. The data outside the analysis range will be grayed out.
