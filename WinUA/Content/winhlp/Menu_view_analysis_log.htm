Analysis Log

View/Analysis Log allows you to view all of the steps used to manipulate and view the data for each open data file. The analysis log captures these steps and allows you to view and print them. You can also convert the analysis log to a program macro using the Macros/Copy/Analysis Log menu item.

 

NOTE: The analysis log will hold up to 800 steps. Any additional steps are lost and a message is displayed, "Maximum number of analysis log steps have been recorded."

 

The analysis log for saved PDF and BMP records can be viewed through the Database Viewer. Select File/Open within Universal Analysis or Tools/Database Viewer within the Instrument Control window. Highlight the desired record. A preview of that record is shown in the right-hand pane. Then select the view analysis log icon to display the associated analysis log for that saved record.
