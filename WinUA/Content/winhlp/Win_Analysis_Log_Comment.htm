Analysis Log Comment Window

This window is used to add a comment to the analysis log for the current file. The comment appears in the analysis log and is identified by a " / / " in front of the entered text. To view the analysis log, select View/Analysis Log from the menu.

 

Enter the desired comment. The maximum length of each comment line is 60 characters and up to 30 lines of text can be entered or you can click the Clipboard button to paste the information from the clipboard into the description box.

 

NOTE: You can clear the description box by clicking the Clear button.

 

About the Analysis Log

The analysis log captures every step used to manipulate and view the current data file and allows you to view and print the steps. You can also convert the analysis log to a program macro using the Macros/Copy/Analysis Log menu item.

 

NOTE: The analysis log will hold up to 800 steps. Any additional steps are lost and a message is displayed, "Maximum number of analysis steps already defined."
