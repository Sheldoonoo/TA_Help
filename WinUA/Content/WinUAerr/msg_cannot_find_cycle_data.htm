Cannot Find Cycle Data

 

Cannot find cycle data.

 

This message is displayed when you attempt to open a Mass Spectrometer file in Universal Analysis and the program cannot locate or read the cycle data. The cycle data that the program is attempting to read starts on the next line after the line with the header "Cycle Date Time RelTime[s]."

 

This particular Mass Spectrometer file cannot be opened with Universal Analysis. Use another file that contains the appropriate cycle data.
