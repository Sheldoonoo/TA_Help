DMA Calibration Error – Dynamic Calibration Unsuccessful

Bad dynamic calibration results obtained. The calibration was not saved. This error can be caused because the lowest compliance, or thinnest sample, used in the dynamic calibration procedure was too weak. Use a steel sample that is between 0.12 mm (0.005 inch) and 2.0 mm (0.08 inch) thick.

 

Run the dynamic calibration procedure again, but make sure that you do not use the weak sample to gather your data.
