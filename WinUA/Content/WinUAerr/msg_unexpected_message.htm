Unexpected Message

Unexpected message in macro.

 

Problem:

An unexpected response to a message dialog has been encountered while running a macro. The dialog that the answer applies to is not posted on the monitor. This usually results when you make inappropriate modifications to the macro or when the macro runs with the Universal Analysis program in a different state from the state in which the macro was captured.

 

Solution:

Be careful when modifying macros. Make sure that the steps can still run in an uninterrupted manner. Place the program into the state needed to run the macro successfully.
