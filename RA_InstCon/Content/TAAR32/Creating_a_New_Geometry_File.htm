Creating a New Geometry File

 

Follow these steps to create a new geometry file:

  1. Select Geometry on the menu.
  2. Click on New.
  3. Select the type of measuring geometry required and click the Next button.
  4. Enter a description under the Name entry. Record any additional information in the Notes section.
  5. Select the correct Material for the geometry. If the geometry is a solvent trap type, select the check box for this.
  6. Click on the Next button.
  7. Enter the dimensions of the geometry into the relevant sections. Press the Next button.
  8. Select the option to automatically calibrate the inertia. You can then leave the actual value as zero.
  9. Enter the back-off distance. Press the Next button.
  10. Enter a name for the geometry file and select the Store to file option.
  11. Press the Finish button. The new geometry is created. If prompted, accept the option to invalidate any reference gaps.
  12. Ensure it is attached to the rheometer and then perform the calibration procedures.
