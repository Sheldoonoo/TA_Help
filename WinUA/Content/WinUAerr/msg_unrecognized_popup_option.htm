Unrecognized Popup Option

 

Unrecognized popup option.

 

Problem:

When you capture a right-click pop-up menu option an edit field, the macro step is captured using the string <popup> followed by the name of the menu item (e.g., "Edit Name <popup>Copy"). If that step is edited to something that is unrecognizable by the program, then this error is posted.

 

Solution:

Do not edit the <popup> macro step to name an unrecognized function. Recognized function include: Undo, Cut, Copy, Paste, Delete, and Select All.
