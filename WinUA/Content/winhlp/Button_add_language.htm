Language Add Button

Press this button to add or update a localization language. The program will display a file open dialog. Follow these instructions:

 

  1. Locate the Universal Analysis language string file for the language you want to add. The default location is in a folder labeled "Language" in the folder where the program executable (WinUA.exe) file is located.

  2. Select the new language file and press OK. Universal Analysis will read the specified file and verify that it is correctly formatted. Any errors detected will be displayed. If the file is accepted, Universal Analysis will add the language name to the Other Language List and store the location of the language file in the Windows registry.
