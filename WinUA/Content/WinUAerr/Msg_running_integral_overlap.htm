Running Integral Overlap

Limits overlap existing result.

 

Problem:

You attempted to calculate a running integral of a region of the curve that was already used in a previous running integral calculation.

 

Solution:

Do one of the following:

 

1. Redefine your limits so that they do not include a portion of the curve that was already used in a previous integral calculation.

2. Delete the current running integral curve (use Delete Results in the Edit menu), then repeat the analysis.
