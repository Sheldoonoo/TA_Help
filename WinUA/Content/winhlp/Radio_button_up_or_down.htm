Radio Button Up or Down

Select the direction to plot your heat flow or temperature difference data for RMX-created data files only.

 

NOTE: Files created using theThermal Advantagesoftware are not affected by this selection. To change the exotherm direction for those files, select the desired direction before running your experiment using the Parameters/Mode function found in the Instrument Control program.

 

  • Select the Exotherm Up radio button to plot your heat flow or temperature difference data in the traditional TA Instruments format, with the exothermic portion of the curve in the upward direction.

 

Example: Exotherm Up

  • Select Exotherm Down to plot your heat flow or temperature difference data with the exothermic portion of the curve in the downward position, inverting the signal from the traditional method.

 

Example: Exotherm Down
