Temp/Time at Weight %

Use the Temp/Time at Weight % option to locate the X-axis value on a TGA curve (either time or temperature) that corresponds to a selected Y-axis value, and label the graph with the weight percent.

 

Follow these steps to perform this analysis:

 

1. Select Temp/Time at Weight % from the Analyze menu. A marker will be displayed on the graph.

2. Position the horizontal marker at the weight on the curve for which you want the corresponding X-axis value (either temperature or time).

3. Right click to display the Analyze pop-up menu, then select Accept Limits (or press Enter).

 

The program calculates the results and marks the graph with a cross at the select point and labels it with the corresponding x- and Y-axis values.

 

NOTE: This analysis is only available when the selected Y-axis units are in % weight.
