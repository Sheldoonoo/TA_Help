Oracle Error

There is an error in finding or creating the Oracle service name [NAME] in tnsnames.ora.

This message can be displayed when you are running the Database Setup Wizard. The wizard is trying to set up the configuration file tnsnames.ora in Oracle\Network\Admin directory, but failed to do so.

Please make sure the computer has Oracle Client Version 9 or above installed. Run the Database Connection Wizard again to recover the file. If it still fails, rename the file and run the Database Connection Wizard again. If the problem persists, contact TA Instruments for assistance.
