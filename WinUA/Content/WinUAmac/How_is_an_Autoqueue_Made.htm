How is an Autoqueue Made?

 

Creating an Autoqueue Automatically

Thermal Advantagehas been designed to allow you to take advantage of a technique that can be used to completely automate the collection and analysis of multiple data files. This Autoanalyze technique can be set up to automatically run as follows:

 

(1) Create an Analysis Macro:

The first thing you need to do is to open the Universal Analysis program to create the macro that will be used to analyze your data files. For this macro to work successfully, it must start with an "Open File" step (File/Open). When you create the macro you should use a file similar to the type to be autoanalyzed (e.g., DSC, MDSC, TGA, etc.). Later, when the experiment is complete, the autoanalysis record (containing the finished data filenames) will be appended to the autoqueue. (NOTE: No spaces can be used in the directory or file names.)

 

NOTE: For more details on creating and editing macros, see the Universal Analysis online manual (Help/Manual) or use Acrobat Reader to access the Uamanual.pdf file contained on the CDROM.

 

(2) Set Up Autoanalysis in Instrument Control:

Open the Instrument Control program for the desired instrument. Set up your experiment as you normally would. Access the Experimental Parameters window (for Autosamplers, use the Sequence Page). At the bottom of the window, check the Autoanalyze box, use the Browse button to select the macro file desired then select the macro you created in step (1) above. After you have finished setting up the experiment(s) and loaded the sample(s), start the experiment. As the controller finishes performing the experiment, it will create the record and add it to the autoqueue. See also: What is an Autoqueue?

 

When a record is added to the autoqueue from the controller, a message box is displayed asking if you wish to process the autoqueue (autoanalyze) at this time. If you select No, then the record will be added to the autoqueue, but not processed. If you select Yes, then the record will be added to the autoqueue and the queue will begin to process starting with the first entry. If no response to this message box is made, after 60 seconds it will be assumed that Yes is the response and the autoqueue will be processed beginning with the first record. The results will be collected in the Autoqueue Log.

 

 

Creating an Autoqueue Manually

You can also use existing data files to manually create and run an autoqueue. This is a good way to take a collection of similar data files and perform the same analysis on each one.

 

(1) Create an Analysis Macro:

The first thing you need to do is to open the Universal Analysis program to create the macro that will be used to analyze your data files. For this macro to work successfully, it must start with an "Open File" step (File/Open). When you create the macro you should use a file similar to the type to be autoanalyzed (e.g., DSC, MDSC, TGA, etc.). Later, when the experiment is complete, the autoanalysis record (containing the finished data filenames) will be appended to the autoqueue.

 

After the first "Open File" step, you will need to insert a "File Name" step using the Insert/File Name menu item on the Macro Editor window. Select the radio button "Use default file name" as the file option. This will allow the files that are placed into the records to be automatically opened. Then you can proceed to capture the appropriate steps to perform the analysis that you desire for your data files. Save the macro to the macro file when finished. (NOTE: No spaces can be used in the directory or file names.)

 

NOTE: For more details on creating and editing macros, see the Universal Analysis online manual (Help/Manual) or use Acrobat Reader to access the Uamanual.pdf file contained on the CDROM.

 

(2) Manually Create an Autoqueue:

Open the Universal Analysis program. Select Macros/Autoqueue to open the Autoqueue window. This window will be empty if there are no records currently in the autoqueue. If there are records in the autoqueue already, you can either delete them or append your new records to the existing list.

 

To create a new record, select Edit/Append from the Autoqueue menu. The Autoqueue Parameters window is displayed. Use the Browse button to locate the first data file you want to add to the queue. Use the second Browse button to locate the macro file used in step (1) above. Click on the arrow next to the list of macros and select the macro created in step (1) above. Click the OK button. Your new record will be created and appended to the list (if any) in the autoqueue. You can reorder the list of records using the cut and paste options found in the Edit menu.

 

When you have finished adding all the records that you want, select File/Run from the Autoqueue menu. The data files will be automatically processed (autoanalyzed) beginning with the first record. The results will be collected in the Autoqueue Log.

 

NOTE: To stop autoanalysis of the autoqueue, press the Escape key.
