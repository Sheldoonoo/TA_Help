Cannot Find the Target for Macro Step

Cannot find target for macro step.

 

Problem:

The target control (e.g., button, edit field, combo box, menu item, etc.) for the last executed macro step cannot be accessed because it is not visible or enabled. This usually results when you make inappropriate modifications to the macro or when the macro runs with the Universal Analysis program in a different state from the state in which the macro was captured. This message can also occur when you run a macro that was captured on an newer version of the program on an older version of the program that does not have all of the features of the newer program.

 

Solution:

  • Be careful when modifying macros. Make sure that the steps can still run in an uninterrupted manner. Place the program into the state needed to run the macro successfully.

  • Do not attempt to run newer macros on an older version of the program. Recapture the macro steps using the program you wish to run it on.
