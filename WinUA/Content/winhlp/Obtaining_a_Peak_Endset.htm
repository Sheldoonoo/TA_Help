Obtaining a Peak Endset

(Backside Onset)

 

There are two approaches to obtaining a peak endset (onset analysis on the backside of a peak).

 

Combined Peak Integration and Onset Point

 

This approach involves combining a normal Peak Integration analysis with an Onset Point analysis. The normal peak integration analysis is performed first, followed by an onset point analysis, using the peak maximum point and the peak end point as cursor positions for the onset analysis. This will produce both an onset point and an endset point on the graph.

 

A frequent problem with combining a peak integration and onset point analysis is that the baselines for the two analyses may not overlay. This can be corrected visually as follows:

 

  1. Maximize the graph and then zoom in on the onset analysis so that all three points of the onset analysis are visible on the plot.

  2. Select the Adjust Result function (right-click on the onset point label on the plot and select Adjust Result from the menu). Three cursor points will appear defining the end point of the onset baseline, the end point of the onset line, and the intersection point of the two lines.

  3. Using the left mouse button, click on and drag the onset intersection point to a point on the peak integration baseline such that the onset and peak integration baselines overlay and the onset line has the same slope as before the adjustment.

  4. Accept the cursor position and the new onset line will be drawn.

 

Manual Onset Tangent Selection

 

This approach involves moving the normal peak integration onset to the backside of a peak as follows:

  1. Check the Manual Onset Tangent Selection checkbox on the Onset/Peak Integration Page of Analysis Options before performing the peak integration. After the cursor positions have been accepted for the peak integration, two additional cursors will appear for selection of the onset line.

  2. Move these cursors to two points on the backside of the curve that straddles the curve inflection point. The endset line will be drawn through these two points intersecting the peak baseline.

 

To identify the curve inflection point more precisely, it may be helpful to plot the derivative of the curve being analyzed. The inflection point on the curve being analyzed will coincide with a peak in the derivative curve. If the derivative curve is not desired in the final plot, it can be removed after the analysis is complete (see Signals menu item in the Graph menu). The analyses on the other curves will not be affected.
