Maximum Number of Variables Defined

 

Maximum number of macro variables defined.

 

Problem:

The running macro cannot have any more than 200 defined variables. This total includes both the current macro and any macros called from within that macro containing additional defined variables.

 

If a variable is never assigned a value in a running macro, then it is not considered to be defined and therefore is not counted as one of the 200 variables. Variables become defined at macro run time, only. All previous variables are set as undefined when a macro starts executing from the beginning.

 

Solution:

Keep the total number of variables included in both the current macro and any macros called from that macro to less than 200. If you are using an indexed array (name [index]), then check that the array index is being incremented properly. Each element of an array counts as one of the 200 maximum variables.
