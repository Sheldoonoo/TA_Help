Entering Sample Information Using the Wizard

 

Use this dialog  to enter or select additional information to be stored with the selected file. These items cannot be changed once the experiment is in progress.

  1. Select the type of Sample Shape that you will be using for your sample from the list shown. For additional information on each type of geometry, see the following list:

    1. Rectangular - sample is a rectangular shape. Enter the length or thickness, width, and thickness or height, as is appropriate for the clamp type.

    2. Cylindrical - sample is in the shape of a cylinder. Enter the length or thickness and diameter, as is appropriate for the clamp type.

    3. Tubular - sample is tube-shaped (hollow in the middle). Enter the length and diameter of the inside and outside of the tube.

    4. Fiber (2 types) - sample is a single fiber or fiber bundle. Enter either the Length and Denier (linear density) or the Length and Tex units.

    5. Generic - sample is unusually shaped. Enter the dimensions appropriate for the chosen clamp type.

    6. Ring – sample is round in shape. Enter an inner and outer diameter, and a thickness.

    7. Square – sample is in the shape of a square. Enter the thickness and width.

    8. Unknown – this will be displayed when the clamp type is changed. You will need to reselect the sample shape.

  1. Enter the Dimensions appropriate to the sample shape selected above.

  2. Enter the Sample Name. Enter up to 32 alphanumeric characters to identify the material.

  3. Enter any Comments. Enter additional text to help describe the experiment, for example, sample treatment or experimental conditions could be included. Up to 64 characters may be entered.

  4. Click the Browse button, then select the desired storage location and enter the Data File Name. Click OK when finished. Typical example \\{computer name}\TA\Data\DATA.001.

  5. Enter a Data File Name by clicking on the button, then select the desired storage location and enter in a filename. Click OK when finished. Typical example \\{Computer name}\Program Files\TA\Data\DMA\Data.001. (UNC filenames are used.)

  6. Check the Network Drive checkbox if you want to send your instrument data to another location on your system (e.g., network drive). Click the Browse button to the right, then select the desired archive location. Click Select when finished. The data will be transferred at the end of the experiment.

  7. If you want to automatically analyze the resulting data file using a macro (created with the Universal Analysis Autoanalysis software option), check the Autoanalyze box. Then locate the desired macro filename using the Browse button. Enter the drive, directory, file location, and name for file.

    1. If you want to automatically have the results of the autoanalysis (checked above) sent to a remote computer, which is one  other than the controller computer attached to the instrument (also called the local computer), check the Start Remotely box. Then enter the same path and data filename chosen above. When the data file has been collected, Universal Analysis will automatically open and begin the autoanalysis of the file on both the local and remote computers.

    2. Click on the Analysis Macro list and select a macro from those displayed. This macro will be used to automatically analyze the data file that will result from this experiment.

NOTE: See the Autoanalysis documentation accompanying the Universal Analysis software for further information on creating macros.

When you have finished setting up this run summary, click Next to proceed.
