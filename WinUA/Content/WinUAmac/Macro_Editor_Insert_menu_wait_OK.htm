Wait OK (Macro Insert Menu)

Select the Wait OK option to insert a step that will stop the running of the macro to wait for the operator to press the OK button on a window. Once the OK button has been pressed, the macro will continue.

 

This step replaces the standard "Button OK." step captured for a window. If you want to post a dialog that tells the operator to make certain selections on a window and press OK, it is best to use the Insert/Dialog function instead of the Insert/Wait OK function. When you set up the dialog, select "Stop macro, wait for next OK" as the action to get the desired results.
