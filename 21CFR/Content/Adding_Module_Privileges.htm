Adding Module Privileges

Adding DMA, TMA and SDT to an Existing Integrity Database

If any of the newly supported instrument modules (TMA, DMA, and SDT) are added to an existing Integrity database, their privileges must be imported to the database using the TA System Manager program. To import these privileges the system manager should follow these instructions:

  1. Log into the TA System Manager.
  2. Click on the Groups Manager icon in the left-hand page.
  3. Select the Privileges tab.
  4. Click the Import Items button
  5. Navigate to C:\Program Files\TA Instruments\Integrity.
  6. Select the Item Access Script file for the new module
  7. Click the Open button and then the Continue button to install the privileges.
  8. Repeat this process for each additional new module.
