Using the Statement Editor

 

Although using the Record Mode is generally the easiest way to construct scripts, some functions that you may wish to include in a script cannot be accessed directly from within the Record Mode (e.g., Loop commands).

To use this functionality, you will need to enter and edit scripts directly using the Statement Editor.

When you have a script listed on the screen, you can call up the statement editor to either edit an existing entry or to add a new line. Select the line in the script you wish to work with and either double mouse click on the line number or select the Edit statement item from the Script menu.

This will start the statement editor. All Navigator commands are split into sections for easier access. Click the required function on the left hand window to show the available functions in the right hand side. Double click the required function to move it up to the Statement window. Here you can either edit it directly, or use the assist button to start a small wizard that will guide you through entering the required parameters.

For further information on the available functions, see List of Navigator Commands.
