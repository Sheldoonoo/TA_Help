Main Options Window  Font Page

Use the Font page of the Main Options window to set up the fonts that appear on the printed curve. Changing fonts here will not affect the graphs displayed on your monitor. To check the size and position of the text on the graph before it is printed, use the File/Print Preview option.

 

Follow these steps to set up the fonts:

  1. Click on the button bearing the name of the part of the graph that will have font adjustments. Click here to see an example showing the location of these graph parts.

  2. Select the desired font, font style, and size using the Font Window displayed.

  3. Repeat steps 1 and 2 for the desired parts of the graph.

  4. Click the Save button, if you want to use these as your new settings or click the OK button to use these settings for this session only. The font(s) will be applied to the selected portion(s) of the graph.

 

NOTE: The Save button only stores the settings for the selected page in the initialization (.ini) file. To save all settings, access theMain Options  Initialization Pageand click the Save As button. The Default button will also only recall the default factory settings for the selected page.

 

NOTE: It is recommended that only True Type (TT) fonts be used. True Type fonts are always scalable to all sizes on all output devices.
