Message 995

The auto-diagnostics test failed.  Please check the diagnostics report for further details.

Problem:

The auto-diagnostics test has been completed, but with one or more failure(s).

Solution:

Check the diagnostics report to determine which tests failed and take action as needed. Perform the auto-diagnostics test again after items have been corrected as appropriate. Consult the instrument error log for additional information.

If this error persists, contact TA Instruments for service.
