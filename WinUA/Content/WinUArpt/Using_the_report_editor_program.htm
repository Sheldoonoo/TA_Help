Using the Report Editor Program

The Report Editor is a text editor program that is contained within the TA Instruments Universal Analysis program. It functions very much like other simple text editors that you may have used.

 

NOTE: Only one Report Editor window at a time is available within the Universal Analysis program. If this window is minimized or opened, and the View options are accessed again, the current information displayed in the window is replaced with the newly selected information.

 

Report Editor Menus

You can select the font type, size, and effects (bold, underline, and italics) from the items found below the menu bar.
