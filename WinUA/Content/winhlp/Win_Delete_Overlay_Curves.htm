Delete Overlay Curves Window

Use this window, accessed by clicking the Delete Curves button on the Curves Page, to delete selected curves from the overlay graphing for the selected Y axis. The overlay axis and its label are shown at the top of the window.

 

Check the box next to curve(s) that you want to remove or click the Select All button. Then select OK.

 

NOTE: If you are having problems with the size of your curve overlay legend, either on the monitor, printing, or otherwise, seeOverlay Legend Sizefor more information.
