Spreadsheet View Pop-up Menu

When the Graph/Spreadsheet function has been selected and the data is displayed in a spreadsheet format, you can right-click to reveal a pop-up menu to allow you to edit the spreadsheet.

 

The menu contains the following static menu items:

Copy

Paste

Select All

Modify

Exclude Data

Spreadsheet View (toggles on/off)

 

Also displayed on the Spreadsheet View Pop-up Menu is a list of signals that can be checked or unchecked as desired to display or remove the signal data values on the spreadsheet.

See also: Spreadsheet View Pop-up Signals
