Examples Auto Line Checkbox (Curve Overlay)

When using the Auto Line checkbox on the Auto Configure window, the following applies:

 

Auto Line Checked

Checking Auto Line for Curves (see the figure below) is equivalent to pressing the Auto Line button on the Curve Overlay Options  Curves Page for each overlay axis.

Checking Auto Line for Axis (see the figure below) assigned the first Auto Line type to all curves on overlay axis Y1, the second Auto Line type is assigned to all curves on axis Y2, etc.

Auto Line Unchecked

If the Auto Line box is not checked, the line type for each curve in the overlay will be the same as that used in the individual file graph.
