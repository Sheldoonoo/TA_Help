Data File Saved

The data file will be saved to the destination. If there is a file with the same name, it will be appended a time stamp in its name. Do you want to continue?

This message is displayed when the data file is saved to a remote computer via the Data File Acceptor (DFA). If the computer already has a file with the same name, a time stamp will be appended to the filename.

  • Select Yes to continue under the conditions stated above.
  • Select No to cancel the operation.
