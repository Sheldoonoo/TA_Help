Message 687

The controller where experiment data is being sent cannot process Unicode data files.  Please upgrade Advantage for Q Series before running experiments or data will be corrupted.

Problem:

The controller software has not been updated to handle Unicode data which is being sent from the instrument.

 

Solution:

Install the latest version of the Advantage for Q Series software (V3.3 or higher), following the instructions to install both the latest instrument control software and instrument software. Retry your operation when all software has been updated.
