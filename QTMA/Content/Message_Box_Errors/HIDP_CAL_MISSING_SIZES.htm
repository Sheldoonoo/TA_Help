No TGA Calibration Weight Sizes Specified

No calibration weight sizes specified.

While performing a TGA Weight Calibration on the SDT instrument, the second file (with weights) specified for this calibration does not have any calibration weight sizes specified and therefore cannot be used.

 

  • Conduct the experiment again and make sure that you enter the sizes, then use the new file for calibration.

or

  • Edit the parameter block with the Universal Analysis program to change the sizes to the correct values, then retry the operation.
