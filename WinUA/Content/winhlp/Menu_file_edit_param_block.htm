Edit Parameter Block Menu

Use the Edit Parameter Block function to open a window that allows you to edit the data file parameter block, then create and save it as a new data file. We recommend that you do not overwrite the original data file after editing the parameter block information. The altered file may not open properly if you edit the parameter block information incorrectly. If the file is not overwritten, you can go back to the original data file without a problem.

 

NOTE: If you edit and save the parameters of an open data file, the changes will not take effect until you reopen the saved file.
