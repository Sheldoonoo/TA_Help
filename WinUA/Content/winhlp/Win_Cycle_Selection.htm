Data Cycle Selection Window

Use this window to check the cycles that you want to use for plotting from the list of those contained in the data file. Cycles are defined by any breaks in the data (data on/off regions) or by gas switch and event markers, if they are enabled on the Main Options - Data Page. When cycles are deselected they are marked as "invalid data," but the data is still retained in memory. Enabling a deselected cycle does not reread the data file.

 

Select the desired value from the list displayed. You can use one of these methods:

 

  • Click on the box next to the value desired to check the item.

  • Click the All button to select all of the values contained in the list.

  • Click the None button to clear the selections. You must choose at least one selection.

 

Click on the OK button when finished.
