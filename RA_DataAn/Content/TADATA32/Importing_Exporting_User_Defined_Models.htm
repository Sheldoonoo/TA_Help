Importing/Exporting User-Defined Models

 

Once you have created a user defined model, you can save it to a file by means of the Export function. This file can then be imported into other copies of the Rheology Advantage data analysis program.

To Export (Save) a User-Defined Model:

  1. Select Analysis on the menu.
  2. Click on User Defined Models.
  3. Click on Export.
  4. Click on the model name you wish to export.
  5. Click on the OK button.
  6. Select a name and folder location for the file.
  7. Click on the Save button.

To Import (Open) a User-Defined Model:

  1. Select Analysis on the menu.
  2. Click on User Defined models.
  3. Click on Import.
  4. Locate the file you wish to open.
  5. Click on the Open button.
