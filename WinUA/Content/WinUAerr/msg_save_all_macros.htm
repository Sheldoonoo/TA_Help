Save Macros?

A program macro has been modified. Do you want to save all macros to a macro file before continuing?

 

This is a warning message that appears if you attempt to exit the program without saving modified or new macros to the macro file first.

  • Select Yes to save all of your macros to a macro filewhen you choose this option the Save As dialog is displayed to allow you to choose a macro file and location.

  • Select No to exit the program without saving the macros.

  • Select Cancel to abort the exit operation and leave the program open.
