Setting Up a Strain Rate - Strain Ramp Experiment

 

In a Strain Ramp experiment the sample is exposed to a constant strain rate while the temperature is held isothermal.

If you want to set up a DMA Strain Rate - Strain Ramp procedure, you can choose one of two techniquesâ€”the DMA Wizard or the Experimental View - Procedure Page. See the instructions below for each technique.

When you perform a Strain Ramp experiment, the following parameters need to be chosen:

  • Preload Force: This is the force applied to the sample prior to initiating the experiment.  

    • No preload force is needed when you select a nontensioning clamp (i.e., single/dual cantilever, shear sandwich).  

    • When using tensioning clamps for stress/strain measurements, an initial preload force is applied to remove slack from the sample and to allow an accurate initial sample length to be measured (films and fibers) or to ensure good contact of the movable clamp (3-point bend nose or compressional upper plate) with the sample.  The preload force used should be the minimum required to achieve the desired starting point since this preload force could affect the measurement (it is not added to the measured stress).  0.001N is the lowest force which can be applied.

  • Initial Strain:  This value specifies the strain percentage that will be established as an initial condition for the run.  The first strain or displacement ramp segment of the method will start at this strain value.

  • Initial Displacement:  This value specifies the displacement that will be established as an initial condition for the run.  The first strain or displacement ramp segment of the method will start at this displacement value.

  • Isothermal Temperature:  This value sets the constant temperature to be maintained during the experiment.

  • Strain Rate:  This value sets the rate (in %/min) at which the sample length will be changed during the experiment.

  • Final Strain:  The experiment will end when the sample length has been changed by this percentage.

  • Additional Conditions:  There are additional experimental conditions that can be changed to optimize specific situations.  These are accessed using Advanced Parameters and Post-Test Conditions.  For most experiments, the system default values shown are recommended.

  • See also:  Guidelines for Running Subambient Experiments.

Using the DMA Wizard :

Using the Experimental View Procedure Page:

  1. Display the Procedure Page. Select the desired experiment from the list of available preprogrammed templates.

  2. Enter the desired parameters. When you have finished setting up this procedure, you can enter more information by selecting the Notes Page and Summary Page for this run.

  3. Click the Apply button when finished to save these changes or the Append button to save the changes to this run and add another run to the end of the sequence.

NOTE:  You can use any test template as the basis for a custom test by selecting the desired test template from the list of tests, then click the Apply button and change the test to Custom. Then you can click on the Procedure Page and then the Editor button to make the changes desired.
