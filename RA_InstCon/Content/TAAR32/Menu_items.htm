Instrument Control Program Menus

 

You can use menus and tool bars to give Rheology Advantage instructions about what you want to do.

A menu displays a list of commands. Most menus are located on the menu bar at the top of the Word window. Shortcut (or pop-up) menus are available when you right-click text, graphs, or other items.

A menu that shows a list of commands relevant to a particular item. Among items for which you can display a shortcut menu are text, tables and graphs. To display a shortcut menu, right-click an item, or press SHIFT+F10.

A short summary of the available menus is given below.

  • File:  Allows you to perform basic file tasks such as open, save etc.
  • Edit:  Editing functions such as cut, copy paste.
  • Instrument:  Provides direct access to instrument control functions. Many of these can also be accessed from the Instrument and Head control tool bars. See also:  Customizing Tool Bars.
  • Geometry:  From here you can open, save, view and edit geometry information.
  • Procedure:  Allows you to open, save, view and edit procedure information.
  • Notes:  From here you can open, save, view and edit the sample notes.
  • Experiment:  You can configure the way information is presented when an experiment is run and control the run progress.
  • Navigator:  This menu is only active if the software has been started via the Rheology Navigator program.
  • Options:  Modify the various options and configuration settings. To find out more about these items, select one and then use the button to find out more.
  • Wizard:  Allows you to start either the Geometry or Procedure wizards. See also: Measuring System Selection Guide or Concepts of Test Design.
  • Help:  Access the help system contents page and index system. The About item shows information about the program and file versions
