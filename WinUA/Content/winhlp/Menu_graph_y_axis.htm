Y Axis

Use the Y Axis option to select a Y-axis signal for functions such as rescaling, analysis, etc. Emphasis will be switched to that signal (as seen by the small box enclosing the Y-axis number). This option is only available when multiple Y-axis signals are displayed.

 

You can either use the Graph/Y Axis menu function or one of the following methods to select a Y-axis signal:

 

  1. Click on the signal name label of the Y-axis shown on the graph to select that axis.

  2. Click on the Y-1, Y-2, label displayed above the corresponding axis on the graph to select that axis.
