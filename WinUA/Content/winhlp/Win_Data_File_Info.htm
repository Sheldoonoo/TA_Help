Data File Information

Use this window to verify the information contained in the current data file. Some of the information is not editable here since it is a fixed part of the data file. To change the editable information, follow the instructions below:

 

1. Verify or edit the desired sample name, sample size, operator name, method description, and comments .

2. Select the exotherm direction for your plotted curve, if applicable.

3. Enter the desired cell constant where appropriate.

4. If the data file contains saved analysis results, you can check the Restore analysis checkbox to return the data file back to the state it was in when Save Analysis was selectedit cancels any analysis and rescale conditions selected after the analysis was saved.

5. Set up or change the parameters used for analysis by clicking on the Inst Params, Signals, Units, or Data Limits buttons.

6. Select OK when finished.

 

To automatically use the same parameters as another open data file, click the Same As button, then click on the desired file to copy parameters from.

 

NOTE: To return to the Open Data File window and choose a different data file, click the Previous button.
