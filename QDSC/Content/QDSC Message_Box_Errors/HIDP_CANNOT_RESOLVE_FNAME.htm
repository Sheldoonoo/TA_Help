Cannot Resolve Filename

Can not solve the remote file name. Use default file name in your local directory.

This message is produced when you attempt to use the Start Remotely function on the Summary Page. This function is used to have the results of the autoanalysis sent to a remote computer, which is one  other than the controller computer attached to the instrument (also called the local computer). When the data file has been collected, Universal Analysis will automatically open and begin the autoanalysis of the file on both the local and remote computers.

The problem arises when the system cannot find the filename on the remote computer. The default filename in the local directory should be used.
