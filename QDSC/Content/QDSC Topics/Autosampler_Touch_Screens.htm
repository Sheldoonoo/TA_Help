Autosampler Touch Screens

 

When an Autosampler is installed on your DSC, there are touch screen functions that are available to allow you to control and calibrate the Autosampler. The Autosampler touch screens are not accessible during an active experiment.

The configuration of the touch screen is different depending on the instrument model you are using. Select the appropriate link from the list below to access information on your particular model's Autosampler options.

Autosampler Control Touch Screen

The Autosampler Control Mode Menu is accessed by touching the AUTOSAMPLER key on the Control Menu touch screen. The DSC AutoLid is automatically opened when the Autosampler Control menu is accessed. The keys shown in the figure below are displayed.  A brief description of the function of each key is provided in the table below.

Back to top

Key Name

Description

LOAD

Loads the selected pan position number from the Autosampler tray to the cell.

PAN NUMBER

Touch the up or down arrow key to increase or decrease the pan position number to select a pan for loading or unloading.

UNLOAD

Unloads the selected pan position number from the cell to the Autosampler tray or waste bin.

PAUSE

Temporarily stops the Autosampler motion without cancelling any prior commands.  Stops operation and freezes motion.

RESUME

Resumes the previous command when the Autosampler has been stopped with the Pause key.

CANCEL

Terminates any Autosampler touch screen key command in progress and parks the unit.

RESET

Clears all commands, instructs the unit to locate all home positions, and returns the Autosampler to the park position. Also instructs the Autosampler to "forget" the current lid and sample pan locations (i.e., assumes the cell is empty).

AUTO CAL

Press this key to automatically calibrate the Autosampler tray positions and gripper positions in the cell. Click here for further information more.

TRAY CAL

This key is used to calibrate the Autosampler tray positions. Click here for further information more.

CELL CAL

This key calibrates the gripper positions in the cell. Click here for further information more.

MANUAL CAL

Press this key to display the Autosampler Manual Calibration touch screen, which allows you to finely tune the positioning after using the AUTO CAL function. For further information, click here more.

TEST MODE

Displays the Autosampler Test Options touch screen.

EXIT

Exits this touch screen and returns to the DSC Control Menu touch screen.

Autosampler Test Options Touch Screen

The Autosampler Test Options touch screen  is accessed by touching the TEST MODE key on the Autosampler Control  touch screen. The keys used to perform a cycle test, test each of the individual Autosampler motors, and access the sensor touch screen, are displayed as shown in the figure below. A brief description of the function of each key is provided in the table below.

Back to top

Key Name

Description

CYCLE TEST

Cycles the Autosampler continuously through the loading and unloading of the Autosampler tray. To stop the Cycle Test you must touch the STOP TEST key. The current pan number and cycle number will be displayed while cycling.

STOP TEST

Stops the cycle test.

SENSORS

Displays the Autosampler Sensor Status touch screen, which displays the current sensor conditions.

EXIT

Leaves the Autosampler Test Options touch screen menu.

Autosampler Sensor Status Touch Screen

The Autosampler Sensor Status touch screen is accessed as follows:

  1. Press AUTOSAMPLER on the Control options touch screen.
  2. Press TEST on the Test Options touch screen.
  3. Touch the SENSORS key on the Autosampler Test Options touch screen.

The items indicating the sensor status of the Autosampler, shown in the figure below, are displayed. A brief description of each sensor item and key displayed is provided in the table below. This touch screen also provides buttons to place the Autosampler in the home, park, and ship positions.

Back to top

Sensor

Description

UP/DN

Indicates the up/down position of the arm. Zero (0) is the up (home) position and one (1) is the down position. The number after the colon indicates the status. On the touch screen above, the status is arm position down.

XT/RT (EXTEND/RETRACT)

Indicates the extend/retract position of the finger assembly. Zero ( 0) is the home (retracted) position and one (1) is the extended position.

LEFT/RT

Indicates the left/right position of the automatic arm in relationship to the home position. Zero (0) is left of home and one (1) is right of home.

TRAY

Indicates the position of the Autosampler tray. Zero (0) is the home position.

GRPR (GRIPPER)

Indicates the gripper finger position. Zero (0) is the home or closed position and one (1) is the open position.

OPTC (OPTIC PAN)

Indicates the fiber optic amplifier output condition, which senses the presence of a pan. Zero (0) equals no pan present and one (1) means a pan is present.

ELEC (ELECTRIC) PAN

Indicates the electric pan sensor condition, which senses the presence of a pan. Zero (0) equals no pan present and one (1) means a pan is present.

TRAY

Indicates whether or not an Autosampler tray is present. Zero (0) equals no tray present and one (1) means a tray is present or on.
