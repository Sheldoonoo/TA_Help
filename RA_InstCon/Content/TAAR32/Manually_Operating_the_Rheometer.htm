Setting the Temperature

Follow the steps below to set the temperature of your experiment. Choose the steps based on the current state of your system.

Setting the temperature from the Instrument Status screen

  1. Select Instrument from the menu.
  2. Click on Set Value.
  3. Click on Temperature. You can access the temperature item either from the button or from the temperature entry in the instrument status view.

Setting the temperature in a procedure

In the Pre-experiment step, set the required temperature for the test run. If the Wait for correct temperature option is selected, the software will not move onto the next part of this step until this temperature is achieved. When you change the value here, all currently defined steps will be updated to use this temperature. If you wish to use different temperature in the subsequent steps, you can change them after you have completed setting up the pre-experiment step.

Setting the Shear Stress

Follow the steps below to set the shear stress:

  1. Ensure that the instrument status page is active.
  2. Select Instrument from the menu.
  3. Click on Set Value.
  4. Click on Stress. You can access the item either from the button or by double clicking the shear stress entry in the instrument status view.
  5. Enter the value required. (You can select between shear stress or torque.)
  6. Press the Send button.

Setting the Shear Rate

Follow the steps below to set the shear rate:

  1. Ensure that the instrument status page is active.
  2. Select Instrument from the menu.
  3. Click on Set Value.
  4. Click on Rate. You can access the item either from the button or by double clicking the shear rate entry in the instrument status view.
  5. Type in the value required. (You can select between shear rate or velocity.)
  6. Press the Send button.

Zeroing the Displacement

Follow the steps below to zero the displacement:

  1. Ensure that the instrument status page is active.
  2. Select Instrument from the menu.
  3. Click on Set Value.
  4. Click on Zero displacement. You can also double click the displacement entry in the instrument status view to zero the reading.

Raising the Head

Follow the steps below to raise the rheometer head:

  1. Select Instrument from the menu.
  2. Click on Gap.
  3. Click on Raise head. You can access the item using the button.

Stopping the Head's Movement

Follow the steps below to stop the rheometer head's movement:

  1. Select Instrument from the menu.
  2. Click on Gap.
  3. Click on Stop head. You can access the item from the button.

Setting a Gap

Follow the steps below to set a gap:

  1. Ensure that the instrument status page is active.
  2. Select Instrument from the menu.
  3. Click on Gap.
  4. Click on Enter gap... You can access the item either from the button or by double clicking the gap entry in the instrument status view.
  5. Type in the gap required.
  6. Press the Send button.
