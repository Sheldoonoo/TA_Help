Compaction Failed

Compaction failed. You must reduce data limits or increase data buffer size.

 

The number of data points in this file is too large to store in the current size data buffer, even with automatic data compaction (averaging). You can solve this problem by 1) limiting the range of data read from the data file (see Data Limits), by 2) closing all files and increasing the size of the data buffer (see Main Options/Data page), or by 3) reducing the number of "Data On/Off" or Event" segments in the thermal method and rerunning the experiment.

 

NOTE: If this data file is currently being collected (the file is not closed), then this error can occur if a large number of additional data points are added to the file at the same time that data is being read by Universal Analysis. In this case, simply reading the data file again should solve the problem. If the problem persists, then refer to the above solutions.
