Message 122

Probe travel exceeded limit. Check probe and stage.

Problem:

This message is caused by one of the following:

  • The probe and stage are improperly installed.
  • The probe type does not match the instrument mode (e.g., a film/fiber probe is installed and instrument is not set to a film/fiber mode).
  • A problem exists with the hardware.

Solution:

Try one or more of the following operations to solve the problem, then restart your experiment.

  1. Check the probe and stage for proper installation. The probe should move freely with low (or no) force applied.
  2. Ensure that the instrument mode matches the probe type.
  3. If the above items are correct, call TA Instruments Service.
