Cannot Connect to Instrument

The destination computer can not connect to the instrument. Please check the connection between that computer and the instrument and then try again.

This message occurs when you try to save the data file to a computer other than the local one and that computer cannot connect to the instrument.

Check all cable connections between the Ethernet hub and the instrument and between the computer on the Ethernet hub to make sure nothing has become disconnected. Ensure that all network capabilities are fully functional. Try again to select the data file on the destination computer.

If this message persists after all efforts to resolve the communication problem have been attempted, call TA Instruments for service.
