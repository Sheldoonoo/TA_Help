Merging Data

Follow the steps below to merge data:

  1. Load two or more results files.
  2. On the Results view window, select the files you wish to merge.
  3. Select File from the menu.
  4. Click on Merge.
  5. Select the file that you wish to use as the base or master file.
  6. If you wish to produce a graph that is an averaged version of all selected curves, make sure the Average points of same x-variable value is selected .
  7. The software will produce a new entry in the Results view window. This file contains the result of the merge. The file name is the base file name with an added digit.
  8. This new file can be treated as any other file loaded into the data analysis software.
