Setting Up an Isostrain Experiment

 

In an Isostrain experiment a specific known strain is applied to the sample and the temperature is then ramped. The amount of stress (force) required to maintain that strain is monitored. The Isostrain test is intended for shrinkage force measurements where the amount of shrinkage that occurs when the sample is heated is very large relative to the thermal expansion of the system (sample, clamps, and posts).  If the sample does not shrink, do not use this test. This mode only works in conjunction with the film tension clamp.

If you want to set up a DMA Isostrain procedure, you can choose one of two techniques—the DMA Wizard or the Experimental View - Procedure Page. See the instructions below for each technique.

When you perform a DMA Isostrain experiment, the following parameters need to be chosen:

  • Preload Force:  When using tensioning clamps for isostrain measurements, an initial preload force is applied to remove slack from the sample (films and fibers) and/or to eliminate sample movement when the offset stress required to achieve the desired isostrain level is instantaneously applied. The preload force used should be the minimum required to achieve the desired starting point since this preload force could affect the measurement (it is not added to the measured stress).  0.001N is the lowest force which can be applied.  

  • Strain:  The objective in this mode is to measure the amount of stress required to maintain a constant sample length (amount of strain).  Therefore, the strain chosen is usually small enough to ensure that measurement is occurring within the linear viscoelastic region and large enough to ensure that fibers and films are fully elongated (i.e., slack is removed). A strain between 0.1 percent and 0.5 percent may be a good starting point.

  • Start Temperature & Final Temperature:  The DMA temperature range is -145 to 600°C.  Attaining the subambient portion of this range depends on the presence of a Gas Cooling Accessory (GCA). Enter the temperature range of interest.

  • Ramp Rate:  The thermal mass of the DMA furnace and the large samples limits the recommended heating and cooling rates.  1 to 5°C/minute is common.  Heating rates in excess of 5°C/minute are not recommended.

  • Additional Conditions:  There are additional experimental conditions that can be changed to optimize specific situations.  These are accessed using Advanced Parameters and Post-Test Conditions.  For most experiments, the system default values shown are recommended.

  • See also:  Guidelines for Running Subambient Experiments.

Using the Experimental View Procedure Page:

  1. Display the Procedure Page. Select the desired experiment from the list of available preprogrammed templates.

  2. Enter the desired parameters. When you have finished setting up this procedure, you can enter more information by selecting the Notes Page and Summary Page for this run.

  3. Click the Apply button when finished to save these changes or the Append button to save the changes to this run and add another run to the end of the sequence.

NOTE:  You can use any test template as the basis for a custom test by selecting the desired test template from the list of tests, then click the Apply button and change the test to Custom. Then you can click on the Procedure Page and then the Editor button to make the changes desired.
