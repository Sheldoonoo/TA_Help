Calibrating the SDT for DTA Baseline

 

DTA Baseline Calibration is based on analyzing Delta T data collected from a baseline run conducted over the temperature range expected in subsequent experiments. (This experiment usually utilizes the same baseline run obtained for TGA Weight Calibration.)

The DTA baseline is corrected by a linear (slope and offset) function of the sample temperature. This results in shifting and rotating the baseline so that the calibrated portion is near 0°C.

NOTE:  The DTA baseline calibration is not required when using the SDT as a DSC-TGA (i.e.,  it is only required when saving the Delta T signal).  This is because the DSC Heat Flow Calibration includes a baseline subtraction.  However, the shape of the DTA baseline is valuable as a quick verification that the SDT beams are correctly positioned and that DSC heat flow calibration can be successfully completed.  

Gathering DTA Baseline Data

Use the following instructions to gather the data file needed for DTA baseline calibration. This process uses the "DTA baseline" test template. If you have already performed TGA weight calibration, you can utilize the first calibration run (without weights) for DTA Baseline calibration and proceed directly to the analysis steps.

  1. Check that your purge gas is connected.

  2. Open the furnace. Remove all SDT cups, sample, and weights from the sample and reference beams.

  3. Close the furnace

  4. Select Experiment View and access the Summary Page. Set the instrument to "calibration" mode using the Mode drop-down list. (In the calibration mode the current temperature table data is ignored.)

  5. Select the "DTA Baseline" test procedure.

  6. Enter the requested sample information, entering zero for the Sample Weight and Reference Weight.

  7. Specify a data file name using the Browse button.

  8. Click on the Procedure tab.

  9. Check Use Current as the starting temperature or enter the desired Start Temperature (e.g., 50.00°C).

  10. Enter your upper temperature (or 1500°C) as the Final Temperature.  

  11. Enter the Heating Rate that will be used in subsequent experiments (e.g., 20°C/min).

  12. Select the Advance button to access and select the Advanced Parameters.

  13. Click the Post Test button to access the Post-Test condition parameters that allow you to specify the furnace and air cool options to use once the experiment is complete. (Select "Furnace Closed" and "Air Cool to 50°C.")

  14. Click on the Notes Page.  Enter the requested information, including the desired flow rate (typically 100 mL/min).

  15. Click on Apply to store the changes made to these pages.

  16. Schedule this run, if more than one run is in the sequence list.

  17. Select Start to begin the calibration run.

  18. Analyze the calibration data as directed in the next section.

Analyzing DTA Baseline Data

  1. Select Calibrate/DTA Baseline from the menu and specify the file created above.

  2. Analyze the baseline data using Analyze from the Analyze menu.

  3. Position the two analysis markers in the desired location (bracketing the analysis region of interest) then select Limits OK from the pop-up menu or Analyze menu.    

  4. Save the calibration results using Analyze/Accept results.

  5. Close the Calibration window.

The Calibration Report menu can be used to view the date this calibration was last conducted, the temperature range selected, and the associated data filename.
