Cannot Find Label in Macro

Cannot find label in macro.

 

Problem:

The target label for a "Select Label" macro step cannot be located. Thus usually happens when you make inappropriate modifications to the macro or when the macro runs with the Universal Analysis program in a different state from the state in which the macro was captured.

 

Solution:

Be careful when modifying macros. Make sure that the steps can still run in an uninterrupted manner. Place the program into the state needed to run the macro successfully.
