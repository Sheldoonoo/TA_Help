Auto Limits

Use the Auto Limits function to store and then automatically reuse stored analysis limits. This option is good for repeating a particular analysis on similar data files and take the uncertainty factor out of trying to choose the correct analysis limits. The desired position of the cursors relative to the peak can be predetermined using the Auto Limits feature. Then when you analyze similar data files, you can recall the appropriate cursor position and perform accurate analyses quickly.

 

This option is only available for the following analyses: Integrate Peak/Linear and Sigmoidal Horizontal, Glass or Step Transition, Onset Point, and Weight Change.

 

When you select this option a window is displayed that allows you to either use the cursor (marker) position that the operator selects or to use an offset (or differential fraction).
