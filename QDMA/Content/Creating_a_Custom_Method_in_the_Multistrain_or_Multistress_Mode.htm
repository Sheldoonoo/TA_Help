Creating a Custom Method in the Multistrain or Multistress Mode

 

The steps needed to create a "custom" method of your own for the DMA are slightly different depending upon the mode selected.

Follow these steps to create a custom method for the Multistrain or Multistress Mode:

  1. Access the Experimental View/Summary Page.

  2. Select the "Multistrain" or "Multistress" Mode from the drop-down list available. See also:  Selecting a DMA Operating Mode.

  3. Select the signals that you want the software to store in the data file by clicking the button. This displays the Select Signals to Store dialog. Check the desired signals.

  4. Select the "Custom" type of Test from the drop-down list available.

  5. Select the name of the installed Clamp from the drop-down list. See Available DMA Clamps for more information.

  6. Select the type of Sample Shape that you will be using for your sample from the list shown.

  7. Enter the Dimensions appropriate to the sample shape selected above.

  8. Enter the Sample Name. Enter up to 32 alphanumeric characters to identify the material.

  9. Enter any Comments.

  10. Enter a Data File Name by clicking on the button, then select the desired storage location and enter in a filename. Click OK when finished.

  11. Check the Network Drive checkbox if you want to send your instrument data to another location on your system (e.g., network drive). Click the Browse button to the right, then select the desired archive location. Click Select when finished. The data will be transferred at the end of the experiment.

  12. If you want to automatically analyze the resulting data file using a macro (created with the Universal Analysis Autoanalysis software option), check the Autoanalyze box. Then locate the desired macro filename using the Browse button. Enter the drive, directory, file location, and name for file.

    1. If you want to automatically have the results of the autoanalysis (checked above) sent to a remote computer, which is one  other than the controller computer attached to the instrument (also called the local computer), check the Start Remotely box. Then enter the same path and data file name chosen in step 9 above. When the data file has been collected, Universal Analysis will automatically open and begin the autoanalysis of the file on both the local and remote computers.

    2. Click on the Analysis Macro list and select a macro from those displayed. This macro will be used to automatically analyze the data file that will result from this experiment.

     

NOTE: See the Autoanalysis documentation accompanying the Universal Analysis software for further information on creating macros.

  1. Click on the Procedure Page tab.

  2. Type any Notes that you want to write about the test you will be running.

  3. Enter the Frequency.  The frequency must be held constant in these modes.

    Frequency, in Hz, is the number of oscillation cycles completed in one second.

  4. For Tensioning Type Clamps Only:

    1. Enter the Preload Force.

      Preload Force is the force applied to the sample prior to initiating the experiment.  

      • Preload force is not available when you select a nontensioning clamp (i.e., single/dual cantilever, shear sandwich).  

      • Preload force is required with the tensioning clamps (i.e., 3-point bending, compression, and tension film/fiber) when running multifrequency and multistrain experiments in order to sustain oscillation (ensure that force is applied to the sample throughout the complete oscillation).  Typical preload forces for those experiments are 0.01 to 0.5 N.

       

    2. Choose one of two methods (a or b) to maintain a positive force on the sample:  

      1. Check the Force Track box and enter the desired ratio of static and dynamic forces, if you want the total force that is applied to the sample to be automatically adjusted during the experiment.

        Selecting Force Track will cause the instrument to automatically adjust the force as the sample changes modulus, in order to minimize permanent sample deformation. This parameter is used for 3-point bending, compression, penetration, and film/fiber tension clamps. These clamps allow sample deformation in one direction only.

        Force track automatically adjusts the total force (preload and dynamic) applied to the sample during multifrequency and multistrain experiments.  

        • Force track is only used when one of the tensioning clamps (i.e., 3-point bend, compression, tension film and tension fiber) is selected. It is not needed for nontensioning clamps (i.e., single/dual cantilever, shear sandwich).

        • Force track essentially determines the dynamic force required to oscillate at the desired amplitude and frequency; it applies an additional static force that is a constant multiple of that dynamic force. This is necessary to avoid sample buckling in tension clamps and loss of contact with the sample for compression and 3-point bending clamps.

        • Recommended force track values are 120% for thin films and low denier fibers (tension film and tension fiber clamps), 150% for thermoplastics (3-point bending clamps), and 125% for general applications.

        • As the temperature increases, the modulus of the material drops (provided that there is no cross-linking reaction). Since the static force is adjusted in proportion to the dynamic force, the sample does not creep or undergo permanent deformation during the experiment.

        After checking the Force Track box, enter the desired value in percent.

      2. Uncheck the Force Track box, if you want the to use a constant force (the Preload Force) on the sample.

NOTE:  If there is a need to maintain a constant force on the sample throughout the experiment, enter that force in the Preload Force field and uncheck Force Track.

  1. For Multistress Only:

Enter the desired Initial Amplitude.  This parameter is used to evaluate the stiffness of the sample prior to running the experiment in order to establish the limits of stress that the instrument will be able to apply at certain frequencies.  This should be a small number, in the range of 2 to 5 micrometers.

  1. For Multistrain Only:

    1. Set up the amplitude table. Click on the Amplitude tab at the bottom of the page. The amplitude table is displayed to allow you to enter these parameters.  See also: Using the Frequency/Amplitude/Stress/Force Tables.

  1. If you want to edit the current method (the name and segment is displayed), click the Editor button to display the Method Editor window.

  2. To change the parameters that the instrument will use to perform the experimental run, click the Advanced button  to display the Advanced Parameters window applicable to the current instrument.

  3. To change the conditions that will apply at the end of the run (method-end conditions), click the Post Test button to display the current Post Test window. See also:  Guidelines for Running Subambient Experiments.

  4. When you have finished setting up this procedure you may add notes or verify the information on the Notes Page.

  5. Click the Apply button when finished to save these changes or click the Append Run button to add another run to the end of the sequence.

NOTE:  After a run is appended, the DMA clamp type cannot be changed.

 

To see a summary of the current procedure for this run, open the Procedure Summary window using the  button located on the Summary Page. To view a summary of a saved procedure file, open the File Utility window using Tools/File Utility.

NOTE:  You can use any test template as the basis for a custom template by selecting the desired test template from the list of tests, then click the Apply button and change the test to Custom. Then you can click on the Procedure Page and then the Editor button to make the changes desired.
