Checking the Installation

Once the installation is complete, you should start the machine to check that everything is working and that all parts of the system are communicating with each other. This is done by using the following method:

  • Turn on the air supply to the instrument.
  • Turn on the water supply to the instrument.
  • Remove the air-bearing clamp.
  • Turn on all electrical parts of the system (rheometer, PC, etc.). A system check will be initiated as shown by the LCD on the rheometer.
  • Start the software.
  • Select the Instrument Status screen in the software.
  • If everything is installed correctly, the instrument will display continually updating figures.
  • Lower the head. If the installation is OK, the head will operate.
  • Input a temperature slightly different to that displayed. If the installation is OK, the temperature will change to that which you have just input.
  • Raise the head.

If all of these actions result in the correct response, you can be confident that you have installed the system correctly and it is ready for use. If you have problems please contact your local TA Instruments office or appointed agent.
