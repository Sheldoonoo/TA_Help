Event Connections

 

The AR2000 event outputs are configured as open drain drivers with a 48-volt supply rail and include a diode to enable use with inductive loads (see diagram below).

It is recommended that any user equipment is connected to the AR2000 via a 48-volt relay operated from the event output to isolate any direct connection to the AR2000.

CAUTION:  Any connection to the event outputs is at the users own risk. TA Instruments cannot be held responsible for any damage or injury that may arise from the use of this facility.

The AR2000 has been tested to the relevant safety and electromagnetic compatibility directives. This does not include any user-supplied equipment, and it is up to the user to ensure that the complete system complies with all relevant regulations.
