Message 231

DSC calibration not available. Cannot start run. Perform DSC calibration.

Problem:

The SDT instrument must have heat flow calibration information in order to perform TGA-DSC experiments.

Solution:

Perform the following steps:

  1. Select the calibration mode on the Summary Page.
  2. Select Calibrate/DSC Heat Flow from the main menu.
  3. Complete the steps needed. See the online help for information.
  4. Select the standard mode on the Summary Page. Perform the desired experiments.
