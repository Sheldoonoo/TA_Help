Bad Macro File

Errors detected while reading macro file.

File is corrupted or not compatible with this program.

Reading file: <macro_file_path>

 

Problem:

There is a problem with the macro file that you are attempting to open. The file could be corrupted or it may have been created with a different program. It cannot be used.

 

Solution:

Select a different macro file or try opening the macro again. If the problem persists, you may have to recapture and resave the macro operation to a different file.
