Selecting a DMA Operating Mode

 

There are eight DMA operating modes available for the cantilever, 3-point bending, shear sandwich, compression, and film/fiber tension clamps.  Four dynamic modes (multifrequency controlled strain, multifrequency controlled stress, multistrain, and multistress), two transient modes (creep and stress relaxation) and three static modes (isostrain, stress/strain, and strain rate).

As with clamping, similar information about material properties (modulus, damping, transition temperatures) can be obtained from more than one mode.

The following table basic guidelines for selecting a mode. For more information see Understanding the DMA Testing Modes and Their Applications.

Once you have selected a mode, you will need to enter your choice using the instrument control program.  

  • If you are using the experimental wizard, select the radio button for the desired mode, then click the Next button.

  • If you are creating your experiment without the wizard, select the desired mode from the drop-down list on the Summary Page.

For further information, click on the name of the mode in the table above.
