Analyzing TMA Calibration Data

 

Before analyzing the TMA cell constant or temperature calibration data you will need to conduct the necessary calibration experiments. Once the calibration experiments are complete, as indicated by a check mark next to the run number(s) on the sequence list, The steps needed to perform a calibration analysis are:

  • Gather calibration data:
    • For information on gathering temperature calibration data, see Calibrating the TMA Temperature.
    • For information on gathering cell constant data, see Calibrating TMA Cell Constant.
  • Select a TMA Calibration File
  • Choose the TMA Calibration Type and Standard
  • Set Calibration Limits
  • Rescale the Plot (if desired)
  • Perform the Analysis
  • Save the results.

Selecting a TMA Calibration File

After you have gathered your calibration files follow the instructions below to select the appropriate calibration file for analysis:

  1. Select Calibrate/Analysis or click the button to access the TMA Open Calibration File  window.
  2. Locate and open a TMA Q400 calibration file.
  3. Click the Open button. A window is displayed to allow you to choose the calibration type as directed in the next section.

Choosing the TMA Calibration Type and Standard

After choosing a calibration file the Calibration Analysis Selection window is displayed to allow you to choose from two types of calibration for the TMA:  cell constant or temperature calibration. Select the standard used for calibration.

Setting Up Temperature Calibration Standards

This function is used  to define the default settings for each TMA temperature calibration standard (e.g., theoretical melting point), add new standards, and reset (reload) the default calibration table.

  1. Place the instrument in calibration mode on the Summary Page.
  2. Access the TMA Calibration Analysis Selection window by selecting Calibrate/Analysis or click the button, then select a calibration file.
  3. Select the Standard Name from the list to edit or verify its default parameters displayed on this window.
    • To edit a standard in this list, click the Edit button to display the Calibration Standard window. Enter the literature Melting Point in °C. Consult your Chemistry handbook for the value. This standard will be edited when you click the OK button.
    • To add a new standard to this list, click the Add button to display the Calibration Standard window. Enter the Standard Name and its literature Melting Point in °C. Consult your Chemistry handbook for the value. This standard will be saved when you click the OK button.
  4. Click the OK button when finished. The calibration file will be plotted. To perform various functions right click to display the pop-up menu. This allows you to rescale the plot, choose another file (Next File), change the data limits, etc. See also:  Rescaling the Plot

NOTE:  To return the standards list back to the default values, click on the Defaults button.

Rescaling the Plot

There are several different ways to rescale the analysis plot:

  • Expand part of the curve
  • Use the rescale options

Expanding Part of the Curve

To zoom in on a particular portion of the analysis curve for calibration purposes:

  1. Position the pointer in the general area that you wish to enlarge and hold down the mouse button  As you move the mouse, a dotted line box will be drawn from the original pointer position.
  2. Move the mouse to draw a dotted line around the portion of the curve to be enlarged.
  3. Release the mouse button. The curve will be rescaled immediately. This process can be used repeatedly to expand a smaller portion of the curve.

To reverse the expansion process, right click the mouse button to display the pop-up menu and select Previous Scale. (Shortcut:  Ctrl-B) You can repeatedly perform this action, until you have stepped backwards through the expansion increments to the desired point.

If you want to return the curve to its original full scale state, right click the mouse button to display the pop-up menu and select Full Scale.

NOTE:  You can not expand the curve when the markers are displayed. Remove the markers by clicking on Cancel from the pop-up menu.

Using the Rescale Menu Options

You can rescale the calibration analysis plot by choosing any of the following items from the Analyze/Rescale menu, which is available after the file is plotted:

  • Previous Scale:  The Previous Scale option redraws the plot as it appeared before the most recent scaling changes. Previous Scale can be used repeatedly until the original limits are displayed. Shortcut: Ctrl-B
  • Full Scale:  The Full Scale option rescales the plot to display the entire curve in the plot window. If you rescale the graph, then decide to go back to the original scaling, choose the Full  Scale option.
  • Data Limits:  See Setting Calibration Limits.
  • X-Axis:  Use the X-Axis/Time option to select the time signal to graph on the x axis. When you select this option the graph will be replotted immediately. No analysis functions are available when the axis is time. Use the X Axis/Temperature option to select the temperature signal to graph on the x axis. When you select this option the graph will be replotted immediately.

Performing the Analysis

To analyze the data in your calibration file, perform the following steps with the calibration file plotted:

  1. Right click and select Analyze from the pop-up menu). The current limit points will be indicated with markers on the graph.

 NOTE:  The active marker is indicated by a full-length vertical red line. Other markers are indicated by smaller red cross-hairs.

  1. Move the active marker to the desired position on the curve using either the mouse or keyboard (see the directions below).
  2. Select (activate) the next marker by:  (a) Pressing the TAB key or (b) Positioning the pointer over a marker and clicking the left mouse button. (Pressing back TAB (Shift-TAB) will activate the previous marker)
  3. Repeat steps 2 and 3 until you have selected all the necessary points for analysis.

NOTE:  The limits for TMA Temperature Calibration are chosen by placing the first marker before the dimension change onset and the second marker after the onset.

  1. Select the Limits OK button (or select Limits OK from the pop-up menu) to accept the  points. When calculated, the calibration results will be displayed.
  2. Accept the calibration results by selecting the Accept button (or by selecting Analyze/Accept results from the pop-up menu). You can delete results by selecting the Delete button (or by selecting Analyze/Delete results from the pop-up menu).

You can verify this result by selecting Cell/Temperature Table from the Calibrate menu.

Moving the Markers Using the Mouse:

  1. Position the pointer at the desired location.
  2. Double-click mouse button 1 to move the active marker to this point. This action causes the next marker to become active.

or

  1. Position the pointer over the active marker.
  2. Press and hold down the left mouse button while you drag the marker to the desired location.
  3. Release the mouse button. At this point you may use the arrow keys to position the marker more closely (see the directions below).

Moving the Markers Using the Keyboard:

Press the arrow keys to move the active marker to the desired position.

The marker will move in the following increments:

  • Arrow key  =  smallest step
  • Alt/arrow key  =  second smallest step
  • Shift/arrow key  =  second largest step
  • Control/arrow key  =  largest step
