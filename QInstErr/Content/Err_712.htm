Message 712

Unrecognized method segment. Segment ignored, run continuing.

Problem:

When you created the method, an invalid segment type was entered. The unrecognized segment has been ignored and the run will continue.

Solution:

To correct the problem, perform one of the following actions:

  1. Use the Method Editor to edit the invalid segment in the file. See the online help to determine the temperature limit applicable to the instrument.
  2. Run the edited method file.
