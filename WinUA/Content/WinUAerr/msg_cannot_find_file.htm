Cannot Find File

Cannot find file: <filename>

Do you want to try to find this file yourself?

 

Problem:

This message can occur in the following circumstances:

  • If you have previously saved a session that uses specific data files, which are no longer in the location expected.

  • If you opened a data file in Universal Analysis using the floppy disk drive, then you removed the diskette, so that the file is no longer where it was expected to be.

  • If you opened a file in a directory that was removed during program operation. The program is attempting to write to a file that is longer there.

 

Solution:

  • If you select "Yes," then you can browse and look for the file. This will work if the file was moved to a different location.

  • If you select "No," you must return the source file to its expected location or abort the operation.
