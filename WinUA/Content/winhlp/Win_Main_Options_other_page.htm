Main Options Window - Other Page

Use this window to set up some miscellaneous options that you may want to use as your default settings when you are using the Universal Analysis program.

 

Follow these steps:

  1. Select the direction (orientation) that you want the printer to use each time the program is started for both the plots and reports: Portrait or Landscape. Note that this orientation can be changed during your use of the program, but this setting will be used each time the program is started anew. NOTE: The report orientation applies to the Report Editor program, The Custom Report Editor program, and the Macro Editor.

  2. Choose the Time of Day format you want the program to use for the time data placed in your files.

  3. Choose the Date format that you want the program to use for the date displayed by Universal Analysis.

  4. Select a line type from each drop-down list to apply to the corresponding plot curve. This sets up the Multiline line type option. You can define up to six line types. If more than six cycles exist in the data, the line types are repeated.

  5. Select the desired Method log date-and-time stamp to be displayed at the beginning of each line of the method log.

  6. Check the Confirm file close with unsaved results box, if you want the software to notify you upon exit whether there are any unsaved results.

  7. Check the Enable analysis log box, if you want to enable the capture of the Analysis log for all open data files. Unchecked disables analysis log capture. (NOTE: When logged into the Integrity database this checkbox is grayed out. Analysis log steps are automatically captured.)

  8. Click the Save button, if you want to use these as your new settings or click the OK button to use these settings for this session only.
