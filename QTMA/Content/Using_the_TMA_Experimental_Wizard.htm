Using the TMA Experimental Wizard

 

This is the opening window for the TMA Experimental Wizard that will guide you through the set up of different types of TMA experiments.  These experiments can be performed on TA Instrumentsâ€™ TMA Q400.

TMA (thermomechanical analysis) is a thermal analysis technique that primarily measures dimensional changes in a material with temperature.  Therefore, TMA can measure transition temperatures where such changes occur (e.g., glass transition, softening or melting point), as well as provide quantitative measurement of the dimension change (e.g., coefficient of expansion).  Newer generations of TMA's can also measure the force required to achieve a specific amount of deformation, thereby allowing additional measurements such as stress/strain, isostrain, and stress relaxation to be made.  (Both TMA and DMA can provide these latter measurements.  DMA is more commonly used because a wider range of force is available and the sensitivity for smaller deformations is better.)

Before Using the Wizard

The following TMA calibration procedures should be conducted before using this wizard.

  • Calibrating TMA Cell Constant

  • Calibrating the TMA Temperature

  • Calibrating TMA Force

  • Calibrating the TMA Oscillatory Phase (TMA Q400 EM Option Only)

  • Calibrating the TMA Oscillatory Compliance (TMA Q400 EM Option Only)

To verify the current parameters for calibration, view the Calibration Report.

Starting the Wizard

To set up and run an experiment using the Experimental Wizard, select Experiment/Wizard from the menu or click on the button.

Step 1:  Select a Probe

The first window that is displayed lists the types of probes available for use with the wizard. Select the radio button for the desired probe and click Next to proceed.

See also:  Choosing a Probe

Back to top

Step 2:  Choose a Mode

The next window displays a set of radio buttons to allow you to select the operating mode desired.

Click the radio button for the desired operating mode, then select Next to proceed.

See also:  Selecting a TMA Operating Mode

NOTE: Some modes have several tests to choose from and other modes only have one test. In the situation where there are several tests to choose from, go to Step 3. In the situation where a mode only has one test, you will proceed directly from Step 2 to Step 4.

Step 3:  Select an Experiment

The list of tests (experiments) displayed on the window are dependent upon the operating mode and probe already selected.

Click the radio button for the desired test, then select Next to proceed.

See also:  Available TMA Test Templates

Back to top

Step 4:  Enter the Parameters

Parameters will also vary according to the chosen test. Set up your experiment, then select Next to proceed. For information about the test you have chosen, click the Help button.

Step 5:  Enter the Sample Information

A variety of descriptive information can be stored with an experimental data file and recalled later during plot/report generation.  This window allows you to make those entries. Click here for more information Entering Sample Information Using the Wizard

Step 6:  Enter the Notes and Purge Gas Settings

You can also enter or select additional information to be stored with the selected file. These items cannot be changed once the experiment is in progress. Click here for more information Entering Notes and Gas Settings Using the Wizard

Back to top

Step 7:  Review the Checklist

Once you have completed setting up the experiment, review the checklist and complete some additional steps. Click here for more information Experimental Checklist.

Step 8:  Review the Experimental Summary

The experimental procedure you just created using the Wizard is summarized here. If you would like to edit these parameters, use the Back button to return to the experimental parameters windows or press Next to proceed. Click here for more information Experimental Summary.

Step 9:  Start the Experiment

When all of the steps have been completed you can click the Start button to begin the experiment. Click here for more information Starting a Run and/or Sequence.
