Problem with Autoqueue File

Errors detected while reading autoqueue file. File is corrupted or not compatible with this program.

 

Problem:

There is something wrong with the autoqueue file. The program requires that certain information be present within the file. This information may have been tampered with or deleted, or the file was not created using TA Instruments Thermal Advantage.

 

Solution:

Delete the corrupted file (AutoQ.txt). The Instrument Control software will automatically create a new file when an analysis record is complete and ready to add to the queue. See the Instrument Control online documentation or the User Reference Guide for information.
