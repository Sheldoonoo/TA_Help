SetTTSOptions

Sets the options for future TTS sessions.

Syntax

SetTTSOptions(<DensitySource>, <KeyUnits>, <VShiftType>)

DensitySource
 Density field in sample notes = 0
 User variable 1 value = 1
 User variable 2 value = 2

KeyUnits
 Celsius = 0
 Kelvin = 1

VShiftType
 None = 0
 Temperature correction = 1
 Temperature / density correction = 2
 Manual = 3
