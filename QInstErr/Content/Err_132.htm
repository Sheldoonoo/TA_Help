Message 132

Probe not in the closed position. Run continuing.

Problem:

This is a warning that you left the probe up off the sample and/or that the LVDT is not in range. The run will continue, but you will not get valid readings from the probe.

Solution:

Try one or more of the following:

  1. Lower the probe.
  2. Use the MEASURE function before starting the experiment to ensure proper probe position.

NOTE:  If you pressed the PROBE UP key twice, you will need to lower the probe by pressing the PROBE DOWN key twice so that the LVDT is back in range. Alternatively you can press MEASURE.
