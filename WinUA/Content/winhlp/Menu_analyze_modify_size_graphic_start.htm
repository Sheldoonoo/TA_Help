Graphic Start

Use the Graphic Start option to select a volatile-free start weight using a marker to choose a corresponding X-axis point (temperature or time). The modified sample weight is considered the difference between the chosen point and the stop point.

 

Follow these steps to perform this analysis:

 

1. Select Graphic Start from the Analyze/Modify Size menu. A marker is displayed on the graph.

2. Position the marker at the desired start weight.

3. Right click to display the Analyze pop-up menu, then select Accept Limits (or press Enter). The modified size values are displayed for verification using the Modified Size window.

4. Click the OK button to accept the modified weight or Cancel to exit the analysis. If you accept the results, the program adjusts the percent axis (y-axis) to the modified weight.

 

Any analyses that you perform after modifying the size are based on the new modified size; previous analysis results are unaffected.

 

To restore the original size, select Graph/Data Limits/Full Range. All analyses performed will be lost.
