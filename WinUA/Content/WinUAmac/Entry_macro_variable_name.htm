Entry Variable Name

Use the following guidelines when entering the variable names:

 

 Variables names can only have alphabetic characters, numbers, underline character (underbar), and square brackets (to enclose an optional index which must be a simple variable name or number).

 The first character in a variable name must be alphabetic. Alphabetic characters are not case sensitive (i.e., Capital "A" is the same as lower case "a.")

 The variable name cannot exceed a total of 30 characters.

 Imbedded spaces are not permitted in macro variable names. A macro variable array element is specified by following the variable name with an array index enclosed in square brackets ("[<index>]"). The array index may be an integer constant or a simple variable name (no index).

 

Examples: X, Sum, Tg35, average[i], long_name_with_underbars, NameWithoutUnderbars, array_name[subscript],

array_name[123], data_3[index_2]
