Rename String File Failed

Rename string file failed. Do you still want to transfer the program file?

This message is displayed when using the Tools/Instrument Setup wizard.  Click the Next button until the Transfer Program Files to the Instrument window is displayed. When you update the module software you are actually sending two files to the instrument. The first file contains all the text the instrument needs.  The second is the executable code.  When you see the error message above, the transfer of the text file has failed.  The string file and program files are matched sets. You cannot update just one.

Select No in answer to the message above, then click the Send Programs button again to resend the files. If this message persists, call TA Instruments for service.
