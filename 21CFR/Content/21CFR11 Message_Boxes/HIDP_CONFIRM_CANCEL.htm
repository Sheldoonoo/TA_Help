Continuing Will Delete Objects

The Setup Wizard for the database has created the database objects on the server. If you continue, all of the objects will be removed. Do you want to continue?

This is a confirmation message displayed when the Database Setup Wizard is canceled. The Database Setup Wizard has already created the database objects on the server.

  • Select Yes to continue with the abort operation and remove all of the objects just created.
  • Select No to resume the Database Setup Wizard operation.
