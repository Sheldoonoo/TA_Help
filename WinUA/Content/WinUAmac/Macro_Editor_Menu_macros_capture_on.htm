Capture On/Off

Use the Capture On function to begin (checked) or stop (unchecked) the capture process to build to macro. The option toggles on and off. You can use either the Capture On menu option or the shortcut, Ctrl+T, to toggle the capture function on and off.

 

NOTE: The capture function will capture the end result of the changes made to a dialog. It will not capture the whole dialog.

 

When the capture function is on, a message appears in the lower left corner (status line) of the program to tell you that capturing is active, which macro is currently open, and the currently selected step in the macro.

 

Shortcut Keys

Ctrl+T

 

NOTE: For more details on creating, editing, and running macros, see the Universal Analysis online manual (Help/Manual) or use Acrobat Reader to access the Uamanual.pdf file contained on the CDROM.
