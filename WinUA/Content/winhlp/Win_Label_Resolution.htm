Label Resolution Window

Use this window to set the numeric resolution of an individual result value that is displayed on the plot and in the reports generated by the Universal Analysis program.

 

This window is displayed when you position the pointer over a result label, right click, and select Resolution from the pop-up menu or when you click the Resolution button on a dialog.

 

To set the resolution:

1. Click on the drop-down list to set the resolution for all of the time, temperature, or other values to one of the following criteria:

-- 1 to 7 significant figures

-- 0 to 10 digits after the decimal point

-- rounded to an integer

-- automatic resolution (this is the default)

NOTE: The "Other" option refers toanyandallresult values that are not time or temperature.

2. Click the OK button when finished.

 

If you want to set the resolution for all result values that are generated using the Universal Analysis program, select Analyze/Options from the menu, then display the Analysis Options  Resolution Page.
