Desmear Curve Window

The Desmear Curve window allows you to improve the resolution of DSC sharp melting peaks. When properly applied, this approach can generate data that matches the resolution obtained in DSC Q Series T4P heat flow. The desmearing routine is an offline approach and requires you to adjust a value until the desired curve shape is obtained.

 

To desmear a curve, follow these steps:

 

1. Open the desired DSC data file.

2. Select Tools/Desmear from the menu.

3. Enter the desired Time constant.

4. Enter the desired Derivative window.

5. Check the Refresh data before desmearing box, if you want the data file read again before applying the desmearing function. Otherwise, the desmearing is applied to the current data in memory and is in addition to any previous processing of the data.

6. Click the OK button when finished.
