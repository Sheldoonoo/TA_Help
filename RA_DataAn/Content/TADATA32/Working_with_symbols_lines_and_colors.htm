Working with Symbols, Lines and Colors

 

The software allows you to present data on the graph in a variety of formats. When selecting between the various options, you should bear in mind not only how the graph looks on the computer screen, but how it will appear when used as hard copy, for instance, when photocopied or faxed. You can use a combination of color, line styles and symbols to help in identifying the various data sets on the graph. The software has two levels of identification that can be used:  Trace or File, which are described below.

Distinguishing by Trace

This setting allows you to specify a unique identifier for each trace on the graph. It is useful if there are a small number of traces on the screen and only one rheological variable. This method does not allow you to specify any direct mechanism for distinguishing between traces belonging to different data files or rheological variables.

Tip When dealing with a large number of traces, this method is likely to produce a graph that is difficult to follow.

Distinguishing by File

This method allows for far greater control of the graph, although it takes more setting up. You can select between line styles, colours and symbols and apply these to the file, the axis (rheological variable) and the step (in the procedure.)

Overheads

Follow the steps below to configure the settings best used for overheads:

  1. Select Graph from the menu.
  2. Click on the Symbols tab.

  3. Select the Distinguish option to By trace.
  4. Select the Colour style.

Comparison of One Variable in Multiple Data Sets

Follow the steps below to configure the settings needed to display one variable in multiple data sets:

    1. Select Graph from the menu.
    2. Click on the Symbols tab.

    3. Select the Distinguish option to By file.
    4. Select Symbols for the File entry.
    5. Select Colour for the Axis entry.
    6. Select Extra symbols for the Step entry.
