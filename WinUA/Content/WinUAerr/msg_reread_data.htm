Reread the Data?

Changing the sample size, exotherm type or cell constant will cause the data to be reread from the data file. Any unsaved changes or results will be lost. Are you sure you want to reread the data?

 

Problem:

This message is displayed if you select Edit/File Information and change the information relating to the data file. Any changes or results that have not been saved will be lost because the file will be reread.

 

Solution:

Select OK to proceed. The data file will be reread, the changes made, and any unsaved changes or results will be lost.

Select Cancel to abort the operation. You can now save your changes or results, if desired.
