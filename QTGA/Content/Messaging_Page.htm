Using the Messaging Page

(A Platinum™ Feature)

 

Email and Windows® Messaging are built-in Platinum features. This feature provides a mechanism for the controller software to communicate with the operator based on subscribed events.

Configuring & Testing Platinum Mail & Messaging

Proper installation and setup are required for these functions to work properly. To configure and test the Platinum mail and messaging features you will need to perform the following steps in the order shown:

(1) Establishing a Database Connection for Mail Messaging

(2) Configuring Windows® for Mail and Messaging

(3) Configuring Platinum Mail and Messaging Functions

(4) Testing Platinum Mail and Messaging Functions

To find out how to perform steps 1 through 4, you can open the appropriate Acrobat Reader PDF file by clicking here Configuring Platinum Mail & Messaging.

Once the Platinum Messaging has been properly installed and setup, you can begin to use this feature to send emails and notification messages directly from the instrument control program. Messaging is available for the following tasks:  individual runs, sequences (experimental and calibration), auto diagnostics, auto weight calibration.

Setting Up Email and Profile Settings

When you install the Advantage software (V4.3 or higher) you will have the option of installing the Platinum Mail & Messaging Database option.  This component creates a localized database on your system to manage all messaging requests. Follow the instructions below to verify the email program and profiles that will be used with the messaging feature:

  1. Open the instrument control program for instrument that will have this feature. See Using the TA Instrument Explorer for instructions.

  2. Select the Platinum button at the bottom left-hand side of the main instrument control window.

  3. Click E-mail Notification under the Messaging group to display the window.

  4. Check the Enable TA Messaging Service for the Connected Instrument to have the instrument that you are currently running use this service.

  5. Verify the following TA Messenger Settings on the Primary TAMA DB computer:

    • Primary TAMA (TA Message Agent) DB (Database) – Displays name of computer containing TAMA DB for this function. To change or set up a database, see Configuring & Testing Platinum Mail & Messaging.

    • Mail Service Provider – Displays the default Mail Service Provider on the Primary TAMA DB computer. (Outlook or Outlook Express is required. See also: Configuring Outlook Express)

    • Default Profile – Displays the default profile that will be used on the Primary TAMA DB computer.

  6. Verify the following TA Messenger Settings on the Local Computer (if different).  The Mail Service Provider and Profile should match.

    • MS Messenger - Displays the status of this service. The service must be running to use the Windows Messaging features within Advantage Platinum.  To set up this service, see Configuring & Testing Platinum Mail & Messaging.

    • Mail Service Provider – Displays the default Mail Service Provider on this computer. (Outlook or Outlook Express is required. See also: Configuring Outlook Express) To change these settings you will need to change them on your computer.

  7. Click the Profile button to change the Default Profile name. See Choosing a Mail Profile.

  8. Click the Address Book button to specify a default address when using this option. Alternatively, you can enter the name. Click the Test button to verify this entry.

  9. Click the Net Hosts button to specify a default computer host name when using the Windows Messaging option. Alternatively, you can enter the computer name. Click the Test button to verify entry.

  10. Click Apply to save and use these settings or Cancel to exit without saving the settings.

Accessing Task-Specific Messaging Pages

Follow the instructions listed for each type of task to gain access to the messaging function.

  • Run Messaging:  Click on the desired run number from the run sequence list. Then select the Messaging page.

  • Sequence (Experimental and Calibration) Messaging*:  Click the Experiment or Calibration button at the bottom left of the instrument control window.  Then click on Sequence No. # at the top of the run sequence list to open the Scheduling, Summary, and Messaging pages.

  • Auto Diagnostics Messaging*:  Click the Platinum button at the bottom left of the instrument control window.  Then click on Auto Diagnostics from the Automatic Tasks list to open Auto Diagnostics, Report,  Scheduling, and Messaging pages.

*Accessing the Central Messaging Page

You can also access the messaging options for the specific tasks (marked with an asterisk [*] above) at a central location. To access these options:

  1. Click the Platinum button at the bottom left of the instrument control window.

  2. Select Settings from the Scheduling option on the left-hand side.

  3. Select the Messaging page. (If the Messaging tab is not displayed then this option has not been enabled or setup completely. Consult the installation and setup topics.)

  4. Select the desired task from the list. Then set up or verify the messaging options as directed below.

Setting Up Messaging Options

To use the Messaging Page, follow these directions:

(If the Messaging tab is not displayed then this option has not been enabled or setup completely.  Consult the installation and setup topics.)

  1. Select the Subscribed Events that you wish to received notification (list will vary based on task selected):

    • Data file receiving complete:  This event will send notification when the experiment has been completed.

    • Run has been rejected: This event will send notification if a run has been rejected.

    • Data File Acceptor error occurred: This event will send notification if the software encounters a data file error condition during the course of the experiment sequence.

    • Task Started:  This event will send notification when the task (sequence, diagnostics, &/or weight calibration sequence) has been started.

    • Task Stopped/Rejected: This event will send notification if the task (sequence, diagnostics, &/or weight calibration sequence) is stopped or rejected.

    • Task Complete: This event will send notification when the task (sequence, diagnostics, &/or weight calibration sequence) has completed.

NOTE: The following messaging options (e.g., comments, email to, attachments, message to) are unique for each subscribed events, unless Apply settings to all checked events is selected.

  1. Enter any Additional Comments to be included in the message.  

  2. Click on the E-mail To button to select the desired addresses for this option, if email notification is desired.  Alternatively, you can enter these addresses (separate each address by a semi colon).

  3. Click on the Attachments button to specify any items, which you wish to include in the email message.  Options will vary based on selected task.

  4. Click on the Message To button to select the desired computer for this option, if Windows Messaging notification is desired. Alternatively, you can enter a computer name(s).

  5. Check the Apply settings to all checked events box, if you wish to apply all of these settings for all the subscribed events selected.

  6. Check the Apply settings to all runs box, if you wish to apply these messaging settings to all the runs in the sequence.

  7. Click Apply when finished.
