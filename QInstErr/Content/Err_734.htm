Message 734

Quench Cooling Accessory (QCA) can be used only to determine Heat Flow T1. Cannot start run.

Problem:

This message is displayed when a run is started with heat flow T4 or T4P as the selected heat flow, but a Quench Cooling Accessory (QCA) is the selected cooling accessory.  T4 and T4P cannot be used with the QCA, you must use either the FACS, RCS, or LNCS with those heat flow settings.

Solution:

  • Switch to a different cooler (FACS, RCS, or LNCS) that is compatible with the heat flow T4 or T4P.

or

  • Change the heat flow selection to T1 to be compatible with the QCA.
