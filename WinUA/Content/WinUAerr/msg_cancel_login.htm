You have unsaved results. Cancel login and close all files?

 

Problem:

This message is displayed when a login function is canceled in a 21 CFR 11 Universal Analysis session that has had inactivity for a defined period of time, or when you log out of a 21 CFR 11 Universal Analysis session and one or more data files are still open.

 

Solution:

Select OK to close all open files and close the connection to the database.

Select Cancel to log into the database again without losing any results or closing any files.
