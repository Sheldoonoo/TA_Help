Checkbox Restore Saved Analysis

Use this option to return the data file back to the state it was in when Save Analysis was selected.

 

If this checkbox is grayed (disabled) then there is no saved analysis information stored in the data file. In this case, the file will be opened normally as if the checkbox was not checked.

 

You can set the default state of this checkbox on the Main Options - Data Page. If the default state of the checkbox is set to restore analysis, then any Quick Open or multiple file selection will be immediately restored. The default state is checked.
