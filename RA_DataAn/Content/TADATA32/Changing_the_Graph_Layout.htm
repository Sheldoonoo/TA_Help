Adding Graph Labels

Follow the steps below to add a label to your graph:

  1. Right click on the active graph.
  2. Select Add label...
  3. Type in the Text you wish to appear for this label.
  4. Select the Justification required (left, right or centred) for the text.
  5. Select the Border Style and Border Colour... required.

    Borders:

  6. Select the font by pressing the Change font... button.
  7. Press the OK button.

NOTE:  To change an existing label, right click on it and select Properties.

Adding a Key

Follow the steps below to add a key to your graph:

  1. Ensure that you have an active graph screen.
  2. Click on Graph.
  3. Select Key.
  4. Ensure that the Displayed item is selected .
  5. Make any changes required to the appearance of the key.
  6. Click the OK button.

Log / Linear Selection

Follow the steps below to change between log or linear options.

  1. Ensure that you have an active graph screen.
  2. Click on Graph.
  3. Select Lin/Log.
  4. Check each axis you wish to use log scales.
  5. Click on the Set to defaults button, to use the available default options.
  6. Click the OK button.

NOTE:  You can also right click on the graph over an axis to switch it between log and linear.
