Message 727

Experiment temperature range exceeds Tzero calibration range. Run continuing.

Problem:

The experiment temperature range specified for this experiment (done with the FACS cooling accessory) exceeds the Tzero calibration range. The run will continue but the results may not be as expected because the data taken outside the calibration range will be less accurate than that inside the calibration range.

Solution:

All experiments should be run within the valid Tzero calibration temperature range. Calibrate the instrument again, expanding the temperature range if necessary.
