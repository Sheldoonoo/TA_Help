Setting Up Email Notification

(A Platinumâ„¢ Feature)

Follow the instructions below to set up the email program and profiles that will be used with the messaging feature:

  1. Open the instrument control program for instrument that will have this feature enabled on it. See Using the TA Instrument Explorer for instructions.
  2. Select the Platinum button at the bottom left-hand side of the main instrument control window.
  3. Click E-mail Notification under the Messaging group to display the window.
  4. Check the Enable TA Messaging Service for the Connected Instrument to enable mail and messaging for this instrument.
  5. Verify that the TA Messenger Settings displayed on the window are correct.
    • If there is no Primary TAMA DB shown, establish a database by following the instructions found in the Adobe Acrobat PDF document, "Configuring & Testing Platinum Mail & Messaging." Click here Configuring Platinum Mail & Messaging.
    • If the Mail Service Provider is not correct, right click on the Internet Explorer icon on your desktop then select Properties from the menu and select the Programs tab. Select the desired Email handler and from the drop-down list and click OK. Click the Refresh button on upper right side of the Platinum Email Notification window and verify the changes.
    • If the Default Profile is not correct, open Outlook Express, select Tools/Accounts from the menu. Then select the Mail tab and set the desired profile to Default and click OK to exit the window.  Click the Refresh button on the upper right side of the Platinum Email Notification window and verify the changes.
  6. Verify that the Local Computer Settings displayed on the window are correct.
    • If the MS Messenger service is desired and the service is stopped or disabled, open a Command Prompt and type SERVICES.MSC to open the Services window. Scroll down to "Messenger" on the right-hand side of the window and double click on it to open the Messenger Properties window. Click the Start button then set the Startup type to "Automatic" and click OK to exit the window. Close all open windows.  Click the Refresh button on the upper right side of the Platinum Email Notification window and verify the changes.
  7. Use the Address Book and Net Host buttons to select the email recipients and computer names which will receive test messages. You can also type the email addresses and host names in the fields, if desired. Separate multiple entries with a semicolon (;).
  8. Press the Apply button to save the entries as the defaults, if desired, or Cancel to exit without saving the settings.
