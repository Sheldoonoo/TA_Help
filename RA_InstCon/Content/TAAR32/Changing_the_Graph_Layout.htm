Adding Graph Labels

Follow the steps below to add labels to the graph:

  1. Right click on the active graph.
  2. Select Add label…
  3. Type in the Text you wish to appear for this label.
  4. Select the Justification required (left, right or centred) for the text.
  5. Select the Border Style and Border Colour… required.
  6. Select the font by pressing the Change font… button.
  7. Press the OK button.

NOTE:  To change an existing label, right click on it and select Properties..

Displaying the Graph Key

Follow the steps below to display the graph key:

  1. Select Experiment from the menu.
  2. Click on Graph.
  3. Select Properties.
  4. Click on the Key tab.
  5. Ensure that the Displayed item is selected.
  6. Make any changes required to the appearance of the key.
  7. Click the OK button.

Displaying a Title on the Graph

Follow the steps below to display a title on your graph:

  1. Select Experiment from the menu.
  2. Click on Graph.
  3. Select Properties.
  4. Click on the Title tab.
  5. Ensure that the Displayed item is selected.
  6. Make any changes required to the appearance of the title.
  7. Click the OK button.
