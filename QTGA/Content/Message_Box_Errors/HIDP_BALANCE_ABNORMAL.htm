Balance Abnormal

The balance is in ___ state. Make sure you have put correct weight on the sample pan. Do you really want to continue the next step?

This message is displayed during weight calibration. It indicates that the balance is not in the state expected at this point in the procedure. Please check to make sure that the pan is emptied or that the correct weight for the current step is placed on the sample pan before continuing. If the procedure is continued with the incorrect weight, the calibration will not be accurate.
