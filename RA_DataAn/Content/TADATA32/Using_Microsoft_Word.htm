Using Microsoft ® Word

 

To create a custom report from scratch, use the following steps:

  1. Start Microsoft ® Word.
  2. Select File on the menu.
  3. Click on New.
  4. Click on the TA Advantage tab.
  5. Select Custom Report.DOT.
  6. Click on the OK button.
  7. This will load custom report information. You can print out this document to review the information it contains on creating your own custom report.
  8. Edit the document until it is in the format you require.

NOTE: You will often see small text boxes to the left of the tables. These are used by Data analysis and should not be deleted. They will not show up in the final report.

  1. Save this document in a suitable location (it must be accessible to everyone who will use it in the Rheology Advantage Data module.)
  2. Once you have saved it, select Insert on the Word menu.
  3. Click on Add to TA template list.
  4. Exit from Microsoft ® Word.
  5. When you start the Rheology Advantage Data analysis package, the new report will be displayed.

NOTE: You must select Custom reports on the View tab under Options.
