Checkbox No Invalid Data Points

Check this box to remove any invalid data points that are present in the data file from the exported data file. These invalid points will be replaced with a single data break for each group of invalid points. NOTE: If the optionNo data breaks or flagsis checked along with this checkbox, no data break will be inserted in place of the invalid points in the exported data file.

 

Leave this box unchecked (default condition) to export your data file as it is, with all invalid data points included.
