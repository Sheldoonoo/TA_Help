Checkbox No Data Breaks or Flags

Check this box to export your data file without any data break or data flags such as cycle markers, gas on/off changes, or event on/off changes. This allows you to remove data breaks that may hinder your use of the data in spreadsheet applications. When this item is checked, the minus time flags are removed from the data and all points appear connected. Note that if you check this option, the data breaks cannot be restored to the exported file.

 

Leave this box unchecked (default condition) to export your data file as it is, with all data breaks or flags intact.
