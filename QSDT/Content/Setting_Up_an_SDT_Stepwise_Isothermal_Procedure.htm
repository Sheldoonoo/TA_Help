Setting Up an SDT Stepwise Isothermal Procedure

A stepwise isothermal experiment can improve weight change resolution by holding isothermally once a weight change has started.  Because weight changes are time-dependent as well as temperature-dependent, this approach, by allowing more time for a specific weight change to be completed, reduces overlap with subsequently higher temperature events.

If you want to set up an SDT stepwise isothermal procedure, you can choose one of two techniques—the SDT Wizard or the Experimental View - Procedure Page. See the instructions below for each technique.

When you perform an SDT Stepwise Isothermal experiment, the following parameters need to be chosen:

  • Start and Final Temperatures:  The temperature range of the SDT is ambient to 1500°C.  The actual start and final temperatures chosen for a specific experiment must be within that range. You can choose to start your experiments from one of the following choices:

    1. Use Current—the temperature of the SDT at the current moment (usually ambient temperature, 35°C). In this case, the ramped heating begins immediately on starting the experiment. To use this option, check the “Use Current” box.

    2. Another Temperature—you can choose the specific temperature desired (default is 50°C). In this case, the system equilibrates at that temperature before ramped heating begins. To use this option, uncheck the “Use Current” box, then enter a new start temperature.

  • Heating Rate:  The acceptable range of SDT heating rates is 0.1 to 100°C/minute.  10 to 20 °C/minute are generally used for stepwise isothermal experiments.
  • Enter and Exit Isothermal When:  The key to obtaining good results with stepwise isothermal experiments is to properly choose the rate of weight changes used to enter and exit the isothermal conditions.  
    1. Enter isothermal when rate of weight change exceeds—the entry rate of weight change should be large enough to respond only when a "real" event is occurring. The stepwise isothermal entry rate of weight change is selected to be 10 times as large as the exit rate but should not be more than 50% of the maximum rate of weight change in the SDT derivative weight change curve.  
    2. Exit isothermal when rate of weight changes falls below—the exit rate of weight change should be small enough to ensure that the event is actually complete. The stepwise isothermal exit rate of weight change is selected to be equal to the baseline rate of weight change encountered during the onset of transitions of interest in the experiment.

These values are usually chosen by running an experiment and evaluating the weight change derivative curve.  Good default values to use in lieu of a preliminary a experiment are 0.5 (entry) and 0.05 (exit).

  • Additional Conditions: There are additional experimental conditions that can be changed to optimize specific situations. These are accessed using Advanced Parameters and Post-Test Conditions. For most experiments, the system default values shown are recommended.

Using the SDT Wizard :

Enter the desired parameters, then select Next to proceed.

Using the Experimental View Procedure Page:

  1. Display the Procedure Page. Select the desired experiment from the list of available preprogrammed templates.

  2. Enter the desired parameters. When you have finished setting up this procedure, you can enter more information by selecting the Notes Page and Summary Page for this run.

  3. Click the Apply button when finished to save these changes or the Append  button to save the changes to this run and add another run to the end of the sequence.

NOTE: You can use any test template as the basis for a custom test by selecting the desired test template from the list of tests, then click the Apply button and change the test to Custom. Then you can click on the Procedure Page and then the Editor button to make the changes desired.
