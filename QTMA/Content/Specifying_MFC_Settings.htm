Specifying MFC Settings

 

To set up the purge gases used by the mass flow controllers (MFC), select Mass Flow Control from the Calibrate menu and follow the instructions below:

Nitrogen Table

This table is used to enter the calibration values for nitrogen, oxygen, air, and argon. A factor will be applied against the nitrogen table if oxygen, air, or argon is selected as the purge gas.

  1. Click on the Nitrogen Table tab.
  2. Click the Reset button to set all of the values to 0.00.
  3. Enter the calibration values for the Requested Flow and Measured Flow.
  4. Click Set Values.

The Restore Values button restores the table with  the factory calibration values set at TA Instruments.

Helium Table

This table is used to enter the calibration values for helium.

  1. Click on the Helium Table tab.
  2. Click the Reset button to set all of the values to 0.00.
  3. Enter the calibration values for the Requested Flow and Measured Flow.
  4. Click Set Values.

The Restore Values button restores the table with the factory calibration values set at TA Instruments.
