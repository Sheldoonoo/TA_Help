Discrete Relaxation Spectrum - Stress Relaxation Comparison

 

Follow the steps below:

  1. Right click on the file PDMS Extended frequency sweep-0001o-spx.rsl (in the Results list window) and select Stress relaxation… from the pop-up window. Accept the default values. A stress relaxation file is generated.
  2. Send the stress relaxation file (PDMS Extended frequency sweep-0001o-spx-x.rsl) to a new graph, then load the file:
    • PDMS Stress relaxation-0001x.rsl
  3. Send this to the same graph as the generated stress relaxation curve to compare the transformed data to the generated curve.
