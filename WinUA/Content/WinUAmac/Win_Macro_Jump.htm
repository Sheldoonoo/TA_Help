Macro Jump Window

Use the Macro Jump window to move to a different step within the macro as the macro runs.

 

To insert a jump step into a macro follow these steps:

 

1. Open the desired macro.

2. Select the position that you want to insert a jump step into.

3. Select Jump from the Insert menu displayed on the Macro Editor window.

4. Select the desired Jump To option from the drop-down list.

5. Enter the desired step that the macro will jump to, when it is run.

6. Select OK when completed. The jump will be inserted into the macro at the point selected.
