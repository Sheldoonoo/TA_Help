Installing the Thermocouple and Clamp Shields

 

When starting experiments at subambient temperatures using the GCA it is highly recommended that you install both the DMA thermocouple shield (shown here on the right) and the appropriate large clamp shield (see the figures below). The purpose of these devices is to reduce the radiant heating effects on the thermocouple.

   

 

NOTE:  The use of these devices, however, is NOT recommended for experiments that start at or above room temperature.

First Install the Thermocouple Shield

Follow these instructions to install the thermocouple shield. Note that this shield can be installed with or without a clamp already installed.

  1. Locate the thermocouple shield and the .050 hex key (Allen wrench) in the GCA Accessory Kit.

  2. Slide the thermocouple shield over the sample thermocouple (the right side thermocouple) as shown here with a clamp already installed, then position the shield so that the tip of the thermocouple is midway between the top and bottom of the screen.

  3. Tighten the setscrew with the .050 hex key to hold the shield in position.

  4. Position the thermocouple with the installed shield as close to the sample as possible without touching the clamp or the sample.

 

The figure to the right shows the thermocouple shield installed on the sample thermocouple (no clamp installed).

Then Install the Large Clamp Shield

After the thermocouple shield, clamp, and sample have been installed, follow this step:

  1. Gently slide the appropriate large clamp shield over the entire clamp assembly prior to starting the run.
