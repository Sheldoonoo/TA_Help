Specifying a Test Procedure

When running experiments on your instruments you will need to specify a test procedure. A procedure contains all of the information regarding a test protocol (see below). The sample and calibration information are not part of a procedure.

A procedure file (.prc) contains the following information:

  • Instrument
  • Mode
  • Saved Signals
  • Test Name
  • Procedure Notes
  • Method Name
  • Method Segments
  • Advanced Parameters
  • Post Test Parameters

You can either select one of the preprogrammed test templates or select "Custom" to create your own procedure using the Method Editor. The list of templates is based on frequently used experimental methods.

  • To view a summary of the current procedure for this run, open the Procedure Summary located on the Summary Page.
  • To view a summary of the parameters contained in a saved procedure file, access the File Utility window. Select all procedure files to view by entering *.prc as the file type of this window.
