Radio Group Action

Select one of the radio buttons displayed on the window listed below:

 

Macro Edit Clipboard window:

  • Select Clear clipboard before assigning edit text to replace any text currently on the clipboard with the entered text.

  • Select Append edit text to existing clipboard text to append the entered text to any text already on the clipboard.

 

Macro Assign Clipboard window:

  • Select Clear clipboard before assigning value to replace any text currently on the clipboard with the entered value.

  • Select Append value to existing clipboard text to append the entered value to any text already on the clipboard.
