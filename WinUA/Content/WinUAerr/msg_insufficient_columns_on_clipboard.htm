Insufficient Columns

 

Insufficient columns on clipboard.

 

This message is displayed when you attempt to paste an insufficient number of columns from the clipboard into a larger number of highlighted columns in the spreadsheet. For example, if you copy one column of data to the clipboard and attempt to paste it into two highlighted columns on the spreadsheet, you will get this message.

 

When more than one cell is highlighted in the spreadsheet, there must be a one-to-one match between the columns on the clipboard and the columns highlighted for pasting in the spreadsheet. If only one cell is highlighted in the spreadsheet, then the columns and rows from the clipboard will be pasted in as required starting with the column and row of the highlighted cell.
