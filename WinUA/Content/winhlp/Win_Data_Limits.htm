Data Limits Window

Use the Data Limits window to select from three options. These allow you to use the entire data file for analysis, or to place constraints on the amount and type of data.

 

1. Click on the radio button for the desired limits.

2. Click OK. If you select Full Range, the entire data file is plotted. If you select either Time, Temperature, or Cycle Limits, you will then enter the limits desired on the next window displayed.
