Unexpected Filename in Macro

Unexpected file name in macro.

 

Problem:

When the macro was run, it encountered a filename input step where filename input was not active. Macro execution was stopped.

 

Solution:

The filename insertion step is out of sequence due to:

 

1. Deletion of other macro steps.

2. A jump to the wrong macro step.

3. A change in the program state since the original steps were captured.

 

Recheck the macro steps leading up to the filename step to determine if the macro steps are in the correct sequence. See also:Debugging Macro.
