Choosing a Mail Profile

 

When setting up your Messaging options, you will need to have email profiles to use. If your email program is Microsoft® Outlook, you will need to run that program to set up a new profile.

Open Outlook Express, select Tools/Accounts from the menu. Then select the Mail tab and set the desired profile to Default and click OK to exit the window.  Click the Refresh button on the upper right side of the Platinum™ Email Notification window and verify the changes.

If the profiles have already been set up, select one from the drop-down list and click OK.
