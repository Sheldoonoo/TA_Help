Add More File Information?

 

Do you want to add temperature, weight, and parameter information from a currently open data file?

 

This option permits extraction of parameter, sample temperature, and weight information from another data file that is already open in Universal Analysis. The extracted information will be added to the converted data file. The extracted parameters are Sample, Size, Operator, Method, and Comment. The Time of each data point from the converted data file is used to interpolate a corresponding Temperature and Weight signal value from the already open Universal Analysis data file.

 

Typically, the parameter, temperature, and weight information is extracted from a TGA, SDT or DSC-TGA data file that was created simultaneously with the converted data file. The Time values in the open data file should correspond as closely as possible to the time values in the converted data file. If these values do not correspond exactly, then Linear Transform can be used to add or subtract an appropriate offset to the Time signal in the open data file before the data file is converted.
