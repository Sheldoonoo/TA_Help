New File (Macro File)

Select New File to create a new macro file. When you select this function, a new, empty macro file is set up in memory by deleting all currently-loaded macros. The Save As window is displayed. Enter a new name for your macro file, then click the Save button. You will then be ready to create new macros to be stored in this new macro file.

 

NOTE: When you select New File and give the macro file a new name, the program essentially deletes the files in the current macro and retains the new filename, but does not actually write this file to the drive. You must use Save or Save As to actually create the new macro file.
