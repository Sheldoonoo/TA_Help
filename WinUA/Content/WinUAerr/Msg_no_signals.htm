No Signals Selected

No signals selected.

 

Problem:

This error can occur for one of the following reasons:

1. Before a curve overlay graph can be created, you must select signals to plot.

2. Before you can export a data files plot or file signals, you must select signals to export.

3. Before you can perform a baseline subtraction or addition operation, using the Tools/Baseline menu, you must have selected signals from the data file being used.

 

Solution:

Select Graph/Overlay/Manual Setup, then click on the Curves Page. Click on the Add Curves button and select files and signals to use for the curve overlay.
