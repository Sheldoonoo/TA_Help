Available DMA Clamps

 

The DMA utilizes several different types of clamps. These clamps can be classified as either tensioning or nontensioning clamps. Tensioning clamps require that a positive force (preload force) be placed on the sample at all times.

Tensioning clamps are:

  • 3-point bending

  • 3-point bending submersion

  • film tension

  • dual surface film tension

  • fiber tension

  • compression

  • penetration

  • submersion compression

  • submersion film/fiber

Nontensioning clamps are:

  • single cantilever

  • dual cantilever

  • shear sandwich

  • powder clamp

Choosing a Clamp Set

The shape and the modulus of most samples determine the clamp to be used for your experiment.  Choosing the clamp set is obvious for some types of samples (films, for example).  For others, however, there are several possible clamp sets (filled thermoplastic bars, for example).  At times there can be more than one set of conditions that are necessary to make measurements on a single material over a broad temperature range.  The clamping fixtures with usual material types are shown in the table below.

Once the appropriate clamp set is chosen, the DMA must be programmed to reflect the choice of clamps so that the instrument can properly control the experiment and report the correct sample moduli.  The  clamp type is selected from the Instrument Control software.

Additional Clamps

There are several additional clamps available which are subtle modifications of the basic clamps listed above. They include:

  • Film/Fiber Submersion:  A modified version of the standard Tension Film clamp which allows viscoelastic measurements to be made while the sample material is submersed in an ambient temperature fluid (e.g., water, buffers, automotive fluids). Click here for detailed information more.

  • Compression Submersion:  A modified version of the standard Compression clamp which allows viscoelastic measurements to be made while the sample material is submersed in an ambient temperature fluid (e.g., water, buffers, automotive fluids). Click here for detailed information more.
