Message 820

Manufacturing calibrations reinitialized.

This message is displayed after you reset the instrument to use the default calibration that was set at TA Instruments.

To reset the parameters follow these instructions:

  1. Verify that both the instrument and the controller are both powered on and connected to the supplied Ethernet Hub or LAN.
  2. Select Tools\Instrument Setup from the main menu to start the Instrument Setup Wizard.
  3. Click the Next button until the Transfer Program Files to the Instrument window is displayed.
  4. Click the Reset Parameters button. The following warning message is then posted asking if you wish to continue with the resetting the parameters, click OK to proceed. The parameters will now be reset.
  5. Wait ten seconds, then press the reset button on the back of the instrument to complete this process.
  6. Click the Cancel button to exit the wizard or select Next to advance to the Instrument Settings window, which is used to configure the instrument's network settings.
  7. Verify that all of the reset parameter settings have the correct selections/values. Change the settings as required.
