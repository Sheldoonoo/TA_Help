Checkbox Autoscale New Data Limits Y Axis

Check the Autoscale New Data Y-Axis box to have the data files y axis autoscaled. When checked the current files y-axis data will be autoscaled when the program reads in new data for Graph/Data Limits, Tools/Linear Transform, or a new Graph/Y axis. This is the default setting. (This applies to single files onlyit does not apply to curve overlay.)

 

Leave this box unchecked and the y-axis data will not be autoscaled (y-axis scaling is preserved).
