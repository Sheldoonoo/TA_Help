Message 883

Balance N2 flow calibration saved:  xx points

This is a status message recording changes to the balance mass flow controller (MFC) for nitrogen.

Under normal circumstances the mass flow controller (MFC) should not need to be calibrated by you, the customer. Calibration is done at TA Instruments for you. These calibration tables can be viewed by accessing Calibrate/MFC Balance. If you need to return the calibration values to their factory settings for any reason, click the Restore Values button and the original calibration values will be restored.

If you decide that you need to calibrate the MFC, follow the calibration instructions found in the online help or call TA Instruments for service. An external high-precision flow meter is required to calibrate for both nitrogen and helium. (Two separate flow meters may be needed, one calibrated for nitrogen and the other for helium.)
