Getting Assistance While You Work

Screen Tips

Screen Tips show information about different elements on the screen. To see a Screen Tip for a dialog box option, click the question mark and then click the option. (If the dialog box doesn't have a question mark button, select the option and then press the F1 key.)

Tool Tip Help

Position the cursor over a button to see the tool tip or flyover help available for that control.

Contents and Index

You can get Help the usual way. Just click Contents and Index on the Help menu. Click the Contents tab to scroll through a table of contents for the Help file. Click the Index tab to search for topics by using an index of Help subjects. Click on the Find tab to use full-text search and look for specific words or phrases.
