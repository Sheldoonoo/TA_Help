Quench Cooler for Heat Flow T1 Only

Quench Cooler can be used to determine Heat Flow T1 only.

This message is displayed if you have selected Quench Cooler as your cooler type and either Heat Flow T4P or Heat Flow T4 as your heat flow selection on the Instrument Preferences/DSC Page. The Quench Cooler can only be used with Heat Flow T1.

You need to either change the heat flow selection or change the cooler type so that a compatible cooler is used with the heat flow you choose.
