Paste

Use this procedure to retrieve information from the clipboard to the spreadsheet view.

 

To paste:

 

1. Select (highlight) the data and copy it to the clipboard using the Copy function.

2. Open the spreadsheet view by selecting Graph/Spreadsheet View from the main menu.

3. Click to select (highlight) the desired cell.

4. Select Edit/Paste to retrieve the numbers from the clipboard.

 

Short cut:

Ctrl-V
