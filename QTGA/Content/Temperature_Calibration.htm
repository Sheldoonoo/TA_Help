Temperature Calibrating the TGA

 

Temperature calibration is useful for experiments in which precise transition temperatures are essential. To temperature calibrate the TGA, you need to analyze a high-purity magnetic standard for its curie temperature, and then enter the observed and correct values in the temperature calibration table.

Determining Curie Temperature

There are two techniques for TGA temperature calibration recognized by the American Society for Testing and Materials. These techniques are described in ASTM Standards E914-83 and E1582-93. The latter, which is based on the Curie Point of paramagnetic metals, is recommended for the TA Instruments TGA's.

In this technique, a nickel Curie standard is heated in an alumina sample pan. A magnet is placed outside the TGA furnace so that it is just below the sample pan. As the nickel goes through its Curie Point, its attraction to the magnet changes, appearing as a weight loss. The onset of which is adjusted to agree with the material's known Curie Point Temperature.

A general procedure for curie temperature calibration is given below.

  1. Select Calibrate/Temperature Table from the TGA menu. The Temperature Table is displayed. Reset all the temperature calibration data before conducting any new temperature experiments. Click the Reset button, then OK.

  2. Choose standards that encompass your experimental range. Multiple standards may be needed for this purpose. Choose standards whose curie temperature differs substantially; each pair of curie temperatures must differ from all other pairs by at least 10°C.

  3. Make sure that the TGA Autosampler, if installed, has been disabled for this experiment. Verify that Control/Autosampler/Enabled is not checked. If it is checked, click on the menu item to uncheck (disable) the option.

  4. Place the curie temperature sample in a tared TGA sample pan and position the pan on the sample platform.

  5. Select the Experiment View. Enter the requested sample information, including name on the Summary Page. Select the Ramp test from the Test list.

  6. Click on the Procedure tab. Enter the requested test parameters that will program the TGA to: (a) equilibrate to 100°C below the onset of the literature Curie temperature of the material, and (b) heat the material, at the same heating rate that you will use in your subsequent experiments, to above the literature Curie temperature.

CAUTION: Be sure to select "Furnace Closed" as your Post-Test Condition when the magnet is positioned under the TGA furnace. If the Autosampler is enabled, you must select "Leave furnace closed" as the sequence stop option on the Instrument Preferences/Autosampler Page.

  1. Click on the Notes tab. Enter/verify the requested information.

  2. Click on the Apply button to save the experimental and sample parameters entered for this run.

  3. If more than one run is in the sequence list, schedule this run. (An arrow will appear next to the run number in the Sequence Pane for the scheduled run.)

  4. Select Start to begin the calibration run. Observe Signal A (weight loss %) from Signal Display.

  5. Slowly raise the magnet under the furnace until a weight gain (~ 2%) is detected. Secure the magnet is this position for the duration of the experiment.

  6. Determine the curie temperature by analyzing the extrapolated end point of the S-shaped curve using the data analysis program.

  7. Repeat steps 4 through 12 for each temperature calibration standard.

  8. Select Calibrate/Temperature Table again. Enter the Observed and Correct temperatures in the Calibration table when all the experiments are complete, following the directions in the next section.

Analyzing Q500/Q50 Data Files for Curie Temperature

The curie temperature corresponds to the extrapolated endpoint on the “S” shaped thermal curve. This point can be determined using the Universal Analysis program.   Make hard copy printouts of all plots for record-keeping purposes.

  1. Start Universal Analysis.

  2. Analyze the scan and determine the curie point using the desired data file.

    1. Set up the analysis parameters to use manual transition limits (4 points).  

    2. Perform an onset point analysis.  Determine the  curie temperature by analyzing the extrapolated end point (see the figure below). Position two markers on the flat portion of the downward sloping line to form one tangent. Place the two remaining markers on the flat baseline portion after the transition to form the other tangent. The intersection of the two tangent lines—the extrapolated endpoint—corresponds to the  curie temperature.

Entering Temperature Calibration Data

Select Calibrate/Temperature Calibration to access the Temperature Calibration Table, which is used to input or reset temperature calibration data.

This window shows the temperature calibration table that the instrument applies to the collected data. Use this window to enter from one to five temperature calibration points (pairs of observed and correct temperature points). The observed and correct temperature corresponds to the experimental and theoretical transition temperature (e.g., melting point) of the calibrant respectively.

NOTE: A multiple-point calibration is more accurate than a one-point calibration. A single-point calibration shifts the sample temperature by a constant amount. A two- or more-point calibration shifts the temperature by a constant amount below the first point, uses a smooth curve through the calibration points and maintains a constant temperature shift after the last point.

 

 If these values are correct, select OK.

 If new values are to be entered follow these steps:

  1. Select the Reset button to remove all previously stored values.

  2. Determine the curie temperature points.

  3. Enter the Observed and Correct temperature points in the table.

  4. Select OK when all points have been entered to save the settings to the instrument.
