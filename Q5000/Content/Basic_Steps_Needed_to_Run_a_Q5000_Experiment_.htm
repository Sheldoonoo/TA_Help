Basic Steps Needed to Run a Q5000 Experiment

 

All of your Q5000 experiments will have the following general outline. In some cases not all of these steps will be performed.

  • Calibration/verification/validation of the instrument performance. See Calibrating the Q5000 - Overview.

  • Attaching and setting up external accessories as required such as the  purge gas. See Q5000 IR Purge Gases and Flow Rates or Q5000 SA Purge Gases and Flow Rates.

  • Selecting the pan type and material. See Choosing a Pan.

  • Q5000 SA Only: Filling the humidity chamber. See Filling the Humidity Chamber.

  • Taring the empty sample pan. See Taring the Pan(s)

  • Loading the sample into the pan. See Preparing Samples and Loading/Unloading a Sample.

  • Entering experiment information through the TA controller, this includes both sample and instrument information on the Summary and Notes pages. See Using the Summary Page and Setting Up the Notes Page.

  • Creating or selecting the experimental procedure using the instrument control software.  See Specifying a Test Procedure.

  • Creating a Run Sequence. See Creating Custom Methods or Available Q5000 Test Templates.

  • Starting the experiment. See Starting a Run and/or Sequence.

  • Unloading the sample at the end of the experiment.
