Reset Size Window

This window is displayed when the Reset Size function is selected from the Analyze menu. The new size is automatically calculated from the accepted marker position and entered into the New Size entry field. The Original size from the data file is displayed above.

 

To reset the sample size to be analyzed follow these steps:

  1. Open the desired data file.

  2. Select Analyze/Reset Size.

  3. Position the marker at the desired data point.

  4. Right-click and select Accept Limits from the pop-up menu. The Reset Size window is displayed with the selected size automatically entered.

  5. Either enter a New Size or accept the size shown.

  6. Click OK.

 

The default sample size is the size measured at the start of a run.

 

Any analyses that you perform after resetting the size are based on the new size; previous analysis results are unaffected.

 

The sample size and size-related signals (e.g., Weight, Weight Change, Dimension Change, Length) will be modified to reflect the new sample size.
