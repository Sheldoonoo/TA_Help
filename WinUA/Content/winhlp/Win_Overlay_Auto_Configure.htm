Overlay Auto Configure Window

Use this window to select options that the program will automatically configure for you and will be used to create a curve overlay graph. You can use the Auto Configure window to set up a curve overlay graph initially (using the program-optimized settings), then use the Graph/Overlay/Manual SetUp function to refine your overlay graph.

 

Follow these steps to automatically set up your curves for overlay:

  1. First you need to open and plot your files using the desired signals.

  2. Select Graph/Overlay/Auto Configure, then follow these instructions:

    1. Check the desired files for auto configuring.

    2. Check the desired Y-axis signals (Y1, Y2, etc.) that will be automatically used, if they are available in the opened data file.

    3. Check the desired Auto options: Line, Symbol, and/or Color. Then click on the radio button for each option to apply the program-optimized settings to either the Axes (Y1, Y2, etc.) or to the Curves on each axis (1, 2, 3, etc.).

    4. Check any of the remaining four Auto options that you want the program to optimize and apply to your overlay graph: Curve offset, Axis, File selection, and/or Legend.

    5. Type the desired plot heading that you want to have printed on your curve overlay graph each time one is created using the Auto Configure window. The plot heading will not be displayed, it will only be printed or exported.

    6. Click the Save button, if you want to use these as your new default settings for Auto Configure or click the OK button to use these settings for this plot only. The curve overlay results will be displayed.

  1. Then use the Graph/Overlay/Manual SetUp function to refine the options to be used. When you select Manual Setup from the Graph menu, the Curve Overlay Options window is displayed. Click on the tab at the top of the window that will open the page pertaining to the item that you wish to configure for curve overlay.

  2. Save the curve overlay options when you have finished the manual setup above.

  3. Once the curves are displayed on the overlay plot, you can still perform all of the same operations on the selected curve (e.g., analysis, rescaling, etc.).

 

NOTE: A signal from a file may only be used once. Signals containing the same units (e.g., nonreversing and reversing heat flow, flexural and storage modulus) may be overlaid on the same Y-axis, however, the axis label will reflect the signal of the first curve added to the axis. Customize the axis label using theAxes Page.

 

NOTE: The curve is displayed on the overlay graph using the same conditions that were used in the single file mode. For example, the same data limits, analyses, etc. will be used.
