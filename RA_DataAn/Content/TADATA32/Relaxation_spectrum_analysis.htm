Relaxation Spectrum Parameters

 

Final number of modes: This parameter defines what is essentially the maximum number of modes that you wish to use. The program may determine that a lower number of modes are more appropriate.

 

Start number of modes: This parameter specifies the number of modes that will be used initially. Although not necessary, it is usually most effective to start with a number less than the final number of modes.

 

Overrun factor: Given that the program is dynamic, it needs to have the flexibility of being able to exceed the final number of modes during the calculation process. This parameter specifies the factor by which the final number can be exceeded.

In order to reduce the calculation time, the program allows for the gradual improvement from a coarse solution through to a refined solution. Thus the Merge, Remove and Convergence criteria each have two settings, the first corresponding to the search for a coarse solution and the second corresponding to the refined solution.

 

Merge criterion 1, Merge criterion 2: Both the relaxation times and the moduli are variable although the program is structured to guarantee that both remain non-negative positive. One of the manifestations of the ill-posedness of the problem is that two non-trivial modes might approach each other whereas if they are replaced by a single mode the degree of fit to the data might be improved. The merge criteria specify how close modes may get before the program merges them into one. If l1 and l2 (l1>l2) are two relaxation times, the two modes will be merged if l1/l2 < (1+Merge criterion).

 

Remove criterion 1, Remove criterion 2: Modes may be removed for one of two reasons- if the modulus falls below the stress cut-off value (see below) or if the inclusion of the mode has a small effect on the error of fit relative to the number of modes. If the criterion is set to 1, all modes are removed; for a value of zero no modes are removed

 

Convergence criterion 1, Convergence criterion 2: These are standard convergence criteria.

 

Refinement integer: This parameter specifies the number of steps taken to progress from the coarse solution to the most refined.

 

Initial time increment, Initial modulus increment, Increment reduction factor: At each level of refinement, iterations are made on the relaxation times and the moduli. Those iterations are controlled by these three additional parameters. Although the choice of values can change the speed of the calculation, there is no real reason not to use the default values, particularly for the two initial increments. The increment reduction factor controls how fast these increments are reduced as the algorithm converges.

 

Stress cut-off factor: This parameter specifies the value of stress that can be regarded as 'zero'. It may be effectively regarded as the experimental error in the zero stress.

 

Error limit: This is the limiting error if fit, below which further refinement is not required. Except when using 'exact' data this parameter can normally be set to any small number as in practice best fit attainable is more closely related to the experimental error in the data.

 

Time range: This parameter is subtle. It determines the range of times that are permissible for the relaxation times, based on the range of frequencies that are available for the experimental data. 1 = An extended time range, 2 = 'Normal' time range, i.e. based on inverse max and min frequencies, 3 = Restricted range based on the Davies-Andersen sampling localisation theory.
