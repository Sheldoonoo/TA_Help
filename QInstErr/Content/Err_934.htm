Message 934

Probe travel at limit in dynamic segment. Run continuing.

Problem:  

This message indicates that the displacement has reached the upper or lower measurement limit.  Data taken while this condition continues will not be accurate and dynamic measurements may produce zeroes.

Possible causes of this message are:

  • Not zeroing the displacement (or measuring length) before starting the current run.
  • Yielding or melting of the sample.
  • An experiment which intentionally includes a period of out of range displacement.

Solution:

Try the following:

  • Make sure the probe is zeroed before starting the experiment.
  • Check to make sure the temperature is not so high that it softens or melts the sample.
  • Check the period used for the experiment to make sure it is in range.
