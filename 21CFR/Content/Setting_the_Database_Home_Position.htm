Setting the Database Home Position

You can select the way that the database browser window, Database Viewer, opens by choosing View/User Preferences from the window. The User Preferences window is displayed. Choose from the following options:

  • Goto the home item:  Select this item to chose the home item as the highlighted (first selected) item when the database browser window is opened.
  • Goto the last accessed item:  Select this item to chose the item that you last accessed as the highlighted (first selected) item when the database browser window is opened. This lets you pick up where you left off the last time you used the database browser.
  • Goto the first item in tree view:  Select this item to chose the first item in the tree view on the left-hand pane as the highlighted (first selected) item when the database browser window is opened. See also:  Setting the Database Sort Order
  • Make the current selected item as the home item:  Check this box to designate the item currently selected (highlighted) on the database browser window as the "home" item.
  • Automatically display saved analysis:  Check this box to always display an analysis file saved in Universal Analysis.

Click Set when finished or Cancel to abort the operation.
