Snap to Grid (Custom Report Editor/Edit Menu)

This function toggles between checked (on) and unchecked (off).

 

Select (check) Snap to Grid to have the object boxes automatically aligned with the grid dots when inserted. The boxes will "snap" to line up with the grid dots. This function is good for aligning object boxes more precisely.

 

Uncheck this menu item to be able to move the object boxes anywhere on the template page. The boxes can be positioned between grid dots, if desired.

 

NOTE: This option may also be set using theCustom Report Options Window.
