Verifying the Q5000 SA Humidity

 

The Q5000 SA uses an "open loop" arrangement for generating and measuring the humidity level in the chamber around the same and reference. The humidity is generated by controlling the ratio of gas passing through two calibrated Mass Flow Controllers (MFCs). These Mass Flow Controllers are calibrated at the factory prior to shipment using a certified external flow measurement device. Proper calibration is verified at installation (and can be periodically verified by the operator) by running deliquescent salts. TA Service uses sodium bromide, which deliquesces at 57.6 + 2 %RH (absolute) at 25°C for verification.

The humidity sensors located near the sample and reference pans provide an ongoing indication of humidity and provide further confirmation of proper humidity calibration.

These instructions can be used to verify the relative humidity being reported by the Q5000 SA instrument. This verification is based on the deliquescence of readily available, well understood materials evaluated in a ramped humidity experiment. Verification of the relative humidity is usually based on measurements at 25°C. However, verification can be performed at other temperatures using suitable standards and procedures.

Setting Up Platinum for Humidity Verification

The Q5000 SA is equipped with Humidity verification features that can be accessed through the Platinum functions. These features provide the following:

  • Humidity Verification

  • Scheduling of verification experiments

  • Email and/or Windows Messaging of process with results

  • Calibration report

To set up humidity verification using Platinum:

  1. Open the instrument control program for a Q5000 SA instrument. See Using the TA Instrument Explorer for instructions.

  2. Choose the standard that will be used for your experiments. (Only one standard can be used with Platinum.)

  3. Select the Platinum button, , at the bottom left-hand side of the main instrument control window.

  4. Click Humidity Verification from the Automatic Tasks list on the left side of the window. Two options will be displayed. Choose the desired option:

    • Use Existing Sequence:  This option leaves the current sequence in place allowing you to customize it or use it as is. After selecting this option, use the Sequence Pane to customize the experiment. Then proceed to Step 6 below.

    • Create New Sequence:  Used to set up a new test with the Platinum functions. This option will replace the current sequence. When you select this option, proceed to Step 5.

  5. Creating a new sequence:

    1. Enter the Method parameters:

      • Equilibrate at:  Select a temperature that is close to the standard literature temperature of the material.

      • Isothermal for:  Check this box to hold the sample at the equilibrate temperature for the desired amount of time.

      • RH Ramp:  Enter the rate at which the humidity level will be increased in percent relative humidity per minute.

    2. Check the box next to the desired Standard Name. Make sure that you verify the theoretical relative humidity used for each standard. Edit the data if necessary. (Click here for more information on choosing/editing standards Setting Up Calibration Standards.)

NOTE:  The default values are based on the deliquescence point of each standard at 25°C.

    • Pan #: Use the up/down arrows to choose the pan number containing your sample.

    • Verification Criteria:  Enter the %RH that must be attained in order to verify the humidity.

  2. Select Apply to save the test setup.

  1. Alter the Post Run Action, if desired. The default Post Run Action is to DO (perform) the humidity verification. You can change this option by selecting a run, right clicking to display the pop-up menu, and selecting Post Run Action.   3. Select the Schedule page to specify the time, day, and frequency of the verification process.

  4. Select the Messaging page to specify the desired notification options.

Manually Verifying Humidity (Using the Template)

  1. Choose the standard for your experiment. Make sure that you verify the theoretical relative humidity used for each standard. Edit the data if necessary. (Click here for more information on choosing/editing standards Setting Up Calibration Standards.) Multiple standards may be needed for this purpose. Choose standards whose relative humidity differs substantially.

  2. Tare the desired number of pans needed for the number of standards to be used for humidity verification.

  3. Select the button to start a sequence. If you wish to clear the current sequence, select to open a new sequence.

  4. Choose the "Standard" Mode.

  5. Select the "Humidity Salt Deliquescence" test from the Test list. See Setting Up a Humidity Salt Deliquescence Experiment for more information.

  6. Enter the requested sample information, including the Pan #. The sample name MUST be the same as the chosen standard's name.

  7. Place the sample in a tared sample pan and position the pan on the sample tray.

  8. Click on the Procedure tab. Enter the requested test parameters:

    • Temperature:Enter the desired temperature for your experiment.

    • Isothermal Time:  Enter the amount of time that the sample will be held at the initial (lower) humidity before starting the humidity ramp.

    • Lower Relative Humidity:  Enter the lowest humidity (%RH) that the sample will be exposed to.

    • Upper Relative Humidity:  Enter the highest humidity (%RH) that the sample will be exposed to.

    • Humidity Ramp Rate:  Enter the rate at which the humidity level will be increased (ramped) in percent/minute.

  9. Click on the Notes tab. Enter/verify the requested information.

  10. Click on the Apply button to save the experimental and sample parameters entered for this run.

  11. If you wish to run multiple standards, click on the Copy Run/Append to End button, , to add another run to the sequence.

  12. Repeat steps 1 through 10 for each standard.

NOTE: If you want to have the analysis of the data performed automatically using Post Run Actions, skip to Automatic Analysis Using Post Run Actions now. Otherwise, continue with the next step.

  1. If more than one run is in the sequence list, schedule the first run in the sequence. (An arrow will appear next to the run number in the Sequence Pane for the scheduled run.) See Starting a Run and/or Sequence for more information.

  2. Select Start to begin the calibration run.

  3. Analyze the Humidity data as directed in the next section.

  4. Save the verification sequence for future recall, if desired. Click on the Save button, , at the top of the sequence pane.

Automatic Analysis Using Post Run Actions

This method utilizes Post Run Actions. These are defined while setting up the run sequence and executed at the end of each test. The experimental relative humidity is automatically determined and the data is sent to the instrument using this procedure.

  1. Set up the verification run sequence manually as described above.

  2. Right click on the humidity verification run number on the Sequence Pane to display the pop-up menu. Select Post Run Action.

  3. Select one of the following:

    • IF -- Successful Humidity Verification -- GOTO -- Run Number. (This will begin the next run if the results meet the verification criteria.)

    • DO -- NONE. (This will begin the next run after the verification run is complete with no dependency on the outcome of the verification criteria result.)

    • IF -- Successful Humidity Verification -- GO next sequence. (This will advance the standard run sequence if the results meet the verification criteria.)

  4. Repeat steps 1 through 3 for each humidity standard used.

  5. Select Start to begin the verification run.
