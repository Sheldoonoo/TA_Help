Plot Legend Window

Use the Plot Legend window to set up a customized legend that will be placed on the graph. Follow the steps below to configure the legend.

 

1. Enter the desired Heading for the plot legend.

2. Check one or more options that will be applied to the legend:

 Multiplexed Signal Table (DMA, DEA, and µTA only). If you check Multiplexed Signal Table, you must also select either Vertical or Horizontal from the drop-down list.

 BorderNOTE: When the Border box is checked the legend is opaque (nothing behind it will show). When the box is not checked, there is no border and the legend is transparent (anything behind it will show through).

 Module ID

 Multicolor tableNOTE: Multicolor table will only work when the Multiplexed Signal Table is set to "Vertical."

 Multicolor text

3. Enter the desired Text that will be added to the bottom of the legend.

4. Enter an X-axis position coordinate.

5. Enter a Y-axis position coordinate.

6. Click on the desired vertical position: Top, Center, or Bottom.

7. Click on the desired horizontal position: Left, Center, or Right.

8. Click the Save button, if you want to use these as your new default settings or click the OK button to use these settings for this session only. For an example, click here.

 

NOTE: To use these settings each time you display this window, select the Save button. Select the Default button to use the default settings for this window.

 

NOTE: When the plot legend is displayed, you can drag and drop the legend anywhere on the graph.

 

NOTE: If you are having problems with the size of your curve overlay legend, either on the monitor, printing, or otherwise, seeOverlay Legend Sizefor more information.
