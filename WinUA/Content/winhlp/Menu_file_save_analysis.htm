Save Analysis Menu

Use the Save Analysis option to save either just the current analysis (Analysis Only) or the analysis and the data file (Analysis and Data) for future recall. When you select this option, the standard Save As window is displayed, allowing you to select a filename and location.

 

Once an analysis has been saved in a data file, the analysis results can be retrieved by checking the "Restore saved analysis" box on the Data File Information window.

 

The difference between "Save Analysis Only" and "Save Analysis and Data" is whether the original raw data is included to the saved analysis output file.

 

  • The "Analysis and Data" option outputs a complete copy of the original data file with an extra data block appended to the end of the file. This extra data block contains the saved analysis information that is read by UA when an analysis is restored. If you do not check the "Restore" checkbox on the Data File Information dialog, then the data file is read as if there were no saved analysis block present; i.e., the original raw data is read, not the analyzed data.

  • When you save analysis using the "Analysis Only" option, the data file is copied without the original raw data points. The saved analysis block is then appended to this abbreviated file. If you restore the analysis from this file then everything will look just the same as it does from a file that includes raw data, except that you will not be able to reread (New Data Limits, Signals, Units, etc.) the raw data from this saved analysis file. In this case, Universal Analysis will have to locate the original data file and read the data from that file.

 

The file name at the top of a plot of the data shows the name of the original data file when an "Analysis Only" file is restored, but shows the name of the saved analysis file when an "Analysis and Data" file is restored. If the program cannot locate the original data file associated with an "Analysis Only" file, then Universal Analysis will prompt you to locate the original file.

 

NOTE: Attempting to open a "Save Analysis Only" saved analysis file with the "Restore" checkbox unchecked will result in the "Less than two points in file" error message because there is no raw data stored in the file.

 

NOTE: Saved analyses that have been performed using the DOS version of Universal Analysis are restorable using this program. However, if you overwrite the DOS analyses with this program, the original record of analyses can no longer be used for the DOS Universal Analysis program.

 

NOTE: You can also use theExport Data File windowto save a modified file.
