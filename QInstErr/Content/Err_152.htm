Message 152

LNCS segment not valid in current mode. Cannot start run.

Problem:

The mode or instrument that you are attempting to use cannot work with the LNCS. The DSC Q10 is not compatible with the LNCS.

Solution:

Try one or more of the following operations to solve the problem, then restart your experiment.

  1. Install a cell that can use the LNCS.
  2. Change the mode to one that can use the LNCS segment.
  3. Remove the LNCS segment from your method.
