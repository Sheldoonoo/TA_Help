Message 743

The instrument audit log is approximately 80% full.  Please ensure that the controller is connected and receiving audit log data.

Problem:

The audit log buffer in the instrument is almost full.  The controller has stopped transferring messages to the database.

Solution:

  • If the error occurred while the instrument was off line from a controller, and it has been awhile since the controller was last connected, try connecting the controller to an instrument. This may transfer the contents of the log buffer and resolve the issue.  
  • This could also be an indication that the controller has lost it's connection to the Integrity database server.  Ensure the connection and that the server is in working order.  
  • If these suggestions fail to fix the problem, and you have not already done so, try closing down and reopening the controller software.  
  • If the problem persists after trying the steps above, contact TA Instruments for service.
