Box Options Window

Use this window to set up the box options that will apply to the highlighted report object.

 

Follow these instructions to use this window. Click on the green links to get an explanation of the item named.

 

1. Select the report object desired on the template.

2. Select Edit/Box from the menu or right click and select Box from the pop-up menu.

3. Choose the desired options that will affect the selected object box to be inserted into your custom report from the following: Visible in report (for printed box), Transparent, Position Locked, Line color (for printed box), and Line thickness (for printed box).

4. Enter/edit the exact position of the report object in inches: Top, Left, Height, and Width.

5. Click the OK button when finished. The box options will be shown on the final custom report.

 

NOTE: To make global changes to report object parameters, which will be applied each time a new object is inserted, select File/Options from the menu and make changes using theCustom Report Options window.

 

NOTE: The Transparent checkbox does not apply to Excel Spreadsheet objects.
