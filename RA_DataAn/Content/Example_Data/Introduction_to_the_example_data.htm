Introduction to the Example Data

 

Example data files are installed as part of the standard installation and are usually located in the folder:

C:\TA\RHEOLOGY\EXAMPLES.  (You can elect to not install them if you perform a Custom installation.)

NOTE:  If you have Microsoft Windows installed onto a drive other than C:, the location mentioned above will also be on that drive.

 

The files are intended to be used with the Data analysis package and are supplied purely for illustrative purposes only.
