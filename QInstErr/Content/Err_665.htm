Message 665

Balance purge mass flow control is out of service.

Problem:

There is a problem with the Mass Flow Controller (MFC) that controls the balance purge.

Solution:

  • Make sure that the gas selected for the balance MFC (Gas #1) is not set to "None" on the Instrument Preferences/MFC Purge page.
  • If the gas is not set to "None," reset the instrument.
  • If the error message persists, call TA Instruments for service.
