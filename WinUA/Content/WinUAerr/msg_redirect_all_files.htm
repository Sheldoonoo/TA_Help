Redirect All Files?

Do you want to redirect all open files in < > to < >?

 

This message is displayed if you have relocated a file that was previously saved to a Universal Analysis session in a different location. The file is now missing from its original path and the program has determined that there are other open files with the same directory path.

 

Select Yes to move all open files to the new location.

Select No to leave the other open files in their current location.
