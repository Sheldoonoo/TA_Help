Selecting TMA Signals to Store

 

Every analytical instrument produces distinctive signals that may or may not be stored in a data file. To select the signals that you want to be saved in data files produced during this experiment, access the Summary Page in Experimental View. Click the button. Check the signals to be stored from the list shown. Then click the OK button. The signals selected apply only to the current run.

This list will vary depending on the mode chosen.

Some of the signals for the TMA are listed below. See Monitoring Real-Time Signals for a complete listing.

  • Time (min) = Time since run start.
  • Temperature (°C) = Sample temperature.
  • Dimension Change (µm) = Change in sample dimension, measured as the probe moves up or down.
  • Force (N) = Force exerted on the probe when it rests on the sample.
