Incompatible Software Versions

The current version of instrument software is not compatible with the controller. Please go to the Instrument Setup Wizard and download the latest version of instrument software before proceeding.

This message is displayed when you first open the instrument control program if the system detects incompatible software versions between the controller and the instrument.

Update the instrument software following the steps below:

  1. Verify that both the instrument and the controller are both powered on and connected to the supplied Ethernet Hub or LAN.
  2. Select Tools\Instrument Setup from the main menu. The Instrument Setup Wizard will now start.
  3. Click the Next button until the window Transfer Program Files to the Instrument is displayed.
  4. Verify the module software path by clicking the Browse button to the right of the Update Module Software field. Select the appropriate directory where the instrument software files are located. The default directory is Program Files\TA Instruments\Thermal Advantage\Qseries\InstumentSW\[instrument].
  5. Select the instrument .abs file to load (e.g., dsc.abs, tga.abs). Then click Select.
  6. Click the Send Programs button. Downloading of the software will now begin and the status can be viewed via a progress bar.  
  7. When the download has successfully finished, the message above is posted indicating the software download was successfully completed, click OK.

NOTE:  If you receive an error that the download failed, you must click the Send Programs button again before you reset the instrument and proceed to the next step.

  1. Press the reset button on the back of the instrument (shown in the figure to the right) to complete this process.
  2. Select the Reset Parameters button to reset the instrument parameters. For more information, see Resetting Instrument Parameters.

Update the controller software, if needed. Insert the Advantage Q Series software CD into the CD-ROM drive. The install program will start automatically. Follow the prompts displayed.
