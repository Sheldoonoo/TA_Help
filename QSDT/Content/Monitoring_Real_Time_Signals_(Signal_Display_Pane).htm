Monitoring Real-Time Signals

 

Real-time instrument signals may be monitored from either the instrument touch screen or from the controller Signal Display pane. If you choose to use the touch screen to monitor signals, you can access either the Status or Display options. See also:Display Menu Touch Screen.

Controller Signal Display Pane

The Signal Display pane (shown to the right) is a customizable instrument control window that allows the user to select the real time signals they want displayed during an experiment. These same signals are also available for plotting in the RealTime Plot window.

Customizing the Signal Display Pane

The TA Instruments SDT displays a certain set of information on the Signal Display window. Using the pop-up menu (accessed by right clicking), you can customize the signal display by selecting Customize Signal Display.  When the Customize Signal Display window appears, select or deselect the signals then click OK to close the signal selection window and display the new signals. You can reorganize the order of the signals by clicking on a signal name to display a drop-down list of signals to select from.

SDT Instrument Signals

Listed below are all of the possible real-time SDT signals. The available signals are dependent on instrument type and options installed. These signals are available for display within the Signal Display pane, as RealTime plot signals, and they are displayed on the instrument touch screen, if available. A subset of these signals are available as stored signals within the data file.

For a detailed explanation of the signals shown on the window, see the table below:

 

Signal

Description

Delta T

The delta-T signal is displayed when running SDT experiments. It is the current differential temperature between the sample and the reference in µV. This signal is selectable and can be included in the data file.

Delta T Temperature

The delta-T signal is displayed when running SDT experiments. It is the current differential temperature between the sample and the reference in °C. This signal is selectable and can be included in the data file.

Heat Flow

The heat flow signal is the average or total heat flow (in mW) given off or absorbed by the sample. This signal is stored in the data file.

Heater Power, Watts

The amount of electrical power being applied to the heater.

Method Time, min

The total time that the current method has been running.

Remaining Run Time, min

The estimated total time remaining until completion of experiment.

Sample Purge Flow, mL\min

The rate of gas being supplied to the sample.

Segment Time, min

The total time that the current segment has been running.

Set Point Temperature, °C

The temperature that the SDT furnace is requested to attain.

Temperature, °C

The current temperature of the sample.

Uncalibrated Temperature, °C

The current temperature of the sample, without a stored temperature calibration (in calibration mode). See Calibrating the SDT Temperature for information on performing temperature calibration.

Uncalibrated Delta T, µV

The uncalibrated delta-T signal is displayed when running SDT experiments in calibration mode. It is the current uncalibrated differential temperature between the sample and the reference in µV. See Calibrating the SDT for DTA Baseline for information.

Uncalibrated Sample Weight, mV

The uncalibrated sample weight signal is displayed by the SDT instrument in calibration mode. It is the measured uncalibrated weight (in mg) of the material on the sample platform and is used for weight calibration. See Calibrating the SDT for TGA Weight for information.

Uncalibrated Reference Weight, mV

The uncalibrated reference weight signal is displayed when operating the SDT instrument in calibration mode. The measured weight (in mg) of the material on the reference platform used for weight calibration. See Calibrating the SDT for TGA Weight for information.

Weight, mg

The measured weight of the material in the sample pan that is displayed when a method is running. The signal is selectable and can be included in the data file.

Weight Percent, %

The measured weight of the material displayed as a percentage of the starting weight.
