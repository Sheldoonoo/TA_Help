Must Log In as Master Controller

This program requires you log in as a master controller. However, One of the controllers on the network is connected as a master already. You need unregister the master on that controller and try to log in again.

In order to set up experiments and create procedures you need to register as the master controller for that particular instrument. If someone else is already logged in as the master, that person must unregister before you can log in again as the master controller.

To unregister as the master controller close the instrument window or select Tools/Unregister Master Controller form the menu.
