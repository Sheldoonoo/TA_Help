Setting Up a TGA Ramp with Gas Switch Procedure

 

Most TGA experiments are performed by heating the material at a constant rate from ambient temperature to a temperature above the point of total decomposition. The purge gas that surrounds the material is switched at an elevated temperature in a Ramp with Gas Switch procedure to cause additional weight changes to occur. A variety of inert and “reactive” purge gases can be used, including nitrogen, argon, helium (inert) and air, oxygen (reactive). Gas switching experiments generally involve switching from an inert to reactive gas at a specific temperature. For example, carbon black determinations are achieved by switching from nitrogen to oxygen at 600°C.

If you want to set up a TGA ramp with gas switch procedure, you can choose one of two techniques—the TGA Wizard or the Experimental View - Procedure Page. See the instructions below for each technique.

When you perform a TGA Ramp with Gas Switch experiment, three test parameters need to be specified:

  • Heating Rate: The acceptable range of heating rates for the standard TGA furnace is 0.1 to 100°C/minute. The acceptable range for the optional EGA (evolved gas analysis) furnace is 0.1 to 50°C/minute. Obviously, slower heating rates result in longer analysis times. Most experiments covering a broad temperature range (>200°C) are performed at 10, 20, or 50°C/minute.

  • Final Temperature: The final temperature is set above the last weight change of interest or 1000°C (600°C if using aluminum pans). The 1000°C setting is recommended when evaluating materials for the first time.

  • Switch to Gas 2: A variety of inert and "reactive" purge gases can be used, including nitrogen, argon, helium (inert) and air, oxygen (reactive). Switching the gas from inert to reactive at elevated temperatures will cause materials which oxidize such as carbon to decompose, providing additional information such as carbon black content in plastics and elastomers or fixed carbon content in coal. This option can only be used when a Mass Flow Controller is installed.

  • Additional Conditions: There are additional experimental conditions that can be changed to optimize specific situations. These are accessed using Advanced Parameters and Post-Test Conditions. For most experiments, the system default values shown are recommended.

Using the TGA Wizard :

Enter the desired parameters, then select Next to proceed or select Back to return to the previous window.

Using the Experimental View Procedure Page:

  1. Display the Procedure Page. Select Ramp with Gas Switch from the list of preprogrammed templates.

  2. Enter the desired parameters. When you have finished setting up this procedure, you can enter more information by selecting the Notes Page and Summary Page for this run.

  3. Click the Apply button when finished to save these changes or the Append Run button to save the changes to this run and add another run to the end of the sequence.

NOTE: You can use any test template as the basis for a custom test by selecting the desired test template from the list of tests, then click the Apply button and change the test to Custom. Then you can click on the Procedure Page and then the Editor button to make the changes desired.
