Calibrating the SDT for Dual Sample Operation

 

Dual Calibration is an alternative method for providing reference data that frees both pans for use with samples.  A baseline run is made with tared, empty pans on both sides and the weight and temperature profiles observed for each side are stored as cubic spline functions of temperature.  These stored profiles then provide historical reference data for subsequent Dual Sample runs.  For optimum results the same cups should be used in subsequent dual sample experiments.

NOTE:  This calibration is not required when operating in the standard mode.

Gathering SDT Dual Sample Data

Use the following instructions to perform the baseline run for dual sample calibration. This process uses the "Dual Sample" test template.

  1. Check that your purge gas is connected.
  2. Open the furnace. Position two clean SDT cups on the sample and reference beams. (For optimum results the same cups should be used in subsequent dual sample runs.)
  3. Close the furnace.
  4. Press the TARE key.
  5. Select Experiment View and access the Summary Page. Set the instrument to "Dual Sample Calibration" mode using the Mode drop-down list. (In the calibration mode the current temperature table data is ignored.)
  6. Select the "Dual Sample" test procedure. See also:  Setting Up an SDT Dual Calibration Procedure
  7. Enter the requested sample information, entering zero for the Sample Weight and Reference Weight.
  8. Specify a data file name using the Browse button.
  9. Click on the Procedure tab.
  10. Check Use Current as the starting temperature or enter the desired Start Temperature (e.g., 50.00째C).
  11. Enter your upper temperature (or 1500째C) as the Final Temperature.  
  12. Enter the Heating Rate that will be used in subsequent experiments (e.g., 20째C/min).
  13. Select the Advance button to access and select the Advanced Parameters.
  14. Click the Post Test button to access the Post-Test condition parameters that allow you to specify the furnace and air cool options to use once the experiment is complete. (Select "Furnace Closed" and "Air Cool to 50째C.")
  15. Click on the Notes Page.  Enter the requested information, including the desired flow rate (typically 100 mL/min).
  16. Click on Apply to store the changes made to these pages.
  17. Schedule this run, if more than one run is in the sequence list.
  18. Select Start to begin the calibration run.

No analysis is required. The calibration is automatically saved and the instrument is ready for dual sample operation. Select Calibrate/Report to view the date and time of this calibration for future reference.
