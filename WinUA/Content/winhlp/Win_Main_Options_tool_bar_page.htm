Main Options Window  Tool Bar Page

Use the Tool Bar page of the Main Options window to set up the tool bar icons that you want to display every time the Universal Analysis program is opened.

 

Follow these steps to perform set up the tool bar:

  1. Select the buttons that you want to display on the tool bar by using one of two methods:

    1. Check each desired function button name from the scrollable button list shown. This list corresponds to the menu items performing the same function.

    2. Click on the button that selects the default functions that apply to the button label. For example, you can click on the TGA button to select the icons for the most common functions that apply to the TGA instrument.

  1. Check each option that you want to use for your tool bar: Show tool bar, Flat buttons, Flyover help.

  2. Click the Save button, if you want to use these as your new settings or click the OK button to use these settings for this session only. 

NOTE: The Save button only stores the settings for the selected page in the initialization (.ini) file. To save all settings, access theMain Options  Initialization Pageand click the Save As button. The Default button will also only recall the default factory settings for the selected page.
