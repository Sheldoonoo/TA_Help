Signal Selection Window

You can use the Signal Selection window to choose the signal and type for the X and Y axes. These signals will be plotted on the graph for the data file to be used for analysis.

 

To select signals follow these steps:

  1. Click on the arrow to the right of the Signals field then click on the desired signal for the Y1 axis. (You can choose "None," if you do not want to plot a Y1 axis).

  2. Click on the arrow to the right of the Type field then click on the desired signal type for the Y1 axis.

  3. Repeat steps 1 and 2 for each remaining Y-axis.

  4. Click on the radio button next to the desired signal for the X axis. You can choose from Time, Temperature, or from a drop-down list of Signals. If you make a selection from the Signals drop-down list, you need to choose a Type as well. NOTE: You can switch between the X-axis signals specified here (time, temperature, etc.) using theGraph/X Axisfunction.

  5. If you are plotting Time on the X axis, you can add a second X axis to the plot by checking the Show Temperature box.

  6. If you are plotting Temperature on the X axis, you can add a second X axis to the plot by checking the Show Time box.
  7. Click the Save button, if you want to use these as your new default settings or click the OK button to use these settings for this session only. 

All derivative curves are automatically smoothed when the curve is drawn. You can change the smoothing interval with the Tools/Smooth option.
