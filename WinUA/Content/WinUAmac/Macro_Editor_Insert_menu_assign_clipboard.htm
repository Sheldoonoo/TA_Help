Assign Clipboard (Macro Insert Menu)

The Assign Clipboard function is used to insert a macro step that will to copy the current value of a macro variable and/or a text string to the clipboard. This is useful for building custom annotations and tables that are placed on a plot or added to a report. The number of significant figures for the copied value can be specified, as well as whether the value should be rounded to an integer. An optional text string of up to 30 characters can be appended to the end of the value when it is copied. If no variable name is specified, then only the text string is copied. An optional carriage return and line feed (<crlf>) line terminator can be appended to the copied text.

 

When you select this function the Macro Assign Clipboard window is displayed.
