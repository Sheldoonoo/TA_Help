Sample Size Too Small

The file selected for calibration has a sample size of 0 (zero). A zero sample size is not valid for this calibration.

 

  1. Check that the correct file was selected for the selected calibration procedure.
  2. Conduct the experiment again and make sure you enter the sizes, then use the new file for calibration.
  3. Edit the parameter block with the Universal Analysis program to change the sizes to the correct values, then retry the operation.
