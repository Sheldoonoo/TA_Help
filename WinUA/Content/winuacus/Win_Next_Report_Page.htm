Next Report Page Window

Use this window to enter or browse for a Universal Analysis template file that will be used as the next page in your custom report.

 

If you leave the entry field blank, there will be no next page.

 

A single template file can only be used once in a multiple page report to prevent an endless loop linking from one template to another recursively.

 

To use this window:

  1. Enter the location and filename of the desired Universal Analysis template file (.uat) or click the Browse button to locate the template file.
  2. Click OK when finished.
