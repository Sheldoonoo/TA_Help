Setting Up the Notes Page

 

Sample information for each run is entered on both the Summary Page and Notes Page. The Procedure Page is used to set up the experimental test conditions.

Use the Notes Page to enter or select additional information to be stored with the selected file. The gas entries entered here are used to specify the gas and the associated flow rate for each run. These items cannot be changed once the experiment is in progress.

  1. Enter the name of the Operator conducting the thermal analysis experiment. Up to 25 characters may be entered.

  2. Select the sample Pan type, which includes the material and/or type of the pan that will be used in the experiment. Click on the down arrow to display a list of pan types, then select the desired pan type. Choose platinum or alumina (ceramics). 90 µL alumina should be used for DSC-TGA experiments. Alumina (ceramic) is available for larger samples. Aluminum can only be used to 600°C. See also: Choosing a Cup Type.

NOTE:  The pan type also determines the Mode that is displayed on the Summary Page.

  1. Enter any additional comments in the Extended Text field. This may involve information about the sample or experiment, such as sample treatment or data sampling parameters. Up to 500 alphanumeric characters may be entered (approximately 12 lines).

  2. Select the Sample gas from the drop-down list, either Gas #1 or Gas #2, and enter the corresponding Flow rate. If an MFC is installed, these settings will automatically set the gas and flow rate on a run by run basis. If your instrument does not have an MFC, then these entries are information only. To change the gas type, access Instrument Preferences.

  3. Click the Apply button to apply the information on this page to the current run.
