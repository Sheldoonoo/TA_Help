Creating a TMA Table of Alpha

To produce an alpha curve and a table of alpha using TMA data files, follow these steps:

1. Select File/New and open the desired TMA data file.

2. Click the Units button on the Data File Information window (or select Graph/Units) then select the TMA Dimension units list box. Set the units to µm/m. Click OK.

3. Click the Signals button on the Data File Information window (or select Graph/Signals). Set the Y1 signal type to be Derivative (temp). The Y1 signal will automatically change to Alpha (µm/m/°C).

4. Select Temperature as the X-axis signal.

5. Click OK. The plot will display Alpha vs. Temperature.

6. Select View/Data Table/ReportorView/Data Table/Spreadsheet. Enter the desired start and stop limits for the table of Alpha on the Data Table Limits window. Click OK. The table of Alpha will be displayed, see the example below.
