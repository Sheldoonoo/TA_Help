Maximum Analysis Steps

 

Maximum number of analysis log steps have been recorded.

 

Problem:

The analysis log can only record up to 800 analysis steps for a single open data file. Any further analysis steps are not recorded.

 

Solution:

Do not perform more than 800 steps on a single data file if the entire analysis log must be saved. If needed, you can combine routine analysis steps into a macro, then execute the macro during the analysis. Steps from an executing macro are not stored in the analysis log and, therefore, do not affect the 800-step maximum. Only the name of the macro is recorded in the log. A convenient way to combine steps is to use the Macro/Copy/Analysis Log function to automatically create a macro from a partial or complete analysis log.
