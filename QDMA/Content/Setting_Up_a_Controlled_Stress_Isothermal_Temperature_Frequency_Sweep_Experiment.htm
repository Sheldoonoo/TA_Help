Setting Up a Controlled Stress Isothermal Temperature/Frequency Sweep Experiment

 

This experiment is performed isothermally by oscillating the stress sinusoidally at a single frequency or multiple frequencies.  This method can be used to study the stress induced fatigue behavior of materials.

If you want to set up a DMA Isothermal Temperature/Frequency Sweep procedure, you can choose one of two techniques—the DMA Wizard or the Experimental View - Procedure Page. See the instructions below for each technique.

When you perform a Multifrequency Controlled Stress mode, DMA Isothermal Temperature/Frequency Sweep experiment (test), the following parameters need to be chosen:

  • Select between Stress and Force:

    • Stress:  The stress depends on the force being applied, the kinematics of deformation (tension, bending, shear, etc.) and the dimensions of the sample.  The stress applied can be selected to simulate real-life end-use conditions or can be a nominal value sufficiently large to ensure good deformation.  It is recommended that the stress selected be at least equivalent to ten times the minimum stress (based on the minimum force of 0.0001 N).  This will enable better resolution of the sine waves.

    • Force:  It is recommended that the minimum force be 0.001 N (ten times the lowest force achievable by the instrument).

  • Initial Amplitude:  This parameter is used to evaluate the stiffness of the sample prior to running the experiment in order to establish the limits of stress that the instrument will be able to apply at certain frequencies.  This should be a small number, in the range of 2 to 5 micrometers.

  • Preload Force: This is the force applied to the sample prior to initiating the experiment.  

    • Preload force is not available when you select a nontensioning clamp (i.e., single/dual cantilever, shear sandwich).  

    • Preload force is required with the tensioning clamps (i.e., 3-point bending, compression, and tension film/fiber) when running multifrequency and mulitstrain experiments in order to sustain oscillation (ensure that force is applied to the sample throughout the complete oscillation).  Typical preload forces for those experiments are 0.01 to 0.5 N.

  • Force Track:  Force track automatically adjusts the total force (preload and dynamic) applied to the sample during multifrequency and multistrain experiments.  

    • Force track is only used when one of the tensioning clamps are selected (i.e., 3-point bend, compression, tension film and tension fiber). It is not needed for nontensioning clamp (i.e., single/dual cantilever, shear sandwich).

    • Force track essentially determines the dynamic force required to oscillate at the desired amplitude and frequency; it applies an additional static force that is constant multiple of that dynamic force.  This is necessary to avoid sample buckling in tension clamps and loss of contact with the sample for compression and 3-point bending clamps.

    • Recommended force track values are 120% for thin films and low denier fibers (tension film and tension fiber clamps), 150% for thermoplastics (3-point bending clamps), and 125% for general applications.

  • Tolerance:  This parameter is the amount that the experiment can deviate above or below the Stress or Force entered. It is entered as a percentage of the value chosen. For example, if the tolerance is five percent and the experiment uses a force of 1.0 N, the instrument will take valid data as long as the force fluctuates between 1.05 and 0.95 N.

  • Isothermal Temperature:  The temperature range of the DMA is -145 to 600°C. This entire range is available if a Gas Cooling Accessory (GCA) is used. Isothermal experiments can be run at any temperature within this range.  

NOTE:  This parameter is not used for submersion-type clamps.

  • Soak Time:  Because of the mass of the DMA furnace and clamps, as well as the large size for most DMA samples, a "soak time" (i.e., time to equilibrate at temperature) is recommended particularly when starting experiments at subambient temperatures. A five-minute soak time is sufficient for most samples.

NOTE:  This parameter is not used for submersion-type clamps.

  • Number of Sweeps:  A single frequency sweep is sufficient for determination of the viscoelastic response. You may run several sweeps, one after another, in order to determine any time-dependent effects at isothermal conditions.

  • Frequency Table:  All Multifrequency mode tests require that you select one or more frequencies. The frequency range for the DMA  is 0.01 to 200 Hz. "Logarithmically spaced" frequencies (e.g., 10, 5, 2, 1) are preferable rather than "linearly spaced" frequencies (e.g., 9, 7, 5, 3, 1). For more information on frequencies, click here Frequency Table.

  • Additional Conditions:  There are additional experimental conditions that can be changed to optimize specific situations.  These are accessed using Advanced Parameters and Post-Test Conditions.  For most experiments, the system default values shown are recommended.

  • See also:  Guidelines for Running Subambient Experiments and Displaying the DMA Waveform.

Using the DMA Wizard :  

Enter the desired parameters, then select Next to proceed.

Using the Experimental View Procedure Page:

  1. Display the Procedure Page. Select the desired experiment from the list of available preprogrammed templates.

  2. Enter the desired parameters. When you have finished setting up this procedure, you can enter more information by selecting the Notes Page and Summary Page for this run.

  3. Click the Apply button when finished to save these changes or the Append button to save the changes to this run and add another run to the end of the sequence.

NOTE:  You can use any test template as the basis for a custom test by selecting the desired test template from the list of tests, then click the Apply button and change the test to Custom. Then you can click on the Procedure Page and then the Editor button to make the changes desired.
