Dialog (Macro Insert Menu)

Use the Dialog option to add a message box to your macro. This option allows you to display informational text and, based on the way the dialog is configured, you can create branching based on a response, provide a repeat mechanism, allow operator intervention to select analysis limits, etc.

 

For example, a message box giving the allowable range for a specific analysis can be displayed, allowing comparison with the experimental result just obtained. When you select this option, the Macro Dialog Window is displayed. Select the desired macro step for the dialog before selecting this option.
