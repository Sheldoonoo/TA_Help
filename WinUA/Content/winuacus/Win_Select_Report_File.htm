Select File/Select Overlay Window

Use the Select File or Select Overlay window to direct the Custom Report Editor program to the filenames or overlay plot names that correspond with the numbers used for the report objects in the template. The title of the window reflects the number to be applied. For example, if you have a template that uses files or overlay plots numbered 1 and 2, the window displayed when the report is exported will prompt you to "Select File 1" or "Select Overlay 1" and then "Select File 2" or "Select Overlay 2."

 

If you want to use a report template that calls for more files or overlays than you need to output, you can limit the number of files or overlays by clicking the Last button when selecting the last file or overlay for output. Any report blocks for files or overlays beyond the last will be suppressed.

 

1. Click on the filename/overlay plot name that corresponds to the file/overlay plot number displayed in the window title bar.

2. Click the OK button when finished. If there are multiple numbers used in the template, a window will be displayed for each file or overlay plot number. When all file/overlay plot numbers have been accounted for, the custom report is generated in Word.
