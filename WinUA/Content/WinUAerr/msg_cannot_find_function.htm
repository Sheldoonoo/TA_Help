Cannot Find Function for Macro Step

Cannot find function for macro step

 

Problem:

The target function for the last executed macro step cannot be located. This usually occurs when you run a macro that was captured on an newer version of the program on an older version of the program that does not have all of the features of the newer program.

 

Solution:

Do not attempt to run newer macros on an older version of the program. Recapture the macro steps using the program you wish to run it on.
