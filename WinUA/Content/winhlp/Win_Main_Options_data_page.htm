Main Options Window  Data Page

Use this window to set up the way that your data files will be presented each time the Universal Analysis program is opened. You can set these options as desired and they will be applied each time you open the program.

 

Follow these steps to set up the data options:

 

1. Click on the direction for your RMX Exotherm plot: Up or Down.

2. Choose the type of DSC and DTA derivative plot desired (if you are analyzing DSC and DTA files): Normal or Reversed.

3. Choose the type of data compaction that you want to use when the data files are opened: Prompt for option, Discard points, or Linear smoothing.

4. Check the remaining options that you wish to use for your data files:

  • Autoscale new data X-axis
  • Autoscale new data Y-axis
  • Plot data outside of analysis range
  • Ignore invalid data
  • Ignore breaks in data
  • Break data on gas switch or event mark
  • Auto refresh data every xx seconds
  • Restore saved analysis
  • Create PDF file with saved analysis
  • Expand heat flow signal names
  • Exact Scaling X/Y Axis

5. Enter the default Data Buffer Size. NOTE: You cannot change the data buffer size when data files are open. Close all files to enable the Data buffer size entry field on this page.

6. Check any desired Open Data File options that will be used whenever a new data file is opened: Preview, All Text, Quick Open and/or Auto Launch.

7. Enter the desired number of tick marks per label on the x axis and y axis. The default is 4 ticks per label.

8. Click the Save button, if you want to use these as your new settings or click the OK button to use these settings for this session only. 

 

NOTE: The Save button only stores the settings for the selected page in the initialization (.ini) file. To save all settings, access theMain Options  Initialization Pageand click the Save As button. The Default button will also only recall the default factory settings for the selected page.
