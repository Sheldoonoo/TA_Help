Message 287

Sample size or geometry unusable for stress/force control.  Cannot start run.

Problem:

This error message is displayed if you try to run in Multifrequency Controlled Stress mode without selecting a geometry and setting sample dimensions. The run will not be started.

Solution:

Make sure that you select a sample geometry and enter the sample dimensions (sizes).  For best results, Poisson's ratio should also be set to the correct value for the sample.

NOTE:  If you change clamps, the geometry will be reset to UNKNOWN. You must then select a sample geometry.
