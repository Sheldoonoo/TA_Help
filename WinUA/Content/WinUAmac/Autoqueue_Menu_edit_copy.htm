Copy (Autoqueue)

Use the Copy menu item to copy a record from the autoqueue to the clipboard for retrieval by pasting.

 

Shortcut Keys

Ctrl+C

 

1. Select Autoqueue from the main Macros menu to display the Autoqueue Window.

2. Click on the record that you want to copy. (Use Select All to copy all records.)

3. Select Edit/Copy from the menu (or press Ctrl+C). The record will be copied to the clipboard for retrieval, if desired. (Use Paste or Paste At End to retrieve the record on the clipboard.)
