Autoqueue Window

Use this window to view, edit, export, print, or run the autoqueue, which is processed to perform Autoanalysis.

 

The autoqueue is made up of a list of records. Records are made up of the data filename that will be processed, the macro file used, and the macro to be applied for analysis. If there are no records in the autoqueue, this window will be empty.

 

To perform the various functions on the autoqueue, use the desired menu from the list below:
