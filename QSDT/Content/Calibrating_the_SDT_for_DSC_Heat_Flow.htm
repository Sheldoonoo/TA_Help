Calibrating the SDT for DSC Heat Flow

 

Heat flow calibration is based on analyzing the heat capacity curve for sapphire over the range 200 to 1500°C. Three experimental runs are required:  two runs to generate the heat flow curve and another run to refine that calibration through cell constant calibration using a known metal standard (zinc). The heat flow  calibration curve experiments are conducted in the calibration mode—one using empty alumina cups (90 µL) for both the reference and sample (baseline run) and another using a sapphire standard as the sample.  The measured heat capacity for sapphire is compared with the literature value at multiple temperatures across the range and mathematically fitted to generate a heat flow calibration curve. This calibration curve can be further refined through cell constant calibration (e.g., running zinc and determining the heat of fusion). With proper heat flow calibration, the experimental value should be 108.7 J/g + 2.17 J/g (2%).

NOTE:  The calibration data for the first two experiments must be obtained with the instrument in the calibration mode.  The calibration for cell constant is performed in the standard mode.

Performing the DSC Heat Flow Calibration

Use the following instructions to gather and analyze the data files needed for DSC heat flow calibration.

Gathering DSC Heat Flow Calibration Data

  1. Check that your purge gas is connected.  

  2. Open the furnace. Remove all SDT cups, sample, and weights from the sample and reference beams.

Setting Up the First Calibration Run (baseline run):  

  1. Place two clean, empty alumina (ceramic) cups on the sample and reference beams. Adjust the position of the cups on the platinum sensor to be as close to the beams as possible. Use the same cup position in all subsequent experiments.  

NOTE:  Obtaining the best heat flow reproducibility and accuracy requires careful attention to details like cup positioning.  The cups should be placed on the sensor in the same position for both calibration and regular experiments.  The “same position” in this context refers, not only to lateral position, but also rotational position.  In addition, to ensure that the alumina cups are clean and free of adsorbed water, it is a good idea to heat them in a propane flame prior to use.

  1. Close the furnace.

  2. Select Experiment View and access the Summary Page, and then set the instrument to "calibration" mode using the Mode drop-down list.

  3. Select "DSC Heat Flow" from the Test list.

  4. Enter the requested sample information, entering zero for both the Sample Weight and Reference Weight.

  5. Specify a data file name for the first run (without weights) using the browse button.

  6. Click on the Procedure Page.

  7. Check Use Current as the starting temperature or enter the desired Start Temperature (e.g., 50.00°C).

  8. Enter your upper temperature (e.g., 1400°C) as the Final Temperature.  

  9. Enter the Heating Rate that will be used in subsequent experiments (e.g., 20°C/min).

  10. Select the Advance button to access and select the Advanced Parameters.

  11. Click the Post Test button to access the Post-Test condition parameters that allow you to specify the furnace and air cool options to use once the experiment is complete. (Select “Furnace Closed” and Air Cool to 50°C.)

  12. Click on the Notes Page.  Enter the requested information, including flow rate. (The same flow rate, as used in step 14, should be used.)

  13. Click on Apply to store the changes made to these pages.

  14. Schedule this run, if more than one run is in the sequence list.

  15. Select Start to begin the calibration run. Wait for the experiment to finish, and allow the SDT to cool (<50°C).

Setting Up the Second Calibration Run (with sapphire):  

  1. On the list of runs in the Sequence Pane, double click on the complete icon (check mark) next to reschedule the run.  This will store the existing information from the previous run and automatically increment the filename.  Alternatively, you could just change the filename or append a new run.

  2. Press the FURNACE key on the SDT touch screen to open the furnace, and place the sapphire standard (PN 915079.903) into the sample (front) cup.  Adjust the cup positions, if needed, as described in step 2 above.

  3. Close the furnace.

  4. Select Experiment View and access the Summary Page. Alter the sample information to specify sapphire as the sample. Enter the weight of sapphire and leave the reference weight equal to zero. The filename will automatically be incremented. Repeat the same procedure as used above (e.g., steps 10-15).

  5. Click on Apply to store the changes made to these pages.

  6. Schedule this run, if more than one run is in the sequence list.

  7. Select Start to begin the next calibration run.  Wait for the experiment to finish.

  8. Analyze the calibration data as directed in the next section.

Analyzing DSC Heat Flow Calibration Data

  1. Once the calibration runs are completed, select Calibrate/DSC Heat Flow on the SDT menu. The DSC Heat Flow Calibration window is displayed.

  2. Select the file for the baseline run experiment (empty alumina cups) by clicking on the Browse button.  Select the file for the second experiment (sapphire) by clicking on the Browse button.

  3. Select OK when both the calibration files have been entered. The Calibration program will open and display both files.

  4. Select Analyze from the Analysis menu.

  5. Position the two analysis markers in the desired location (bracketing the analysis region of interest) then select Limits OK from the pop-up or Analyze menu.  

NOTE:  When you are selecting your analysis limits, the default limits of 200 and 1400°C are recommended as the optimum limits to use when calibrating the SDT up to 1500°C.

  1. Save the calibration results using Analyze/Accept results.  (DSC Heat Flow calibration values are not displayed, only the resultant curve.)

NOTE:  The baseline file data must be greater than the sapphire data to complete the analysis.

  1. If results are not satisfactory, rerun your calibration experiments.  Pay careful attention to make sure that the beams are positioned within the center of the furnace, that the ceramic cups are seated properly on the platform, and that the purge flow is steady. If the results are still not satisfactory after rerunning the calibration, contact your service representative.

  2. Close the Calibration window.

The Calibration Report can be used to view the date this calibration was last conducted, the temperature range used, and the associated filenames.

Performing the DSC Heat Flow Cell Constant Calibration

Use the following instructions to gather and analyze the data files needed for DSC heat flow cell constant calibration.

Gathering DSC Heat Flow Cell Constant Calibration Data

Set up the cell constant calibration experiment (using zinc):  

  1. Check that your purge gas is connected.

  2. Select Cell/Temperature Table from the Calibrate menu.  Enter a value of 1.00 for the cell constant value.

  3. Select Experiment View and access the Summary Page. Verify that the "standard" mode has been selected. Alter the sample information to specify zinc as the sample.

  4. Specify a data file name using the browse button.

  5. Select "Custom" from the Test list.

  6. Click on the Procedure Page, then click on the Editor button. Create the following method:

1 Sample interval:  0.5 sec/pt

2 Data storage:  Off

3 Equilibrate at  250.00°C

4 Data Storage:  On

5 Ramp  20.00°C/min to 500.00°C

  1. Click on the Notes Page.  Enter the requested information, including flow rate.

  2. Click on Apply to store the changes made to these pages.

  3. Press the FURNACE key on the SDT touch screen to open the furnace. Remove all SDT cups, sample, and weights from the sample and reference beams.

  4. Obtain two clean, empty 90 µL alumina cups, and then position them on the platforms. Position the cups as close as possible to the beam side of the sensor.  Mark each cup and make sure that you position the cups so that the mark is rotationally in the same position each time you place the cups on the sensor.

  5. Close the furnace. Once the balance has stabilized, press the TARE key on the touch screen.

  6. Press the FURNACE key on the SDT touch screen to open the furnace.

  7. Place approximately 3 to 10 mg of the calibration standard (zinc) into the sample cup and replace it on the SDT sample (front) platform.  Position the cup as close as possible to the previous position.

  8. Close the furnace.

  9. Schedule this run, if more than one run is in the sequence list.

  10. Select Start to begin the next calibration run.  Wait for the experiment to finish.

  11. Analyze the calibration data as directed in the next section.

Analyzing DSC Heat Flow Cell Constant Calibration Data

  1. Once the calibration experiment is completed, double click the Universal Analysis icon to start the program.

  2. Open the zinc data file gathered for the SDT calibration.

  3. Select Analyze/Options from the Analyze menu.  Then, click on the Onset/Peak Integration Page and verify/set up the following options:

Weight: original

Peak Label: area and normalized area

  1. Integrate the melting endotherm using the Analyze/Peak Integration/Linear baseline function.  The normalized area of the melting endotherm is then used to calculate the cell constant.

  2. Return to the DSC instrument control program and select Cell/Temperature Table from the Calibrate menu.  Enter the calculated value of the cell constant into the Cell Constant field.
