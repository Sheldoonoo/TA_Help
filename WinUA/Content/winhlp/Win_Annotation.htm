Annotation Window

Use the Annotation window to edit or place text on your graph as follows:

 

1. Enter or edit the desired text in the Text box. The maximum number of characters in a single line is 60. The maximum number of lines allowed is 20. NOTE:Text may be copied to or pasted from clipboard by clicking the right mouse button in Annotation text box and selecting the desired function or by typing the appropriate control key. (See also:Edit Text pop-up menu.)

2. Click on the desired text color. The FG (foreground) mark will be displayed in the selected color block.

3. Check the Arrow to Point box, if you want to have an arrow pointing from the text to the reference point selected on the plot.

4. Check the Draw Border box to draw a border box around the label text.

5. Check the Transparent box to make the background of the label text transparent.

6. Click the Params (Parameters) button, if you want to choose common parameters to insert in your annotation. The Insert Data File Parameters window is displayed. Make your selections and click OK. The selected text is inserted at the current position of the cursor in the text box of the Annotation window.

7. Click the Clipboard button, if you want to copy text that is contained on the clipboard (copied from another source) to the annotation text box.

8. Click on OK. The graph is displayed again with crossed marker lines displayed.

9. Position the markers so that they intersect at the desired reference point on the curve.

10. Right click to display the pop-up menu and select Accept Limits (or press Enter). The text will be displayed on the graph at the chosen point.

 

NOTE: When the annotations are displayed, you can drag and drop the annotations anywhere on the graph using the left mouse button. You can edit or delete the annotations by placing the pointer over the annotation and clicking the right mouse button to display theannotate pop-up menu.

 

You can use tabs to separate or align columns of text in an annotation. Tab stops are set every eight characters. If you use a proportional font, each tab stop represents the average character width of eight characters.
