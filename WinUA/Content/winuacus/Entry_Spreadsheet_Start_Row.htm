Entry Spreadsheet Start Row

Enter the number of the first row that should receive entries when a Microsoft Excel® spreadsheet is generated in a custom report.

 

NOTE: This item is best used when you have an Excel template that you want to pull data into. For example, you may have set up pre-defined rows with certain information in them and want to insert the Universal Analysis-generated spreadsheet to fit that template. So you may not want to start a row 1, you may want to start at row 2, leaving row 1 for the pre-defined information. See the Spread Sheet Options window.
