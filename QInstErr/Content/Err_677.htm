Message 677

Run filename incorrectly formed.

Problem:

The data filename does not conform to Microsoft Windows® standards.

Solution:

Enter a valid filename for the data file. Windows NT and Windows 2000 supports both the FAT and NTFS file systems with filenames up to 255 characters. In general, avoid the following symbols when you create filenames:

“, /, \, :, *, ?, |, <, >, -, &, and non-printing characters.
