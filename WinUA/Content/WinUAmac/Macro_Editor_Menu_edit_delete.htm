Delete (Macro Edit Menu)

Use the Delete menu item to remove a step from a macro.

 

Shortcut Keys

Ctrl+D

Delete key

 

1. Ensure that the macro you want to edit is active by opening the desired macro.

2. Select Macro Editor from the main Macros menu to display the Macro Editor window.

3. Select the macro step(s) that you want to cut. To select multiple steps, hold down the Shift key while you click on the desired series of steps or click on a step and drag over a range of steps.

4. Select Edit/Delete from the menu (or press Ctrl+D or press the Delete key). The step will be removed from the macro.
