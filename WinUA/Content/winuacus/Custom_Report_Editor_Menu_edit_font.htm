Font (Custom Report Editor/Edit Menu)

Use the Font option to access the Font window, which allows you to select a font, font type, font size, font color, and effects that will be applied to the highlighted report object.

 

To change the font to all report objects at once, select Edit/Select All from the menu, then select Edit/Font or right click and select Font from the pop-up menu. Then the font specifications chosen will apply to all highlighted report objects.
