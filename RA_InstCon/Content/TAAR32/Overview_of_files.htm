Overview of Files

 

The Rheology Advantage Instrument Control program uses various file types to store the configuration information for procedures, geometries, results etc. As you create new files, follow the naming convention listed below or click here Naming Files.

Creating a Backup File

Follow the steps below to create a backup file copy:

  1. Select View from the Geometry, Procedure or Notes menu (depending on what you wish to back up.)
  2. Select File/Save As....
  3. Enter a name for the backup file.
  4. Click the Save button.

Viewing the Default File Locations

  1. Select Options from the menu.
  2. Click on Settings.
  3. Click on the Directories tab.

The last used folder locations for various file types are displayed. The software sets this to the location used on the last file save.

Using/Displaying Reference Data

When running a procedure on a sample, it is often useful to be able to compare the results against some previously generated data. This allows you, for example, to display upper and lower bounds for QC pass/fail testing.

Displaying Reference Data

  1. Select Experiment from the menu.
  2. Click on Previous results.
  3. Select Previous results of same type and/or Control results .
  4. If you are using Control results, select the first (and optional second) files to display. Enter label(s) if required.
  5. Press the OK button.

Naming Files

To make it easier to find files, you can use long, descriptive file names. The complete path to the file, including the drive letter, server name, folder path, and file name, can contain up to 255 characters. File names cannot include any of the following characters: forward slash (/), backslash (\), greater than sign (>), less than sign (<), asterisk (*), period (.), question mark (?), quotation mark ("), pipe symbol (|), colon (:), or semicolon (;).
