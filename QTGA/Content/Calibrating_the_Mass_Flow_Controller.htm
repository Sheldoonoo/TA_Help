Calibrating the Mass Flow Controller

 

Under normal circumstances the mass flow controller (MFC) should not need to be calibrated by you, the customer. Calibration is done at TA Instruments for you. These calibration tables can be viewed by accessing Calibrate/MFC Sample or Calibrate/MFC Balance. If you need to return the calibration values to their factory settings for any reason, click the Restore Values button and the original calibration values will be restored.

If you have the QNX/Platinum features, you can verify the functioning of the MFCs by using the Auto Diagnostics page.

If you decide that you need to calibrate the MFC, follow the calibration instructions below or call TA Instruments for service. An external high-precision flow meter is required to calibrate for both nitrogen and helium. (Two separate flow meters may be needed, one calibrated for nitrogen and the other for helium.)

To calibrate the mass flow controllers, follow these directions for either the Nitrogen Tables or Helium Tables.

Calibrating the MFC Nitrogen Tables

This table is used to enter the calibration values for nitrogen, oxygen, air, and argon. A factor will be applied against the nitrogen table if oxygen, air, or argon is selected as the purge gas.

  1. Select Calibrate/MFC Sample. Click on the Nitrogen Table tab.

  2. Click the Reset button to set the observed values to zeroes. The Requested Flow and Measured Flow values will display the recommended flow rates for calibration. These values may be changed, if desired.

  3. Click OK to close this window.

  4. Select Calibrate/MFC Balance. Click on the Nitrogen Table tab. Repeat steps 2 and 3 for this table also to reset the values to zero.

  5. Connect a high-precision, external flow meter to the Gas 1 port along with Nitrogen.

  6. Select Tools/Instrument Preferences/MFC Page, then select Nitrogen as the Gas 1 purge gas.

  7. Access the Notes Page for the "scheduled" run and select #1 Nitrogen from the drop-down list for both the Balance and the Sample. Click Apply.

  8. Enter the Balance Flow Rate as 0.0 mL/min and the Sample Flow Rate as 10.0 mL/min and click Apply.

  9. Observe the measured flow rate on the external flow meter. Wait until the flow rate stabilizes and record this value.

  10. Repeat steps 8 and 9 until all of the calibration values are obtained, changing the flow value for only the Sample Flow Rate (leaving the Balance Flow Rate as 0.0). Typical calibration flow rates are 10, 50, 100, 150, and 190 mL/min.

  11. Select Calibrate/MFC Sample. Click on the Nitrogen Table tab.

  12. Enter the Requested Flow and Measured Flow calibration values obtained. Click OK.

  13. Access the Notes Page for the "scheduled" run and enter the Balance Flow Rate as 10.0 mL/min and the Sample Flow Rate as 0.0 mL/min and click Apply.

  14. Observe the measured flow rate on the external flow meter. Wait until the flow rate stabilizes and record this value.

  15. Repeat steps 13 and 14 until all of the calibration values are obtained, changing the flow value for only the Balance Flow Rate (leaving the Sample Flow Rate as 0.0). Typical calibration flow rates are 10, 50, 100, 150, and 190 mL/min.

  16. Select Calibrate/MFC Balance. Click on the Nitrogen Table tab.

  17. Enter the Requested Flow and Measured Flow calibration values obtained. Click OK.
