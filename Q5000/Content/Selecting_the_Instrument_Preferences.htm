Selecting the Instrument Preferences

 

You can set up certain operating parameters for your instrument through the instrument control software by selecting Tools/Instrument Preferences. Several pages will be displayed to allow you to set these parameters. To display the desired page of Instrument Preferences, click on the corresponding tab.

Select from the following topics for more information:

  • MFC Purge Page - Applies to Q5000 IR instruments only: Specifying the Connected Q5000 IR Purge Gases

  • LCD Signals Page - Setting Up Signal Display Touch Screen

  • Autosampler Page - Selecting Autosampler Options

  • TGA Page - Setting Up the Instrument

  • Humidity Page - Applies to Q5000 SA instruments only: Selecting the Q5000 SA Humidity Options

  • Autoweight Cal - Automatically Calibrating Weight

Specifying the Q5000 IR Connected Purge Gases

To set up the gases select Tools/Instrument Preferences, then click on the MFC Purge page. The appearance of this page differs slightly depending upon whether your instrument is equipped with mass flow controllers (MFC) or not.

Use Gas #1/#2 to specify the purge gases that are connected to the corresponding ports on the instrument. A purge gas is a gas, often an inert gas such as nitrogen, that flows through the instrument during the experiment.

  • If there are no mass flow controllers installed, Gas #1 is designated strictly as the Sample Purge and Gas #2 strictly as the Balance Purge.

  • If there are mass flow controllers installed, Gas #1 can be used as both the Sample and Balance Purge and Gas #2 as the Sample Purge, giving you the option of using two gases as the sample purge during different points in your experiment.

  • In addition, you can check the Stop experiment when flow rate deviates from the set value checkbox, if you would like to have the experiment stop in the event the flow rate cannot be maintained during an experiment (e.g., if the purge gas tank becomes empty).

Select the flow rate of the purge gases on the Notes Page.

NOTE: If you are using laboratory purge gas, rather than bottle purge gas, you will need to install an external drier.

CAUTION: Use of corrosive gases will shorten the life of the instrument. NEVER use a corrosive gas in the balance purge. The MFC is not designed to work with corrosive gases.

WARNING: Use of an explosive gas as a purge gas is dangerous and is not recommended for this instrument.

Setting Up the Signal Display Touch Screen

To determine which signals coming from the instrument will be displayed on the Signal Display touch screen (touch the Display Menu key to display), select them from the dialog that is opened through the Tools/Instrument Preferences/LCD Signals page.

Select up to six signals. These signals will appear on the instrument touch screen.

NOTE:  This does not affect the signals displayed on the instrument control Signal Display Pane. Those signals are customized using another function.

Selecting Autosampler Options

Use the Tools/Instrument Preferences/Autosampler page to set up the actions of the Autosampler. The stop options instruct the Autosampler on the action to take when the run sequence has been stopped or runs to completion. The error handling options determine whether the Autosampler will keep going or halt when a nonfatal error is encountered. If you want to use the Autosampler instrument as a single sample instrument, check the Disable Autosampler box.

Autosampler Sequence Stop Options

Select the desired stop sequence option (click for a definition). These conditions will override the Post-Test conditions for the last run in the sequence.

  • Open furnace and unload

    Select to have the furnace automatically open at the end of the sequence and unload the last sample pan in the run sequence. The furnace will not open until the temperature is below 600°C.

  • Leave furnace closed

    Select to leave the furnace closed at the end of the sequence. (Air Cool will not turn on when the furnace is closed.)

Autosampler Error Handling

Use the Error Handling options to specify when to continue or halt an Autosampler sequence when a recoverable error occurs. Select the radio button next to the desired setting to turn it on (click for a definition):

  • Recover on error

    Select to have the Autosampler attempt to continue operation when it encounters a recoverable, nonfatal error condition (such as not picking up a sample pan from the platform).  Two attempts will be made to pick up a pan, if unsuccessful, the sequence will continue to the next run in the sequence.  An error entry will be added to the instrument log.

  • Halt on error

    Select to have the Autosampler suspend all further operation when it encounters any Autosampler error condition.  When an error occurs, you must first correct the problem, then you can restart the sequence from that point.  An error entry will be added to the instrument log.

Disabling/Enabling the Autosampler Functions

If you prefer to work with the TGA without using the Autosampler to load pans, you may do so by disabling the Autosampler. This can be done using one of these methods:

  • Select the Instrument Preferences/Autosampler page, then uncheck the option Enable Autosampler.

  • Select Control/Autosampler/Enable, then uncheck the menu item. (If there is a check mark next to Enable, the Autosampler is enabled and will operate as normal.)

NOTE: The Autosampler may not be disabled during an active sequence.

Setting Up the TGA

Choose the following options for the Q5000:

  1. Enter the desired Balance Housing Temperature (should be slightly above room temperature). This entry will maintain your balance at the desired temperature (40°C is recommended for the Q5000 IR and 35°C is recommended for the Q5000 SA).

  2. Choose the type of Sample Platform that is present on your instrument. Typically the IR furnace (Q5000 IR) instruments use the Standard (25 pans) tray and the Humidity (Q5000 SA) instruments use the Quartz (10 pans) tray.

  3. Check the Enable Evolved Gas Heater box to enable the heater inside the Mass Spectrometer Interface Assembly to facilitate the transfer of gases evolved from the sample. (See Connecting a Spectrometer for details.)

  4. Set the % of Full Power for the heater (used with the spectrometer interface) at 40% (the default value). Please contact TA support for advice when considering an increase in the power.

  5. Click OK when finished.

Selecting the Q5000 SA Humidity Options

This page is only available when you have the Q5000 SA instrument installed. Set up the following options for your experiments:

  1. Enter the desired Standby Temperature, which is the temperature that the instrument will maintain between experiments.

  2. Enter the desired Standby Humidity, which is the percent relative humidity (%RH) that will be maintained between experiments.

  3. View the default gas to the Balance Purge  and the Humidity Purge. This page contains a place for selection of the purge gases, however, only the default option, based on nitrogen gas for both purges, is currently available. The Q5000 SA only has one gas inlet port on the back of the instrument. The dry nitrogen source should be connected to that port. See Connecting Purge Gas Lines for information.

    • You can check the Stop experiment when flow rate deviates from the set value checkbox, if you would like to have the experiment stop in the event the flow rate cannot be maintained during an experiment (e.g., if the purge gas tank becomes empty). The default for this selection is "unchecked." This prevents a short-term upset in flow from stopping the experiment unnecessarily.

  4. Select the Flow Rate for each purge gas.

  5. Click OK when finished.

NOTE: If you are using laboratory purge gas, rather than bottle purge gas, you will need to install an external drier.

CAUTION: Use of corrosive gases will shorten the life of the instrument. NEVER use  corrosive gases for the Q5000 SA balance purge. The MFCs are not designed to work with corrosive gases.

WARNING: NEVER use an explosive gas as a purge gas.
