Introduction to the Analysis Database

An analysis database allows you to keep a record of the results of an analysis on various files. The database can be updated when you generate new data and is thus useful for spotting time related trends (over days, weeks or even longer) in your data.

Click on the topics below for more information:

Creating a Database

Follow the steps below to create a new database:

  1. Select Database from the menu.
  2. Click on New.
  3. Select the location for the database you wish to create and then enter a new name.
  4. Click on the Save button.

Once you have created the database, the first analysis you perform will set the analysis type for this database.

Loading an Existing Database

Follow the steps below to load an existing database into the software:

  1. Select Database from the menu.
  2. Click on Open.
  3. Select the database you wish to open.

Once you have a database open, all subsequent analysis of the type set in the file will be added to the database.

Setting the Default Database

When you open a database file, all model analysis that you perform will be added to the database, provided that it is of the same type as that set in the analysis database file. You can open further database files that contain different analysis types, and the software will automatically add the correct file analysis to the correct database

If, however, you open more than one database with the same analysis type, you must specify which is the active one.

  1. Highlight the database in the Results list window that you wish to set as default.
  2. Select Database from the menu.
  3. Click on Set Default.
  4. The Results list window will show the default database symbol against the active file.

QC Functions of the Database

The database allows you to use QC criteria on the data. Follow these steps:

  1. Select Database from the menu.
  2. Click on QC...
  3. Ensure that Perform check is selected.
  4. Select the required QC parameters and then press the OK button.
