Checkbox Transparent

Check this box to make the background of an object transparent or "see-through" for any objects behind it in the final Microsoft Word® custom report. For example, if you check this box for a Text object, you can position the text label over a curve and the curve will show through the background of the text characters.

 

Leave this box blank to make an object opaque.

 

NOTE: This setting does not apply to Microsoft Excel® Spreadsheets. They must always remain opaque.
