Setting Up TTS Options

 

You can set up several Time-Temperature Superposition program options by selecting TTS/Options on the menu.  The window shown here is displayed.  Select the following options from the drop-down list for each item:

  • Key units:  The key shows the temperature for each curve. You can select between Kelvin and Celsius.
  • Density source:  Select the source of the sample density figure.
  • Vertical shifting:  If you wish to use vertical shifting, select one of the three options:
    • Temperature correction - The curves are automatically shifted vertically to compensate for G µ T
    • Temperature / density correction - For most systems, both the modulus and the volume change with temperature. This correction contains terms for both temperature and density and will generally give the best results. However, you must have a value for the sample density to use this correction.
    • Manual - Allows you to manually shift the curves.
