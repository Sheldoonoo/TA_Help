Select Multiplex (Frequency or Amplitude)

Use the Select Frequency option to separate the data according to a chosen frequency. This option is very useful when performing a function on an individual frequency.

 

1. Choose Select Frequency from the Analyze/Analysis Range menu. The Data Frequency Window is displayed.

  1. Click on the button to display the drop-down list of frequencies available.

  2. Click on the desired frequency, then click the OK button. The data in that frequency range will be available for analysis. The data outside the analysis range will be grayed out.
