Starting a Run and/or Sequence

If more than one run is contained in the run sequence, the scheduled run (run with the arrow pointing to it) will be executed. To schedule a run, prior to starting the experiment, move the pointer (cursor) along the left of the run numbers, you should see the pointer change to an arrow. When it becomes an arrow, you can then click the mouse to position the arrow next to the selected run number. This run becomes the scheduled run, which is the experiment that will be run when Start is selected.

Once the sample has been loaded, the experimental parameters set up, and the run scheduled you can start your experiment one of four ways:

  • Select Control/Start on the main menu
  • Click on the tool bar.
  • The scheduled run can also be executed by double clicking on the red run arrow in the Sequence Pane.
  • Schedule the desired run using the controller software (as outlined above), then press the START key on the instrument keypad or touch screen.
  • If you have an Autosampler and it is enabled, the run sequence will be executed, starting with the scheduled run, when Start is selected.

For starting conditions pertaining to particular cooling accessories, see these topics: About the RCS or About the LNCS.
