Print Preview Window

Use this window to view the current graph as it would appear when printed.

 

NOTE: If you want to change the text font or position, select File/Options from the main menu. Then set up the fonts as desired on theMain OptionsFont Page.

 

The menus shown on this window allow you to perform several functions:

  • Select File/Close to close the window.

  • Select File/Export Plot to export the existing plot to the clipboard, file, printer, or plotter.

  • Select File/Export PDF File to export the existing plot and save it as an Adobe PDF file.

  • Select File/Print to print the graph.

  • Select File/Print Setup to select print options, such as the printer, paper, and orientation to use when printing.

  • Select Help/Help Topics to obtain help for the program.

  • Select Help/Help on Print Preview to obtain help for the window.
