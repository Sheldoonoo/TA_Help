Word Template (Custom Report Editor/File Menu)

Use the Word Template function to base your custom report, when it is generated, on an existing Microsoft Word® template file. When this menu item is selected you can browse for and select a Word (.dot) file using the Word Template window.

 

For example, you may have a Word template already configured with the headings, styles, and fonts that you want to use for your reports generated with Universal Analysis Custom Report Editor program.
