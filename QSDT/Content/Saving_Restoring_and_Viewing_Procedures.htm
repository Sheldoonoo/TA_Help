Saving, Restoring, and Viewing Procedures

 

When running experiments on your instrument, you will be either creating a new custom procedure files, selecting a preprogrammed procedure, or restoring a saved procedure file. A procedure contains all of the information regarding the test protocol (mode, signals, method segment, advanced parameters, and post-test options).

Select from the following topics for more information:

  • Saving a Procedure File

  • Restoring an Existing Procedure File

  • Viewing a Procedure File

  • Specifying a Test Procedure

 

Saving a Procedure File?

Saving a custom procedure for future recall is easy. After you have created one, access the Procedure Page, then click the button. The Save Procedure File window is displayed to allow you to choose the location for your procedure files (.prc) and name the file before saving it.

Restoring an Existing Procedure

To restore a saved custom procedure file, access the Procedure Page in Experimental View and click on the button. This allows you to select the desired procedure file from the Open Procedure File window. A preview window shows you the contents of the procedure file before you open it. The selected file will set the mode, signals, test, advanced and post-test conditions for the run.

Viewing Procedure Files

To view a summary of a saved procedure file follow these steps:

  1. Select Tools/File Utility to open the File Utility window.

  2. Select the procedure file type (*.prc) from the available list. a summary of the procedure will be displayed on the right in a preview pane.

  3. Right-click to display a pop-up menu that allows you to either print the procedure file or display the file in an editor.
