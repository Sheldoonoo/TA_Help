Computer Controlled Fluid Circulator (CCFC)

(Applies to AR 1000/550/500 Only)

If accurate temperature control is desired when using a concentric cylinder system, you need to use an external fluid circulation system. The system recommended by TA Instruments uses a Computer Controlled Fluid Circulator (CCFC), which can be interfaced with the software. It provides very stable temperature control of the sample (and not the circulating fluid temperature). This involves the use of an external Pt100 placed in the cup of the concentric cylinder system. The use of such a system is detailed in a separate instruction booklet available from TA Instruments.

Temperature range: -20°C to 150°C
