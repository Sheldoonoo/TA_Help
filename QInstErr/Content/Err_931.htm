Message 931

It is recommended to enable Go To Temp to avoid frosting the cell.

Problem:

The instrument is not currently maintaining a temperature control window, which could result in frost accumulating on the cell.

Solution:

Select Control/Go to Standby Temp from the menu.

This temperature is set using the Tools/Instrument Preferences/DSC Page to enter the desired temperature value. This option is used when the selected cooler is LNCS or RCS and the Post Test temperature window is not used or stopped. It prevents the cooling temperature from dropping too low and causing frost to build up in the cell.
