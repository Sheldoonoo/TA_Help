Insert Data File Parameters Window

Use this window to select data file parameter block information for insertion into your annotation as follows:

  1. Uncheck any items that you do not want to appear in your annotation or click the All button to select every item displayed, or click the None button to uncheck every item displayed.
  2. Click the OK button when you have finished making your selection. The Annotation window is displayed again.

NOTE: Some parameter block entries may not appear in the parameters check list. To insert these parameters, first copy them to the clipboard from theReport Editor windowor from theEdit Parameter Block window, then paste them into the Annotation text. See also:Edit Text pop-up menu.)
