Constructing Scripts Using the Record Mode

 

Script construction is very simple using the "Record" Mode. The Record Mode enables you to construct scripts in a logical fashion directly from within Rheology Advantage software. All actions in the Instrument and Data modules are recorded and put into script format. Additional lines for prompts, etc., can be added as the script is developed or during editing after recording is completed.

Accessing the Record Mode

To access the Record Mode, use the icons shown below or select Start from the Record menu.

If you select the Instrument icon, for example, the Rheology Advantage instrument control module will be started. The screen is the same as the conventional module with an additional toolbar and the Navigator option on the menu bar.
