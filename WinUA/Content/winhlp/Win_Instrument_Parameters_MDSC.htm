Instrument Parameters Window  MDSC®

Use this window to enter the instrument parameters that are used when performing an MDSC® phase correction procedure. Click here for details on how to perform MDSC® phase correction.

 

NOTE: This is available to DSC 2920 operators only.

 

Follow these steps to use this window:

 

1. Click on the radio button desired for the MDSC® phase correction: No correction, Time limits, or Temperature limits.

2. Enter the start and stop limits for the phase correction. A precise phase correction calculation will be performed on this data contained within this range and extrapolated to the data outside the range. The calculations done on the data outside of the phase correction range may not be as quantitatively accurate as that contained within the range. We recommend that you place the data limits: (a) in a region of zero phase lag (i.e., in a region where no transition or kinetic heat flow is occurring), (b) so that they include the transitions of interest, and (c) provide at least 10°C of baseline on either side of the transition(s). See also: Choosing the Phase Correction Range.

3. Click the Save button, if you want to use these as your new default settings or click the OK button to use these settings for this session only.
