Previous Limits

The Previous Limits option redraws the graph as it appeared before the most recent scale changes. Only the scaling options selected from the Rescale menu are affected by this menu option, any changes made in color or annotation are not affected. Selecting Previous Limits a second time will return the graph back to the most recent changes.

 

Shortcut Key

Ctrl-U
