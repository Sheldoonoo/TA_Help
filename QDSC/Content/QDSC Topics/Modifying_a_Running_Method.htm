Modifying a Running Method

 

To change an active method, select Experimental/Modify Running Method from the main menu. The Modify Running Method window is displayed. It allows you to perform one of the following operations for an active experiment.

Select from the following topics for more information:

  • Going to next segment
  • Changing the data sampling interval
  • Modifying the current method segment
  • Modifying the running  method

Any changes made to the method segments will be maintained after the experiment is completed.

Go to Next Segment

If you would like to advance the active method to the next segment, just click on the Go to Next Segment button. The method will automatically advance to the next segment.  If this option was selected while the last segment was active, the method will end.  Alternatively, you can select Go to Next Segment by right clicking on the running segment pane and selecting this option from the pop-up menu.

Changing the data sampling interval

During an active experiment the current data sampling interval can be changed from the Modify Running Method window.  To change this value, simply enter a new value and click on the Send Changes button. If an experiment is not currently in progress, you can specify the data sampling interval for the selected run by accessing the Advanced Parameters or by using a data sampling interval segment in your method. See also:  Segment List.

Modifying the current method segment

To change the parameters for the segment that is currently being performed open the Modify Running Method window, then follow these steps:

  1. Double click on the current method segment in the segment description list.  Enter the new values.  Use the TAB key to move between the segment parameters, if there is more than one parameter.
  2. Click on the Send Changes button to send the new segment parameters to the instrument for execution.  ([If the current segment completes before the new parameters are sent, the changes will not be applied.)

Modifying the running method

The Modify Running Method window allows you the flexibility to alter a running method by changing segment parameters, as well as adding, inserting and removing segments.  However, you can only make changes to the currently executing segment or those segments below the current segment.  Changes cannot be made to segments that are already completed.  These operations are the same as within the Method Editor window when you are creating this method, but you need to click on the Send Changes button to send the updated method to the instrument for execution.  [If the current segment is completed before the new method is sent (using Send Changes), changes may be lost.]
