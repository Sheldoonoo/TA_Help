Overlay Legend Size

The curve overlay legend is a block of information that can be used to identify the curves used on your curve overlay graph. Because there are many options that you can select or set up to appear on this legend, it may get too large. You may not detect that the legend is too large on your display since font size changes do not appear on the display screen, but will appear when the plot is printed.

 

To help you solve this problem, a message will appear in the curve overlay legend box when appropriate. For example, if your legend is too large for printing, but not too large for display, then the message will not appear until the curve overlay graph is printed. The opposite will also be true, if the legend is too large for display, the message will appear there.

 

Solving the Problem

 

Try one or more of the following solutions to solve the size problem:

 

  • Always check the appearance of your curve overlay graph before printing it by using the File/Print Preview option. Then adjust the legend using one of the suggestions below.

 

  • Uncheck a few of the "Show" items on the Curve Overlay Options  Legend Page to reduce the number of entries. Check each axis for redundant entries.

 

  • If the legend is too wide, then reduce the number of text characters appearing in the "Filename" and/or "Show Legend Text" fields on the Curve Overlay Options  Curve Page.

 

  • Reduce the font size used in the legend by selecting the Legend button on the Main Options  Font Page, then reducing the font size on the Font window.

 

  • Reduce the number of curves appearing in your legend by removing one or more, as needed, using the Delete Curves button on the Curve Overlay Options  Curve Page.
