Entry Audit Trail Comment

Enter the desired comment to add to the audit trail log for the current data file or click the Clipboard button to paste the information copied to the clipboard in the comment entry field. The maximum length of each line is 60 characters and up to 30 lines of text can be entered for a single comment. NOTE: You can clear the description box by clicking the Clear button.

 

The comment is described in the Audit Trail as a "User Comment."
