Save Cell Constant?

The heat flow signal used to generate this calibration file does not match the current heat flow selection.  Cell constant and temperature calibration experiments should be performed using the same heat flow as sequent experiments.  Do you want to save the cell constant?

As the message indicates, the heat flow signal for the instrument as currently configured does not match the heat flow signal for the calibration file currently used for cell constant calibration. They are inconsistent. If you save the cell constant calibration using these two different heat flow signals, the calibration may not be reliable for your experiments.

  • Select Yes to save the cell constant. If you select Yes, you may want to consider changing the heat flow signal to match that currently used in the calibration file.

  • Select No to cancel the operation.
