Delete (Autoqueue)

Select Delete to delete the selected record. (If you want to copy a record to the clipboard rather than delete it, use the Edit/Cut option in the Autoqueue menu.)

 

Shortcut Key

Delete

 

1. Select Autoqueue from the main Macros menu to display the Autoqueue Window.

2. Click on the desired record. (Use Select All to select all records.)

3. Select Edit/Delete from the menu (or press the Delete key). The record(s) will be deleted from the list of records in the autoqueue.
