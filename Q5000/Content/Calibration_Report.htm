Calibration Report

 

If an instrument has been properly calibrated, a calibration report can be generated. Select Calibrate/Report from the main menu to display a window such as the one shown to the right.

This reports lists information obtained from the calibration procedures.

 

Select from the following topics for more information:

  • Calibrating the Q5000 - Overview

  • Calibrating the TGA Weight

  • Calibrating the TGA Q5000 IR Temperature

  • Verifying the Q5000 SA Humidity

All of these calibrations can be performed from items found in the Calibrate menu.
