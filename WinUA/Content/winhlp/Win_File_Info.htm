File Information Window

Use the File Information Window to modify the parameters on the Data File Information Window. If only text items are changed, the data in memory is not reloaded (i.e., current analyses are not lost). If any other items are changed, the data is reread from the file so the new values (e.g.,size, cell constant) can be applied. These changes are not saved to the original data file.

 

At the top of the File Information Window is the data files run date, number of data points read from the file, number of data points used to plot the data (points saved in the data buffer), number of data cycles in the file, and the data compression ratio (points read divided by points used).

 

NOTE: The data buffer size can be changed using theMain Options - Data Page.
