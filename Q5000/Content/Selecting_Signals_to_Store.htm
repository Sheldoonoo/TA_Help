Selecting Signals to Store

 

Every analytical instrument produces distinctive signals that may or may not be stored in a data file. To select the signals that you want to be saved in data files produced during this experiment, access the Summary Page in Experimental View. This list will vary based on whether your options. Click the button. Check the signals to be stored from the list shown. Then click the OK button.

Some examples of available signals stored are:

  • Time

  • Temperature

  • Weight

  • Temperature Amplitude

  • Derivative Weight Amplitude

  • Ln (Rate Ratio)

  • Activation Energy

  • Log (Pre-Exponential)

  • Modulated Temperature

  • Modulated Weight

  • Balance Purge Flow

  • Sample Purge Flow

The signals selected apply only to the current run. For a description of these signals, see Using the Signal Display Pane.
