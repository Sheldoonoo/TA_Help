Instrument Type Not Supported

The instrument type is not supported or the required component is missing for the instrument!

This message indicates that something unusual has happened to the instrument control software. One of the following events may have occurred:

  1. A software component was accidentally deleted.
  2. The registry entry that defines where the dll resides has been changed.  This means the component can't be found.
  3. Some type of error occurred during the installation.

If this message persists, install the software again following the instructions displayed on the screen carefully.

If the message still persists after reinstalling the software, call TA Instruments for service.
