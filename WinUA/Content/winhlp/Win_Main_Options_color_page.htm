Main Options Window  Color Page

Use this window to set up the colors that you want to use as your default settings when you are using the Universal Analysis program.

 

Follow these steps to set up the color options:

  1. Click the down arrow to display the drop-down list of colors for the item that you wish to set up.

  2. Click on the desired color for that item.  

  3. Repeat steps 1 and 2 for each item that you want to set up. Multicolor is used to set up the "multicolor" option that is available to apply colors to curves in normal and overlay plots. (This choice is available on the Graph Options window, Curve Overlay Options - Curves Page, and normal Plot Legend window.) You can select a color to apply to up to 12 corresponding cycles in the data. If more than 12 cycles exist in the data, the colors in the list will be repeated.

  4. Click the Save button, if you want to use these as your new settings or click the OK button to use these settings for this session only. The color(s) will be applied to the selected portion(s) of the graph.

Or

  1. Click on the down arrow to display the drop-down list of color schemes that you can use for the entire program.

  2. Click the Save button, if you want to use these as your new settings or click the OK button to use these settings for this session only. The color(s) will be applied to the selected portion(s) of the graph.

  3. Click the Default button to return all of the colors to their original factory settings.

NOTE: The Save button only stores the settings for the selected page in the initialization (.ini) file. To save all settings, access theMain Options  Initialization Pageand click the Save As button. The Default button will also only recall the default factory settings for the selected page.
