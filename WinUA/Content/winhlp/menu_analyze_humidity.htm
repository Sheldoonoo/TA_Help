Analyze Humidity

Select this menu item from theAnalyzemenu to perform the analysis on the current humidity data file. The following guidelines should be considered when analyzing:

 

  • Data is generally plotted first with Weight Change Percent on the Y1 axis, Relative Humidity on the Y2 axis, and Time on the X axis.

  • Select Analyze/Reset Size to select the dried weight of the sample.

  • Before analyzing the data, set up the analysis options by selecting Analyze/Humidity/Options.

  • When you select Analyze/Humidity there are three ways to display and analyze the data. You can select Humidity Sorption Isotherm, Humidity BET Analysis, or Humidity GAB Analysis. Each type of analysis will give you a new results plot.

  • The original data plot remains intact after analysis. No analysis results are posted on the original plot. Analysis results are posted on separate plots. These plots can be saved using the Save Analysis function.

  • Enter the molecular weight of the sample in g/mol when needed (to determine moles of water adsorbed/desorbed during the experiment).
