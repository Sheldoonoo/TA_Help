Choosing the TMA Purge Gases

You can control the sample atmosphere during  experiments by connecting a purge gas to the TMA system.

NOTE: Please make sure that you run your experiments with the same gas that you used to calibrate the system. For example, if you calibrate using nitrogen, make your runs with nitrogen.

  • MFC Calibrated Gases:  air, argon, helium, nitrogen, or oxygen
  • If you are planning to run subambient experiments helium is recommended for use as the purge gas.
  • Purge gas can be obtained from a pressurized cylinder or an in-house supply source. Gas supplied from an in-house source should be passed through a sieve dryer to remove any trace of moisture before it enters the TMA.
  • If you have an instrument with an MFC, the gases are passed through a mass flow controller (MFC), which is inside the cabinet. The flow rate for the  MFC is set through the instrument control software. See Specifying the Connected Purge Gas for information.
  • If your instrument does not have an MFC, the gases must be controlled by a flow meter connected to each gas outlet (unless you upgrade your instrument and install a  mass flow controller [MFC]).
  • If you are using a purge gas that is not one of the available options, you can select "outside purge," which fully opens the MFC valve requiring that the flow rate be controlled using an external flow meter. Or you can select a purge gas type from the drop-down list that has similar characteristics to the one you want to use (e.g., conductivity) and use that calibration table for controlling the flow rate via the MFC.

CAUTION: The mass flow controller is not designed to work with corrosive gases.

WARNING:  Do not use hydrogen or any other explosive gas in the TMA. If you are using samples that may emit harmful gases, vent the gases by placing the TMA near an exhaust.

WARNING:  Do not use any liquid in the purge lines.

Setting the Flow Rates

The gas entries on the Notes Page of the Experiment View are used to specify the gas and the associated flow rate for each run.

  • If an MFC is installed, these settings will automatically set the gas and flow rate on a run by run basis. The scheduled run's purge settings will take effect immediately once the Apply button is clicked.
  • If your instrument does not have an MFC, then these entries are for information purposes only.

To change the gas type, access the Instrument Preferences/MFC Purge Page.

Recommended Flow Rate:   50 mL/min
