Delete Curve Results?

This curve contains saved results that will be deleted when the curve data is modified. Are you sure that you want to delete the saved results for this curve?

 

This message appears when you select Tools/Shift Curve after performing analyses on the curve. If you shift the curve, the results will be lost. If you want to proceed, regardless of the lost results, select Yes. If you do not want to lose the results, select No and the shift will not occur. Select Cancel to abort the operation.
