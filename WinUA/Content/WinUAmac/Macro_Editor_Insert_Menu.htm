Macro Insert Menu

Use the Insert menu to add dialogs (message boxes), pauses (delays), jumps to other steps, beeps, and other functions that can be used to further customize the steps in your macro. Select the desired position for the insert option before selecting the menu item from the drop-down Insert menu.
