Cannot Save Analysis

Cannot save analysis to text file.

 

Problem:

Analysis and set-up information cannot be saved to an ASCII or Spread Sheet format data file.

 

Solution:

To save the analysis in the file you must analyze the binary format version of this file. If the binary format version of the file is not available, you can create one by using the File/Export Data File/File and Plot Signals function and selecting the Binary data file radio button. Alternately, you can return to the data file selection screen and save the entire session of the program in a saved session file. This does not save the analysis in the data file, it just restores the current state of the program.
