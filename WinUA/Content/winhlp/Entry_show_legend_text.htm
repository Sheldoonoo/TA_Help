Entry Field Show Legend Text

Use this field to edit the legend text that will be applied, if the Text box has been checked. This is a customizable label used to distinguish curves in the graph legend. The text defaults to the sample identification found in the data file. The maximum number of characters is 32.

 

NOTE: If you are having problems with the size of your curve overlay legend, either on the monitor, printing, or otherwise, seeOverlay Legend Sizefor more information.
