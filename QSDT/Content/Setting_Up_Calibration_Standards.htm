Setting Up Calibration Standards

 

This function is used to define the default settings for each temperature calibration standard, add new standards, and reset (reload) the default calibration table.

Access the Calibration Standard window by selecting Calibrate/Analysis/Temperature from the main menu. Choose a file on the Open Calibration File window. The Calibration Analysis Specification window will be displayed.

Then set up the calibration standard as follows:

  1. Select the Standard Name from the drop-down list to edit or verify its default parameters displayed on this window. To add a new standard to this list, type in a new name and define its settings. This standard will be saved when you click the OK button.

  2. Enter the literature Melting Point in Â°C. Consult your Chemistry handbook for the value.

  3. Select OK when completed.

NOTE:  To add another standard, click the Add button. To edit the standard values, click the Edit button. To return the standards list back to the default values, click on the Default button.

Resetting the Calibration Standard Settings

In the event that you want to restore the entire calibration table back to the original manufacture defaults (set at TA Instruments), click on the Reset All button. A confirmation message will be displayed. Select Yes to continue operation or select No to cancel the reset function.
