Data Buffer Too Small

The current data buffer size of xxxxx points is too small to restore this analysis. The buffer size must be increased to a minimum of xxxxx points.

 

Problem:

This message is displayed if you are attempting to restore a saved analysis that requires larger data buffers than the current buffer size was improved to read.

 

Solution:

Increase the minimum buffer size by closing all open files, then selecting File/Options and editing the data buffer size on the Main Options/Data Page. Add a margin of ten percent over the buffer size displayed in the message.
