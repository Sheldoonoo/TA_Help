Calibrating the Q5000 SA Mass Flow Controllers

 

The Q5000 SA mass flow controllers (MFCs) are precise devices that are factory calibrated and verified using a rigorous procedure. The resultant calibration tables can be viewed by accessing Calibrate/MFC Dry or Calibrate/MFC Humidified. If you need to return the calibration values to their factory settings for any reason, click the Restore Values button and the original calibration values will be restored.

Verification of continuing satisfactory  performance (calibration) of the MFCs can be achieved by evaluating a known deliquescent salt. If the deliquescence value obtained lies within the acceptable range, the MFC flow calibration is still okay. If the deliquescence value obtained is outside the acceptable range, please contact TA Instruments Service.

When entering data from the MFC manufacturing tag, the following key applies.

E3     MFC (dry) Measured Flow

E5     MFC (dry) Secondary Flow

E2     MFC (humidified) Measured Flow

E4     MFC (humidified) Secondary Flow

E1     MFC Balance Flow
