Setting Up a Curie Point Temperature Calibration

(Q5000 IR Only)

Temperature calibration is useful for TGA experiments in which precise transition temperatures are essential.  There are two techniques for TGA temperature calibration recognized by the American Society for Testing and Materials. These techniques are described in ASTM Standards E914-83 and E1582-93. The latter, which is based on the Curie Point of paramagnetic metals, is recommended for the TA Instruments TGA's.

In this technique, a Curie standard is heated in a sample pan in a magnetic field. The Q5000 IR furnace is equipped with an electromagnet to facilitate this operation.  As the standard goes through its Curie Point, its attraction to the magnet changes, appearing as a weight change. The extrapolated endpoint of this weight change is adjusted to agree with the material's known Curie Point Temperature.  Up to five calibration points can be entered into the temperature calibration table.  A multi-point calibration is more accurate than a one-point calibration.  

When you perform a TGA Curie Point Temperature Calibration experiment, these test parameters need to be specified:

  • Equilibrate at:  Select a temperature that isup to 100°C below the literature Curie temperature of the material.

  • Isothermal for:  Check this box to hold the sample at the equilibrate temperature for the desired amount of time BEFORE the magnet is applied for calibration. This is done to observe the effect of the magnet on the sample.

  • Heating Rate:  Heat the material at the same heating rate that you will use in your subsequent experiments to 100°C above the literature Curie temperature.

For more information on determining the Curie Point Temperature, see Calibrating the TGA Q5000 IR Temperature.

Using the Experimental View Procedure Page:

  1. Display the Procedure Page. Select the test from the list of preprogrammed templates.

  2. Enter the desired parameters. When you have finished setting up this procedure, you can enter more information by selecting the Notes Page and Summary Page for this run.

  3. Click the Apply button when finished to save these changes or the Append Run button to save the changes to this run and add another run to the end of the sequence.

NOTE: You can use any test template as the basis for a custom test by selecting the desired test template from the list of tests, then click the Apply button and change the test to Custom. Then you can click on the Procedure Page and then the Editor button to make the changes desired.
