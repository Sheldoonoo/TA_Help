Message 192

Autosampler platform did not reach destination. Cannot load or unload.

Problem:

The sample platform did not move to the correct position to allow the pan to be loaded or unloaded.

Solution:

Try one or more of the following operations to solve the problem, then restart your experiment.

  1. Make sure that there is a sample platform installed.
  2. Make sure that the platform rotates freely.
  3. Make sure that nothing is laying on the arm behind the platform.
  4. If none of the steps above solve the problem, call TA Instruments for service.
