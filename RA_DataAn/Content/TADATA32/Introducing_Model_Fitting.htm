Introducing Model Fitting

 

In order to perform a quantitative comparison of materials, it is generally required that you fit the experimental data to some form of best -fit mathematical equation or 'model'. These models range from the basic prediction of viscosity as a function of shear stress or shear rate, to rigorous constitutive equations that attempt to deal with all rheological properties over a wide range of conditions. In the majority of cases, there is no need to obtain a constitutive equation, since an existing model will give useful results. In addition to those built in to the program, Rheology Advantage also allows you to define your own models.

Models can be fitted to part of a curve by use of the limits function.

Selecting an Analysis Range

Normally, a model fit or analysis uses all of the data points in the results file. In some cases, you may wish to limit the range of values used. Select Analysis from the menu and then click on Cursors or click the button. You can then select the upper and lower limits to use by dragging the cursors on the graph or clicking on the limits button

NOTE:  If you have one or more bad points, you may wish to delete them or apply a smoothing to the curve. You can not exclude an individual point in the limit range. You must delete it instead.

Adjusting an Analysis

When you have fitted a model to a data set, you may wish to "fine tune" the results obtained by the software.

  1. After you have fitted a model to the data, ensure the model fit is selected and then right mouse click in the Analysis Results window.

    Analysis Results Window:  To display this window, select Options from the menu bar, then Settings. Click on the View tab and ensure that Analysis results is highlighted in the tool bars listing.

  2. Select the Adjust all option from the pop up menu.
  3. Click on the term you are interested in altering and then move the slide control to adjust the term. You should be able to see the graph being adjusted in real time.

NOTE: If you wish to reset the model to the value obtained by the software, simply delete the model fit and then re-apply the model to the data.

Calculating a Point

 

Once you have used a model fit on a result file, you can then use it to calculate point values. For example, if you wish to determine the viscosity at a particular shear rate, but there is no data point at this shear rate.

 

  1. Load the required results file.
  2. Apply the required model to the curve.
  3. Right click in the Analysis Results Window.
  4. Select point calculation from the drop down menu shown to the right.
  5. Click on the Insert button to add a value.
  6. Double click on the new value to edit it.

Type in the independent value and the software will calculate the dependant using the model. The new value will be added to the values shown in the Analysis Results window.
