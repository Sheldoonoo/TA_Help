DMA Calibration Report

 

If an instrument has been properly calibrated, a calibration report can be generated. Select Calibrate/Report from the main menu. The following information is displayed.

Instrument Calibration Report

Type               Date                      Residual

Electronics:     24-Oct-01 10:13      0.00005

Force:             24-Oct-01 14:37      0.0118 Enabled

Dynamic:        26-Oct-01 07:44       0.0100 Enabled

Position:         Calibrated

Clamp Calibration Report

Type:             Tension: Film            

Date:              09-Nov-01 12:05

Mass:             17.8 gm

Offset:            -10.23 mm

Compliance:  0.659 um/N

 

This reports lists the date, time, and values as applicable to the following calibrations:

  • Calibrating DMA Clamps

  • Calibrating the DMA Instrument - Dynamic, Electronic, and Force

  • Calibrating the DMA Position

Select OK when you have finished reviewing the report.

Select Print to send the report to a printer.

Select Back, when available, to reanalyze the Dynamic Calibration results or to add additional samples to the Dynamic Calibration sample set.
