Tzero Values are Zero

Tzero slope or offset data is equal to zero.

This message is received during the use of the Calibration Wizard for Tzero (Cell Resistance & Capacitance) calibration. The Tzero slope and/or offset results from this calibration are equal to zero.  This is not a valid calibration result. Rerun the calibration data again.  If the problem persists, call TA Instruments.
