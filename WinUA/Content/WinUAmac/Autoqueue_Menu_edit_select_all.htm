Select All (Autoqueue)

Use the Select All Labels option to highlight all of the records contained in the autoqueue. You can then cut, copy, or delete all of the records.

 

Shortcut Key

Ctrl + A

 

To select all records follow these steps:

 

1. Select Autoqueue from the main Macros menu to display the Autoqueue Window.

2. Choose Select All to select all records.

3. Select the desired operation from the Autoqueue Edit menu.
