Purity Basic Steps

Follow these basic steps when performing a DSC Purity analysis:

 

1. Collect your data.

2. Obtain calibration values for cell constant and slope at 0.5°C/min. Also, obtain the molecular weight for the sample to be analyzed. (If you dont know this value, you could use a molecular weight of 100 g/mol.)

3. Select File/Open, then locate and open the desired purity data file.

4. Select Analyze/Purity/Options and set up the desired analysis options.

5. Select Analyze/Purity/Analyze and analyze the data.

6. Select View/Results Spreadsheet or Report to view and/or print results.

 

When selecting peak integration limits, be sure to select the left (or leading) limit to the baselinewell below (usually about 5°C) the apparent onset to melting.
