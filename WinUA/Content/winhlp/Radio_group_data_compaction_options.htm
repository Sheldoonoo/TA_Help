Radio Buttons - Data Compaction Options

Choose from one of the following ways to handle data compaction for your files.

  • Radio Button Prompt for Option: Click on the Prompt for Option radio button to display theData Compression windowthat enables you to choose the data compaction each time the program opens a large data file that exceeds the data buffer size and must be compacted.

  • Radio Button Discard Points: Click on the Discard Points radio button and the program will remove data points linearly throughout the data file when the file exceeds the data buffer size. The number of points removed is based on the amount of data reduction needed.

  • Radio Button Linear Smoothing: Click on the Linear Smoothing radio button and the program will divide the points into groups and compute an average for each group when the file exceeds the data buffer size. The size of the groups depends on how much data compaction is needed.
