Entering Sample Information Using the Wizard

 

Use this dialog  to enter or select additional information to be stored with the selected file. These items cannot be changed once the experiment is in progress.

  1. Enter the Sample Name. Enter up to 32 alphanumeric characters to identify the material.

  2. Enter any Comments. Enter additional text to help describe the experiment, for example, sample treatment or experimental conditions could be included. Up to 64 characters may be entered.

  3. Click the Browse button, then select the desired storage location and enter the Data File Name. Click OK when finished. Typical example \\{computer name}\TA\Data\DATA.001. (UNC filenames are used.)

  4. Select the Pan number from the list. Use this field to specify the number of the TGA Autosampler pan that will be used in the experiment. Click on the down arrow to scroll through the numbers.

  5. Integrity Users Only:  View the SUID#. Instead of defining the data storage path, the database will automatically assign a SUID# to the original, raw data record.  This number can also be seen on the Summary Page of the Experimental View.

  6. If you want to automatically analyze the resulting data file using a macro (created with the Universal Analysis Autoanalysis software option), check the Autoanalyze box. Then locate the desired macro filename using the Browse button. Enter the drive, directory, file location, and name for file.

NOTE: See the Autoanalysis documentation accompanying the Universal Analysis software for further information on creating macros.

  1. Click on the Analysis Macro list and select a macro from those displayed. This macro will be used to automatically analyze the data file that will result from this experiment.

When you have finished setting up this run summary, click Next to proceed.
