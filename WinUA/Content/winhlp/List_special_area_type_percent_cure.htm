List Special Area Type - Percent Cure

Percent cure is available for DSC data files only. If you select Percent cure from the drop-down list, the percent cure (%c) is calculated using the following equation:

 

%C = (1  area/standard heat) x 100

 

When you click on Percent Cure the standard heat entry field is displayed, allowing you to change the value, if desired.

 

See also: Analysis Options - Special Area Page.
