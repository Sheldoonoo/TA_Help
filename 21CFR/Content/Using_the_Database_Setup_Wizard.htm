Using the Database Setup Wizard

 

Before beginning this procedure, perform the following steps:

  • Power on the database server and make sure it is properly configured and visible to the controller. (It should be visible when browsing the network.)
  • Determine the Netbios name (name used in the Browse windows) of the database server.
  • Obtain the user name and password of the database administrator account.
  • Power on all licensed Integrity instruments and make sure they are connected and visible to the controller.
  • Determine the name to be given to the TA Instruments Database created during this procedure. (Ten [10] characters or less is recommended.)

After the previous items have been accomplished, run the Database Setup Wizard as follows:

  1. Insert the TA Instruments Advantage Integrity software CD into your controller CD-ROM drive. The installation program will automatically begin.  Press Next to continue.
  2. Accept the software license, enter the requested information, and read the installation information. Then press Next to proceed.
  3. Select Run Integrity Database Setup Wizard, select the desired language from the Select Language list, then select Next to begin database configuration. (See also: Selecting a Language.)

NOTE:  If the Oracle Client software is not installed on this computer, the Oracle Universal Installer will appear to install this component before running the Database Setup Wizard.

  1. Select Next on the Database Setup Wizard when you are ready to proceed.
  2. Enter the database information requested.. (This information can be found in the box with the database server when supplied by TA Instruments.):
    • Database Server: Computer name (also called the Host Name) of server.
    • Database Administrator Account User Name and Password: Database account that has Administrative (SYSDBA) rights so that program can create database objects on the server.
  3. Select Next when all the database information has been entered.  If the database information is correct, the setup program will advance to the next window.
  4. Enter the desired database name (using 10 or less alphanumeric characters).  Press Next to proceed.
  5. Select Start Tasks to begin creation of the database objects as listed in the task list.  (You do not need to select [check] any of these items.)  The progress of this operation can be viewed in the task log.  Once the database creation is complete, a message box will be displayed.
  6. Select OK to close the message box, then select Next to advance to the next window.

The Integrity-licensed instruments must be imported in the database as system modules of the CFR system.  If the instruments are connected and visible from this location, you will see them in the Import Modules into the Database System window.  

  1. Select Import to add the displayed instruments (modules) in the database.  (You do not need to select [check] any of these items.)  All the instruments displayed in the list will be added.  If an instrument is already in the database, it will not be added again.  (You can select Refresh to update this list as needed.)
  2. Notice that a check mark will be displayed next to its name as each one of the instruments is added to the database.  Once all the instruments have been imported, select Next to advance to the next window. A listing of all the database creation audit events appears in this window.  
  3. Select Finish to complete this process.  A message box displaying the default user account information will be displayed.

Important:  Record this default user account information so that you can obtain access to the TA Instruments System Manager program.
