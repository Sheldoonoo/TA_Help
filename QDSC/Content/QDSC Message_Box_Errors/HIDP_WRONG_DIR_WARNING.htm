Inappropriate Files. Transfer Anyway?

The files in currently selected directory are not for the instrument. Do you still want to transfer them to the instrument?

This message is displayed when you attempt to restore backup files to the instrument. The MAC address of the instrument is compared with the MAC address contained in the backup files. If the addresses do not match, this message is displayed. You can proceed to transfer the files to the instrument even if they did not originate from that instrument, if desired. However, this is not recommended.
