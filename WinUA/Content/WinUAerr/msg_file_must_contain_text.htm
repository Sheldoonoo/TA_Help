File Must Contain Text

 

File must contain plain text.

 

Problem:

The text file specified does not have a .txt extension, indicating a plain text (text only) file. No conversion to plain text is provided for non-text file types (e.g., .rtf, .doc, etc.). If the specified file is a plain text file, but does not have a .txt extension, then this warning message can be ignored.

 

Solution:

If the file already contains plain text then add a .txt extension to the file name. If the file does not contain plain text, then use a text processing program like Microsoft® Word® to convert your file to plain text. Open the file with the text-processing program and resave it with a .txt (text only) extension.
