Message 514

Cell lid sensor error.

Problem:

This message is displayed when the cell lid close and open sensors are both triggered simultaneously.

Solution:

Try the following:

  • Select the Lid/Open function from the instrument control menu to open the lid, then Lid/Close to close the lid.
  • Reset the instrument parameters as follows:
    1. Verify that both the instrument and the controller are both powered on and connected to the supplied Ethernet Hub or LAN.
    2. Select Tools\Instrument Setup from the main menu to start the Instrument Setup Wizard.
    3. Click the Next button until the Transfer Program Files to the Instrument window is displayed.
    4. Click the Reset Parameters button. A warning message is then posted asking if you wish to continue with the resetting the parameters, click OK to proceed. The parameters will now be reset.
    5. Press the reset button on the back of the instrument to complete this process.
    6. Click the Cancel button to exit the wizard.

NOTE: This operation has no effect on your saved calibrations.

  • If the problem persists, contact TA Instruments for service.
