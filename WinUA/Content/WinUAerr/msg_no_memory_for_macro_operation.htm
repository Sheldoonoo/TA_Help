Insufficient Memory for Macro Operation

Insufficient memory for macro operation.

 

Problem:

There is not enough free memory in the system to perform the macro operation requested.

 

Solution:

Try one of the following:

 

1. Close one or more nonessential applications.

2. Close one or more open files in the analysis program.

3. Close the Universal Analysis program and one other application. Then restart the Universal Analysis program before restarting the other application.

4. Add additional RAM to your system.
