Radio Buttons Add/Subtract Operation

Select the desired radio button as follows:

 

  • Add Operation: Click the Add Operation radio button to permit the addition of the same signal(s) from another file so that the signal(s) may be averaged over multiple runs. Use the Linear Transform function to average the signal.

 

  • Subtract Operation: Click the Subtraction Operation radio button to permit subtraction of the same signal(s) from another file so that subtle transitions can be emphasized.
