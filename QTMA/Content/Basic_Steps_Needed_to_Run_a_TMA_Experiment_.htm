Basic Steps Needed to Run a TMA Experiment

 

All of your TMA experiments will have the following general outline. In some cases not all of these steps will be performed.

  • Calibrate the instrument. See Calibrating the TMA
  • Choose and install the probe appropriate for the sample shape. See Choosing a Probe.
  • Dilatometer Accessory
    • Expansion Probe
    • Film/Fiber Probe
    • Flexure Accessory
    • Hemispherical Probe
    • Parallel Plate Rheometer Accessory
    • Penetration Probe
  • Calibrate the probe. See Calibrating a TMA Probe
  • Zero the probe by selecting Control/Length/Zero from the menu or ZERO LENGTH on the Control touch screen. See also:  Introducing the Touch Screen
  • Select the desired mode through the TA instrument control software. See Selecting a TMA Operating Mode
  • Measure the sample dimensions and load the sample into the probe. See Preparing and Loading Samples
  • Adjust the thermocouple position.
  • Create or select the experimental procedure using the instrument control software.  See Specifying a Test Procedure.
  • Close the furnace by pressing the FURNACE key on the touch screen or selecting Control/Furnace/Close.
  • Enter the experiment information through the TA controller, this includes both sample and instrument information on the Summary and Notes pages. See Using the Summary Page and Setting Up the Notes Page.
  • Set the purge gas flow rate. See Purge Gases and Flow Rates
  • Add coolant to the furnace reservoir for subambient operation, if applicable. See  Running Subambient TMA Experiments
  • Start the experiment. See Starting a Run and/or Sequence.
