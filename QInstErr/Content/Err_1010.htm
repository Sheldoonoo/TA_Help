Message 1010

Weight calibration failed.

Problem:  

TGA Auto weight calibration failed.

Solution:

Try the following steps:

  1. Check to make sure that you have used the correct pans for this procedure and that they are in the correct pan locations (i.e., the pans are not reversed). See the online help for detailed information.
  2. Ensure that the pan differential is correct and has been entered as required.
  3. Check the verification criteria to make sure the value is accurate.
  4. Repeat the calibration procedure.
  5. If the calibration fails again, contact TA Instruments for service.
