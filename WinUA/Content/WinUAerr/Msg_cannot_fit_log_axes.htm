Cannot Fit Log Axes

Cannot fit data on log axes.

 

Problem:

The entire data file cannot be plotted using log axes.

 

Solution:

Try one of the following:

 

  1. Change the Y-axis type to normal scaling (i.e., not log scaling) using the Signal Selection window.
  2. Change the data compaction option to reduce the number of points that are plotted using the Main Options  Data Page, then reopen the file.
