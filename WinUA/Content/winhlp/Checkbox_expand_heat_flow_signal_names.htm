Checkbox Expand Heat Flow Signal Names

Check the Expand heat flow signal names box to automatically replace the heat flow signal name in a Tzero data file with the full signal name from the parameter block (e.g., Heat Flow T4). The signal name is replaced in memory only when the data file is initially opened. The "SelHeatFlow" parameter that specifies the full signal name must appear in the data file parameter block. The data file is not modified.

 

Leave this box unchecked to use the default signal name, Heat Flow.
