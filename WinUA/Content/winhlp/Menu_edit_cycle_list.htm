Edit Cycle List

Use this menu item to select the cycles for plotting from the list of those contained in the data file. The list of cycles is displayed on the Data Cycle Selection Window.

 

Cycles are defined by any breaks in the data (data on/off regions) or by gas switch and event markers, if they are enabled on the Main Options - Data Page. When cycles are deselected they are marked as "invalid data," but the data is still retained in memory. Enabling a deselected cycle does not reread the data file.
