Analyze Humidity BET Analysis

Select this item to fit the sorption isotherm data from the Q5000 SA to the BET model.

 

To use this function:

  1. Collect your data following the guidelines suggested. See Humidity Analysis.
  2. Select File/Open, then locate and open the desired humidity data file.
  3. Select Analyze/Humidity/Options and set up the desired analysis options.
  4. Plot Weight Change versus Time for the desired Q5000 SA data.
  5. If sample drying was conducted during this experiment, then the dry weight needs to be determined before generating this plot. Use the Analyze/Reset Size function.
  6. Select Graph/Data Limits/Time, and select the Start time as the time after the drying step, and end time as the experiment end time.
  7. Manually mark the adsorption/desorption isotherms, if desired. Select Analyze/Humidity/Mark Adsorption/Desorption then select the desired position on the curve.
  8. Select Analyze/Humidity/BET Analysis. A new window will be displayed with the resultant data, based on the chosen humidity options (see Humidity Analysis Options - BET Analysis. The sorption isothermal data from the selected file is processed using the BET equation and displayed.

This analysis results in a plot of the adsorption and/or desorption data values.

 

A data table can be obtained for this plot by selecting View/Data Table.

 

Background

In 1938 Brunauer, Emmett and Teller demonstrated that, with the help of a few simplifying assumptions, it was possible to carry out an isotherm deviation for multimolecular adsorption layers. The equation, shown below, not only represented the general shape of isotherms (BET Type II and Type IV), but also yielded reasonable values for the average heat of sorption and the amount of gas adsorbed in the monolayer.

 

 

W = weight adsorbed

Wm= weight adsorbed as a monolayer

P/P0= relative adsorbate pressure = activity

C = BET constant (dependent upon the heat of adsorption)

 

The BET equation is used to calculate monolayer values as well as estimate the surface area. The reason for its widespread use includes the simplicity of application and its approval by the International Union of Pure and Applied Chemistray (IUPAC)[1]. A 1985 report of the Commission on Colloid and Surface Chemistry[2] recommends the BET plot for a standard evaluation of monolayer values of adsorbate activity between 0.05 and 0.30.
