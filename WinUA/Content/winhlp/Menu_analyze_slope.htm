Slope

Use the Slope option to calculate the average slope of a selected region of the curve.

 

Follow these steps for perform this analysis:

 

1. Select Slope from the Analyze menu. Markers will be displayed on the graph.

2. Right click to display the Analyze pop-up menu, then select Point 1 to activate the first marker.

3. Position the marker at the beginning of the region you want to measure.

4. Right click to display the Analyze pop-up menu again, then select Point 2.

5. Position the second marker where you want to stop measuring the slope.

6. Right click to display the Analyze pop-up menu. Select Accept Limits. The program calculates and displays the slope.
