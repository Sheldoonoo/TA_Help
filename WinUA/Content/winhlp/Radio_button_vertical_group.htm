Radio Buttons Vertical

  • Top Radio Button (Vertical Position): When the Top Vertical radio button is selected, the legend/inset view is positioned on the graph using the top of the vertical side of the box as the handle.

  • Center Radio Button (Vertical Position): When the Center Vertical radio button is selected, the legend/inset view is positioned on the graph using the center of the vertical side of the box as the handle.

  • Bottom Radio Button (Vertical Position): When the Bottom Vertical radio button is selected, the legend/inset view is positioned on the graph using the bottom of the vertical side of the box as the handle.
