Accessing Records in the Database

 

After logging into the Advantage Integrity-licensed Universal Analysis program, you will need to locate the experiment data in the database for analysis.

 

You can either browse for data in the Database Viewer using the tree view functions in the left-hand pane, using the Record ID number, or by using the Database Search Wizard (which employs SQL). These three methods are briefly explained in the sections that follow.

Browsing for Data in Tree View

The left-hand pane of the Database Viewer can be used to locate data in tree view. Click on each item to open the items below it until you come to the desired record. The sort order in which these items are displayed can be customized using the Set Sort Order function, . (See: Setting the Sort Order for Tree View for information on setting the sort order.)

 

  • Dates, User, Project, Instrument, Run Number: When you click on one of these icons, the records that apply are displayed in the right-hand pane.

  • Raw Data files, Calibration files, Saved Analysis files, Saved Session files, Text files: When you select the data file record, the data file information is displayed in the right-hand pane. To open the file, click the tool bar button or select File/Open. (These files can be opened in Universal Analysis.)

  • PDF files: When you select an Acrobat PDF file, the Acrobat Reader program will open and the file will displayed. (This type of file cannot be opened in Universal Analysis.)

  • BMP files: When you select a BMP file, the image is displayed. (This type of file cannot be opened in Universal Analysis.)

 

Browsing the Database by Record

 

To browse around the database by a particular record you can use several different methods:

 

  • Using the Record Search Option: At the top of the Database View window is an entry field called Record No. Enter the number of the record you wish to locate and click the Go button.

  • Using the Pop-up Menu: If you wish to search for records that are related, you have several options to choose from as follows:

 

 1. Open the Database Viewer.

 2. Select the desired record.

 3. Right click to display the pop-up menu.

 4. Choose from the following menu items depending on the course of action desired:

 

  • Copy record ID to the clipboard: Used to copy the current record ID to the clipboard so it can be pasted into the Record No. field at the top of the window.

  • Find data source of the item: Used to find the original data files associated with the current item.

  • Go find audit trail of the item: Use to open the Audit Trail Viewer displaying the audit trail used to create the selected item.
