Setting Up a Creep Experiment

 

In a creep experiment, a constant force (stress) is instantaneously applied to the material and the resulting deformation (strain) is measured as a function of time.  Creep compliance is calculated by dividing the time-dependent strain by the applied stress.  After creep has occurred for a specified period of time, the stress is removed and strain recovery of the sample is measured.

If you want to set up a DMA Creep procedure, you can choose one of two techniques—the DMA Wizard or the Experimental View - Procedure Page. See the instructions below for each technique.

When you perform a DMA Creep experiment, the following parameters need to be chosen:

  • Preload force: This is the force applied to the sample prior to initiating the experiment.  

    • A preliminary preload force is available for creep measurements using the tension (film) and tension (fiber) clamps.  This preload force ensures that the sample material is fully elongated (has no slack).  A preliminary preload force is also available when using the compression and 3-point bend clamps for creep measurements.  In those clamps the preload force ensures that the movable part of the clamp is in contact with the material prior to starting the experiment.  In all cases, the preload force should be limited to the minimum force required to achieve the desired starting condition.  Otherwise, it may affect the creep results.  (NOTE:  Preload force will also affect the creep recovery results.  Since creep recovery assumes “zero force,” the impact of improper preload force is greater during that portion of the experiment.)  0.001N is the lowest force which can be applied.  

    • Preload force is not used when you select a nontensioning clamp (i.e., single/dual cantilever, shear sandwich).

  • Stress:  The stress applied during creep measurements can be selected to simulate real-life end-use conditions or can be a nominal value sufficiently large to ensure good deformation. See also:  Calculations Based on Clamp Type.

  • Isothermal Temperature:  Creep experiments are run isothermally.  Any temperature within the temperature range of the DMA (-145 to 600°C) is acceptable provided a gas cooling accessory (GCA) is used.  However, most creep experiments are run either at the temperature of actual end-use (e.g., ambient) or at a temperature where sufficient creep occurs to enable comparisons between materials.  

  • Soak Time:  Because of the mass of the DMA furnace and clamps, we well as the large size of most DMA samples, a "soak time" (i.e., time to equilibrate at temperature) is recommended.

  • Creep Time & Recovery Time:  The creep time is the time that the stress is applied to the material.  The recovery time is the time after the applied stress is removed and the material is allowed to recover from the strain resulting during creep.  The times required for both of these conditions is usually selected by trial and error.  These times can also be selected to reflect real-life end-use conditions.  However, 10 and 20 minutes respectively are good initial values to try.

  • Additional Conditions:  There are additional experimental conditions that can be changed to optimize specific situations.  These are accessed using Advanced Parameters and Post-Test Conditions.  For most experiments, the system default values shown are recommended.

  • See also:  Guidelines for Running Subambient Experiments.

Using the DMA Wizard :  

Enter the desired parameters, then select Next to proceed.

Using the Experimental View Procedure Page:

  1. Display the Procedure Page. Select the desired experiment from the list of available preprogrammed templates.

  2. Enter the desired parameters. When you have finished setting up this procedure, you can enter more information by selecting the Notes Page and Summary Page for this run.

  3. Click the Apply button when finished to save these changes or the Append button to save the changes to this run and add another run to the end of the sequence.

NOTE:  You can use any test template as the basis for a custom test by selecting the desired test template from the list of tests, then click the Apply button and change the test to Custom. Then you can click on the Procedure Page and then the Editor button to make the changes desired.
