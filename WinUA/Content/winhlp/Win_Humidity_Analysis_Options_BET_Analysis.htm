Humidity Analysis Options - BET Analysis

Use this page to set up the default parameters for the program to use when you perform the TGA Humidity BET analysis.

 

Follow the steps below to set up these options:

  1. Select the type of signal to be plotted on the Y Axis from the drop-down list.

  2. Select the type of signal to be plotted on the X Axis from the drop-down list.

  3. Select the line type from the drop-down list for both the BET data and fit line.

  4. Select the color for the graph and label from the drop-down list for both the BET data and fit line.

  5. Select the grid type from the drop-down list.

  6. Select the symbol type from the drop-down list. If you select a symbol, it will be displayed on the dialog next to the type of plot chosen (e.g., next to "Plot adsorption data"). You can edit the symbol list by clicking the Edit Symbols button.

  7. Select the type of differentiation from the drop-down list.

  8. Check one or more options that will be applied to the humidity analysis plot:

  • Show annotation
  • Label curves
  • Border

NOTE: When the Border box is checked the legend is opaque (nothing behind it will show). When the box is not checked, there is no border and the legend is transparent (anything behind it will show through).

  •  Y4 Axis
  • Plot BET data
  •  Plot BET fit line
  •  Show legend: When you check this box, you can either use the default legend displayed or type a new legend in the entry field (50 characters maximum length).

  1. Enter the lower and upper range for the desired Constraints on relative humidity (RH%).

  2. Click the Save button, if you want to use these as your new default settings or click the OK button to use these settings for this session only. 

 

NOTE: Select the Default button to use the default settings for this window.
