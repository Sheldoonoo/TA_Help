Checkbox Auto Refresh Data

Check the "Auto refresh data every xx seconds" box to automatically refresh the plotted data from an active data file (real time plot), without losing completed results. If you select this checkbox, you must enter the minimum amount of time delay between refresh actions. The default time is five seconds.

 

  • Auto refresh is automatically suspended when activity is detected (e.g., mouse movements, keyboard input, active dialogs, etc.).

  • Auto refresh will continue in the background if Universal Analysis is not the active application.

  • Auto refresh can be toggled on and off using the Graph/Auto Refresh Data menu item.
