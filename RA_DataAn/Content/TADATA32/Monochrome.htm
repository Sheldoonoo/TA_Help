Monochrome

To configure a monochrome graph follow the steps below:

  1. Select Graph from the menu.
  2. Click on the Symbols tab.

  3. Select the Distinguish option to By file.
  4. Select Extra symbols for the File entry.
  5. Select Symbols for the Axis entry.
  6. Select None for the Step entry.
