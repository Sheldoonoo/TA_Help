Setting Up Calibration Standards

 

This function is used  to define the default settings for each calibration standard, add new standards, and/or reset the existing standards.

Access the Calibration Standard window by selecting Calibrate/Analysis from the main menu. The Calibration Analysis window is displayed.

  1. Select the desired standard displayed.

NOTE:  To add a new standard to this list, click the Add button then type in a new name and define its settings. This standard will be saved when you click the OK button.

  1. Accept the standard's Curie Temperature as displayed then go on to step 3 or edit the value as described below.

  1. Click the Edit button to open the Calibration Standard window.

  2. For Q5000 IR:  Enter the literature Curie Temperature in Â°C. Consult your Chemistry handbook for the value.

ForQ5000 SA:  Enter the literature Theoretical Relative Humidity in %. Consult your Chemistry handbook for the value.

  1. Select OK to save the edited value and return to the Calibration Analysis window.

  1. Select OK to save the changes.  Any entries that are made here will be saved for future recall.

NOTE:  To return the standards list back to the default values, click on the Default button.

NOTE:  The Delete button can be applied only to a standard that you have entered. You cannot delete the program standards.

Resetting the Calibration Standard Settings

In the event that you want to restore all of the standards back to the original manufacture defaults (set at TA Instruments), click on the Reset All button. A confirmation message will be displayed. Select Yes to continue operation or select No to cancel the reset function.
