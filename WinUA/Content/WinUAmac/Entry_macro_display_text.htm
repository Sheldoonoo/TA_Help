Entry Macro Display Text

Enter the text to be displayed in the dialog (up to a maximum of 30 characters).

 

NOTE: If more than 30 characters of text must be displayed, or if more than one value must be entered in a single dialog, then consider using theMacro Edit Clipboardstep instead. After the Macro Edit Clipboard window is accepted, the individual values that were entered can be extracted from the clipboard by using one or more Macro Assign Variable steps, specifying Clipboard (value) or Clipboard (variable, value) as the operation.
