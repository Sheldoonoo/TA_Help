Viewing a Procedure Summary

 

This is a view-only window that displays the contents of the current procedure file. This window contains the following information:

  • Instrument name

  • Instrument mode

  • Saved signals list

  • Procedure name

  • Notes (relating to selected procedure)

  • Method Name

  • Method Segments

  • Advanced Parameters

  • Post Test Parameters

You can right click to display a pop-up menu that allows you to copy this information to the clipboard or view it from your editor application.

  1. Click the Print button to print this information on your default printer.

  2. Click Cancel to close the window without printing the procedure file summary.
