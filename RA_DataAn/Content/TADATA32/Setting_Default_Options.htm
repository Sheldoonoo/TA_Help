Setting Default Options

Follow these steps to set the program default options:

  1. Select Options from the menu.
  2. Click on Settings.
  3. Click on the Miscellaneous tab
  4. Select either CGS or SI for the Display units.
  5. Select the action required when you double click a filename in the results list window.
  6. Click on the View tab.
  7. Select required toolbars.
  8. Click the OK button.
