Checkbox Create Integrity Item Access Files

Check this box to create a new Universal Analysis item access file for use with the Integrity database system. This checkbox is not saved in the initialization file. The item access file is created immediately when the OK button is pressed. The files are stored in the directory containing the Universal Analysis program executable file (WinUA.exe). Refer to the TA System Manager program for more information on using these files.

 

NOTE: This feature is intended for use by trained Service and Administration personnel only.

 

NOTE: You must save this setting to the current initialization file by clicking the Save button on theMain Options  Initialization Page, or it will not take effect.
