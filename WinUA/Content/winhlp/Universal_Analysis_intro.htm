Welcome to Online Help for Universal Analysis

 

Introducing TA Instruments powerful tool for analyzing data obtained from any of the following instruments:

 

DSC (Differential Scanning Calorimeter)

High-Temperature DTA (Differential Thermal Analyzer)

DMA (Dynamic Mechanical Analyzer)

SDT (Simultaneous Differential Technique) Analyzer

TGA (Thermogravimetric Analyzer)

TMA (Thermomechanical Analyzer)

DEA (Dielectric Analyzer)

µTA (Micro Thermal Analysis)

 

Using Universal Analysis you can take any combination of signals for plotting and analyzing. You can select the type and number of curves, region to be graphed, the type of transition analysis, which transitions to analyze, and the limits of transitions.
