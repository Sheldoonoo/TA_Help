Single Step (Macro)

Select Single Step to run the current macro one step at a time. When you choose Single Step, you will need to press any key (except Escape) to proceed to the next step in the macro. If the macro is not currently on the first step, you have the option to go to the beginning or continue the macro from its current position.

 

Shortcut Keys

Ctrl+G

 

NOTE: You can press the Escape key to stop a running macro.
