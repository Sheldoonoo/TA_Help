Message 529

Autosampler control task exists.

Problem:

This message is displayed when the Autosampler is not found by the controller or there is a communication error between the Autosampler and the controller.

Solution:

  • Reset the Autosampler by touching the RESET key on the Autosampler Control Menu or by selecting Control/Autosampler/Reset on the instrument control program, then retry your operation.
  • If the problem persists, call TA Instruments for service.
