Entering Notes and Gas Settings Using the Wizard

 

Use this dialog to enter or select additional information to be stored with the selected file. These items cannot be changed once the experiment is in progress.

  1. Enter the name of the Operator conducting the thermal analysis experiment. Up to 25 characters may be entered.

  2. Enter any additional comments in the Extended Text field. This may involve information about the sample or experiment, such as sample treatment or data sampling parameters. Up to 500 alphanumeric characters may be entered (approximately 12 lines).

  3. Select the desired Sample Purge Gas to be used with your TMA experiment. Helium is recommended as a purge when performing experiments below 0°C.  Enter the desired Flow Rate to be used for that gas. The recommended purge rate is 100mL/minute. Because of the significant volume of air trapped around the sample when closing the furnace, let the purge gas flow for 3-5 minutes before adding coolant or starting a run.

  4. When you have finished setting up this information, click Next to proceed.

NOTE:  If you are using laboratory purge gas, rather than bottled purge gas, you will need to install an external gas drier and a 5-µm filter.

NOTE:  If an MFC is installed, these settings will automatically set the gas and flow rate on a run by run basis. (The scheduled run's purge setting will take effect immediately once the apply button is clicked.) If your instrument does not have an MFC, these entries are for information only. (In this case an external gas regulator and flow meter are required to regulate the purge flow rate.) To change the gas type access Tools/Instrument Preferences.
