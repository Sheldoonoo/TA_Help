Setting Up a Stress Relaxation TTS Experiment

 

Stress relaxation experiments apply an instantaneous strain to the sample and measure the force required to maintain the strain as a function of time. The primary result is stress relaxation modulus during the strain step.

Time-temperature superposition (TTS) experiments can be performed using stress relaxation tests that were run at different isothermal temperatures, but using the same strain. This allows you to evaluate long-term properties (stress-relaxation modulus) in an experimentally feasible time.

If you want to set up a DMA Stress Relaxation Time-Temperature Superposition (TTS) procedure, you can choose one of two techniques—the DMA Wizard or the Experimental View - Procedure Page. See the instructions below for each technique.

When you perform a DMA Stress Relaxation TTS experiment, the following parameters need to be chosen:

  • Preload Force:  This is the force applied to the sample prior to initiating the experiment. A preliminary preload force is available for stress relaxation measurements using the tension (film) and tension (fiber) clamps.  This preload force ensures that the sample material is fully elongated (has no slack).  A preliminary preload force is also available when using the compression and 3-point bend clamps for stress relaxation measurements.  In those clamps the preload force ensures that the movable portion of the clamp is in contact with the material prior to the start of the experiment.  In all cases, the preload force should be limited to the minimum force required to achieve the desired starting condition.  Otherwise, it may affect the stress relaxation results.  0.001N is the lowest force which can be applied.  

  • Strain:  The % strain applied during stress relaxation measurements can be selected to simulate real-life end-use conditions or can be a nominal value (usually 0.1 - 1%).

  • Start/Final Temperature:   The DMA temperature range is -145 to 600°C.  Attaining the subambient portion of this range depends on the presence of a Gas Cooling Accessory (GCA).

  • Temperature Increment:   The temperature increment should be small enough to provide good resolution of events. Usually a 5°C increment is sufficient if each stress relaxation segment is about ten minutes long.

  • Isothermal Soak Time:  Because of the mass of the DMA furnace and clamps, as well as the large size of most DMA samples, a "soak time" (i.e., time to equilibrate at temperature) is recommended.

  • Relaxation Time:  The relaxation time is the time that the material is strained. Ten minutes is a good initial value to use.

  • Additional Conditions:  There are additional experimental conditions that can be changed to optimize specific situations.  These are accessed using Advanced Parameters and Post-Test Conditions.  For most experiments, the system default values shown are recommended.

  • See also:  Guidelines for Running Subambient Experiments.

Using the Experimental View Procedure Page:

  1. Display the Procedure Page. Select the desired experiment from the list of available preprogrammed templates.

  2. Enter the desired parameters. When you have finished setting up this procedure, you can enter more information by selecting the Notes Page and Summary Page for this run.

  3. Click the Apply button when finished to save these changes or the Append button to save the changes to this run and add another run to the end of the sequence.

NOTE:  You can use any test template as the basis for a custom test by selecting the desired test template from the list of tests, then click the Apply button and change the test to Custom. Then you can click on the Procedure Page and then the Editor button to make the changes desired.
