Initialize (Macro Insert Menu)

Use the Initialize option to insert a step that will open an initialization (.ini) file into your macro. When this option is selected, the Macro Initialize window is displayed to allow you to choose an .ini file. After you select OK, the program will insert the initialize step that recalls this file. It is best to insert an Initialize step as the first step in your macro.

 

Before you start to capture your macro, save the program settings to an initialization file, then load that .ini file. Insert an Initialize step using the saved .ini file as the first step in your macro. This will eliminate problems that can occur due to a change in settings or dialogs made before this macro is run.
