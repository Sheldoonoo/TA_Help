Choosing the Advanced Parameters  

 

Click the Advanced button on the Procedure Page to define more advanced instrument parameters that may not normally be changed for experiments, once they are set. The information displayed on this window varies based on the mode and clamp selected.

  1. Enter the desired values for the parameters displayed or select an option.

Click on the following Advanced Parameter topics for more information:

    • Data Sampling Interval

      This instrument parameter defines how often the system reads and stores a data point per second.  For example, if 2.0 is entered, the system stores a data point every two seconds. Consult the list below to determine the range allowable for the mode used.

      NOTE:  If 1/Frequency is greater than the data sampling interval, then the data point will be recorded after the minimum time interval for the frequency.

      Range between 1.0 and 1000.0, divisible by 1.0:

      DMA Multifrequency - stress

      DMA Multifrequency - strain

      DMA Multistrain

      DMA Multistress

      Range between 0.1 and 1000.0, divisible by 0.1:

      DMA Creep

      DMA Stress Relaxation

      Range between 0.5 and 1000.0, divisible by 0.5:

      Controlled Force

      Isostrain

      Strain Rate Mode

      In the DMA stress relaxation mode, data collection is performed on a logarithmic-time basis, with a factor of 1.2. If a data sampling interval of 1 second is selected, data will be collected at 1,2,3,4,5,6,8,10,12,15,18, etc.

      Recommended values for the data sampling interval are 2 to 10 seconds, if data is to be plotted on a linear scale (time or decay time), or 0.2 to 1 second, if data is to plotted on a logarithmic scale.

    • Minimum Dynamic Force

      Enter the Minimum Dynamic Force, the minimum force that will be applied to the sample.

      The minimum dynamic force setting has two uses:

      • If the minimum dynamic force is set to a value greater than zero, it sets the minimum dynamic force that will be applied to the drive coil.

      • If the minimum dynamic force is set to a negative value, it locks the dynamic force to the absolute value of the value entered. This setting bypasses all internal amplitude and stress control loops and can be useful for testing samples that are too fragile to support closed loop amplitude and stress control. The force actually measured at the sample when using this option can be somewhat less than the value entered because some dynamic force is required to move the DMA slide assembly. The losses for moving the slide assembly can be minimized by using low frequencies at low displacement amplitudes.

    • Auto Zero Displacement

      Selecting Yes automatically sets the signal value to zero displacement before the method begins.

      Selecting No maintains the current signal value at method start.

    • Equilibrium Criteria

      Equilibration criteria are used at the beginning of a displace/recover method to allow the sample to mechanically and thermally equilibrate before the sample is displaced.

      • Select the Time button if you wish to have the equilibration criteria expressed in minutes.

      • Select the Rate button if you wish to have the equilibration criteria expressed in the rate of deformation (Âµm/min).

    • Measure Again After Method Equilibration

      Select Measure Again After Method Equilibration to measure the sample after the initial equilibration segment in the method. This features only applies when using the film/fiber tension, compression, and penetration clamps. It is very useful when performing subambient experiments to correct for dimension changes as a result of temperature changes.

  1.   Click the OK button when finished.
