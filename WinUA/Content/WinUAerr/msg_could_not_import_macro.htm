Could Not Import Macro(s)

Could not import one or more macros.

 

Problem:

When you tried to import macros from a specified macro file, the macros could not be imported into another macro file. This could occur if the macro file was not created using Universal Analysis or if the file is corrupted.

 

Solution:

Check your macro file to make sure it was created using Universal Analysis and try the operation again. If the import function still fails, the file may be corrupted and you may have to recreate it.
