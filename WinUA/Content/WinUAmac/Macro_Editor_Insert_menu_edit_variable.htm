Edit Variable (Macro Insert Menu)

The Edit Variable macro step is used to display a dialog box in which you can view and edit the contents of a macro variable. This is useful for entering or changing a result or macro parameter. See also: Macro Edit Variable Window.

 

 If the variable name is already defined at the time that the Edit Variable step is executed, then the current value is displayed in an edit field to accept or modify.

  • If the variable name is undefined, then the variable is created and initialized to zero prior to opening the dialog box. The numeric value entered in the dialog box may be an integer or a floating-point number.
