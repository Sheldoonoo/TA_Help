Using Auto Diagnostics

(A Platinum™ Feature)

 

Your Q2000/Q200/Q20 is equipped with an Auto Diagnostics routine. This routine verifies the basic operation of your instrument to ensure that it is ready for operation. (See the complete list of tasks below.)

  1. Select the Platinum button at the bottom left-hand side of the main instrument control window.
  2. Click Auto Diagnostics from the list of Automatic Tasks, a set of pages is displayed. The Auto Diagnostics Page can be used to set up a number of tasks that can be executed on demand or can be automatically performed at the time(s), date(s), and frequency selected on the Schedule Page. The items in this list will vary based on the instrument accessories such as the Autosampler and cooler types.

The results of the auto diagnostics tasks are shown on the Auto Diagnostics Report Page. The results can be sent via email or Windows Messaging. See Using the Messaging Page for more information.

  1. Check the desired auto diagnostics tasks then select Apply to set them up to run when scheduled or select Start Now to execute the diagnostics immediately:

NOTE:  If you want to capture only the real-time information found in the diagnostics report, select Start Now without choosing any of the tasks.

  • Autosampler:  (Applicable to DSC instruments with an Autosampler assembly only.) This diagnostics routine resets the Autosampler motors, tests the home sensor, verifies that the Autosampler tray is installed, and tests all of the motion control electronics. See Calibrating the Autosampler for further information on Autosampler maintenance.
  • Lid Check:  (Applicable to DSC instruments with an AutoLid mechanism only.) This diagnostics routine detects the presence of the AutoLid then tests and verifies that the lid moves up and down unobstructed. See Aligning the AutoLid for instructions when needed.
  • Impedance:  This diagnostics routine tests the furnace to measure the impedance, tests and verifies the safety relay circuitry for proper functioning, and tests the high voltage circuitry within the power control unit.  
  • Mass Flow Controllers:  This diagnostics routine detects the presence of the mass flow controllers, tests and verifies that the MFCs are working properly, and verifies the MFC's ability to maintain both high and low flow rates. Verify the purge gas connection prior to using this operation.
  • Pan Load/Unload:  (Applicable to DSC instruments with an Autosampler assembly only.) This diagnostics routine tests and verifies that a sample and reference pan can be loaded and unloaded and verifies that the Autosampler has been properly calibrated. Routine requires a pan in position #1 and R#1 for this operation. See Calibrating the Autosampler for further information on Autosampler maintenance.
  • LNCS Cooler:  (Applicable to DSC instruments with an LNCS only.) This diagnostics routine tests and verifies that the cooler can reach -180°C and tests the cooler communications with the instrument.
  • RCS Cooler:  (Applicable to DSC instruments with an RCS only.) This diagnostics routine tests and verifies that the cooler can reach -80°C and that the cooler event cable is connected and operational.
  1. Click on the Report tab, after the Auto Diagnostics routine has been completed, to display the associated report. Any errors that occurred during this routine can be viewed by displaying the Instrument Log .
