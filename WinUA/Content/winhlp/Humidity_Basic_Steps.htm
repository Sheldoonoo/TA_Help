Humidity Basic Steps

Follow these basic steps when performing a TGA Humidity analysis:

 

1. Collect your data following the guidelines suggested. See Humidity Analysis.

2. Select File/Open, then locate and open the desired humidity data file. Data is generally plotted first with Weight Change Percent on the Y1 axis, Relative Humidity on the Y2 axis, and Time on the X axis.

  1. Select Analyze/Reset Size to select the dried weight of the sample.

  2. Select Analyze/Humidity/Options and set up the desired analysis options.

  3. Manually mark the adsorption/desorption isotherms, if desired. Select Analyze/Humidity/Mark Adsorption/Desorption then select the desired position on the curve.

  4. Obtain the molecular weight for the sample to be analyzed, if you wish to plot your data using a mole-based signal.

  5. Select Analyze/Humidity then choose the desired analysis type: Sorption Isotherm, BET Analysis, or GAB Analysis.

  6. Select View/ResultsSpreadsheet or Report to view and/or print results.

  7. Select View/Data Table/Spreadsheet or Report to view and/or print the data values.
