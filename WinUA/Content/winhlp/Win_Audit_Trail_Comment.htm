Audit Trail Comment Window

You can add a comment to an existing audit trail entry or add a new audit trail entry to further explain an action or event.

 

Click on the following topic for instructions or scroll down to the desired topic:

 

  • Adding a New Audit Trail Entry

  • Adding a Comment to an Existing Audit Trail Entry

  • Viewing an Audit Trail Entry Report

Adding a New Audit Trail Entry

 

To an a new audit trail entry ["User Comment"} to the audit trail, follow these steps:

 

Select Edit/Audit Trail Comment. Use the Audit Trail Comment window to add a comment to the Advantage Integrity Audit Trail for the current data file. The comment is described in the Audit Trail as a "User Comment."

 

  1. Select the arrow to the right of Project and select an Advantage Integrity project from the drop-down list to associate your comment with a project or select "Unassigned" for no project association.

  2. Select an item to associate the comment with from the radio buttons: Current File or No Association.

  3. Enter a description of the database file or click the Clipboard button to paste the information copied to the clipboard in the comment entry field. The maximum length of the comment is 255 characters. NOTE: You can clear the description box by clicking the Clear button.

  4. Click the OK button when finished.

 

Adding a Comment to an Existing Audit Trail Entry

 

Open the Audit Trail Viewer by selecting View/Audit Trail.

Browse the audit trail to find the desired entry for the additional comment. See also: Searching for Audit Trail Events. Once the audit entry is highlighted, double click on the audit entry to display the Additional Comments to an Existing Log window.

 

Type the desired comment to be added to the comments section for this audit trail entry.

 

Select OK when finished.

 

Viewing an Audit Trail Entry Report

 

Comments can be seen in the Comment column within the audit trail viewer, along with the data and time of entry, and user name. Multiple comments may be entered for any entry. You may need to refresh the viewer to see it after immediately adding the entry.

 

To view any audit entry in textural format, browse the tree-view to the lowest level and highlight the desired Audit Entry. A textural report for this entry will be displayed in the right-hand table view pane.
