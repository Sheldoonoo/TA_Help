Time-Temperature Superposition

 

Rheology Advantage will shift flow, creep, and oscillation files. The Instrument control software has an oscillation template (Assist option in procedure or the Wizard button) that will result in a single multi-step file being generated. TTS can also be done on multiple files.

  1. From the File menu, click on Open and select the required file(s).

NOTE: You can select more than one file in the selection box.

  1. Highlight all files in the selection box and then right click with the mouse and select To Graph.

NOTE: You can also press the button.

  1. Click on Graph from the menu and select Change variables. Set the X-axis variable to ang.frequency.
  2. Set  Y1 to G' and Y2 to G"  (use None for all other axes).
  3. Click the OK button.
  4. Click on Graph from the menu and select Lin/Log.  Check the Log box for all axes.

NOTE:  If the Y axes have been locked, changing the setting for Y1 will automatically change Y2. To lock the graph:  (1) Select Graph from the menu. (2) Click on Miscellaneous. (3) Select Lock related y-axes to lock the scaling.

  1. Click the OK button.  Click here for an example TTS Example 1

  2. Select TTS from the menu.
  3. Click on Begin session.
  4. Highlight a curve [G', G" etc.] on the graph or select a temperature and variable in the key and then click the Set reference curve button on the TTS tool bar or select from the TTS menu.

  5. Click on the Shift button or select Shift curves from the TTS menu.  The curves will shift automatically and a new TTS type file appears in the file list.
  6. Perform the following optional procedures, if desired:
  1. Adjust the shift manually, if desired, by clicking on a curve and using the buttons.
  2. Highlight the new file and select View/Table to get the table of shift factors.
  3. Highlight the new file and select View/Send Data to Graph to get a graph of the shift factors. Click here for an example TTS Example 2

  4. Analyze the shift factors by selecting WLF or Arrhenius from the list of models, select the curve, and click on the 'Go' icon.
  5. Select Windows/Cascade to see that all three windows are open and then you can easily switch between them.
