Message 655

Run terminated - specified flow rate cannot be maintained.

Problem:

There is a problem with the gas supply. The specified flow rate needed for the balance cannot be maintained for the duration of the run. The run has stopped because the option, "Stop experiment when flow rate deviates from the set value" was selected on the Instrument Preferences MFC Purge Page.

Solution:

Check to make sure there is an adequate supply of gas and that it has not been turned off, or that the gas line has not been blocked. Replace the purge gas tank, if needed, then retry the operation.
