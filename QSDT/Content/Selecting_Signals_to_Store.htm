Selecting SDT Signals to Store

 

Every analytical instrument produces distinctive signals that may or may not be stored in a data file. To select the signals that you want to be saved in data files produced during this experiment, access the Summary Page in Experimental View. Click the button. Check the signals to be stored from the list shown. Then click the OK button.

Some examples of available SDT signals stored are listed below. These are the signals available in SDT standard mode.

  • Time (min)  =  Time since run start

  • Temperature (°C)  =  Sample temperature

  • Weight (mg)  =  Sample weight

  • Delta Weight (mg) = Weight difference between sample and reference

  • Heat Flow (mW) = DSC heat flow data

  • Delta T (°C)  =  Temperature difference between sample and reference

  • Delta T (µV)  =  Temperature difference between sample and reference

  • Sample Purge Flow (mL/min) = Purge flow rate of the sample

When using dual sample mode a different saved signal list is available. These signals include Temperature Difference (°C), Temperature Difference (°C/mg), Weight (mg), and Weight Percent (%) for both side A (sample beam side) and side B (reference beam side).  It is important to select the normalized signals, if these signals are desired for display and analysis within the Universal Analysis program.

The signals selected apply only to the current run. For a description of the available SDT signals, see Using the Signal Display Pane.
