Digitally Verifying Data

 

This procedure is utilized if you have existing data that has not been digitally verified. If you have installed Advantage Integrity software V 1.1 or later, this step will have already been performed during the software upgrade procedure.

This procedure was implemented to increase security of data. Depending on the amount of information in the database this process will take a few minutes to complete.

  1. Open the TA Instrument System Manager and log in using the System Manager account.
  2. Enter your company name in the window as requested then select OK.
  3. A message will be posted to allow you to digitally verify the data. Select Yes. (This is a one time procedure.)
