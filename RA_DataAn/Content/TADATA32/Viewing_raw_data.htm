Viewing 'Raw' Data

 

The Rheology Advantage Instrument control software lets you store 'raw' or point data as part of a results file. Use the following steps to display this information.

  1. Select Options from the menu.
  2. Click on Settings.
  3. Click on the View tab.
  4. Highlight Point graph to show steady state information for flow data and sine waves for oscillatory data.
  5. Highlight Strain control to show the stain history used in strain controlled oscillatory data.
  6. Click on the OK button.
  7. You can move the new windows around on the screen. If you move them to the edge of the screen, they will dock.
  8. Right clicking on these windows allows you to switch the display between graph mode and table mode.
