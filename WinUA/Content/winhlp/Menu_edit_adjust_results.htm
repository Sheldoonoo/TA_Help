Adjust Last Result

Use the Adjust Last Result option to fine tune the position of the analysis tangent lines. This option is available after you have performed an operation such as:

Integrate Peak

Glass Transition

Step Transition

Onset Point

Slope

Oxidation Time/Temperature

Alpha Fit X1 to X2

Annotate

Add Scale Bar

 

1. Select Edit/Adjust Last Result from the main menu (or right click on a label associated with one of these results and select Adjust Last Result from the pop-up menu) immediately after one of the analyses listed above. Markers for each of the analysis tangent lines will be displayed.

2. Select the desired marker and reposition it to the new location, as desired. The revised analysis results will be shown when you are finished.

3. Right click to display the Analysis pop-up menu, then select Accept Results (or press Enter) when you have completed the adjustment.

 

To Cancel this operation, select Cancel from the Analysis pop-up menu or press the Esc key.
