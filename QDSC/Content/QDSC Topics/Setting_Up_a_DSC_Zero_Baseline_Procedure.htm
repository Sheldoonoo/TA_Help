Setting Up a DSC Zero Baseline Procedure

 

This template can be used to adjust the absolute value of the heat flow baseline to zero mW at the lower and upper temperature limits without requiring the complete Tzero calibration procedure.  This template should be used between routine Tzero calibration whenever the offset and slope have drifted from zero.  It is not necessary to use this template if the full Tzero calibration is performed.

Using the Experimental View Procedure Page:

  1. Display the Summary Page and select the Standard mode.
  2. Display the Procedure Page. Select "Zero Baseline" from the list of available preprogrammed Test  templates.  Enter the desired parameters.
    • Lower and Upper Temperatures:  These temperatures should bracket the region where sample transitions are expected to occur and be the same temperature limits as used when performing Tzero calibration.

    • Additional Conditions: There are additional experimental conditions that can be changed to optimize specific situations. These are accessed using Advanced Parameters and Post-Test Conditions. For most experiments, the system default values shown are recommended.  
  3. When you have finished setting up this procedure, you can enter more information by selecting the Notes Page and Summary Page for this run.
  4. Click the Apply button when finished to save these changes.
  5. If more than one run is in the sequence list, schedule this run. (An arrow will appear next to the run number in the Sequence Pane for the scheduled run.)
  6. Select Start to begin the experiment.  When the second segment executes the appropriate slope and offset values are calculated to produce a level baseline between the two temperatures.
