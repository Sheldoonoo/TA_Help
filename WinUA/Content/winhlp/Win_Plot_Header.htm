Plot Header Window

Use the Plot Header window to set up a customized header that will be placed on the top of an exported or printed graph. Follow the steps below to configure the header.

 

To view the header, select File/Print Preview after you have created one.

 

1. Select the radio button for the desired header type: Standard, Custom, Blank, or No Header.

2. Type in the desired text in the "Custom header text" area, if you have selected the Custom radio button in step 1. A maximum of four lines of text can be entered. You can set the font for this text by clicking the Font button.

3. Click on the desired button to choose the alignment of the text from the following justifications: Left, Center, or Right.

4. Click the OK button when finished.

 

NOTE: The options that are set on this window will reset theMain Options  Export Plot Pageand vice versa.
