Save Analysis Results?

Save analysis results for <filename>?

 

This message prompts you to save the analysis results that were generated using the current data file.

 

If you select "Yes," a second message, "Save file data with analysis?" is displayed to allow you to save the data file with the analysis results.

If you select "No," the data file is closed without saving the analysis or data file.

If you select "Cancel," the operation is aborted and you will return to the open data file.
