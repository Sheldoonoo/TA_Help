Cannot Find Multiplex Signal

Cannot find multiplexed signal in data file.

 

Problem:

The "Multiplex" parameter, in the data files parameter block, specifies the name of a multiplexed signal that was not found in the data file.

 

Solution:

Examine the data files parameter block using the View/Parameter Block function. Check the "Multiplex" parameter and its designated signal name for accuracy. If the data file has been modified, or was not generated by a Thermal Advantage program, then correct the error in the file modification/generation process, otherwise, contact TA Instruments for assistance.
