Must Recalibrate When Heat Flow Changed

Changes to this heat flow selection require that you recalibrate cell constant and temperature. Are you sure you want to make this change?

This message is displayed when you change the DSC heat flow selection. It reminds you that you will need to recalibrate for cell constant and temperature if you make this change. Proceed if desired. If you change the heat flow signal, use the Calibration Wizard to calibrate the cell constant and temperature again before running experiments with this new setting.
