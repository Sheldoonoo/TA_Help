Basic Steps Needed to Run a DMA Experiment

 

All of your DMA experiments will have the following general outline. In some cases not all of these steps will be performed.

  • Make sure the instrument is fully prepared and that the air bearing and cooling gases have been connected. See Connecting the Air Cool Outlet, Connecting the Cooling Gas Line.

  • Select the sample material.

  • Choose and install the clamp appropriate for the sample shape and modulus range. See Available DMA Clamps.

  • Calibrate the clamp. See Calibrating the DMA.

  • Measure the sample dimensions and load the sample into the clamp. See Loading a DMA Sample.

  • Position the thermocouple near the sample. See Aligning the DMA Thermocouples.

  • Close the furnace by pressing the FURNACE key on the touch screen or selecting Control/Furnace/Close.

  • Enter the experiment information through the TA controller, this includes both sample and instrument information on the Summary and Notes pages. See Using the Summary Page and Setting Up the Notes Page.

  • Create or select the experimental procedure using the instrument control software.  See Specifying a Test Procedure.

  • Create a run sequence, if desired. See Creating Custom Methods or Available DMA Tests.

  • Start the experiment. See Guidelines for Running Subambient Experiments or Starting a Run and/or Sequence.
