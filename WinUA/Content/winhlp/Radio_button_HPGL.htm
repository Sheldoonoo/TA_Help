HPGL Radio Button

Click on this button to export your plot to a file using the HPGL format. If you select this option, you must also choose a HPGL symbol type from the HPGL list.

 

NOTE: The ASCII only type is generally more universally recognized than the UC (user characters) type.

 

  • Special symbols (such as degrees symbols, etc.) appear as lower case equivalents when you use the ASCII only type.

 

  • Special symbols are drawn as vector equivalents of the symbols when you use the UC type. The UC type should be used with plotters or programs that can recognize UC symbols.
