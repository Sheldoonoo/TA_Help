File Information

Use the File Information option to modify the parameters on the Data File Information Window. When you select this menu item, the File Information Window is displayed to allow you to edit the information contained in the data file. If only text items are changed, the data in memory is not reloaded (i.e., current analyses are not lost). If any other items are changed, the data is reread from the file so the new values (e.g.,size, cell constant) can be applied. These changes are not saved to the original data file.
