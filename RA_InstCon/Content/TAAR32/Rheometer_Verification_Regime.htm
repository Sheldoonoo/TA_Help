Rheometer Verification Regime

 

The following is an example of a rheometer verification regime that could be implemented as part of a quality audit procedure. This procedure could be built up and operated using the Navigator software for an automated verification method.

Rheology Navigator Software:  This is a separate software package available from TA Instruments. It allows you to build up and operate the rheometer using 'scripts.' These scripts allow you to fully automate the operation of the instrument.

  • Verify the calibration of the rheometer air bearing. See: Calibrating Instrument Inertia
  • Verify the calibration of the standard measuring geometry. See: Calibrating the Measuring System Inertia
  • Perform a rotational mapping. See: Rotational Mapping
  • Verify the calibration of the air-bearing friction factor. See: Determining the Friction Correction Factor
  • Verify the temperature calibration of the rheometer. See: Calibrating the Temperature
  • Obtain the measuring system 'zero datum.' See: Finding the Zero (Datum) Point
  • Run a flow curve using a rheology standard fluid (verify viscosity.) See: System Verification
