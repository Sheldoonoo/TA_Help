Abort Autoqueue Changes?

The autoqueue file has been modified since the last update. Saving this change will discard any new records in the file. It is recommended that you abort this change and reload the autoqueue file before continuing.

 

Abort change and reload the autoqueue file?

 

Problem:

The system has detected the fact that the autoqueue file was manually modified since the last time it was updated by Instrument Control. If you decide to save the manual modification, any new records that were appended in the meantime by Instrument Control will be lost.

 

Solution:

Choose one of the following:

 

  • Select Yes to abort (discard) the manual changes to the autoqueue and allow Instrument Control to append the new records to the file. You can go back into the autoqueue file and make the manual changes later.

  • Select No to save the manual changes that you made to the autoqueue file. If you select this option, the new records from Instrument Control cannot be saved.
