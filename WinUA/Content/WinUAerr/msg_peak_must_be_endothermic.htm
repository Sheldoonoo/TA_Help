Peak Must Be Endothermic

Peak must be endothermic.

 

Problem:

In order for this program to work effectively, there must be a complete endothermic peak within the limits that you have specified for analysis.

 

Solution:

Try one of the following:

 

(1) Reselect the limits so that the endothermic peak is contained within the area under analysis.

(2) Use a different data file that does contain a complete endothermic peak.
