LoadVariablesFromFile

Loads and sets the values for all of the variables contained in the given variables file. Any variables already defined will be overwritten with the value contained in the variable file.

Syntax

LoadVariablesFromFile(<FileName>)

The default extension for navigator variable files is NAVVAR
Variable files can be created by means of the StoreVariablesToFile command. They are simple text files and so can also be created by, for example, the program Notepad

Example

Code

 V0=1.23
 S1="Hello"
 V1=-6.3e-3
 V2 =V0 * V1
 StoreVariablesToFile ("c:\ta\rheology\scripts\myvars.navvar")

Variables file ( c:\ta\rheology\scripts\myvars.navvar )

 V0=1.230
 S1="Hello"
 V1=-6.300E-3
 V2=-7.749E-3
