Introducing the Auto TGA

 

The TGA Autosampler, known as the Auto TGA, is an accessory to the TA Instruments Thermogravimetric Analyzer (TGA) Q500. It allows you to place up to 16 samples at one time on the sample platform to measure the amount and rate of weight change in the samples one by one. Experiments are performed as they normally would be using the TGA with a single sampleâ€”but now you can run samples on a continual basis.

In the Autosampler configuration, the sample tray is modified to accommodate up to 16 samples. Furthermore, up to four trays (64 samples) can be pretared and experimental conditions queued to run unattended for increased productivity.

Using the Auto TGA

In general, the operation of a TGA with the Autosampler option installed is very similar to a single-sample TGA instrument.

NOTE:  If you have the QNX/Platinum features installed, many of these functions may be scheduled to run automatically. See Introducing Platinum for more details.

See the instructions below:

  • Calibratethe Auto TGA for weight and temperature at least once a month or when replacing the hang-down wire. See Calibrating the TGA Weight and Temperature Calibrating the TGA for instructions.

  • Adjust the sample platform if the sample hang-down wire fails to pick up a sample pan. See TGA Platform Calibration.

  • Select the type of pan. The six(6) standard TGA pans can be used on the Auto TGA. See also Choosing a TGA Pan. Once you have selected the type of pan that you wish to use, you must use the same type of pan for all of the samples on the sample platform disk. The same type of pan that you use for experiments must also be used as the tare pan.

  • Prepare the samples. When using the Auto TGA you can prepare from 1 to 64 different samples. Each sample platform can hold 16 numbered samples. The four sample platforms are numbered in the following ranges: 1 to 16, 17 to 32, 33 to 48, and 49 to 64 samples.

  • Tare the sample pans. When you tare the sample pans, the TGA reads the weight of the empty pans in their numbered order and then stores these weights as a set of offsets. Fill the platform with the desired number of empty, clean pans. These offsets are subtracted from subsequent weight measurements for each numbered sample. When you select Calibrate/Tare from the menu or click the button, the Tare Utility window is displayed to allow you to tare any or all of the 16 pans on the tray. (Alternatively you can select TARE TRAY from the Control Menu on the touch screen to tare all of the positions on the current tray. Review the Instrument Log after taring to verify that all of the pans were successfully tared. If a pan load error occurred, simply tare that pan again. Once the pans are tared, do not switch their locations on the sample platform.

NOTE: View the Autosampler Log after taring to determine which pan, if any, does not tare during the procedure. If a tare error occurs, replace the pan and manually retare it.

NOTE: Do not switch the pans around on the sample platform because they have already been tared in their number position.

  • Load the samples. After you have finished taring the pans, load each sample in the correct sample pan, returning the pan, if removed, to its proper original pan location.

  1. Make sure that the wire on the bottom of the sample pans align with the groove in the pan hole, so that the sample can be picked up by the sample hang-down wire.

  2. Verify the thermocouple position. Position the thermocouple at the edge of the sample pan, rather than in the middle for the best results. The position of the thermocouple should be the same as it was during temperature calibration.

  • Set up and start the experiment. Use the Wizard, if desired. See also: Basic Steps Needed to Run a TGA Experiment

  • If you want to stop an Auto TGA run that is already in progress, use either Stop or Reject, found on the Control Menu touch screen or the instrument control menu. If STOP is selected from the instrument control menu, you can choose to either stop the current run and continue with the sequence or stop the sequence. Selecting STOP from the touch screen will automatically stop the sequence.

  • Enter the desired text conditions for each of the samples and its pan number using the instrument control software. Each sample may have a unique experiment.

  • Start the sequence using the instrument control Start function.
