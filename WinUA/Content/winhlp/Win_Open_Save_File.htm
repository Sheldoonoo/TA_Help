Open/Save/Save As/Load File Window

Use this window to locate and open, save, or load a file.

 

1. Select the arrow to the right of Look in: to display all the folders on your system. Optional: You can directly type the full path and filename of the desired file in the File Name field and skip to step 5.

2. Select a folder from the list.

3. Select the type of file from the list appearing at the bottom of the window.

4. Select a filename from the File list. This name will appear in the File name field at the bottom of the window.

5. Click the Open, Save, or Load button.
