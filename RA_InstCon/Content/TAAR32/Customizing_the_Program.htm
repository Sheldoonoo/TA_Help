Displaying Raw (Point) Data

Use the following steps to display the point graph for Oscillatory and steady state flow procedures.

Flow Procedures

  1. Select Options from the menu.
  2. Click on Experiment.
  3. Click on the Flow tab.
  4. Select the option Display point graph during run.
  5. Click on the OK button.

Oscillation Procedures

  1. Select Options from the menu.
  2. Click on Experiment.
  3. Click on the Oscillation tab.
  4. Select the option Display waveform during run.
  5. Click on the OK button.

Change the Button Size

  1. Select Options from the menu.
  2. Click on General.
  3. Click on the View tab.
  4. Use the Large buttons check box to select between large and small buttons.
  5. Click the OK button.

Viewing the Default File Locations

  1. Select Options from the menu.
  2. Click on Settings.
  3. Click on the Directories tab.

The last used Set folder locations for various file types are displayed. The software sets this to the location used on the last file save.
