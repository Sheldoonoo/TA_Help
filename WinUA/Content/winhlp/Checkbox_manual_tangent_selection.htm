Checkbox Manual Tangent Selection

Check the Manual tangent selection box if you want to manually select the start and stop limits for each tangent line used in the analysis. If you choose the manual mode, the program uses the limits you select to fit the tangent lines to the curve.

 

If you do not check this box, the analysis will be done in automatic mode. This means that you will only need to enter the start and stop data limits to be used in the analysis and the program will take care of the rest.

 

NOTE: When manual tangent mode is selected, theOnset at Inflectionoption is ignored. In this mode the start and stop points for the onset line are entered by the user.
