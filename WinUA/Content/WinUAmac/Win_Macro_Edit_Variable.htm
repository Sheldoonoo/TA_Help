Macro Edit Variable Window

The Edit Variable macro step is used to display a dialog box in which you can view and edit the contents of a macro variable. This is useful for entering or changing a result or macro parameter.

 

 If the variable name is already defined at the time that the Edit Variable step is executed, then the current value is displayed in an edit field to accept or modify.

 If the variable name is undefined, then the variable is created and initialized to zero prior to opening the dialog box. The numeric value entered in the dialog box may be an integer or a floating-point number.

 

1. Select (highlight) the desired position (step number) for the macro step insertion.

2. Select Insert/Edit Variable from the Macro Editor menu. The Macro Edit Variable window is displayed.

 

To use this window:

 

3. Enter the Display Text to be used in the dialog (up to a maximum of 30 characters). NOTE: If more than 30 characters of text must be displayed, or if more than one value must be entered in a single dialog, then consider using theMacro Edit Clipboardstep instead. After the Macro Edit Clipboard window is accepted, the individual values that were entered can be extracted from the clipboard by using one or more Macro Assign Variable steps, specifying Clipboard (value) or Clipboard (variable, value) as the operation.

4. Check the Beep box, if you want the beeper to sound as the dialog opens.

5. Enter a Variable name. A macro variable name may consist of up to 30 characters. The first character must be alphabetic. The remaining characters may be alphabetic, digits, or the underline (underbar) character. Imbedded spaces are not permitted in macro variable names. A macro variable array element is specified by following the variable name with an array index enclosed in square brackets ("[<index>]"). The array index may be an integer constant or a simple variable name (no index).

6. Enter the number of Significant figures for the entered value.

7. Check the Round to integer box, if you want to restrict the users entry to an integer value.

8. Click OK when finished. The step will be inserted at the selected position.
