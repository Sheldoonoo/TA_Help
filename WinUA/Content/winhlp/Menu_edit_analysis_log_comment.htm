Analysis Log Comment

Use this menu item to access the Analysis Log Comment window, which allows you to add a comment to the analysis log file for the current data file. The comment is identified in the log by " / / " in front of the entered text.

 

About the Analysis Log

The analysis log captures every step used to manipulate and view the current data file and allows you to view and print the steps. You can also convert the analysis log to a program macro using the Macros/Copy/Analysis Log menu item.

 

NOTE: The analysis log will hold up to 800 steps. Any additional steps are lost and a warning message is displayed, "Maximum number of analysis steps already defined." Analysis steps from an executing macro are not included in the log and, therefore, do not affect the 800 step maximum. Only the name of the macro is included in the log.
