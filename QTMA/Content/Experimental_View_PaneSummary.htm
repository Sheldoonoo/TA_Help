Using the Summary Page

 

Sample information for each run is entered on both the Summary Page and Notes Page. The Procedure Page is used to set up the experimental test conditions.

When you select View/Experiment View from the menu, the middle pane displays the Summary Page. The Summary Page is used to enter the basic information needed to create a run. The summary information that is currently displayed is applicable to the run number that is highlighted on the Sequence Pane. The run that has the arrow pointing to it is the scheduled run. (i.e., The run that will be executed when Start is selected.) This information should be updated with the correct information prior to starting each experiment.

To use the Summary page, follow these directions:

  1. Verify that the Mode displayed is the one you want the instrument to use. Additional modes are available when you have the EM (Enhanced Mode) Option installed. See:  Selecting a TMA Operating Mode
  2. Select the signals that you want the software to store in the data file by clicking the button. This displays the Select Signals to Store dialog. Check the desired signals.
  3. Select the type of Test you want to use from the drop-down list available. The list contains both preprogrammed procedures (test templates) and a custom test. To open a saved procedure file, click the Open File button, . This file will be added to the test list once you have selected it. You can view a summary of the chosen procedure by selecting the Summary button, . To change any of the summary information, except for the mode and clamp, select the Procedure Page.
  4. Select the Probe from the drop-down list available. See Choosing a Probe for information. If you are switching probes, you will need to perform probe calibration before proceeding with the experiment. See Calibrating a TMA Probe for information.
  5. Standard TMA Only:  Enter the sample Size. (Size is the measurement of the sample between the probe and the stage.)
  6. EM Option Only:  Select the type of Sample Shape that you will be using for your sample from the list shown. The available choices depend on the probe selected. For additional information on each type of geometry, see the following list:
    1. Rectangular - sample is rectangular in shape. Enter the length or thickness, width, and thickness or height that applies to the selected probe.
    2. Cylindrical - sample is cylindrical in shape. Enter the length or thickness and diameter that applies to the selected probe.
    3. Tubular - sample is tube-shaped (hollow in the middle). Enter the length and diameter of the inside and outside of the tube.
    4. Generic - sample is unusual in shape. Enter the dimensions appropriate for the selected probe.
    5. Ring – sample is round in shape. Enter an inner and outer diameter, and a thickness.
    6. Square Disk – disk type sample is in the shape of a square. Enter the thickness and width.
    7. Round Disk – disk type sample is round. Enter the thickness and diameter.
    8. Unknown – this will be displayed when the probe type is changed. You will need to reselect the sample shape.

  1. EM Option Only:  Enter the Dimensions appropriate to the sample shape selected above.
  2. Enter the Sample Name. Enter up to 32 alphanumeric characters to identify the material.
  3. Enter any Comments. Enter additional text to help describe the experiment, for example, sample treatment or experimental conditions could be included. Up to 64 characters may be entered.
  4. Enter a Data File Name by clicking on the button, then select the desired storage location and enter in a filename. Click OK when finished. Typical example \\{Computer name}\Program Files\TA\Data\DMA\Data.001. (UNC filenames are used.)
  5. Check the Network Drive checkbox if you want to send your instrument data to another location on your system (e.g., network drive). Click the Browse button to the right, then select the desired archive location. Click Select when finished. The data will be transferred at the end of the experiment.
  6. If you want to automatically analyze the resulting data file using a macro (created with the Universal Analysis Autoanalysis software option), check the Autoanalyze box. Then locate the desired macro filename using the Browse button. Enter the drive, directory, file location, and name for file.
    1. If you want to automatically have the results of the autoanalysis (checked above) sent to a remote computer, which is one  other than the controller computer attached to the instrument (also called the local computer), check the Start Remotely box. Then enter the same path and data filename chosen above. When the data file has been collected, Universal Analysis will automatically open and begin the autoanalysis of the file on both the local and remote computers.
    2. Click on the Analysis Macro list and select a macro from those displayed. This macro will be used to automatically analyze the data file that will result from this experiment.

NOTE: See the Autoanalysis documentation accompanying the Universal Analysis software for further information on creating macros.
