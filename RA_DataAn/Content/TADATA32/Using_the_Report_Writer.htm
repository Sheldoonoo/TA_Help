Using the Report Writer

 

This feature integrates report layouts (created using Microsoft ® Word 97 or above) with data analysis.

The easiest way to get started is to modify one of the pre-installed templates. Alternatively, you can create a template from scratch in Microsoft ® Word (File\New).
