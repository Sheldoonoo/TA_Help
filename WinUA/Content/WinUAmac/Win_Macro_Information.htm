Macro Information Window

Use the Macro Information window to enter two items regarding the macro file you are about to capture: The name, which is restricted to 30 characters and the Description, which is restricted to 300 characters.

 

When you have finished entering the name and description, click the OK button. The capture function is automatically turned on and the Macro Editor window is displayed.

 

This window can be displayed by selecting Macro/New or Macro/Information from the main menu.

 

NOTE: For more details on creating, editing, and running macros, see the Universal Analysis online manual (Help/Manual) or use Acrobat Reader to access the Uamanual.pdf file contained on the CDROM.
