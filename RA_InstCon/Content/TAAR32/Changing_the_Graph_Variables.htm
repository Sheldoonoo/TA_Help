Changing the Graph Variables

Follow the steps below to change the graph variables:

  1. Select Experiment from the menu.
  2. Click on Graph.
  3. Select Change variables…
  4. Select the required variable for each axis from the list. If you do not wish to show anything on an axis, select <None>.
  5. If you wish to use the default settings, click on the Set to defaults button.
  6. Click on the OK button.
