Label Size Window

Use the Label Size window to choose a point size for the annotation text you have already created. The size change will not affect the size of the annotation text displayed on the monitorit will affect the size of annotation text on the PRINTED graph.

 

1. Click on the arrow to display the drop-down list of text point sizes.

2. Click on the desired point size.

3. Click the OK button to save your selection. The annotation text will be PRINTED in the size chosen. The annotation that is displayed on your monitor will not change.

 

NOTE: When the annotations are displayed, you can drag and drop the annotations anywhere on the graph using the left mouse button. You can edit or delete the annotations by placing the pointer over the annotation and clicking the right mouse button to display theannotate pop-up menu.
