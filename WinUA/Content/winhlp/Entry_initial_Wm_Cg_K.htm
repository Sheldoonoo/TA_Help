Entry Initial Wm, Cg, and K Constants

Enter an initial value for each of the GAB constants. The regression analysis will start with these values and then try to improve the fit of the GAB equation to the raw data.

 

Because the GAB modeling equation is not a simple polynominal, it can have more than one solution, depending upon the experimental data. If the moisture data is sparse or not completely appropriate for the GAB model, then the non-linear regression analysis can generate a solution that fits the data, but does not make sense from the perspective of moisture analysis. Entering initial values that are close to the expected results can help prevent the return of any inappropriate solutions.
