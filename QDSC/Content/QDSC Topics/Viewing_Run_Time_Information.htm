Viewing Run Time Information

 

You can get run information about your instrument by either running a diagnostics report using the Platinum Auto Diagnostics feature or by selecting View/Run Time Information from the menu.

When you select View/Run Time Information a dialog is displayed allowing you to view run time, number of runs, and time above a specified temperature (as applicable) to provide you with an overview of the experiments run on your instrument.

The list of items displayed on this window vary based on your instrument type. An general overview of the available parameters is listed below.

You can reset the time by clicking the Reset button next to the desired parameter.

  • Run Time:  The sum total of the amount of time the instrument has been run since the last time you clicked the Reset button.
  • Runs: The number of standard runs performed by the instrument since the last time you clicked the Reset button.
  • Time above x °C: The amount of time the instrument has been operated at temperatures above x °C since the last time you clicked the Reset button. This information can be used to help evaluate the performance of the furnace/heater.

These values are also displayed in the diagnostic report. See also:  Introducing the Auto Diagnostics Report.
