Auto Scale New Data X Menu

Click on Autoscale New Data X to select this function. When checked the current files x-axis data will be autoscaled when the program reads in new data for Graph/Data Limits, Tools/Linear Transform, or a new Graph/X axis. This is the default setting.

 

Leave this box unchecked and the current x-axis data will not be autoscaled when the program reads in new data (x-axis scaling is preserved).

 

NOTE: This option applies only to thecurrentdata file. The "Autoscale new data: X axis" checkbox found on theMain OptionsData Pageis the default option that applies to each file as it is first opened.
