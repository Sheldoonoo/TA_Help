Symbol Differentiator List

Select the differentiator to be used for symbol designation from the drop-down list.

 

  • Curves is used to have the symbol that is used for a curve change to the next symbol in the list at a break in the data. (E.g., data breaks, change in multiplexed signal value.)

  • Axis/Axes is used to have the symbol that is used for an axis change when the next axis is drawn.

  • Subcurves is used to have the symbol that is used for a curve change to the next symbol in the list when a subcurve is displayed. This option is only available for curve overlay plots.
