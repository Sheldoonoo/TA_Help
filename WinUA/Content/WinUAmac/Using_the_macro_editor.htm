Using the Macro Editor

Select Macro Editor from the Macros menu to display the Macro Editor window for the current macro. This window displays a list of all of the steps contained in the macro as they are captured and provides valuable information when you capture, view, edit, and debug the macro steps.

 

Shortcut Key

Ctrl + E

 

NOTE: If the Macro Editor window gets covered up by another window, you can bring it back to the front by pressing Ctrl+E two times.
