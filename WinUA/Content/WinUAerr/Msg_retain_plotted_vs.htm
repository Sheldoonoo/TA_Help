Retain Plotted vs <X Type>

Retain <result type>? [Result is plotted vs <x type>.]

 

This message occurs when you perform an analysis that is valid for a certain X-axis signal, then you change the X-axis signal and choose Edit/Delete Results/Select. When the program comes to the result that is no longer displayed on the window, it will display the message above. Select Yes to keep the result named, No to delete the result, or Cancel to abort this operation.
