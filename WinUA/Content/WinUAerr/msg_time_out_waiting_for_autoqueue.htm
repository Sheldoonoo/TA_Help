Timed Out While Waiting for Autoqueue

Time out while waiting for autoqueue access.

 

Problem:

The system is trying to add records to the autoqueue from experiments that have been completed on the instrument. The autoqueue file is in use. Perhaps you are editing the autoqueue while the system is trying to add records for autoanalysis. The process has timed out while waiting to access the autoqueue.

 

Solution:

Do not access the autoqueue file while Instrument Control is in the process of appending records to the autoqueue for autoanalysis. If a time out does occur, close the autoqueue window in Universal Analysis.
