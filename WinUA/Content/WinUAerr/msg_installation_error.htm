Installation Error

 

Installation error 0, 1, 2, etc.

 

Problem:

The Universal Analysis program was not installed from the distribution CD-ROM or the installation was corrupted.

 

Solution:

Install or reinstall the Universal Analysis program from the distribution CD-ROM.

 

If you are attempting to upgrade just the WinUA.exe file, you must still have an existing version of Universal Analysis on your system that was installed from the distribution CD-ROM. After installing Universal Analysis from the distribution CD-ROM, you can replace the WinUA.exe file with a newer version of the program.

 

It is recommended that you install Universal Analysis from the distribution CD-ROM to ensure that up-to-date version of all associated and required files are present on your system.
