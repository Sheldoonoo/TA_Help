More Than Ten Curves

The number of curves per axis exceeds the limit of 10 curves. Do you want to plot the remaining curves using additional axes? Selecting "No" will plot only the first 10 curves on each axis.

 

This message is displayed if you attempt to use Overlay/Auto Configure when more than ten curves, with the same signal, units, and Y axis, are selected.

 

  • Select "Yes" to plot the first ten curves on the first Y axis and the remaining curves on the second Y axis. After plotting them in this manner, you can set the scaling of both axes so that they are similar by using the Rescale/Common Scale function.

  • Selecting "No" will give you an overlay graph that contains only the first ten curves.
