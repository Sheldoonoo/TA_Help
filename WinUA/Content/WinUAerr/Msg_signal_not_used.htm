Signal Not Used

Baseline signal "<signal name>" not used.

 

Problem:

A selected baseline file signal does not match any of the current data file signals and will not be used.

 

Solution:

Try one of the following:

 

1. Ignore the error, if the signal is not required for operation.

2. Plot the same signals for both the sample and reference files, then retry the operation.

3. Select another baseline file with signals that match the current data file signals.
