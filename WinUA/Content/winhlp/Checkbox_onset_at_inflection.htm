Checkbox Onset at Inflection

This box applies to Onset and Oxidation analysis with automatic (2-point) cursors.

  • Check the box to fit the onset line to the curve at the inflection point on the curve. If no inflection point is encountered between the two cursors, the onset line is fitted to the data at the point of highest slope. In this case, the onset line is fitted to a data region whose width is 12.5 % of the vector length of the curve between the two cursors.

  • Uncheck the box to fit the onset line to a data region whose width is 12.5 % of the vector length of the curve between the two cursors.

 

Default is checked.

 

NOTE: Onset at inflection cannot be used with manual tangent selection.
