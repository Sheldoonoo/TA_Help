Changing Your Password

 

The Advantage Integrity system relies on password protection to maintain the integrity of the system and the audit trail. Your log in password may be changed at any time desired and must be changed when it expires.  The expiration date is set by your TA Instruments System Manager. A few days before your password expires, you will begin to receive “password expiration” notices each time you log in. To change your password, follow these instructions:

  1. Open the desired program. Once the Log in window is displayed, type in your User ID and old Password, then select New Password.  
  2. Type in the New Password. This password must follow these rules:
    • The password must contain the number of characters defined by the system setup. Any combination of numbers, symbols, or letters is allowed.  
    • Passwords cannot contain spaces.  
    • The password is case sensitive (recognizes upper and lower case letters) by default, unless changed by the system administrator.
    • Passwords cannot be reused for a specific user account.
  3. Type in the new password again to confirm previous entry.
  4. Click the OK button.

See the Advantage Integrity Getting Started Guide or the online help for additional information.
