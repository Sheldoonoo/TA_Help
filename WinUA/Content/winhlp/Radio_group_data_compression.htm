Data Compression Options

Select a radio button as follows:

  • Linear Smoothing: Click on the Linear Smoothing radio button and the program will divide the points into groups and compute an average for each group when the file exceeds the data buffer size. The size of the groups depends on how much data compaction is needed.

  • Discard Data Points: Click on the Discard Points radio button and the program will remove data points linearly throughout the data file when the file exceeds the data buffer size. The number of points removed is based on the amount of data reduction needed.

  • Select New Data Limits: Click on this radio button to select new data limits for plotting using the Data Limits Window, which is displayed after you select OK.
