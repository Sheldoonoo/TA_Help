Introducing Platinum™

 

Platinum™ features are designed to assist you in ensuring that your instrument is in proper working condition, calibrated, and has the latest software available. These features can be performed with minimal operator interaction and can be scheduled to occur on weekends or other times, which do not interfere with normal experiments. The main features include the following:

Click the Platinum button to display a list on the left-hand side of the instrument control window. A brief description of the Platinum components is provided below.

Automatic Tasks

(DSC Autosampler Only)

  • Automatic Calibration Routines:  Used for automatic execution of calibration and/or validation sequence for the DSC Q2000/Q200 equipped with an Autosampler.
  • Cell/Cooler Conditioning: Used for automatic execution of a cell/cooler conditioning experiment. This experiment is run when an instrument and/or cooling accessory is first installed.  This procedure should be run to dry the DSC cell and cooler before you turn on the cooler and start running any experiments.  During this test, the flange temperature limits are raised.  See also: Cell/Cooler Conditioning
  • Baseline Conditioning: Used for automatic execution of a baseline conditioning experiment.  This experiment is run to condition the cell before performing Tzero calibration. See also: Baseline Conditioning
  • Tzero Calibration: Used for automatic execution of Tzero calibration when using Heat Flow T4P or T4. This allows calibration to be performed on demand or scheduled during quiescent instrument times.  See also: Calibrating Tzero. The Platinum features provide the following functions:
    • Scheduling of Tzero calibration. See also: Using the Schedule Page
    • Email and/or Windows Messaging notification of the calibration/verification with the results. See also: Using the Messaging Page
    • Calibration report. See also: Calibration Report
  • T1 Baseline Calibration: Used for automatic execution of baseline calibration when using Heat Flow T1. This allows calibration to be performed on demand or scheduled during quiescent instrument times.  See also: Calibrating the T1 Baseline. The Platinum features provide the following functions:
    • Scheduling of Baseline calibration. See also: Using the Schedule Page
    • Email and/or Windows Messaging notification of the calibration/verification with the results. See also: Using the Messaging Page
    • Calibration report. See also: Calibration Report

  • Cell Constant/Temperature Calibration:  Used for automatic execution of cell constant/temperature calibration and/or validation sequence. This allows calibration and/ or verification of cell performance to be performed on demand or scheduled during quiescent instrument times.  See also: Calibrating the DSC Enthalpy (Cell Constant) and Temperature. The Platinum features provide the following functions:
    • Scheduling of cell calibration with verification. See also: Using the Schedule Page
    • Verification of cell performance with scheduling of automatic cell constant/temperature calibration, if verification does not meet specified acceptable criteria. See also: Using the Schedule Page
    • Email and/or Windows Messaging notification of the calibration/verification with the results. See also: Using the Messaging Page
    • Calibration report. See also: Calibration Report
  • Auto Diagnostics:  Used to perform the diagnostics routine, which provides verification that the instrument subassemblies are in proper operating condition. See also:Using Auto Diagnostics

Scheduling

  • Scheduled Events:  Allows sequences and procedures (autoweight calibration, autotemperature calibration, & diagnostics) to be executed at a specified time/day and frequency interval. See also: Using the Schedule Page
  • Settings: This function allows you to check the schedule settings for all automatic tasks in one place. This is a convenient way to set up the timing of these tasks so that they do not conflict with each other.

Messaging

  • EMail Notification: Provides email and/or Windows messaging notification of subscribed events including experiment and procedure completion with results. Enables you to follow the instrument's activities. (Email notification requires Microsoft Outlook and a dedicated mail account.) See also: Using the Messaging Page

More...
