Calibrating the TGA Q5000 IR Temperature

 

Temperature calibration is useful for experiments in which precise transition temperatures are essential. To temperature calibrate the TGA, you need to analyze a high-purity magnetic standard for its curie temperature, and then enter the observed and correct values in the temperature calibration table.

There are two techniques for TGA temperature calibration recognized by the American Society for Testing and Materials. These techniques are described in ASTM Standards E914-83 and E1582-93. The latter, which is based on the Curie Point of paramagnetic metals, is recommended for the TA Instruments TGA's.

In this technique, a Curie standard is heated in a sample pan in a magnetic field. The Q5000 IR furnace is equipped with an electromagnet to facilitate this operation. As the standard goes through its Curie Point, its attraction to the magnet changes, appearing as a weight change. The extrapolated endset of this weight change is adjusted to agree with the material's known Curie Point Temperature. Up to five calibration points can be entered into the temperature calibration table.  A multi-point calibration is more accurate than a one-point calibration.

Setting Up Platinum for Curie Temperature

The Q5000 is equipped with Curie temperature verification/calibration features accessed through the Platinum functions. These features provide the following:

  • Curie Temperature Verification

  • Scheduling of verification/calibration experiments

  • Email and/or Windows Messaging of process with results

  • Calibration report

To set up Curie temperature verification/calibration using Platinum:

  1. Open the instrument control program for a Q5000 IR instrument. See Using the TA Instrument Explorer for instructions.

  2. Choose the standard that was used for your experiments. (Only one standard can be used with Platinum.)

  3. Select the Platinum button, , at the bottom left-hand side of the main instrument control window.

  4. Click Curie Calibration from the Automatic Tasks list on the left side of the window. Two options will be displayed. Choose the desired option:

    • Use Existing Sequence:  This option leaves the current sequence in place allowing you to customize it or use it as is. After selecting this option, use the Sequence Pane to customize the experiment. Then proceed to Step 6 below.

    • Create New Sequence:  Used to set up a new test with the Platinum functions. This option will replace the current sequence. When you select this option, proceed to Step 5.

  5. Creating a new sequence:

  1. Enter the Method parameters:

    • Equilibrate at:  Select a temperature that is up to 100°C below the literature Curie temperature of the material.

      Isothermal for:  Check this box to hold the sample at the equilibrate temperature for the desired amount of time BEFORE the magnet is applied for calibration. This is done to observe the effect of the magnet on the sample.

    • Heating Rate:  Heat the material at the same heating rate that you will use in your subsequent experiments to 100°C above the literature Curie temperature.

    • Pan #: Use the up/down arrows to choose the pan number containing your sample.

  2. Check the box next to the desired Standard Name. Make sure that you verify the theoretical relative Curie temperature used for each standard. Edit the data if necessary. (Click here for more information on choosing/editing standards Setting Up Calibration Standards.)

  3. Choose your Settings:

    • If you want to determine if your current calibration is acceptable, choose the Verification (Verification is defined as a double-check of calibration, performed immediately after calibration, usually on the same sample to verify that the instrument has accepted the adjustments made during calibration.) radio button.

    • If you wish to perform a full Curie calibration, choose the Calibration (Calibration is defined as the adjustment of an instrument so that it provides correct theoretical value for a specific measurement on a well-characterized [certified] standard.) radio button.

    • Check the Perform Calibration if Verification Fails box to automatically perform the Curie calibration in the event the validation test does not fall within the acceptable validation criteria. Select the Pan # by using the up/down arrows to choose the pan number containing your sample for the calibration experiment.

    • Check the Perform Verification after Calibration box to automatically verify the new calibration, if desired.

    • Enter the desired Verification Criteria.

  4. Select Apply to save the test setup.

  1. Alter the Post Run Action, if desired. The default Post Run Action is to DO (perform) the Curie calibration. You can change this option by selecting a run, right clicking to display the pop-up menu, and selecting Post Run Action. For example, you could select IF -- Successful Curie Calibration -- GO next sequence -- to begin your experimental sequence. See Selecting Post Test Actions for more information.

  2. Place the curie temperature sample in a tared TGA sample pan and position the pan on the sample tray in the numbered position chosen above.

  3. Select the Schedule page to specify the time, day, and frequency of the calibration process.

  4. Select the Messaging page to specify the notification options desired.

  5. Select Start to execute the sequence immediately, if desired. You may also wish to save this calibration sequence for future use. (Click on the Calibration button on the left-hand side. Click on the Save button, , at the top of the sequence pane.

Manually Determining Curie Temperature (Using the Template)

  1. Choose the standard that was used for your experiments. Make sure that you verify the theoretical Curie temperature used for each standard. Edit the data if necessary. (Click here for more information on choosing/editing standards Setting Up Calibration Standards.) Multiple standards may be needed for this purpose. Choose standards whose curie temperature differs substantially; each pair of curie temperatures must differ from all other pairs by at least 10°C.

  2. Tare the desired number of pans needed for the number of standards to be used for temperature calibration.

  3. Select the button to start a calibration sequence. If you wish to clear the current sequence, select to open a new sequence.

  4. Choose the "Calibration" Mode.

  5. Select the "Curie Point Cal" test from the Test list. See Setting Up a Curie Point Temperature Calibration for more information.

  6. Enter the requested sample information. The sample name MUST be the same as the chosen standard's name.

  7. Place the curie temperature sample in a tared TGA sample pan and position the pan on the sample tray.

  8. Click on the Procedure tab. Enter the requested test parameters:

    • Equilibrate at:  Select a temperature that is up to 100°C below the literature Curie temperature of the material.

    • Isothermal for:  Check this box to hold the sample at the equilibrate temperature for the desired amount of time BEFORE the magnet is applied for calibration. This is done to observe the effect of the magnet on the sample.

    • Heating Rate:  Heat the material at the same heating rate that you will use in your subsequent experiments to 100°C above the literature Curie temperature.

  9. Click on the Notes tab. Enter/verify the requested information.

  10. Click on the Apply button to save the experimental and sample parameters entered for this run.

  11. If you wish to run multiple standards, click on the Copy Run/Append to End button, , to add another run to the sequence.

  12. Repeat steps 1 through 10 for each temperature calibration standard.

NOTE: If you want to have the analysis of the data performed automatically using Post Run Actions, skip to Automatic Analysis Using Post Run Actions now. Otherwise, continue with the next step.

  1. If more than one run is in the sequence list, schedule the first run in the sequence. (An arrow will appear next to the run number in the Sequence Pane for the scheduled run.) See Starting a Run and/or Sequence for more information.

  2. Select Start to begin the calibration run. Observe the Weight Loss % from Signal Display. The procedure will automatically turn on the magnet when required and run the experiment.

  3. Analyze the Curie Temperature data as directed in the next section.

  4. Save the calibration sequence for future recall, if desired. Click on the Save button, , at the top of the sequence pane.

Automatic Analysis Using Post Run Actions

This method utilizes Post Run Actions. These are defined while setting up the run sequence and executed at the end of each test. The experimental Curie Temperature is automatically determined and the calibration data is sent to the instrument using this procedure.

  1. Set up the calibration run sequence manually as described above.

  2. Right click on the temperature calibration run number on the Sequence Pane to display the pop-up menu. Select Post Run Action.

  3. Select one of the following:

    • IF -- Successful Curie Calibration -- GOTO -- Run Number. (This will begin the next run if the results meet the verification criteria.)

    • DO -- NONE. (This will begin the next run after the verification run is complete with no dependency on the outcome of the verification criteria result.)

    • IF -- Successful Curie Calibration -- GO next sequence. (This will advance the standard run sequence if the results meet the verification criteria.)

  4. Repeat steps 1 through 3 for each temperature calibration standard used.

  5. Select Start to begin the calibration run. Observe the Weight Loss % from Signal Display. The procedure will automatically turn on the magnet when required and run the experiment.

Entering Temperature Calibration Data

Select Calibrate/Temperature Table to access the Temperature Calibration Table, which is used to input or reset temperature calibration data.

This window shows the temperature calibration table that the instrument applies to the collected data. Use this window to enter from one to five temperature calibration points (pairs of observed and correct temperature points). The observed and correct temperature corresponds to the experimental and theoretical transition temperature (e.g., melting point) of the calibrant respectively.

Data is automatically entered into this table when you select Accept Results through the Calibration Analysis function (described above) or when using the AutoCurie Platinum setup.

NOTE: A multiple-point calibration is more accurate than a one-point calibration. A single-point calibration shifts the sample temperature by a constant amount. A two- or more-point calibration shifts the temperature by a constant amount below the first point, uses a smooth curve through the calibration points and maintains a constant temperature shift after the last point.

 

 If these values are correct, select OK.

 If new values are to be entered follow these steps:

  1. Select the Reset button to remove all previously stored values.

  2. Determine the curie temperature points.

  3. Enter the Observed and Correct temperature points in the table.

  4. Select Apply when all points have been entered to save the settings to the instrument.
