Mass Number Problem

 

No mass numbers found.

Invalid mass numbers found.

 

This message is displayed when you attempt to open a Mass Spectrometer file in Universal Analysis and the program cannot locate or read the mass numbers. The mass number data that the program is attempting to read starts on the next line after the line with the header "Enabled m/e Statistic Info."

 

Check that the "mass numbers" in the file are actually numbers, and not mass names or text descriptions.

 

This particular Mass Spectrometer file cannot be opened with Universal Analysis. Use another file that contains the appropriate mass numbers.
