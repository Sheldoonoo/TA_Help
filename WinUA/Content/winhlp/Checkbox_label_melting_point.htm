Melting Point

Check this box to display the melting point in the results annotation. Melting point is defined as the point at which the crystals of a pure substance are in equilibrium with the liquid phase at atmospheric pressure. This point is commonly referred to as melting point when a substance is approaching it from its solid state and is referred to as freezing point when a substance is approaching it from its liquid state.

 

Depending on whether this value is determined by the data or a known value is used (which is set on the Purity Analysis Options  Purity Calculation Page), this value will be labeled as (determined) or (entered) respectively.

 

Example: Melting Point 134.90°C (determined)
