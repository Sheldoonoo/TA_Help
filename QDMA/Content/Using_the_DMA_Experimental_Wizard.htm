Using the DMA Experimental Wizard

 

DMA is a thermal analysis technique which measures viscoelastic properties including modulus (stiffness) and damping (energy dissipation) of solids.  It is one of three complimentary techniques for characterizing viscoelastic properties.  The other two are dielectric analysis (DEA) and rheology (rotational and torsional).

The DMA Experimental Wizard that will guide you through the set up of different types of Dynamic Mechanical Analyzer (DMA) experiments. These experiments can be performed on TA Instrumentsâ€™ DMA Q800.

Before Using the Wizard

The following DMA calibration procedures should be conducted before using this wizard.

  • Clamp Calibration (mass, zero, and compliance calibrations)

  • Position Calibration

  • Instrument Calibration (electronics, force, and dynamic calibrations)

To verify the current parameters for clamp and instrument calibration, view the Calibration Report.

Starting the Wizard

To set up and run an experiment using the Experimental Wizard, select Experiment/Wizard from the menu or click on the button.

Step 1:  Select a Clamp

The first window that is displayed allows you to choose the type of clamp installed.

Click the radio button for the type of clamp desired, then select Next to proceed. See also:Available DMA Clamps

NOTE:  This wizard is not intended for use with the Penetration, Film/Fiber Submersion, or Compression Submersion clamps.   Use the Experimental View to set up experiments for those clamps.

Step 2:  Choose a Mode

The next window displays a set of five radio buttons to allow you to select the operating mode desired.

Click the radio button for the desired operating mode, then select Next to proceed. See also:  Selecting a DMA Operating Mode

Step 3:  Select an Experiment

The list of tests (experiments) displayed on the window are dependent upon the operating mode and clamp already selected.

Click the radio button for the desired test, then select Next to proceed. See also:  Available DMA Test Templates

Step 4:  Enter the Parameters

Parameters will also vary according to the chosen test. Set up your experiment, then select Next to proceed. For information about the test you have chosen, click the Help button.
