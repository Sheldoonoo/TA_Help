Editing Data

Follow the steps below to edit the data:

  1. Display the data in File View.
  2. Click on the point you wish to edit.
  3. Select Edit from the menu.
  4. Click on Modify Value.
  5. Type in the new value.

NOTE:  Due to the interrelationships between associated rheological variable, changing one data point may alter others in the same row.

 

Any results file that has been modified will be marked with the symbol in the Results view window.

 

NOTE: You can also delete points directly from the graph screen by highlighting the required point and pressing Ctrl D (or select Delete from the Edit menu.)
