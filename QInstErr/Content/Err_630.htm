Message 630

Bad entry in calibration file.

Problem:

When analyzing a calibration file, the file path length for one or more of the Tzero calibration files has exceeded the instrument's character limit of 125 characters.

 

Solution:

  1. Shorten the file path length by moving the Tzero calibration files to another location whose total file path is less than 125 characters.
  2. Analyze the Tzero calibration files again.
