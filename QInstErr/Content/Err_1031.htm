Message 1031

MCA temperature switch open. Run terminated.

Problem:

You have attempted to use the Mechanical Cooling Accessory (MCA70), but it is not operating or there is a problem with the accessory resulting in the temperature going too high. This message occurs during a run.

Solution:

Check the connection between the MCA70 and the TMA to make sure they are fully connected. Make sure the MCA70 is turned on. Try to run your experiment again. If the error persists, call TA Instruments for service.
