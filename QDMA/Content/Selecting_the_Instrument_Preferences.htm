Selecting the Instrument Preferences

 

You can set up certain operating parameters for your instrument through the instrument control software by selecting Tools/Instrument Preferences. Several pages will be displayed to allow you to set these parameters. To display the desired page of Instrument Preferences, click on the corresponding tab.

Select from the following topics for more information:

  • Bearing Gas Page - Specifying the Bearing Gas

  • LCD Signals Page - Setting Up Signal Display Touch Screen

  • Instrument Touch Screen Page - Setting Up the Instrument Touch Screen

  • DMA Page -Setting Up DMA Specifications

Specifying the Air Bearing Gas

To set up the air bearing gas select Tools/Instrument Preferences, then click on the Bearing Gas page and select the gas from the list of available gases. A purge gas is always used in the DMA.  Unlike the other thermal analysis modules, however, the purge in DMA is essentially a purge "by default."  That is because the purge is the gas used to "float" the air bearing slide.  Therefore, air at about 2 liters/minute (either from an in-house air line, a gas cylinder, or Air Compressor Accessory) is the most common purge gas.  Other "inert" gases like nitrogen can also be used.  

NOTE:  Dryness of the purge gas is important.  Moisture condensation in the air bearing will adversely affect performance.

Select the flow rate of the purge gases on the Notes Page.

Setting Up the Signal Display Touch Screen

To determine which signals coming from the instrument will be displayed on the Signal Display touch screen (touch the DISPLAY MENU key to display), select them from the dialog that is opened through the Tools/Instrument Preferences/LCD Signals page.

Select up to six signals. These signals will appear on the instrument touch screen.

NOTE:  This does not affect the signals displayed on the instrument control Signal Display Pane. Those signals are customized using another function.

Setting Up the Instrument Touch Screen

You can set up and calibrate the touch screen by selecting Tools/Instrument Preferences, then clicking on the Touch Screen page to perform the following steps:

  1. Select the desired Screen Saver Selection from the drop-down list. If you select "Custom," enter the desired Text using 20 characters or less.

  2. Enter the desired Delay time in seconds.

NOTE:  The screen saver will turn off automatically after one hour. Simply touch the screen to activate it again.

  1. Calibrate the touch screen, if desired, by checking the Start Instrument Touch Screen Calibration box and click OK. A series of prompts will be displayed. To continue with the calibration, follow the directions displayed. Reset the instrument as directed and follow the prompts using a non-sharp, pointed object (such as a blunt pencil or pen) to tap the touch screen at the indicated points.

  2. Click the OK button when finished.

Setting Up DMA Specifications

You can set up certain DMA options by selecting Tools/Instrument Preferences, then clicking on the DMA page to perform the following steps.

NOTE:  These parameters are applicable to certain clamps and certain modes only. See the information below to determine whether these DMA specifications are appropriate for your experiments.

  1. Used for Multifrequency modes only:  Select Oscillation Cycle Times. The DMA data collection is affected by several parameters. These are oscillation frequency, the number of cycles required to stabilize the system (Stabilization cycles), and the number of cycles averaged to generate a point. The DMA averages three cycles to generate a point and four additional cycles to stabilize the system when stepping between frequencies. In some applications (such as fast curing experiments), faster Data collection cycle times are needed. Use these parameters to adjust the cycle times. A value of zero uses the instrument-defined cycle times and is the default value for these parameters. (When adjusting these values, too few cycles may result in "noisy-appearing" data.)

  2. Used for Multifrequency Stress, Multifrequency Strain, Multistress, and Multistrain modes only:  Choose the parameter Collect data only when stable, if desired. By default the DMA waits until the oscillation force is stable within the predefined limits before proceeding to take a measurement. In some applications (such as fast curing experiments), this approach may produce unacceptable time gaps in the data. Unchecking this box will prevent the wait for stable force and therefore eliminate any time gaps in the collected data. (Overriding the default may result in "noisy-appearing" data.)

  3. Used for Multifrequency Stress and Multistress modes only:  Enter the Stress/Force Accuracy in percent. The DMA waits until the selected stress or force is within the predefined limits before proceeding to take a measurement. This may lengthen the data sampling time. This setting allows you to adjust this accuracy value. A value of zero uses the instrument-defined limits and is the default value for this parameter. If Collect data only when stable is not selected, this value is not applicable.

  4. Used for Film/Fiber Tension Clamps and Compression Clamps only:  Check Ability to Track Length and Adjust Modulus for Delta-L, if desired. This option can be used to track the sample dimension changes with respect to temperature and continuously adjust the modulus to reflect the changes in size. If this option is not checked, only the initial sample length is used for these calculations. When this option is selected, stress and strain calculations are based on "large displacement/constant volume" equations, rather than "small displacement" equations.

  5. Used for Fiber Tension Clamps only:  Check Calibrate for Fiber Clamp Compliance to enable this option, if desired. This fiber tension clamp option uses piano wire as the calibrant. If this option is not enabled, a default compliance factor will be used. See Calibrating DMA Clamps for more information.

  6. Click the OK button when finished.
