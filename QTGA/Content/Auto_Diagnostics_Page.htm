Using Auto Diagnostics

(A Platinumâ„¢ Feature)

 

Your QNX/Platinum TGA may be equipped with an Auto Diagnostics routine. This routine verifies the basic operation of your instrument to ensure that it is ready for operation. (See the complete list of tasks below.)

NOTE:  The Auto Diagnostics tasks that are available depend upon the instrument model.

  1. Select the Platinum button at the bottom left-hand side of the main instrument control window.

  2. Click Auto Diagnostics from the list of Automatic Tasks, a set of pages is displayed. The Auto Diagnostics Page can be used to set up a number of tasks that can be executed on demand or can be automatically performed at the time(s), date(s), and frequency selected on the Schedule Page.

The results of the auto diagnostics tasks are shown on the Auto Diagnostics Report Page. The results can be sent via email or Windows Messaging. See Using the Messaging Page for more information.

  1. Check the desired auto diagnostics tasks then select Apply to set them up to run when scheduled or select Start Now to execute the diagnostics immediately:

NOTE:  If you want to capture only the real-time information found in the diagnostics report, select Start Now without choosing any of the tasks.

    • Autosampler:  (Q500 Only) This diagnostics routine resets the Autosampler motors, tests the home sensor, verifies that the Autosampler tray is installed, and tests all of the motion control electronics. See TGA Platform Calibration for further information.

    • Furnace:  This diagnostics routine tests and verifies the furnace sensors, tests the furnace motor, checks the electronic board for proper functionality, and tests the travel of the furnace slide.

    • Impedance:  This diagnostics routine tests the heater to measure the impedance, tests and verifies the safety relay circuitry for proper functioning, and tests the high voltage circuitry within the power control unit.

    • Mass Flow Controllers:  This diagnostics routine detects the presence of the mass flow controllers, tests and verifies that both the balance and sample MFCs are working properly, and verifies the MFC's ability to maintain both high and low flow rates. Verify the purge gas connection prior to using this operation. See Calibrating the Mass Flow Controller for additional information.

    • Sample Pan Load/Unload:  (Q500 Only) This diagnostics routine tests and verifies that a pan can be loaded and unloaded, verifies that the balance can detect the presence of a loaded pan, and verifies that the Autosampler has been properly calibrated. See TGA Platform Calibration for further information.

  1. Click on the Report tab, after the Auto Diagnostics routine has been completed, to display the associated report. Any errors that occurred during this routine can be viewed by displaying the Instrument Log .
