Computer Not Set Up Correctly

The computer has not been set up to connect to TA database. The attempt to run the TA Connection wizard is failed.

The application failed to establish a connection between an Integrity-licensed instrument control computer and the TA database. The computer that has been targeted has not been correctly set up with the appropriate licensed software.  

Make sure the computer has the latest Advantage Integrity software installed as well as the Oracle client software. Set up the computer as needed for the Advantage Integrity system and run the Database Connection Wizard. Contact TA Instruments for assistance, if needed.
