Using the Database Connection Wizard

Before using the Database Connection Wizard, perform the following steps:

  • Power on the database server and make sure it is properly configured and visible to the controller.
  • Determine the Netbios name (name used in the Browse windows) of the database server and the type of database (SQL or Oracle).
  • Power on all Advantage Integrity-licensed instruments and make sure they are connected and visible to the controller.
  • Determine the name of the TA Instruments database that will be connected during this procedure.

After the items above have been accomplished, run the Database Connection Wizard as follows:

  1. Insert the TA Instruments Advantage Integrity software CD into your controller CD-ROM drive.  The installation program will automatically begin.  
  2. Select Database Connection Wizard, then select Next to begin database configuration.
  3. Enter the database information requested (this information can be found in the box with the database server when supplied by TA Instruments):
  1. Database System: Select from the drop-down list
  2. Database Server:  Computer name of server
  1. Select the desired language from the Select Language list.
  2. Select Next when all the database information has been entered.  If the database information is correct, the connection program will advance to the next window.
  3. Select the desired database name from the list of Existing Databases.
  4. Select Next to establish the database connection and advance to the next window.

The Advantage Integrity-licensed instruments must be imported in the database as system modules of the system.  If the instruments are connected and visible from this location, you will see them in the Import Modules into the Database System window.  

  1. Select Import to add the displayed instruments (modules) in the database.  (You do not need to select [check] any of these items.)  All the instruments displayed in the list will be added.  If an instrument is already in the database, it will not be added again.  (You can select Refresh to update this list as needed.)
  2. Notice that a check mark will be displayed next to its name as each one of the instruments is added to the database.  Once all the instruments have been imported, select Next to advance to the next window. A listing of all the database creation audit events appears in this window.  
  3. Select Finish to complete this process. A message box displaying the default user account information will be displayed.

Important:  Record this default user account information so that you can obtain access to the TA Instruments System Manager program.
