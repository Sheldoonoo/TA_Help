Macro Comment Window

Use this window to add a comment to the currently displayed macro. Comment steps in an executing macro are skipped over and produce no action or display. They are for documentation purposes only.

 

  1. Open the desired macro.

  2. Select the desired step position for the Comment.

  3. Select Insert/Comment from the Macro menu.

  4. Enter the desired text in the comment entry field. The maximum length of each line is 60 characters and up to 30 lines of text can be entered for a single comment.

  5. Click the OK button when finished. The comment is added to the macro and is identified by " / / " in front of the entered text.
