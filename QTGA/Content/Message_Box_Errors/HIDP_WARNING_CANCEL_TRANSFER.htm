Instrument Currently Receiving Files

The instrument is receiving files. If you cancel this operation, the instrument will not reboot and run correctly. You must resend the files to instrument BEFORE YOU CAN REBOOT it. Are you sure you want to continue?

This message is displayed during an operation that transfers files to the instrument, such as instrument software downloading or during the sending of calibration files. If you cancel this operation before its completion, the instrument will not run as expected. You will need to resend the files in question before you can reboot the instrument.

  • Select Yes to continue cancelling the file transfer. File transfer is now incomplete and the instrument may not reboot and run correctly.

  • Select No to abort the cancel operation and continue file transfer to its completion.
