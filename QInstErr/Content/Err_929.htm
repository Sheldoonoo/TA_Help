Message 929

Audit log checksum failure.  Corrupt message discarded.

Problem:

The checksum for a message computed when the message was put into the Integrity audit log buffer does not match when pulled from the buffer to send to the controller.  This was most likely caused by shutting down the instrument without using the proper shutdown procedure. A power outage may also cause this error.

Solution:

Check the audit log from the time the instrument was last powered down or reset. Confirm that Message #911 (Shutdown) is in the database from that time. If it is not, then the instrument was not properly shutdown, which resulted in Message 929.

Make sure that you use the SHUTDOWN key on the touch screen or Control/Shutdown Instrument from the menu to shut down the instrument properly.

If you feel that improper shutdown was not the cause of this message, contact TA Instruments for service.
