Change X-Axis?

This message is displayed if you have already completed at least one calibration analysis and then you select change the x-axis time. Changing the X axis to Time will cause the analysis results to be removed because you cannot perform a calibration analysis when the X axis is Time.

  • If you wish to change the X-axis signal and lose all calibration analyses, click the Yes button.

  • If you want to retain your analysis results, click the No button. The current X-axis signal will not be changed.
