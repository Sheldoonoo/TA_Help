Analyzing Data

 

The Rheology Advantage data module allows you to perform the following analysis functions:

  • Comparing data:  The software allows you to place data from more than one results file onto a single graph. Click here for more information.
  • Merging data sets:  You may have two or more data sets that contain related data, for example two halves of a long time-scale test. The data software allows you to merge these files into one new results file. You can also use this feature to produced averaged data when performing repeat measurements on samples. Click here for more information.
  • Editing data points:  The software allows you to delete or change data points in results files. This is helpful when manually smoothing data. In addition, you can add extra data points. This makes it possible to generate ‘ideal’ data for use as a reference curve, for example. Click here for more information.
  • Smoothing data:  This function allows you to apply a mathematical filtering of noise from your data. It is useful when editing of discrete points would be impractical. Click here for more information.
  • Area determination:  The area bounded by a rheological data curve can often be used as part of the characterization process. The area under the curve calculates the area under the selected curve. The calculated area is displayed in the Analysis results window.

Basic Analysis Steps

You can carry out ‘post processing’ on generated data files using the data analysis module. The basic operational steps are as follows:

  1. Open one or more results files (select File, then Open on the menu).
  2. Send the data to one or more graphs.
  3. Customize the graph layout.
  4. View the results / test procedure.
  5. Perform any required analysis on the data.
  6. Send the data to the required output format (printer, file or other application).

  • Derivative

    Calculates and displays the derivative of any variable. This is useful, for example, when looking at change of states of materials with time.

    You can select between using a central difference or a cubic spline analysis.

  • Finding Peaks and Valleys

    The software allows you to locate peaks, valleys or both. The analysis takes an input parameter (analysis Input parameter window) specified as the sensitivity. This can be set to a value of 1(high), 2 (medium) or 3 (low).

    Depending on the quality of your data, you can adjust the sensitivity to get the best fit. The lower the sensitivity value, the more likely the analysis is to locate a peak. Therefore, if you find that the software is detecting peaks on noisy data, select a higher value (e.g. 3). The results are displayed in the Analysis results window.

     

  • Locating "Cross Over" Points

    The software allows you to locate the point at which two similar variables "cross over," i.e., are equal. An example is when G’ = G" (Tan d = 1)

    You can locate the cross over for G’ v G", J’ v J" or h’ v h" The results are displayed in the Analysis results window.
