Report Text Window

Use the Report Text window to enter or edit the text that will be included in the custom report. Follow the steps below:

 

To enter new text:

  1. Select Insert/Text from the Custom Report menu.
  2. Click the Font button to open a window allowing you to choose the desired font.
  3. Enter the desired text.
  4. Apply the desired Text alignment or Fill color from the drop-down list of choices.
  5. Use the Clipboard button to import any text from the clipboard, if desired.
  6. Click the OK button when finished.

 

To edit existing text:

  1. Highlight the desired text report object on the template.
  2. Select Params from the Edit menu or from the pop-up menu.
  3. Make the desired changes to the text.
  4. Click the Font button to open a window allowing you to choose the desired font.
  5. Apply the desired Text alignment or Fill color from the drop-down list of choices.
  6. Use the Clipboard button to import any text from the clipboard, if desired.
  7. Click the OK button when finished.
