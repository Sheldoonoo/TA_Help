Entry Field Alpha at a Point Smoothing Width

For TMA data file analysis only.

Alpha is calculated as the slope of a least squares linear fit to the dimension and temperature data region around the given temperature. The size of this linear fit region is the "alpha at a point smoothing width" analysis parameter. The region is truncated if it extends outside the analysis limits or data collection region. If the given temperature is outside the analysis limits or is in a region where data was not collected, then the calculation of alpha is suppressed.
