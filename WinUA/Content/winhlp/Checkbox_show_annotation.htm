Checkbox Show Annotation

Check this box to display the result legend or result label on the plot.

  • For Sorption Isotherm, a legend box is displayed showing the line type, color, and symbol type for both the adsorption and desorption curves.

  • For BET analysis, a result label is displayed showing the slope and intercept of the BET fit, the monolayer coverage Wm, the BET constant C, and the value of R².

  • For GAB analysis, a result label is displayed showing the monolayer coverage Wm, the GAB constant Cg, the GAB constant K, and the value of R².
