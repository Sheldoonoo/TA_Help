Parameter Block Error

Parameter block error. Cannot read file.

 

The parameter block for this data file does not contain the minimum required information, or the data file is stored in a format that cannot be read by this version of Universal Analysis. Check that this is a valid data file by opening the file with a text editor program, such as NotePad, and examine the beginning of the file for textual parameter information.

 

WARNING: If the data file is a binary file then you cannot edit or save it with a conventional text editor program. Doing so will damage the format of the binary file. Contact Technical Support for assistance.
