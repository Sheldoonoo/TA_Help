Data Table Limits Window

Use the Data Table Limits window to select the range of data that you want output to the data table report or data table spreadsheet. This lets you restrict the data to a particular range and set the spacing between the reported values.

 

Follow these steps to perform this function:

1. Enter the point at which the program will start using the data to create the data table report.

2. Enter the point at which the program will stop using the data to create the data table report.

3. Enter the increment to set the spacing between the reported values. For example, an increment of 0.1 causes the program to report a data point from every 0.1 unit of time or temperature.

4. Check the All Data Points box to use all of the data points available in the file.

5. Check the Time and Temperature box to output the time and temperature data to the data table report or spreadsheet. Leave the box blank to output only the currently displayed x- and y-axis signals.

6. Check the No Invalid Data box to exclude all invalid data values from the data table report or spreadsheet.

7. Click the Resolution button, if you want to set the numerical values to a particular resolution. The Label Resolution window is displayed.

8. Click the OK button to display the data table report in the Report Editor window.
