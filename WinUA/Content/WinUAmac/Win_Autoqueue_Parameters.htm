Autoqueue Parameters Window

This window is used to select the items to be included in a record that will be contained in the autoqueue for autoanalysis.

 

To set up or change a record, follow these steps:

 

1. Use the Browse button to locate the first data file you want to include in the record.

2. Use the second Browse button to locate the macro file desired.

3. Click on the arrow next to the list of macros and select the macro desired.

4. Edit the comment text, if desired. This information is for display purposes only. The comment usually (by default) reflects the date and time the record was added to the autoqueue. It can be edited as desired.

5. Click the OK button. Your record will be created or modified and placed in the autoqueue according to the function already chosen. (A modified record does not have a change in position. An appended record is added to the end of the list.)

 

When you have finished adding/modifying all the records that you want, select File/Run from the Autoqueue menu. The data files will be automatically processed (autoanalyzed) beginning with the first record. The results will be collected in the Autoqueue Log.

 

NOTE: To stop autoanalysis of the autoqueue, press the Escape key.
