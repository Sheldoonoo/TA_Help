Modified Size Limits Window

Use the Modified Size Limits window to enter the start and final sample weights as follows:

 

NOTE: The current sample size is displayed for reference.

  1. Enter the desired start weight.

  2. Enter the desired final weight.

  3. Click the OK button. The entries are shown in the Modified Size window. Click OK again to accept the weights displayed.

 

The program adjusts the percent axis (y-axis) to the modified weight and alters the axis label to reflect the changes.

 

Any analyses that you perform after modifying the size are based on the new modified size; previous analysis results are unaffected.

 

To restore the original size, select Graph/Data Limits/Full Range. All analyses performed will be lost.
