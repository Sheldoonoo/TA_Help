Analyze Sensitivity Window

The Sensitivity option allows you to analyze curve sensitivity using a least-squares averaging technique. The curve data is reread from the data file, smoothed using the analysis region window, and subtracted from the original raw data. The resultant plot is a measure of the signal sensitivity during any portion of the experiment.

 

Follow these steps to perform this function:

 

1. If more than one Y-axis is displayed, choose the desired Y-axis to smooth by clicking on the desired Y number displayed on the graph.

2. Select Sensitivity from the Tools menu.

3. Enter the width of the desired analysis region. For example, if you enter 5 for an X-axis showing temperature, data from every 5 degrees is grouped together and averaged by the method of least squares.

4. Click on the OK button when completed.

 

NOTE: To use the default settings, click on the Default button.
