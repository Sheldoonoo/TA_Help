Main Options Window  Export Plot Page

Use this window to set up the plot items that you want to include as your defaults when you are exporting a plot, either as a file to or a printer. See Export Plot Window for more information on exporting plots.

 

Follow these steps to set up the export plot options:

  1. Check the options that you want to appear in the heading on the exported plot: Sample, Size, Method, Comment, Module, File, Operator, Run Date, Instrument, or Directories. You can also click on the buttons to select All or None of these header options.

  2. Check the options that you want to appear on each axis (Y1 to Y4 and X) of the exported plot: Axes names, tick labels, or tick marks. You can also click on the buttons to select All or None of these header options.

  3. Select the radio button for the desired header type: Standard, Custom, Blank, or No Header. NOTE: The plot header options selected here can be reset by using the Graph/Plot Header option and vice versa.

  4. Check the frame option if you want the plot to be printed with a boxed frame around it.

  5. Check the options that you want to appear in the footer on the exported plot: Exotherm direction and/or Program version.

  6.  If you are exporting your plot to an HPGL plotter, perform the following steps:

    1. Select the appropriate radio button for the output port connected to your HPGL plotter.

    2. Enter the X min and X max values.

    3. Enter the number of pens used on your plotter.

    4. Check the Initialize plotter box if you want the program to reset the plotter prior to drawing a plot.


  7. If you are exporting your plot to a printer, choose the default orientation that you want to use for the printing of the plot: Portrait or Landscape. You can override this default orientation using the File/Print Setup window.

  8. Click the Save button, if you want to use these as your new settings or click the OK button to use these settings for this session only. 

NOTE: The Save button only stores the settings for the selected page in the initialization (.ini) file. To save all settings, access theMain Options  Initialization Pageand click the Save As button. The Default button will also only recall the default factory settings for the selected page.

 

After you have set up these export plot options, each time you select File/Export Plot these settings will be used, if applicable.
