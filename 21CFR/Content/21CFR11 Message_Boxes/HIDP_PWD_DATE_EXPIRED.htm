Password Will Be Expired

The date you selected will cause your User account status to change from Enabled to Password Expired.

This message is displayed when you access the User Account Manager/General Page and modify the password expiration date, Pwd. Expire, to a date prior to the today's date. This will automatically make the password expire on this user account.

To keep the password enabled, do not set the expiration date to a past date. Set the expiration date to a future date, then click Apply or select  Reload.
