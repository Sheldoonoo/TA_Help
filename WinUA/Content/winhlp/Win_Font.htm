Font Window

Use this window to set up the font, font style, font size, and type of script used.

 

Follow these steps to set up the font characteristics:

1. Select the font name from the list. (Scalable True Type fonts (TT) are recommended.)

2. Select the font style desired.

3. Select the size of the font (chosen in point size).

4. Select the effects desired (if applicable to this window).

5. Select the type of script desired (usually depending upon the language and usage desired).

6. Click the OK button when completed.
