Setting Up a TMA Force Ramp Experiment

 

In a TMA Force Ramp experiment the material is exposed to a ramped force and the resultant strain is monitored while the temperature is held isothermal.

If you want to set up a TMA Force Ramp procedure, you can choose one of two techniques—the TMA Wizard or the Experimental View - Procedure Page. See the instructions below for each technique.

When you perform a TMA Force Ramp experiment, the following parameters need to be chosen:

  • Preload Force:  The preload force is an initial force applied to the sample prior to measuring sample and initiating the experiment.  Preload force is required for the film/fiber probe to ensure that all slack is removed from the sample.  This preload force is automatically compensated for in the applied force when the experiment begins.  That is, if a "preload force" of 0.005N is used and the experiment specifies an "applied force" of 0.015N, then when the experiment is initiated an additional 0.010N is applied on top of the "preload force" to achieve the specified 0.015N.  Despite this adjustment, the "preload force" should be the minimum required to achieve well-defined sample dimensions because measurements such as creep can be affected by excessive preloading. The default value is 0.2000 N.
  • Isothermal Temperature:  Strain curves are generated isothermally.  Enter the temperature of interest.  The temperature range for TMA is -150 to 1000 °C.  This entire range is available if liquid nitrogen is used in the reservoir surrounding the furnace.  If the MCA is used for cooling, the allowable range is -70 to 400°C.
  • Soak Time:  The soak time is used to ensure the sample has reached thermal equilibrium prior to applying the stress.  A five minute soak time is adequate for most materials.
  • Force Ramp Rate:  The force rate entered here in N/min is applied to the sample over the course of the experiment until the Final Force Value is reached. The default value is 0.100 N/min.
  • Final Force Value:  The final force value is the force at which the experiment will end. The default value is 1.000 N.
  • Additional Conditions:  There are additional experimental conditions that can be changed to optimize specific situations.  These are accessed using Advanced Parameters and Post-Test Conditions.  For most experiments, the system default values shown are recommended.

Using the TMA Wizard :  

Enter the desired parameters, then select Next to proceed.

Using the Experimental View Procedure Page:

  1. Display the Procedure Page. Select the desired experiment from the list of available preprogrammed templates.
  2. Enter the desired parameters. When you have finished setting up this procedure, you can enter more information by selecting the Notes Page and Summary Page for this run.
  3. Click the Apply button when finished to save these changes or the Append button to save the changes to this run and add another run to the end of the sequence.

NOTE:  You can use any test template as the basis for a custom test by selecting the desired test template from the list of tests, then click the Apply button and change the test to Custom. Then you can click on the Procedure Page and then the Editor button to make the changes desired.
