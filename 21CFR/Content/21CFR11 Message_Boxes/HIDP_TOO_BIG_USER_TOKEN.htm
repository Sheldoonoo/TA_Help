User Token Too Large

The size of user access token has exceeded the 40 KB limit, therefore the token is not accessible.

This message can occur if you have granted a particular user access and rights to an excessively large amount of groups, projects, and/or modules. Each user has a limit of 40 KB for their user access token size and this user has exceeded that limit.

The System Manager should remove all options assigned to that user and reassign them a few at a time, then click Apply to reduce the size of the token. It may be necessary to leave some options unassigned to that user.
