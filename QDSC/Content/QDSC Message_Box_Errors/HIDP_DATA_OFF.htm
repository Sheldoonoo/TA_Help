Not All Data in Range is On

If you turn the data collection off and then back on during a calibration experiment, you can get some gaps in your data file. The resulting file has some “off” regions in it. When you analyze that data for calibration purposes, and the “off” regions are within the analysis range, you will get this error message.

 

It is best to calibrate your instrument using complete data. Either select the analysis region so that it contains continuous data or repeat the experiment.
