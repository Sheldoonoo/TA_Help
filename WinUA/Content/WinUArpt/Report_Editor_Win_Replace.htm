Replace Window (Report Editor)

Use this window to locate and replace a string of characters in the open document. To use this window, follow these steps:

 

Use this window to locate a string of characters within the open document. Follow these instructions to use this window:

 

1. Type the string of characters you want to locate in the Find what field..

2. Type the string of characters that you want to use to replace the existing string.

3. Check Match whole word only, if you want to find and replace that exact word without any prefixes, suffixes, etc.

4. Check Match case, if you want to find and replace the given set of characters in only the case typed.

5. Click the Find Next button to locate the characters.

6. Click the Replace button to replace the located set of characters.

 

NOTE: After you have completed both entry fields, you can use the Replace All button to automatically locate and replace the characters.
