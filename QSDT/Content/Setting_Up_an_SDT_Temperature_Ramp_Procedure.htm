Setting Up an SDT Temperature Ramp Procedure

 

SDT ramp experiments heat or cool the material at a constant rate.  The results obtained are affected by the previous thermal history (often imparted on the material during processing [e.g., extrusion]).  Therefore, the results are designated "as received.”

If you want to set up an SDT ramp procedure, you can choose one of two techniques—the SDT Wizard or the Experimental View - Procedure Page. See the instructions below for each technique.

When you perform an SDT Temperature Ramp experiment, the following parameters need to be chosen:

  • Start and Final Temperature:  The temperature range of the SDT is ambient to 1500°C.  The actual start and final temperatures chosen for a specific experiment obviously must be within this range.

When evaluating materials for the first time, the entire range is recommended.  Once more is known about the material, these temperatures can be adjusted to bracket the region where the transitions of interest occur but should be far enough below and above the transition to establish a stable baseline.  At a heating rate of 20°C/minute, that equates to about 50°C buffer.

The optimum temperature range for DSC-TGA studies is 200 to 1500°C.

You can choose to start your experiments from one of the following choices:

  1. Use Current—the temperature of the SDT at the current moment (usually ambient temperature, 35°C). In this case, the ramped heating begins immediately on starting the experiment. To use this option, check the Use Current box.

  2. Another Temperature—you can choose the specific temperature desired (default is 50°C). In this case, the system equilibrates at that temperature before ramped heating begins. To use this option, uncheck the Use Current box, then enter a new start temperature.

  • Heating Rate:  The acceptable range of SDT heating rates is 0.1 to 100°C/minute.  Slower heating rates improve the resolution of closely spaced thermal events but increase the time of analysis.  20°C/minute is generally used.

  • Switch to Gas 2 at:  A variety of purge gases can be used including nitrogen, argon, helium, air, and oxygen.  Since the purge gas in the SDT passes through the balance housing before flowing across the sample and reference, reactive (corrosive) gases (e.g., SO2) are not recommended.  Switching the purge gas at an elevated temperature, by checking this option, (e.g., nitrogen to oxygen) can provide additional information such as carbon black content in plastics.

  • Hold Time at Final Temperature:  Check the Hold time at final temperature option to study the sample response at the final temperature (e.g., oxidative stability, curing, studies etc.).

  • Additional Conditions: There are additional experimental conditions that can be changed to optimize specific situations. These are accessed using Advanced Parameters and Post-Test Conditions. For most experiments, the system default values shown are recommended.

Using the SDT Wizard :

Enter the desired parameters, then select Next to proceed.

Using the Experimental View Procedure Page:

  1. Display the Procedure Page. Select the desired experiment from the list of available preprogrammed templates.

  2. Enter the desired parameters. When you have finished setting up this procedure, you can enter more information by selecting the Notes Page and Summary Page for this run.

  3. Click the Apply button when finished to save these changes or the Append  button to save the changes to this run and add another run to the end of the sequence.

NOTE: You can use any test template as the basis for a custom test by selecting the desired test template from the list of tests, then click the Apply button and change the test to Custom. Then you can click on the Procedure Page and then the Editor button to make the changes desired.
