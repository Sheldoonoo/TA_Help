Operation Failed. Do Not Reset. Try Again.

Verify transferred program file failed. You shall not reset your instrument. Please try to send files again.

This message is displayed during the downloading of instrument software using the Tools/Instrument Setup wizard. The file transfer (download) was not successful. Do not reset the instrument.

Click the Send Programs button again. Once the software has been successfully loaded to the instrument reset the instrument by pressing the reset button on the back of the instrument. If this error persists, call TA Instruments for service.
