Message 420

Power supply furnace impedance error

Problem:

This error could be caused by a faulty furnace or by removing the DSC cell. In addition, this error can occur falsely after loading instrument software or resetting instrument parameters. This message lists the impedance value that triggered the error as well as the range that the value fell outside.

Solution:

Try the following:

  • If you have just installed new instrument software or reset parameters, no action is required.
  • If you have reinstalled a DSC cell, try removing and installing the cell again.
  • If the furnace is faulty, replace the furnace.
  • If the error persists, contact TA Instruments for service. Please note the impedance value listed in the error and provide this information when calling.
