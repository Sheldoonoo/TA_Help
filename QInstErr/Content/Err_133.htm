Message 133

Force not calibrated or calibration in progress. Cannot start run.

Problem:

You have tried to start a run; however, one of the following situations has occurred:

  • The force was not calibrated as required.
  • Force calibration is in progress now.

Solution:

Try one or more of the following operations to solve the problem, then restart your experiment.

  1. Calibrate the force before beginning a run.
  2. Allow the calibration to be completed.
  3. Typically, this problem will occur because the probe is not calibrated. This happens whenever the instrument mode is changed. Perform the Calibrate Probe function.
  4. The force may become uncalibrated when the values in the saved memory region are lost or the TMA instrument software is revised. Perform the Force Calibration.
