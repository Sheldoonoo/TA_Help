Analysis Options - Onset/Peak Integration Page

Use this page to set up the parameters for the program to use when you perform a onset or peak integration analysis.

 

Follow the steps below to set up these options:

 

1. For DSC-TGA Mode SDT Files Only: Select a Weight option to be used to calculate peak integration results from the drop-down list.

2. Check the Manual Onset tangent selection option, if you wish to manually select the start and stop limits for the tangent lines. The program will use the limits you select to fit the tangent lines to the curve.

3. Click on the arrow to display the drop-down list and select the signal used to Calculate peak onset versus.

4. Check the Onset at inflection box, if desired.

4. Check the desired boxes to label these onset values: X-axis, Y-axis, Time, Temperature, X-signal. These options apply to Onset Point and Peak Integration analyses.

5. Check the desired options to be used when labeling the peak: Limits, Area, Height, Width at half height, Signed heat flow area and/or Normalized area.

6. Click on the arrow to display the drop-down list and select the desired peak maximum label option.

7. Click the Save button, if you want to use these as your new default settings or click the OK button to use these settings for this session only.

 

NOTE: You can choose the default options by clicking on the Default button.
