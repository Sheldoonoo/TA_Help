Q5000 Autosampler Cycle Test

 

The Q5000 Autosampler Cycle Test functions are used to test the Autosampler pan loading, unloading, punching, and furnace functionality. To perform an Autosampler Cycle test, touch the CALIBRATION key on the touch screen then touch the Autosampler button on the Calibration screen. Select the Cycle Test tab, the touch screen shown in the figure below is displayed. This screen is provided to cycle the autosampler through its actions to test each function. A brief description of the function of each item is provided in the table below.

 

Item

Description

Test Type List

Use the up/down arrows to scroll through the list and choose the type of cycle test to perform.

  • Cycle Load: Loads the pan(s), moves the furnace up, then down, and then unloads the pan(s) for the set number of cycles.

  • Cycle Punch/Load: Punches and loads the pan(s), moves the furnace up, then down, and then unloads the pan(s) for the set number of cycles.

  • Cycle Furnace: Moves the furnace up and down for the set number of cycles.

  • Cycle Punch: Punches the selected pan(s) for the set number of cycles.

Pan Number List

Select a particular pan number or use the Select All/Deselect All buttons to choose the pan(s) that will be tested.

Multiple pans may be selected by using the  Select Allbutton or by clicking on the desired pan positions.

Count

Displays the number of cycles that have been performed for the active test.

Set

Press the # button to specify the number of cycles to perform for the specified test.

Start

Begins the cycle test.

Stop

Stops the cycle test.
