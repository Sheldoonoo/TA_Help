Delete Peak

Use the Delete Peak option to delete the results of a selected peak integration so that you can reanalyze the peak without deleting all of the results. If the deleted peak is a tangent or perpendicular drop subpeak, its area is returned to the peak that was subdivided. If the deleted peak contains any subpeaks, they too are deleted.

 

Follow these steps to delete peaks:

 

1. Select Delete Peak from the Analyze/Integrate Peak menu. A marker is displayed on the graph.

2. Position the vertical bar of the marker within the limits of the peak integration results you wish to delete.

3. Right click to display the Analyze pop-up menu, then select Accept Limits (or press Enter). The selected results are deleted.
