Weight Calibration Coefficients Incorrect

While performing a TGA Weight Calibration on the SDT instrument, the calculated weight coefficients are outside the allowable limits. This calibration cannot be conducted using the specified data files.

 

Try one or more of the following:

  1. Check that the correct files are selected.

  2. Make sure the SDT beams are not contaminated or touching anything.

  3. Check the counterweights inside the SDT balance for tightness.

  4. Make sure that none of the four locking screws are touching the balance arms inside the SDT balance.

If you have found that problems # 2, 3, or 4 in the list above are in effect, you will need to conduct the calibration experiments again. Then perform the TGA weight calibration with the new files.
