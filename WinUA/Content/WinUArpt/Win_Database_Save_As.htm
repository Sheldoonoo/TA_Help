Save As Window

Use this window to save a database file.

 

  1. Select the arrow to the right of Project and select a 21 CFR 11 project from the drop-down list.
  2. Select the Type of database record from drop-down list. You can save the database record as a text (.txt) file or a Portable Document Format (.pdf) file.
  3. Enter a description of the database file or click the Clipboard button to paste the information copied to the clipboard in the Description box. This description appears in the browse menu at the end of each record. The maximum length of this description is 255 total characters. NOTE: You can clear the description box by clicking the Clear button.
  4. Click the OK button when finished.
