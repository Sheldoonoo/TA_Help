Calibrating the SDT Temperature

 

Temperature calibration is useful for experiments in which precise transition temperatures are essential. To temperature calibrate the SDT you need to analyze a high-purity metal for its melting temperature, and then enter the observed and correct values in the temperature calibration table. The temperature calibration parameters (observed and correct values) are automatically updated after temperature calibration has been performed, accepted, and saved. They may also be manually entered. Up to five calibrants may be used.

Temperature calibration should be performed on the SDT at least once a month.

NOTE: A multiple-point calibration is more accurate than a one-point calibration. A single-point calibration shifts the sample temperature by a constant amount. A two- or more-point calibration shifts the temperature by a constant amount below the first point, uses a smooth curve through the calibration points and maintains a constant temperature shift after the last point.

NOTE:  The calibration data must be obtained with the instrument in the calibration mode.

Use the following instructions to gather and analyze the two data files needed for SDT temperature calibration. This process uses the "Temperature" test template.

Gathering Temperature Calibration Data

  1. Choose standards with known melting points that encompass your experimental range.  Multiple standards may be needed for this purpose. Up to five temperature calibration standards may be used. Choose standards whose melting points differ substantially; each pair of melting points must differ from all other pairs by at least 10°C. Repeat the process below for each standard.

  2. Check that your purge gas is connected.

  3. Select Experiment View and access the Summary Page. Set the instrument to "calibration" mode using the Mode drop-down list. (In the calibration mode the current temperature table data is ignored.)

  4. Select the "Temperature" test from the Test list.

  5. Enter the weight of the standard into the Sample Weight field. Enter 0.0 as the Reference Weight.

  6. Specify a data file name using the Browse button.

  7. Click on the Procedure tab. Enter the requested test parameters that will program the SDT to:

  1. Equilibrate to a temperature below the onset of the literature melting temperature of the material. Use the following equation to determine that temperature:

5 min. X Heating Rate (°C/min) = Temperature (°C)

Typical example: 5 min X 20°C/min = 100°C (below the onset of the literature melting temperature)

  1. Heat the material, at the same heating rate that you will use in your subsequent experiments, to 50°C above the literature melting temperature.

  1. Select the Advance button to access and select the Advanced Parameters.

  2. Click the Post Test button to access the Post-Test condition parameters that allow you to specify the furnace and air cool options to use once the experiment is complete.

  3. Prepare approximately 3 to 10 mg of the calibration material in a sample cup, and then position it on the sample platform (front). Enter the weight of the material on the Summary Page.

  4. Prepare a reference cup using the same cup type as that used for the sample, and position it on the reference platform.

  5. Close the furnace.

  6. Click on the Notes tab. Enter/verify the requested information, including the flow rate (typically 100 mL/min).

  7. Click on the Apply button to save the experimental and sample parameters entered for this run.  

  8. If more than one run is in the sequence list, schedule this run. (An arrow will appear next to the run number in the Sequence Pane for the scheduled run.)

  9. Select Start to begin the calibration run.

Analyzing Temperature Calibration Data

  1. Wait until the run is complete, then access the calibration program using Calibrate/Analyze/Temperature on the SDT menu. Select the SDT temperature data file created above.

  2. Choose the calibration standard used from the Standards Specification window and select OK. See also:  Setting Up Calibration Standards.

  3. Analyze the temperature data using Analyze on the Calibration menu. Integrate the melting endotherm to determine onset to melting temperature. (Position the analysis markers to bracket the melting transition, then select Limits OK from the pop-up menu or Analyze menu.)

  4. Save the calibration results by clicking the Accept button (or by selecting Analyze/Accept results from the calibration menu). This will update the current temperature parameters with this calibration data.

  5. Close the Calibration window.

You can verify this result by selecting Cell/Temperature Table from the Calibrate menu, then clicking on the Temperature Table tab.

Repeat this process for each temperature standard (up to five standards may be used).
