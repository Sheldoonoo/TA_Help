Setting Up a Time-Temperature Superposition Test

 

Follow the steps below to set up a time-temperature superposition test.

  1. Select Procedure from the menu.
  2. Click on New.
  3. Select the Templates option
  4. Select an Oscillation procedure type and click the Next button.
  5. Click on the Template (TTS) radio button. Set up the parameters desired in your experiment and click Next.
  6. Follow the rest of the steps in the wizard. Select Finish when completed.
