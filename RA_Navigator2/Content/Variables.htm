Variables

 

Navigator supports two basic variable types, String and Numeric. Numeric variables can be Integer, Real or Time.

Variables are defined in the script and have their values assigned at run time. When the script ends, all variables are reset. If you wish to transfer values from one script run to another (or to share variables between scripts) you can save the values into a Navigator Variable file.

String Variables

String variable names consist of the letter S followed by a number (0 to 99). This gives up to 100 variable names that can be used (S0 to S99).

Numeric Variables

Numeric variable names consist of the letter V followed by a number (0 to 99). This gives up to 100 variable names that can be used (V0 to V99).
