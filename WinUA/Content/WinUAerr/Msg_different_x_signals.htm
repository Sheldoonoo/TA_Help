Different X Signals

X signal types are not compatible.

 

Problem:

You attempted to create a curve overlay plot that contains files with different X-axis units [e.g., Temperature (°C) in one file and Temperature (°F) in the other], or different X-axis signal types [e.g., Heat Flow (W/g) in one file and Heat Capacity (J/g/°C) in the other], or different X-axis scaling types [e.g., Storage Modulus (MPa) in one file and Log of Storage Modulus (MPa) in the other].

 

Solution:

The X-axis signal units for each curve in a curve overlay plot must be the same. Use the Units button on each data files Data File Information window to display the Units Selection window. Then select the same signal units for the overlay X-axis signal. Whenever files with different X signals are selected for the overlay plot, verify that the X signals have the same units, even though their names may be different. Also, verify that the axis scaling type (linear, log, inverse, inverse x 1000, or derivative log) is the same.
