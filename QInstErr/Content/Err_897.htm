Message 897

Weight failed to stabilize. Cannot start run.

Problem:

The SDT run cannot start until the weight on the beams has stabilized.

Solution:

Several things could be causing the problem. Try the following.

  • Make sure the instrument is sitting on a vibration-free surface.
  • If the sample is volatile and causing the problem, choose another sample.
  • If the error persists, call TA Instruments for service.
