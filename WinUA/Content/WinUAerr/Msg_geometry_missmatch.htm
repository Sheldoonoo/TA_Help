Geometry Mismatch

Sample geometry calculations do not match data file.

 

Problem:

This is a warning message that the calculations used in the instrument to compute sample geometry are different from the calculation in the data analysis program. If there is no sample geometry in the data file, or there is an unsupported geometry in the data file, this message will occur.

 

Solution:

The geometry factor calculations from the data analysis program will be used to adjust the data. However, you should check the versions of the instrument software and the data analysis software. This message can occur if an old version of software is in use. Make sure that you have the most recent software installed.
