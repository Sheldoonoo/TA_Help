Checkbox Label Peak with Signed Heat Flow Area

Check this box to label the peak with the appropriate exotherm/endotherm designation. If the peak area is exothermic, the value will be labeled as positive (implied no sign displayed). If the peak area is endothermic, the value will be labeled as negative (minus sign displayed).

 

NOTE: You must direct the program to label the peak with the area value before the sign can be indicated on the plot (see above) by checking the "Area" box on the page. If Area is not checked, the result will not be displayed on the plot.
