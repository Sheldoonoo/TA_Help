Message 627

Calibration file is for a different cooling unit.

Problem:

The Tzero calibration results were obtained from a cooling device different from the one currently selected.

Solution:

This is just a warning message. For the best results, Tzero calibration should be conducted using the same conditions as the subsequent sample experiments. (For example the same cooling device.)

Try one of the following:

  1. Repeat Tzero calibration.
  2. Load a saved Tzero calibration results file which was conducted using the currently selected cooling device.
