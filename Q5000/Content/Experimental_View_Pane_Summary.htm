Using the Summary Page

 

Sample information for each run is entered on both the Summary Page and Notes Page. The Procedure Page is used to set up the experimental test conditions.

When you select View/Experiment View from the menu, the middle pane displays the Summary page. The Summary page is used to enter the basic information needed to create a run. The summary information that is currently displayed is applicable to the run number that is highlighted on the Sequence Pane. The run that has the arrow pointing to it is the scheduled run. (i.e., The run that will be executed when Start is selected.) This information should be updated with the correct information prior to starting each experiment.

To use the Summary page, follow these directions:

  1. Verify that the Mode displayed is the one you want the instrument to use. The mode is determined by the pan type, which is selected on the Notes Page.

  2. Select the signals that you want the software to store in the data file by clicking the button. This displays the Select Signals to Store dialog. Check the desired signals.

  3. Select the type of Test you want to use from the drop-down list available. The list contains both preprogrammed procedures (test templates) and a custom test. To open a saved procedure file, click the Open Procedure File Button, . This file will be added to the test list once you have selected it. You can view a summary of the chosen procedure by selecting the Procedure Summary button, .

  4. Enter the Sample Name. Enter up to 32 alphanumeric characters to identify the material.

  5. Select the sample Pan Type, which includes the material and/or type of the pan that will be used in the experiment. Click on the down arrow to display a list of pan types, then select the desired pan type. See also:  Choosing a TGA Pan

NOTE:  The pan type also determines the Mode that is displayed.

  1. Select the Pan No. from the list. Use this field to specify the number of the TGA Autosampler pan that will be used in the experiment. Click on the down arrow to scroll through the numbers.

  2. Enter any Comments. Enter additional text to help describe the experiment, for example, sample treatment or experimental conditions could be included. Up to 64 characters may be entered.

  3. Integrity Users Only:  View the SUID#. Instead of defining the data storage path, the database will automatically assign a SUID# to the original, raw data record.

  4. Enter a Data File Name by clicking on the button, then select the desired storage location and enter in a filename. Click OK when finished. Typical example \\{Computer name}\Program Files\TA\Data\TGA\Data.001. (UNC filenames are used.)

  2. When you have finished setting up this run summary, you should verify the information contained on the Procedure Page and the Notes Page.

  3. Click the Apply button to save these changes or click the Append button to save these changes and add another run to the end of the sequence.
