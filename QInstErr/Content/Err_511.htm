Message 511

The Autosampler failed to load a pan from tray to cell. Pan number xx.

Problem:

This could be caused by one of several problems:

  • A pan that the Autosampler is attempting to load is missing from the tray.
  • The Autosampler needs the be calibrated.
  • The gripper fingers made poor electrical contact with the pan or did not optically sense the pan.

Solution:

Try the following:

  • The Autosampler will try two times to pick up a missing pan. If it is unsuccessful and is in "Recover on Error mode," it will abort the current run and go on to the next run in the AS run sequence.
  • If the Autosampler is in “Halt on Error mode” it will attempt two times to load the pan and, if not successful, it will immediately halt operation. In this case, take one of the following actions:
  1. Place the missing pan in the tray, then resume the motion.
  2. Reset the Autosampler, then restart the AS run sequence at the next run.
  3. Calibrate the Autosampler. See the online help for detailed instructions.
  4. Use alcohol on a Q-tip swab to carefully clean the gripper fingers and the optical lens.
  5. Reset the Autosampler by touching the RESET key on the Autosampler Control Menu or by selecting Control/Autosampler/Reset on the instrument control program, then retry your operation.
  6. If the problem persists, contact TA Instruments for service.
