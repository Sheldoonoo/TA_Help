Cycle (Data Limits)

Use the Cycle (Limits) option to limit the data used for analysis to a specific data cycle range. This option is valid if you have any data breaks or markers in your file, these can include the following:

 

 Cycle marks (due to Mark Cycle End segments in a procedure)

 Data breaks due to gas switch or event mark (selected on the Main Options  Data Page)

 Other data breaks
