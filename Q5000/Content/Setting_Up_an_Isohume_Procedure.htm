Setting Up an Isohume Procedure

 

This experiment maintains the relative humidity at a constant level while holding the temperature constant.

If you want to set up an isohume procedure, you can choose one of two techniquesâ€”the Wizard or the Experimental View - Procedure Page. See the instructions below for each technique.

When you perform an Isohume experiment on the Q5000 SA, these test parameters need to be specified:

Drying:

  • Enable drying at:  Check this box to select the drying option thenenter the desired temperature to be reached in order to dry your sample.

  • Stabilization Criteria:Check this box to abort the maximum isothermal drying time based on a specific weight change over a specified time interval.

  • Maximum Dwell Time:  Enter the maximum amount of time that the sample will be held at the drying temperature.

Method:

  • Temperature: Enter the desired temperature for your experiment.

  • Stabilization Criteria:Check this box to abort the maximum isothermal time based on a specific weight change over a specified time interval.

  • Maximum Dwell Time:  Enter the maximum amount of time that the sample will be held at the method temperature and humidity.

Additional Conditions:

  • There are additional experimental conditions that can be changed to optimize specific situations. These are accessed using Advanced Parameters and Post-Test Conditions. For most experiments, the system default values shown are recommended.

Using the Wizard :

Enter the desired parameters, then select Next to proceed or select Back to return to the previous window.

Using the Experimental View Procedure Page:

  1. Display the Procedure Page. Select Humidity from the list of preprogrammed templates.

  2. Enter the desired parameters. When you have finished setting up this procedure, you can enter more information by selecting the Notes Page and Summary Page for this run.

  3. Click the Apply button when finished to save these changes or the Append Run button to save the changes to this run and add another run to the end of the sequence.

NOTE: You can use any test template as the basis for a custom test by selecting the desired test template from the list of tests, then click the Apply button and change the test to Custom. Then you can click on the Procedure Page and then the Editor button to make the changes desired.
