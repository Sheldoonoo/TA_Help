Entering Post Test Options

 

Select the Procedure Page, then click the Post Test button to access the Post Test Parameters window, which is used to define functions that will occur at the end of a run. Method-end conditions are functions that can be automatically executed at the end of a run to increase turn-around time between experiments:

NOTE:  The parameters displayed will vary based on the instrument mode and clamp selected.

  1. For Multifrequency and Multistrain Tests Only:  Select the check box next to Auto End on modulus, if desired, and enter the values.

    Select the Auto End on modulus check box, then enter the modulus value, to automatically end the method when the measured modulus of the sample drops below or rises above the specified amounts.

    • Auto End on modulus  Above  value:  Enter the modulus value of the sample (in MPa) that will be used as the determining factor to automatically end the method. When the measured modulus of the sample goes above this value, the method will end.

    • Auto End on modulus  Below  value:  Enter the modulus value of the sample (in MPa) that will be used as the determining factor to automatically end the method. When the measured modulus of the sample goes below this value, the method will end.

  2. Enter the Return to Temperature Range. The temperature will be kept within this range until the STOP key is pressed, the furnace is opened (furnace closed at method end must be selected to maintain the temperature range), or the next run is started.

    When Return to Temperature Range is selected, the instrument will use the furnace and the GCA or air cool (if connected) to bring the temperature to within the specified temperature limits. The temperature will be kept within this range until the STOP key is pressed, or the furnace is opened, or the next run is started.

    Enter the upper and lower temperature limits that you want the instrument to reach at the end of the method.

    The upper temperature limit is 100°C, the lower temperature limit is 0.0°C.

  3. Select the radio button next to the desired option for the DMA furnace: Open or Closed.   

    Choosing to leave the furnace closed at the end of the method will keep the furnace in the temperature range selected by the Return to Temperature option until it is opened.

    Choosing to open the furnace at the end of the method programs the DMA to raise the furnace after it has reached the specified temperature limits.

  4. Select the radio button next to the desired option for the DMA clamp:  Up, Float, or Closed.

    Select the Clamp Closed button to close the clamp at the end of the method.

    Select the Clamp Float button to release (float) the clamp at the end of the method.

    Select the Clamp Up button to raise the clamp at the end of the method, using a positive force to lift the clamp.

  5. Check the GCA Autofill box if you want the GCA to be automatically refilled at the end of the test.

  6. Select OK to save settings.
