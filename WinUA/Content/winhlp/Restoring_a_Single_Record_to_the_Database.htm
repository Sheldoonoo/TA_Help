Restoring a Single Record to the Database

 

Records that have been archived and purged from the database can also be restored one record at a time using the Database Viewer.

  1. Double click on the icon, if you are not currently logged into Universal Analysis, and log into the database by entering your IntegrityUser ID and Password and then click Login.

 

NOTE: In order to restore database records, the TA System Manager must grant access rights to both the specified database and the DBARCHIVE Module. If you do not have these access rights, the database login will be terminated and the following message will be displayed. Once access rights have been obtained, continue with the next step after logging in.

 

 

  1. Select the button from the Universal Analysis tool bar. The following window is displayed.

 

  1. Locate the record that you would like to restore and double click on that record. The following message will be posted.

 

 

  1. Click Yes to restore the record from the archive. When the selected archived record has been successfully restored to the database, the following message will be posted.

  2. Click the OK button to complete the restore. Records that have been restored are tagged with the status R, indicating they have been successfully restored to the database.

 

 

  1. Verify the status of the records by opening the Database Viewer from either Universal Analysis or the Instrument Control program. Records that have been restored are tagged with the status R indicating they have been successfully restored to the database.
