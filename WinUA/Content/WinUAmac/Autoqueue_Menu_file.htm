Autoqueue File Menu

Use the File menu to perform various functions such as running or printing an autoqueue, importing records from an exported file into the autoqueue, and exporting an autoqueue.

 

NOTE: For more details on autoanalysis and the autoqueue, see the Universal Analysis online manual (Help/Manual) or use Acrobat Reader to access the Uamanual.pdf file contained on the CDROM.
