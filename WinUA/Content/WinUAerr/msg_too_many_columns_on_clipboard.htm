Too Many Columns

 

Too many columns on clipboard.

 

This message is displayed when you attempt to paste too many columns from the clipboard into a smaller number of highlighted columns in the spreadsheet. For example, if you copy two columns of data to the clipboard and attempt to paste it into one highlighted column on the spreadsheet, you will get this message.

 

When more than one cell is highlighted in the spreadsheet, there must be a one-to-one match between the columns on the clipboard and the columns highlighted for pasting in the spreadsheet. If only one cell is highlighted in the spreadsheet, then the columns and rows from the clipboard will be pasted in as required starting with the column and row of the highlighted cell.
