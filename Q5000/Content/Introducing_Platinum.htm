Introducing Platinum

 

Platinum features are designed to assist you in ensuring that your instrument is in proper working condition, calibrated, and has the latest software available. These features can be performed with minimal operator interaction and can be scheduled to occur on weekends or other times, which do not interfere with normal experiments. The main features include the following:

Click the Platinum button to display a list on the left-hand side of the instrument control window. A brief description of the Platinum components is provided below.

Automatic Tasks

Automatic Calibration Routines:  Used for automatic execution of weight and temperature calibration and/or validation sequence for the Q5000 IR equipped with an Autosampler. Alternatively used for automatic execution of weight calibration as well as weight and humidity validation for the Q5000 SA.

  • Weight Calibration:  Used for automatic execution of weight calibration and/or validation sequence (Q5000 IR or Q5000 SA). This allows calibration/verification or validation of weight performance to be performed on demand or scheduled during quiescent instrument times.  See also:Calibrating the TGA Weight. The Platinum features provide the following functions:

    • Scheduling of weight calibration with verification. See also: Using the Schedule Page

    • Verification of weight performance with scheduling of automatic weight calibration, if verification does not meet specified acceptable criteria. See also: Using the Schedule Page

    • Email and/or Windows Messaging notification of the weight calibration/verification with the results. See also: Using the Messaging Page

    • Calibration report. See also: Calibration Report

  • Curie Calibration: Used for the automatic execution of Curie Temperature calibration and verification (Q5000 IR). This allows calibration and verification of temperature performance to be performed on demand or scheduled during quiescent instrument times.  See also:Calibrating the TGA Q5000 IR Temperature

  • Humidity Verification: Used for the automatic execution of deliquescent salt evaluation on the Q5000 SA. This allows verification of total instrument performance (humidity, humidity sensors, temperature) to be performed on demand or scheduled during quiescent instrument times. See also:  Verifying the Q5000 SA Humidity

Auto Diagnostics:  Used to perform the diagnostics routine, which provides verification that the instrument subassemblies are in proper operating condition. See also:Using Auto Diagnostics

Scheduling

  • Scheduled Events:  Allows sequences and procedures (autoweight calibration, autotemperature calibration, & diagnostics) to be executed at a specified time/day and frequency interval. See also:  Using the Schedule Page

  • Settings: This function allows you to check the schedule settings for all automatic tasks in one place. This is a convenient way to set up the timing of these tasks so that they do not conflict with each other.

Messaging

  • EMail Notification: Provides email and/or Windows messaging notification of subscribed events including experiment and procedure completion with results. Enables you to follow the instrument's activities. (Email notification requires Microsoft Outlook and a dedicated mail account.) See also:  Using the Messaging Page

More...

  • Software Update:  Provides automatic notification that software updates are available for download from TA Instruments web site. (Connection to the Internet from the controller is required for this function.) Click the Help button within the TAUpdate program to get more information.

  • TA On the Web:  Provides a direct link to the TA Instruments website, http://www.tainstruments.com.

  • Our Support:  Connects you to the TA Instruments website for software support.

  • Software Suggestions: Connects you to the TA Instruments website for input of suggestions for improvement of our software.
