Temperature Limits Window

Use this window to limit the data used for analysis to a specific temperature range.

 

  1. Enter the temperature at which you want to program to begin retrieving data. An entry of 0.00 stands for zero degrees unless you also enter 0.00 as the stop temperature. If you enter 0.00 as both the start and stop temperatures, the entire data file is read.

  2. Enter the temperature at which you want to program to stop retrieving data. An entry of 0.00 stands for zero degrees unless you also enter 0.00 as the start temperature (see step 1).

  3. Enter the time, in the displayed units, at which you want the program to begin searching for the specified temperature range. For experiments in which the designated temperature range is cycled, the delay time option enables you to skip over earlier cycles and select data from later cycles in the experiment. For example, a delay time of 5.00 minutes tells the program to skip all data generated before the fifth minute of the experiment, even if the first five minutes contain the temperature range specified in steps 1 and 2.

  4. Check the Include all cycles after delay time box, if you want to include data from all cycles after the delay time that has the specified temperature limits.

  5. Check the Reset time to zero box, if you want to reset the Time signal to zero at the start of the specified data limits region.

  6. Click on OK.

The program scans the data file between the points you specify to determine the minimum and maximum values, which are then used to determine the autoscaling factors for the plot. Any data outside these limits is not plotted or analyzed, regardless of the scaling factors you select later.

 

To regain the entire data limits for analysis, use the Data Limits/Full Range option.
