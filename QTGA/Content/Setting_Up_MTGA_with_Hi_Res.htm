Setting Up a Modulated TGA with Hi-Res™ Procedure

You have chosen to perform a Modulated TGA (MTGA) experiment. Now you need to select which type of Modulated TGA experiment to perform from those listed on the window.

Conventional TGA can be used to study the kinetics of volatilization or decomposition processes using a number of experimental approaches including the multiple heating rate approach specified in ASTM (American Society for Testing & Materials) Method E 1641. Modulated TGA allows similar kinetic information to be obtained from a single experiment by applying a modulated heating profile to the material, which in effect is comparable to applying multiple constant heating rates simultaneously. Hence, Modulated TGA saves time. If High Resolution TGA capability is available, it is recommended that it be used in combination the Modulated TGA Survey procedure.

If you want to set up a Modulated TGA with Hi-Res procedure, you can choose one of two techniques—the TGA Wizard or the Experimental View - Procedure Page. See the instructions below for each technique.

The sinusoidal heating profile applied to the material in Modulated TGA depends on the underlying heating rate, the amplitude of modulation, and the period (frequency) of modulation.

When you perform a Modulated TGA with Hi-Res experiment, these parameters need to be chosen:

  • Modulation Amplitude: The modulation amplitude is the maximum positive or negative temperature excursion from the underlying heating profile during one modulation cycle. Acceptable values are 0 to +10°C. Typical amplitudes are +4 to 5°C although smaller amplitudes (+2 to 3°C) may be used for evaluating weight losses which cover a narrow temperature range (<50°C).

  • Modulation Period: Modulation period is the length of time required for one modulation cycle. Acceptable values are 100 to 1000 seconds. In general, the modulation period should be as long as possible (to assure quantitative heat transfer to the sample material) while still providing at least 10 modulation cycles during the weight loss being evaluated. 200 seconds is usually a good starting point.

  • Heating Rate, Resolution Number, & Sensitivity Value: Heating rate, resolution number and sensitivity are interrelated parameters which ultimately determine the underlying heating rate during weight losses. The heating rate value entered is the maximum rate. This rate is obtained between weight losses. The actual heating rate during weight losses will continuously vary. A combination of 5°C/minute heating rate, 6.0 resolution, and 1.0 sensitivity is a good starting point.

NOTE: Not all combinations of heating rate, resolution, and modulation period will result in the desired 10 cycles during the weight loss.

  • Final Temperature: The final temperature is set above the last weight change of interest or 1000°C (600°C if using aluminum pans). The 1000°C is recommended when evaluating materials for the first time.

  • Additional Conditions: There are additional experimental conditions that can be changed to optimize specific situations. These are accessed using Advanced Parameters and Post-Test Conditions. For most experiments, the system default values shown are recommended.

Using the TGA Wizard :

Enter the desired parameters, then select Next to proceed or select Back to return to the previous window.

Using the Experimental View Procedure Page:

  1. Display the Procedure Page. Select Modulated TGA with Hi-Res from the list of preprogrammed templates.

  2. Enter the desired parameters. When you have finished setting up this procedure, you can enter more information by selecting the Notes Page and Summary Page for this run.

  3. Click the Apply button when finished to save these changes or the Append Run button to save the changes to this run and add another run to the end of the sequence.

NOTE: You can use any test template as the basis for a custom test by selecting the desired test template from the list of tests, then click the Apply button and change the test to Custom. Then you can click on the Procedure Page and then the Editor button to make the changes desired.
