Export Plot Window

When you want to use a plot for another application or have it printed, use the Export Plot option to access this window. You can exclude certain parts of the plot, if desired, so that they do not appear in the exported plot.

 

To export a plot, follow these steps:

1. Select the plots destination: Clipboard, File (Enter or locate a directory.), Plotter, or Printer (Use the Setup button, if needed.).

2. Select the format for export: Metafile, Bitmap, HPGL, or PDF.

3. Select the desired colors for export from the drop-down list.

4. Select the size desired for the exported plot, if applicable: Full size (100%), Reduction (Enter a percentage for height and width reduction.), or sized in Pixels (Enter the desired size in pixels.)

5. Enter the desired pen width, if applicable.

6. Click on the Save button, if you want to save the settings for the plot destination chosen. NOTE: You can set up each plot destination with its own settings, then click the Save button each time to save them as the default settings for that specific destination.

7. Click the Export button when completed.

 

NOTE: HPGL is the only format available when you export to a plotter. COM1 is the expected port for the plotter.
