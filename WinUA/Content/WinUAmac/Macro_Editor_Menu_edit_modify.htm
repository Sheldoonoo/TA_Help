Modify Step (Macro Edit Menu)

Use the Modify function to edit certain macro steps. You can edit steps that contain a text entry, insert parameter, cursor or mouse coordinate, checkbox or radio button selection, or color selection. You cannot, for instance, change a button selection; you must delete, then recapture these steps.

 

To modify a macro step, follow these steps:

 

1. Ensure that the macro you want to edit is active by opening the desired macro.

2. Select Macro Editor from the main Macros menu to display the Macro Editor window.

3. Select the macro step you want to modify. Select Modify from the Macro Edit menu (or double click on the macro step). Depending upon the type of step you want to modify, a specific window (see the list below) is displayed.

4. Make the desired corrections and select OK.
