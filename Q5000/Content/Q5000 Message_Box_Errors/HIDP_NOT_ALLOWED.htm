Master Function Only

A non-master user is not allowed to set DCC (ID %d) to the instrument.

In order to perform this function you need to register as the master controller for that particular instrument. If someone else is already logged in as the master, that person must unregister before you can log in again as the master controller.

To unregister as the master controller close the instrument window or select Tools/Unregister Master Controller form the menu.
