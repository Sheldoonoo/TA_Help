Molecular Weight Determination

 

The optional Polymer package gives the Rheology Advantage software the ability to calculate the molecular weight of a polymer, using library values for K and x in the equation

Since viscosity changes with temperature, if your data is not at the reference temperature it will need to be shifted by the Arrhenius or WLF models. (Both of these models are available for all modes.) In flow data is fitted to the Williamson model, in creep the steady state part is fitted to a straight line, and in oscillation Cox-Merz is assumed and the resultant data fitted to the Williamson model. Therefore to get an accurate measurement for ho you need data that can be usefully fitted to these models.

Once you have your data on the graph, select the required Molecular weight model and then call up the polymer library by right clicking in the Analysis Input Parameters window.

Select the polymer from the list and click on Copy Values to automatically update the input parameters with those from the library.

NOTE: These values have been obtained from available literature. If more than one source was found in the literature, there will be more than one entry for that polymer. Select the most appropriate entry.

If the polymer of interest doesn’t have c1 and c2 [WLF], or Ea [Arrhenius] values you will need to perform a TTS experiment to derive them yourself or you will need to be working at the listed reference temperature. This temperature is given in Kelvin in the table [0°C is 273K].
