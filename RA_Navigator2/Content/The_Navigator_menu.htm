The Navigator Menu

 

An additional "Navigator" menu appears on the Main menu bar when you are using the "Record Mode" (under either Instrument or Data Module).

  • Start recording:  Starts recording the script.
  • Finish recording:  Ends the recording session.
  • Pause / Resume recording:  Pauses recording until you are ready to start again. Select again to resume recording at the point where you left off.
  • Record PromptMessage:  Insert a Prompt message command to the script.
  • Record DisplayMessage:  Insert a Display Message command.
  • Record Call:  Insert a Call command into the script.
  • Record Comment:  Add a comment to the script.
  • Options:  Further recording options can be set here. Use the "What's this help" button for an explanation of each item.
  • Go to Navigator:  Switches back to the Navigator program.
