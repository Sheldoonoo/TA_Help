Entering Post Test Options

 

Select the Procedure Page, then click the Post Test button to access the Post Test Parameters window, which is used to define functions that will occur at the end of a run.

Method-End Conditions

Method-end conditions are functions that can be automatically executed at the end of a run to increase turn-around time between experiments:

  1. Select the radio button next to the desired option for the TGA Furnace:Open furnace and unload or Closed. Select the option that best suits your needs:

    • Furnace Open and Unload will automatically open the furnace at the end of the method, or once the temperature reaches 600 °C. If your instrument is equipped with an Autosampler, the Furnace Open and Unload option is always selected and cannot be changed unless the Autosampler is disabled. (Select Tools/Instrument Preferences/Autosampler Options and clear the option, Enable Autosampler to disable the Autosampler.)

    • Furnace Closed leaves the furnace shut at the end of the method. If a standard furnace is attached, Furnace Closed will override the Enable Air Cool option.

CAUTION: During vacuum operation, select "Furnace Closed" to prevent instrument damage. The vacuum must be released before furnace opens. Set this option through both the Post Test AND the Tools/Instrument Preferences/Autosampler Options. You must disable the Autosampler to enable this selection.

NOTE:  The Tools/Instrument Preferences/Autosampler Options will override the Post Test conditions for the last run or the only run in a run sequence when the Autosampler is enabled.

  1. Enter the desired Return to Temp value. This function will allow the furnace to cool until the entered temperature is reached then unload the sample. If you would like to cool the furnace using air, select this option. See Connecting Purge Gas Lines.

  2. Check the Enable Air Cool box, if you want to have the furnace open and allow room air to cool the sample. Air Cool will enable once the temperature is below 500 °C. This function will open (checked) or close (unchecked) the internal solenoid valve connected to the Air Cool connector on the exterior of the instrument.  

Humidity Chamber

  1. Enter the desired temperature for the Humidity Chamber.

  2. Choose the desired action for the end of the experiment:

    • Maintain standby temperature will let the humidity chamber temperature return to the standard standby temperature.

    • Maintain method end temperature will maintain the humidity chamber at the temperature used for the method end.

    • Maintain standby humidity will let the humidity return to the standard standby humidity.

    • Maintain method end humidity will maintain the humidity chamber at the humidity used for the method end.

Select OK to save settings.
