First Time Login

When you open any software component of the Advantage Integrity system, either the TA System Manager Program, an Instrument Control program, or the Universal Analysis program, the Log in window is displayed.

 

The first time that you open the TA System Manager program or access a new user account you will be, entering a default User ID and password. This password must be changed immediately as follows:

 

Enter the case-sensitive default User ID and Password, then click the New Password button and follow the instructions for changing your password.
