Entry Label Offset

Enter the offset to the place on each of the axes where you want the first label to appear. The first label offset enables you to start labeling the graph at any point you choose. For example, if the axis starts at 0.03, but you want the first label to appear at 0.05, enter 0.02 (the difference between the start limit and the point on the axis where you want the first label to appear). (For log axis scaling, the offset must be 0.00.)
