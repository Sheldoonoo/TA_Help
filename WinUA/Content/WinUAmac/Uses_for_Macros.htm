Uses for Macros

Macros can be very useful tools if you find yourself performing a set of steps (including analysis) more than once. A macro can contain a few very basic steps or it could contain a very complex (normally time-consuming) set of instructions. Here are some examples:

 

Simple macro

You routinely place your company name on every data plot that you analyze. In this case you can set up a macro that will automatically type your name in a desired place on the plot.

 

Complex macro

Every day you have to analyze a whole set of DSC files to define their glass transition points. You can define a macro that will automatically open the DSC file, select the specified signals, and plot the data. At this point it can pause and wait for you to choose the data limits or you can set it up to automatically predetermine the limits using the Auto Limits option. Then you can continue the macro with instructions for automatic calculation and printing of the glass transition.

 

NOTE: For more details on creating, editing, and running macros, see the Universal Analysis online manual (Help/Manual) or use Acrobat Reader to access the Uamanual.pdf file contained on the CDROM.
