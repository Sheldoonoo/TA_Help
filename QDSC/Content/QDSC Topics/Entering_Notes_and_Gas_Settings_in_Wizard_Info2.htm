Entering Notes and Gas Settings Using the Experimental Wizard

 

Use this dialog  to enter or select additional information to be stored with the selected file. These items cannot be changed once the experiment is in progress.

  1. Enter the name of the Operator conducting the thermal analysis experiment. Up to 25 characters may be entered.
  2. Enter any additional comments in the Extended Text field. This may involve information about the sample or experiment, such as sample treatment or data sampling parameters. Up to 500 alphanumeric characters may be entered (approximately 12 lines).
  3. Select the Gas to be used for the sample from the drop-down list, either Gas#1 or #2, and enter the corresponding Flow rate (including units). If no purge gas is desired, select None from the list. If an MFC is installed, these settings will automatically set the gas and flow rate on a run by run basis. If your instrument does not have an MFC, then these entries are information only. To change the gas type, access Instrument Preferences.
  4. Select the desired Calibration Set to be used for this experiment from the list. Click on the Browse (â€¦) button to display the available calibration sets and their parameters. For more information see Configuring and Using Multiple Calibration Sets.
  5. When you have finished setting up this information, click Next to proceed.
