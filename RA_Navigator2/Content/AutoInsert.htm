AutoInsert

 

When you develop a script, you should consider including certain rheometer functions. However, because there are so many things to consider, it would be very easy to forget some important functions. The warnings tab found on the Navigator options menu (in Instrument Control) ensures that these functions are recorded automatically in the script at the appropriate places.

  • If you wish, you can override the installed default of "auto-insert if missing" for all items by accessing the pull-down list under each function and choosing from these selections:
  • If the "no action if missing" option is selected, you are not prompted on the need to include those commands.
  • If the "warn if missing" option is inserted, you will have the opportunity to decide whether a particular command should be included.

NOTE: It is advisable for the inexperienced user to keep the "auto-insert if missing" option active as a default.
