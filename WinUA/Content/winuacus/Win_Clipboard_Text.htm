Report Clipboard Text Window

Use the Report Clipboard Text window to specify the justification, color, and font for clipboard text that will be included in the custom report. Follow the steps below:

 

To enter new clipboard text:

1. Select Insert/Clipboard Text from the Custom Report menu.

2. Click the Font button to open a window allowing you to choose the desired font.

3. Apply the desired Text alignment or Fill color from the drop-down list of choices.

4. Click the OK button when finished.

 

To edit existing clipboard text:

1. Highlight the desired text report object on the template.

2. Select Params from the Edit menu or from the pop-up menu.

3. Click the Font button to open a window allowing you to choose the desired font.

4. Apply the desired Text alignment or Fill color from the drop-down list of choices.

5. Click the OK button when finished.
