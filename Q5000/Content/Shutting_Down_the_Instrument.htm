Shutting Down the Instrument

 

Before you decide to power down your instrument, consider the following:

  • All of the components of your thermal analysis system are designed to be powered on for long periods.

  • The electronics of the instrument and the controller perform more reliably if power fluctuations caused by turning units on and off are minimized.

For these reasons, turning the system and its components on and off frequently is discouraged.  Therefore, when you finish running an experiment on your instrument and wish to use the thermal analysis system for some other task, it  is recommended that you leave the instrument on purging, with the recommended flow rates, and the furnace in the up (closed) position.

Shutting Down from the Touch Screen

To ensure proper shutdown of the instrument, you can select the SHUTDOWN command or the button on the Control touch screen.

  1. Select from one of two options:

  2. Shutdown:  Closes down the instrument. Select Apply to initiate.

  3. Restart: Closes down, then automatically reboots the instrument.

  1. Select Apply to initiate.

A confirmation message will be displayed. Select OK (touch screen).  All communication to the instrument will be halted while the instrument saves data to the flash screen. Once this procedure is complete, the instrument will post a message on the touch screen indicating that it is safe to reboot the computer (instrument). To power down your instrument, set the power switch to the OFF (0) position.

Shutting Down from Instrument Control

To ensure proper shutdown of the instrument, you can select Control/Shutdown Instrument from the Instrument Control menu. The Shutdown Instrument window is displayed:

  1. Select the desired Shutdown option from the drop-down menu.

  2. Select from one of two options:

    • Shutdown:  Closes down the instrument.

    • Restart: Closes down, then automatically reboots the instrument.

  1. Select Start to initiate.

A confirmation message will be displayed, then the instrument window will close. All communication to the instrument will be halted while the instrument saves data to the flash screen.  Once this procedure is complete, the instrument will post a message on the touch screen indicating that it is safe to turn off the power to the instrument or reset the instrument, if Shutdown is selected.

To power down your instrument set the power switch to the OFF (0) position.
