Analysis Options - Other Page

Use this page to select special additional analysis parameters.

 

1. For TMA Files Only: Enter the desired Alpha at a point smoothing width. The number entered here will result in the logical smoothing of the data to the specified width.

2. For TMA Files Only: Click on the next checkbox to calculate the percent value by dividing the expansion over the temperature range by the corrected size of the sample. Leaving this box blank (which is the default) indicates that the percent value will be found by dividing the expansion over the temperature range by the original size of the sample.

3. For DSC-TGA Mode SDT Files Only: Enter the desired Corrected heat flow minimum weight that will be used as the minimum value for the weight-corrected SDT heat flow data. The default value is 5 percent.

4. For TGA or DSC-TGA Mode SDT Files Only: Click on the checkbox to Adjust TGA weight derivative for modify size analyses.

5. Click on the checkbox to Label results with curve color, if you want to use the same color for the curve and results label.

6. Click on the checkbox to draw sigmoidal baseline iterations, if desired.

7. Click on the checkbox Suppress arrowheads on result lines, if no arrowheads are desired on the result lines.

8. For TGA and SDT Files Only: Select the desired checkbox(es) that will label your TGA step change, weight changes, or residue analyses with the Limits, Weight, Percent, Transition Label, or with a Signed Change.

9. Click the Save button, if you want to use these as your new default settings or click the OK button to use these settings for this session only.
