Status Message 601

Waiting for cooler to be ready.

This status message is displayed because the LNCS cooler has not stabilized its cooling power. The run will automatically start when the flange, which is part of the cell structure, has reached the correct temperature.  No action is required.  For LNCS the flange must reach -160C with liquid detected for the run to proceed automatically.

Typically, this condition occurs if the cooler was recently turned on and has not had sufficient time to cool the DSC cooling flange.  Check that the cooler is turned on.  Select Control/LNCS/Cool to put the LNCS into the cooling mode before starting a run.
