Delete Results?

 

Changing units will cause the current file results to be deleted. Are you sure that you want to delete the results for this file?

 

This message is displayed if you have already performed some analysis on a data file, the results are displayed on the curve, and you attempt to change signal units.

 

If you select OK, the analysis results will be deleted when the data file is replotted.

If you select Cancel, the results will be retained and the units change will be cancelled.
