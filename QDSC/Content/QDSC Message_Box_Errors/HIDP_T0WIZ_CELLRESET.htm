Changes Require Recalibration. Proceed?

Changes to this heat flow selection require recalibration of cell constant and temperature.  The current calibration values for these parameters will be reset.  Are you sure you want to make this changes?

If you change the heat flow selected on the Instrument Preferences/DSC Page, you will need to calibrate the instrument for cell constant and temperature again. When you change the heat flow signal the current calibration values for these parameters will be reset.

After conducting and saving cell resistance and capacitance (Tzero) calibration, the cell constant calibration must be performed to finish calibrating the DSC. If you close the calibration wizard before completing this calibration you will receive this message.

  • Select Yes, if you wish to change the heat flow selection and recalibrate the instrument for cell constant and temperature.
  • Select No to cancel the operation.
