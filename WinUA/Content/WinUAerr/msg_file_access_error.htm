<various file I/O errors>

 

A file system error occurred while accessing the specified file.

 

This error frequently occurs if the file is currently in use by another application. In that case you can close the file in the other application to free up the file for use by Universal Analysis.

 

If the file is not currently in use and you receive this error, close the file and try the operation again. If the problem persists, you may have a corrupted file or a file with restricted access rights.

 

Another cause for this type of error is restricted access rights to the folder for the target file (e.g., the system Temp or Temporary folder).

 

Right-click on the folder containing the file and check the folders security settings. Files typically cannot be written to a folder that does not have "Modify" and/or "Write" permission enabled. You may need to contact your system administrator to modify a folders access rights.

 

If you cannot resolve the problem, call TA Instruments for assistance.
