Unavailable or Restricted Functions

 

Within the Advantage Integrity system the TA System Manager can define privileges (e.g., restrict actions) within each module. In Universal Analysis this list of controllable privileges includes most of the menu items. Therefore, typical functions may not be available depending on the Access Group(s) assigned to you. Consult with your TA System Manager to learn about your privilege list for Universal Analysis and Instrument Control.

 

In addition, a set of functions, which are available in the standard version of Universal Analysis, are never available when logged into the TA database. These functions include:

  • Editing the Data Parameter Block (File/Edit Param Block)

  • Excluding Data (Edit/Exclude Data)

  • Editing cells within the spreadsheet view (Graph/Spreadsheet View)

  • Editing Textual Reports (View menu)

  • Creating an Autoanalysis Queue (Macros/Autoqueue; View/Autoqueue Log)

  • Automatically reopening records (File/Reopen)

  • Removing saved analysis information (File/Remove Saved Analysis)

  • Editing cells within the spreadsheet view of the data (Edit/Modify, Cut, Paste)

 

NOTE: The ability to reject an experiment is also never available with an Advantage Integrity instrument.
