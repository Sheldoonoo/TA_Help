Units Selection Window

The Units Selection window is used to change the units used for the selected signals. This window displays the units most commonly used during analysis. To select units, follow these steps:

 

  1. Click on the arrow to the right of the field to display a drop-down list of available units for each type of instrument signal. You can click on the Default button to have all units field set to their default units selection.

  2. Click the Save button, if you want to use these as your new default settings for the current instrument type or click the OK button to use these settings for this session only. 

 

NOTE: If units are changed any existing results are deleted.

 

NOTE: If the selected units are normalized units, and a zero sample size is specified, the program will automatically plot the units that arenotnormalized.

 

TMA Offset List

The TMA zero offset drop-down list is used to correct Dimension Change (µm) signals for the initial size of the sample. Click on the desired item in the drop-down list from the following choices:

  • No offset - No size correction is made. The data is used as measured by the probe. This is the default.

  • Zero size - Convert the Dimension Change (µm) signal to Dimension (µm) by adding the sample size parameter to each data point. The data is then a measurement of absolute sample length.

  • Size at 25°C - Perform all Dimension Change result calculations with respect to the size of the sample measured by the probe at 25°C during the experiment. The size correction is not made if there is no Dimension Change data available at 25°C (i.e., the experiment did not include this temperature or the range of the data is restricted).
