Zero Size

Size must be greater than zero.

 

Problem:

The sample size indicated for the file is zero and is needed in a calculation or unit conversion. This will happen if the size was not entered when the data file was created.

 

Solution:

Determine the sample size, then enter it when the program requests it. If you do not know the sample size, press ESCape to abort the calcula∀tion.

 

If you are trying to read a data file in which sample size is not important (e.g., an empty pan was run in a method), then select the signal units that do not contain sample size.
