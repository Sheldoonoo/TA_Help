Cannot Fit Negative Data

 

Cannot fit negative data.

 

Problem:

The humidity weight change data selected for analysis contains negative data values (i.e., weight values below 100%). Fitting of negative weight change data is not supported.

 

Solution:

Try one or more of the following:

 

  • Use the Reset Size function to remove any weight loss due to sample conditioning (such as drying) from the initial sample weight.

  • Change the sample weight on the Data File Information window or File/File Information window to compensate for any initial weight loss.

  • Use the Data Limits function to restrict the range of the Weight signal to values greater than 100% or the range of the Weight Change signal to values greater than zero.
