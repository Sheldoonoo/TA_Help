Selecting the Instrument Preferences

 

You can set up certain operating parameters for your instrument through the instrument control software by selecting Tools/Instrument Preferences. Several pages will be displayed to allow you to set these parameters. To display the desired page of Instrument Preferences, click on the corresponding tab.

Select from the following topics for more information:

  • MFC Purge Page - Specifying the Connected Purge Gas
  • LCD Signals Page - Setting Up Signal Display Touch Screen
  • Instrument Touch Screen Page - Setting Up the Instrument Touch Screen
  • TMA Page - Setting Up TMA Options

Specifying the Connected Purge Gas

To set up the gases select Tools/Instrument Preferences, then click on the MFC Purge page. The appearance of this page differs slightly depending upon whether your instrument is equipped with mass flow controllers (MFC) or not.

Use Gas #1 to specify the purge gas that is connected to the corresponding ports on the instrument. A purge gas is a gas, often an inert gas such as nitrogen, that flows through the instrument during the experiment.

  • Without MFC:  If there is no mass flow controller installed, Gas #1 is routed directly from the port to the furnace and the flow must be regulated by a flow meter.
  • With MFC:  If there is a mass flow controller installed, Gas #1 is routed directly from the port to the furnace and is flow regulated from the instrument control software
  • In addition, you can check the Stop experiment when flow rate deviates from the set value checkbox, if you would like to have the experiment stop in the event the flow rate cannot be maintained during an experiment (e.g., if the purge gas tank becomes empty).
  • If you are using a purge gas that is not one of the available options, you can select "outside purge," which fully opens the MFC valve requiring that the flow rate be controlled using an external flow meter. Or you can select a purge gas type from the drop-down list that has similar characteristics to the one you want to use (e.g., conductivity) and use that calibration table for controlling the flow rate via the MFC.

Select the flow rate of the purge gases on the Notes Page.

NOTE: If you are using laboratory purge gas, rather than bottled purge gas, you will need to install an external drier.

CAUTION: Use of corrosive gases will shorten the life of the instrument. NEVER use a corrosive gas.

WARNING: Use of an explosive gas as a purge gas is dangerous and is not recommended for this instrument.

Setting Up the Signal Display Touch Screen

To determine which signals coming from the instrument will be displayed on the Signal Display touch screen (touch the DISPLAY MENU key to display), select them from the dialog that is opened through the Tools/Instrument Preferences/LCD Signals Page.

Select up to six signals. These signals will appear on the instrument touch screen.

NOTE:  This does not affect the signals displayed on the instrument control Signal Display Pane. Those signals are customized using another function.

Setting Up the Instrument Touch Screen

You can set up and calibrate the touch screen by selecting Tools/Instrument Preferences, then clicking on the Touch Screen Page to perform the following steps:

  1. Select the desired Screen Saver Selection from the drop-down list. If you select "Custom," enter the desired Text using 20 characters or less.
  2. Enter the desired Delay time in seconds.

NOTE:  The screen saver will turn off automatically after one hour. Simply touch the screen to activate it again.

  1. Calibrate the touch screen, if desired, by checking the Start Instrument Touch Screen Calibration box and click OK. A series of prompts will be displayed. To continue with the calibration, follow the directions displayed. Reset the instrument as directed and follow the prompts using a non-sharp, pointed object (such as a blunt pencil or pen) to tap the touch screen at the indicated points.
  2. Click the OK button when finished.

Auto Measure Offset

The Auto Measure Offset parameter is used when you have a film/fiber probe installed. This parameter compensates for the added thickness of the film/fiber probe and stage.  To calculate the value of the offset, follow these steps:

  1. Install the film/fiber probe and stage as directed. See Installing the Film/Fiber Probe for information.
  2. Select the desired TMA Q400 from the TA Instrument Explorer.
  3. Select Experiment View/Summary Page and select the TMA Film/Fiber mode.
  4. Select Tools/Instrument Preferences. Click the TMA Page.
  5. Take note of the value displayed on the window under Auto Measure Offset and record it on a piece of paper as the "Current Offset."
  6. Use calipers to measure the gap in the Auto Measure Offset Gauge and record this value as the "Actual Size."
  7. Press the PROBE UP key on the instrument touch screen to open the probe. Insert the Auto Measure Gauge, which is found in the TMA accessory kit.
  8. Press the MEASURE LENGTH key on the instrument touch screen and record this value as the "Displayed Size."
  9. Calculate the new offset using this equation:  New Offset = (Actual Size - Displayed Size) + Current Offset
  10. Enter the New Offset as the Auto Measure Offset value on the TMA Page.

Poisson's Ratio

Enter the desired value for Poisson's Ratio. This is the parameter that defines the ratio of transverse contraction per unit dimension to the elongation per unit length, when the sample is subjected to a tensile stress. Typical values are:  0.5 for rubber materials, 0.33 for metals, and 0.44 for most plastics. Consult a chemistry reference book such as the CRC Handbook of Chemistry and Physics for applicable values.

MTMA Calibration Constant

Enter the Modulated TMA Calibration Constant, which can be determined by the following experiment:

  1. Verify that the MTMA Calibration Constant, located on the Instrument Preferences/TMA Page, is set to 1.000 before you start this experiment.
  2. Select and install the desired probe on the TMA. We recommend using the Expansion probe, however, you can also run this experiment using the Film/Fiber probe. If you are switching probes you will need to perform the Probe Calibration procedure for the new probe..
  3. Check that your purge gas is connected and set to the desired flow rate, typically 50 to 100 mL/min for both Nitrogen and Helium.
  4. Select Experiment View and access the Summary Page. Then set the instrument to "Modulated TMA" mode using the Mode drop-down list. See: Selecting a TMA Operating Mode.
  5. Select the Modulated Temperature Ramp test from the Test list.
  6. Click on the Procedure tab and set up the following experiment, choosing starting and final temperatures based on the temperature range of your samples.

NOTE: The minimum starting temperature for users who do not have an MCA installed is 100°C. This is the minimum temperature at which the instrument can sufficiently cool without an MCA attached.

    • Preload Force: 0.3 N
    • Start Temperature: 100 °C
    • Final Temperature: 250 °C
    • Ramp Rate: 2 °C/min
    • Temperature Amplitude: ±5 °C
    • Temperature Period: 300 sec
  1. Click on the Notes tab. Enter/verify the requested information. If an MFC is installed, set the desired flow rate.
  2. Click on the Apply button to save the experimental and sample parameters entered for this run.  
  3. Zero the probe by selecting Control/Length/Zero from the menu or by pressing ZERO LENGTH on the TMA touch screen.
  4. Load the sample appropriate for the installed probe:  For the Expansion probe, load the aluminum standard found in the TMA Accessory Kit. For the Film/Fiber probe use aluminum foil as your sample.
  5. Verify the thermocouple position. For all probes, except Film/Fiber, position the tip of the thermocouple so that it is against the platform. The tip should be angled towards the sample location. For the Film/Fiber probe, position the thermocouple tip so that it is at the middle of the sample length.
  6. Click the Measure button on the tool bar or press MEASURE on the TMA touch screen. The size field on the Experimental View will be updated automatically once this process is complete.
  7. If more than one run is in the run sequence list, schedule this run. (An arrow will appear next to the run number in the run sequence list for the scheduled run.) Select Start to begin the calibration run.
  8. When the run is complete, open the Universal Analysis program.
  9. Select Open File from the main menu and then select the MTMA calibration constant data file created by the experiment above.
  10. Select the following Y-axis signals:  Dimension Change and Reversible Dimension Change and plot vs. X-axis signal: Temperature.
  11. Click on the Y-1 signal, Dimension Change, and then select Analyze/ Alpha X1 to X2 and select your limits. It is recommended that this expansion measurement be made over a 100°C temperature interval.
  12. Click on the Y-2 signal, Rev Dimension Change, and then select Analyze/Alpha X1 to X2 and select your limits. It is recommended that this expansion measurement be made over a 100°C temperature interval.
  13. Calculate the MTMA Calibration Constant using the following equation:  

MTMA Constant = Dimension Change Alpha  / Rev Dimension Change Alpha

  1. Using the QTMA Instrument Control program, select Tools/ Instrument Preferences/TMA and enter the MTMA Calibration Constant calculated above.

Cooler Type

Cooling in the TMA can be achieved by adding coolant, usually liquid nitrogen or ice, inside the furnace reservoir. The temperature range available for TMA is -150 to 1000°C when liquid nitrogen cooling, via the reservoir around the furnace, is used.

In addition, TA Instruments offers two different Mechanical Cooling Accessories, the MCA and the MCA70, specially designed for the TMA.  See also:  About the MCA or About the MCA70.

Choose the cooler in use with your system from the drop-down list.
