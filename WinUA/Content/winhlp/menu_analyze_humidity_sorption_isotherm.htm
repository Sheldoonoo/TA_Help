Analyze Humidity Sorption Isotherm

Select this item to plot the Humidity Sorption Isotherm of the current data file.

 

The TGA weight change data is displayed when the file is first opened as a function of humidity. Select Analyze/Humidity/Soprtion Isotherm to plot the data that is obtained at the end of each isotherm for a given humidity level. This plot requires that the experimental method be properly created (see Guidelines for Data Collection for more information).

 

To use this function:

  1. Collect your data following the guidelines suggested. See Humidity Analysis.

  2. Select File/Open, then locate and open the desired humidity data file.

  3. Select Analyze/Humidity/Options and set up the desired analysis options.

  4. Plot Weight Change versus Time for the desired Q5000 SA data.

  5. If sample drying was conducted during this experiment, then the dry weight needs to be determined before generating this plot. Use the Analyze/Reset Size function.

  6. Manually mark the adsorption/desorption isotherms, if desired. Select Analyze/Humidity/Mark Adsorption/Desorption then select the desired position on the curve.

  7. Select Analyze/Humidity/Soprtion Isotherm.

  8. Select View/Data Table/Spreadsheet or Report to view and/or print the data values.

 

This analysis results in a plot of the adsorption and/or desorption data values. No additional analysis of the data is performed.
