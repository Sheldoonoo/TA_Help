Onset at Super Heat/Cool

Onset at superheat/cool point.

 

Problem:

The sample was superheated (or supercooled if a cooling curve) by more than 0.25oC between the peak start and peak maximum, so the program cannot calculate a valid onset tangent.

 

Solution:

No action required. The program will report the point at which the superheating or supercooling reaction begins as the onset.
