Message 730

Saved parameters have been reset.

Problem:

This message is displayed after you reset the instrument parameters using the Instrument Setup wizard. When the instrument parameters are reset, you will need to check all of your program settings.

(The instrument parameters are reset as follows:  Select Tools/Instrument Setup. Then press Next until the Transfer Program Files to the Instrument window is displayed. The Reset Parameters button is clicked and the reset button on the back of the instrument is pressed.)

Solution:

Reset the instrument parameters only when necessary. After resetting, check all program and instrument settings to verify that they are set as desired. For example, you will need to re-enter all run information, preferences, signal selections, purge gas selections and flow rates.
