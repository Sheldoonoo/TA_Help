Unicode Characters Modified

 

Unicode characters were modified.

 

Problem:

One or more Unicode characters in the data file being read could not be converted to the equivalent Windows® (ANSI) character because the character does not exist in the Windows character set. This version of Universal Analysis cannot process Unicode characters that are not contained in the Windows character set.

 

Solution:

Avoid the use of Unicode text characters that are not contained in the Windows character set. Only the first 256 Unicode characters can be converted to equivalent Windows characters.
