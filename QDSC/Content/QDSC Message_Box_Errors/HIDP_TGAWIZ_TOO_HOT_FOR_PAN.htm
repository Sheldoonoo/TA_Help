Too Hot for Pan

XXX °C is too hot for an < > pan.  Change either the pan type or the temperature in the Experimental Parameters page.

This message is displayed during experimental setup using the TGA wizard. The temperature that you have entered is too high for the specified pan. You will need to either change the pan to a type that can be used with higher temperature or lower the temperature to be used in the experiment to a point that is compatible with the specified pan.
