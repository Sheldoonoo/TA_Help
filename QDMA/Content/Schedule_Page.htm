Using the Schedule Page

(A Platinum Feature)

Select the Platinum button at the bottom left-hand side of the main instrument control window.

The Schedule Page can be used to schedule "events" such as auto diagnostics, calibration, and run sequences.

 

Follow these instructions to schedule an event:

  1. Set up the desired event or task. Click the Help button for more information, if needed.

  2. Click the Schedule Page.

  3. Check the box "Schedule _____"

  4. Use the calendar displayed to choose the start date for the event. You can click the left/right arrows to access the different months of the year.

  5. Select the Time that the event should start.

  6. Select the Frequency of the event occurrence(s): One time only, Daily, Weekly, or Monthly.

  7. Messaging: If you want to be notified about the event scheduled, select one or more the following check boxes:

    • Remind xx minutes before [event] start

    • [Notify] When the event starts

    • [Notify] When [event] completed

  8. Click the OK button when finished. The event will be scheduled to begin when chosen.
