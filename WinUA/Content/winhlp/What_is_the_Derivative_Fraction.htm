What is the Derivative Fraction?

Derivation fraction is a percentage of the derivative curve.

 

It is used when performing a TGA Weight Change or Signal Change analysis as follows: The program will first determine the highest derivative peak in the current data file. Then the program will go down the highest peak until it determines where the desired fraction of that derivative peak is reached. Then it will find the middle of the area between the two derivative peaks at that fraction point and use that as the analysis point. For example, if you use the default derivative fraction of 0.05, the program scans down the highest derivative peak until it comes to the 5 percent point of that peak. Then the middle of the area between the highest peak and the one next to it is determined and used at the analysis limit (either Point 1 or Point 2, or both).

 

NOTE: If the program does not locate an adjacent peak to use for the midpoint determination, it will use the end of the data curve.

 

When doing a succession of weight change or signal change analyses on the same curve, the derivative fraction used for the Point 2 of the first step should be the same as the fraction used for Point 1 of the next step.

 

NOTE: The default derivative fraction of 0.05 (5 percent) is recommended for both Point 1 and Point 2. If you do not use the default value, it is recommended that you use the same derivative fraction for both points.
