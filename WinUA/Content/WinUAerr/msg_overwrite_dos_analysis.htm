Overwrite DOS Records?

 

This file contains 16-bit DOS saved analysis records, which will be overwritten by 32-bit Windows analysis records. DOS Universal Analysis cannot read 32-bit analysis records. Are you sure that you want to overwrite the DOS records?

 

Select OK to overwrite the DOS saved analysis records with the 32-bit Windows analysis records. The file will no longer be able to be read by the DOS Universal Analysis program.

 

Select Cancel to abort the overwrite operation. The file will not be changed.
