Invalid Sigmoidal Baseline

 

Sigmoidal baseline may be invalid. Check results.

 

Problem:

The sigmoidal baseline for one or more results may be invalid. The data in memory that was used to calculate the baseline(s) has changed.

 

This problem is typicallly caused by rereading the data with different data limits, or by a real-time update of the data that changes the data compression ratio.

 

Solution:

Delete any peak integration results that contain invalid baselines and repeat the analysis.
