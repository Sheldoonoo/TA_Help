Procedure List

Click on the arrow next to the procedure name to display a drop-down list of the procedures that are available. This contains a list of preprogrammed procedures (also called templates) based on the instrument mode. You can also create a Custom method or open a saved procedure.

NOTE: The procedure used must have been created using the current instrument mode.

To open a saved procedure file, click the Open File Button, , and select the procedure file from the desired directory. This file will be added to the test list.
