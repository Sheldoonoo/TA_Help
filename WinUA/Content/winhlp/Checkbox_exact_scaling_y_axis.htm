Checkbox Exact Scaling Y Axis

Check the Exact Scaling Y-Axis box to have the plot axis using the exact range of the y-coordinates found in the data file or rescale zoom box.

 

Leave this box unchecked and the data will be autoscaled when applicable. When exact scaling is not selected, the plot axis is drawn with evenly spaced axis labels that fall on the corners of the plot.
