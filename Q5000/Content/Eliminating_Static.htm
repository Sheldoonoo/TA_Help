Eliminating Static

 

Some materials readily build up static electricity changes that can create problems during loading. The Q5000 IR is designed with a unique hang-down wire assembly that provides a direct path to ground when metal pans are used. Hence, no static effects will be seen with those pans.

When using ceramic, uncoated quartz, or platinum-HT pans, however, it may be necessary to periodically discharge static build-up using an "anti-static pistol." Cleaning the furnace tube will also help discharge static buildup.
