Message 526

Cannot start run. Lid disabled and closed.

Problem:

The Autosampler cannot start the run because the AutoLid is closed on top of the cell and will not open to allow the sample to be placed inside the cell.

Solution:

Try the following:

  • Check for an obstruction. If one is found, clear it and repeat the command.
  • Make sure that you have not set the temperature so high that the lid begins to soften and adhere to the cell. If that is the case, heat the cell again and repeat the open command.
  • If the problem persists, contact TA Instruments for service.
