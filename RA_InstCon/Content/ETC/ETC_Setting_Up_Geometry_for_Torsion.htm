Setting Up Geometry for Torsion

Follow the steps below to set up the select geometry for torsion:

  1. Select Geometry on the menu.
  2. Click on New.
  3. Select the Solid sample geometry type.
  4. Fill in the Name and Notes fields. Select the type of Material from the drop-down list
  5. Enter the sample dimensions. Note: The actual sample length will be read automatically during the test.
  6. Either type in values for the Geometry inertia and Compliance values or select the option to calibrate after creation.
  7. Enter the final details into the geometry wizard.
