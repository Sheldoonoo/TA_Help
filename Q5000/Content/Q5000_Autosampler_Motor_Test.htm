Q5000 Autosampler Motor Test

 

The individual motors of the instrument can be executed using the Autosampler Motor test functions. To perform an Autosampler Motor test, touch the CALIBRATION key on the touch screen then touch the Autosampler button on the Calibration screen. Select the Motor tab, the touch screen shown in the figure below is displayed. Press the button to move the desired motor.

A brief description of the function of each item is provided in the table below.

 

Key Name

Description

Rotate Left

Moves the sample tray counter-clockwise one position.

Rotate Right

Moves the sample tray clockwise one position.

L/R Load

Moves the sample tray to the load position using the left/right motor.

Left

Moves the sample tray to the left one position.

Right

Moves the sample tray to the right one position.

L/R Park

Moves the sample tray to the park position using the left/right motor.

U/D Load Up

Raises the sample tray to the load upper position using the up/down motor.

Up

Raises the sample tray up one position.

Down

Moves the sample tray down one position.

U/D Load Down

Lowers the sample tray to the load lower position using the up/down motor.

Furnace Up

Moves the furnace up at a slower speed than during normal operation.

Furnace Down

Moves the furnace down at a slower speed than during normal operation.
