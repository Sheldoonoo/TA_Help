Module Name Required

The Module Name field is a required field to add a new Module. Enter the missing information or select Reload to cancel the operation.

The TA Instruments database system requires certain information be supplied for each new module. You must enter the Module Name for each new module created on the Instrument Manager/General Page. If you change your mind after starting to enter a new module, you can select the Reload button to cancel the operation.
