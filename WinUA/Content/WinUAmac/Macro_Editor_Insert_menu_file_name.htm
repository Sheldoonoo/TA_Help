File Name (Macro Insert Menu)

Use the File Name option to insert a macro step that will identify the specified file (data file, session file, etc.) to open or save when the macro step runs. Select the desired position, which should be immediately after a captured "Open File" or "Save File" step. Then select Insert/File Name from the Macro Editor window. The Macro File Name window is displayed to allow you to locate or enter the desired path and filename.
