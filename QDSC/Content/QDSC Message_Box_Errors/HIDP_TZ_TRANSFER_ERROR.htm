Cannot Transfer Tzero Calibration

Unable to transfer Tzero calibration information to the instrument. Please try again.

This message is displayed while using the Calibrate/Calibration Wizard functions. The system is attempting to transfer the Tzero calibration results to the instrument.

To transfer the calibration results again, select Calibrate/Calibration Wizard, then perform the following steps:

  1. Press Next.
  2. Select "Load Saved Cell Resistance & Capacitance File to Instrument." The File Utility window will be displayed.
  3. Select Tzero calibration files (*.tzr) as the file type.
  4. Select the storage directory. (Tzero calibration files are stored in a subfolder "Calibration" located in the data file directory of the original calibration data file.)
  5. Click the Select button.
