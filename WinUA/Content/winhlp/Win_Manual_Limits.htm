Manual Limits Window

Use the Manual Limits window to enter numeric values from the keyboard for the data limit points needed for the analysis function chosen. This window can display from one to six points depending upon the analysis and the analysis options chosen for that analysis function.

 

Follow these steps to use this window:

1. Enter the X and Y data point values desired for each limit point needed in the analysis.

2. Click the OK button when finished.

 

NOTE: You can switch to using markers again by clicking theCursor button.
