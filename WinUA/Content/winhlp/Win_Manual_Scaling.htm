Manually Rescaling

The Manual rescale option is used to select exact start and stop limits for the x- and y-axes to reduce or expand the graph, and to change the label and tick intervals.

 

  1. Enter the new start limits for each of the axes. The start limit determines the signal value at which the axis begins. (For log axis scaling, the limit must be at a full decade interval.)

  2. Enter the new stop limits for each of the axes. The stop limit determines the signal value at which the axis ends. (For log axis scaling, the limit must be at a full decade interval.)

  3. Enter the intervals you wish to appear between labeled points on each of the axes. The label interval determines how often the axis is numbered. (For log axis scaling, the label interval must be a multiple of 9.0.)

  4. Enter the intervals you wish to appear between tick marks on each of the axes. The tick interval determines how often the axis is marked to show increments in the signal.

  5. Enter the offset to the place on each of the axes where you want the first label to appear. The first label offset enables you to start labeling the graph at any point you choose. For example, if the axis starts at 0.03, but you want the first label to appear at 0.05, enter 0.02 (the difference between the start limit and the point on the axis where you want the first label to appear). (For log axis scaling, the offset must be 0.00.)

  6. Click OK when completed. The graph will be rescaled using all of the options chosen.
