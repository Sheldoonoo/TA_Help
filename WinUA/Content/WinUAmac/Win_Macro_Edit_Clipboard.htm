Macro Edit Clipboard Window

The Edit Clipboard macro step is used to display a dialog box in which you can view and edit the contents of the Windows clipboard. This is useful for entering custom text or multiple parameter values.

 

 For example, a previous set of macro steps can be used to copy text from an edit field in a window to the clipboard. (Right click in the edit field and select Copy.) The Macro Edit Clipboard dialog box is the displayed to permit you to edit the text or enter new text. Subsequent macro steps can then retrieve the text from the clipboard. (See Assign Variable and Paste.)

 

 If the clipboard contains text at the time that the Edit Clipboard step is executed, then the current text is displayed in a multi-line edit field for the user to accept or modify.

 

 If the clipboard contains no text, or if the Clear checkbox is checked, then the edit field is cleared prior to opening the dialog. There is no program-imposed limit on the amount of text that can be edited or entered.

 

1. Select (highlight) the desired position (step number) for the macro step insertion.

2. Select Insert/Edit Clipboard from the Macro Editor menu. The Macro Edit Clipboard window is displayed.

 

To use this window:

 

3. Enter the Display Text to be used in the dialog.

4. Check the Append clipboard box to append the current contexts of the clipboard to the dialogs display text.

5. Check the Leading space box to insert a space between the display text and any appended clipboard text.

6. Check the Beep box, if you want the beeper to sound as the dialog opens.

7. Check the Horizontal Scroll Bar box, if you want the edit field to have a horizontal scroll bar.

8. Check the Vertical Scroll Bar box, if you want the edit field to have a vertical scroll bar.

9. Enter the number of Lines to display in the edit field (this defines the vertical size of the edit field).

10. Enter the number of Columns to display in the edit field (this defines the Horizontal size of the edit field).

11. Check the Clear box if you want to clear the edit field (discard any existing clipboard text).

12. Select Clear clipboard before assigning edit text to replace any text currently on the clipboard with the entered text. Select Append edit text to existing clipboard text to append the entered text to any text already on the clipboard.

13. Click OK when finished. The step will be inserted at the selected position.
