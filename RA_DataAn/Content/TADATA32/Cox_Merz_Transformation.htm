Cox-Merz Transformation

Rheology Advantage can have files of different ‘Types’ in the file selection box and so the Cox-Merz transformation is done as follows:

  1. Load the oscillatory file you want to transform
  2. Select the File menu and click on Tools then select Cox-Merz (alternatively, simply right click in the file selection box and select Cox-Merz.)
  3. A new file is added to the file selection box. The file name is the base file name with ‘cm’ added, and its ‘Type’ is changed to Flow.

This file can now be treated like any flow file.
