Adjust Region

Select this menu item to adjust the region on the graph to be displayed in the inset view window as follows:

 

1. Position your cursor over the inset view window. Right click to display the pop-up menu. Click on Adjust Region. Markers will be displayed on the full size plot to indicate the beginning and end points for the area of the curve represented in the inset view window.

2. Use your mouse to drag-and-drop the marker to the desired start point on the curve. Repeat this process for the end point.

3. Move the cursor outside the inset view window and right click to display the pop-up menu.

4. Select Accept Limits. The data region displayed in the inset view window will be adjusted accordingly.
