Linear Transform - X/Y Axis

Use the Linear Transform/X Axis or the Linear Transform/Y Axis function to adjust the slope and/or intercept of the selected X- or Y-axis signal.

 

Before you adjust a curve, you must choose the desired Y-axis by clicking on the desired Y number displayed on the graph. When you choose the Linear Transform/X Axis or the Linear Transform/Y Axis option from the Tools menu, the X/Y Transform window is displayed to allow you to enter the values to use in the general equation. The program will use this equation to perform the linear transformation.
