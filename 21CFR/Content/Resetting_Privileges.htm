Resetting Privileges

 

The Reset Privileges function, found on the Privileges page, allows the System Manager to reset the currently assigned privileges for any module.

To Reset Privileges:

  1. Log in to the System Manager.
  2. Select Groups by clicking on the icon in the left-hand pane. Select the Group Manager/Privileges Page.
  3. Click the module drop-down menu to select the module whose privileges will be reset and then click the Reset Privileges button. Once the privileges have been reset, the default privilege list for that module must be re-imported.
  4. Importing the Default Privilege List: Click the Import Items button on the Privileges Page then locate the directory C:\Program Files\TA Instruments\Integrity. Select the appropriate item access file for the deleted module type (e.g., ItemAcess_WINUA_ENU.scr) then click the Open button.
  5. Once the file is imported, a digital verification certificate will appear as shown in the figure to the right.
  6. Click the Continue button to proceed.
  7. Choose the desired privileges by selectively assigning privileges (checking each one) or enable all privileges by checking the Select All Privileges box on the Privileges Page.
  8. Click the Save button to retain the selections.
