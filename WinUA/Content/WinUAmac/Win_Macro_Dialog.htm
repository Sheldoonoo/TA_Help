Macro Dialog Window

Use the Macro Dialog window to add a message box to your open macro. For example, a message box giving the allowable range for a specific analysis can be displayed, allowing comparison with the experimental result just obtained.

 

A dialog can be customized with several different options to make it perform the actions you desire.

 

To set up your macro dialog, follow these steps:

 

1. Open the desired macro.

2. Select the position in the macro steps for the message box to be inserted into.

3. Select Dialog from the Insert menu.

4. Enter the desired message text in the Text field.

5. Check the Append clipboard box to append the current contexts of the clipboard to the dialogs display text.

6. Check the Leading space box to insert a space between the display text and any appended clipboard text.

7. Check the Beep box to sound a beep as the dialog opens.

8. Click on the drop-down list button to display a list of the Dialog types. Select the desired type.

9. Click on the drop-down list button to display a list of the Message Responses to the text contained in the message. Select the desired response.

10. Click on the drop-down list button to display a list of the Actions that will occur after the response chosen in step 9. Select the desired action. If you selected "Jump to macro step" as your action, continue with step 11. If you do not select this Jump type of action, skip to step 12.

11. For a "Jump to macro step" action only: Type the number of the macro step that the program will jump to, after the message response has been clicked.

12. Enter the Delay Time depending upon the response chosen in step 9 as follows:

  • If "None" is selected as a response in step 9, enter the desired Delay Time that the message dialog will be displayed.

  • If any response other than "None" is selected in step 9, you can enter a nonzero Delay Time to use the response chosen after the entered delay time has passed. You can enter a Delay Time of zero to cause the message to be displayed indefinitely while the program waits for a response to the dialog.

13. Select OK when completed.

 

NOTE: For more details on creating, editing, and running macros, see the Universal Analysis online manual (Help/Manual) or use Acrobat Reader to access the Uamanual.pdf file contained on the CDROM.
