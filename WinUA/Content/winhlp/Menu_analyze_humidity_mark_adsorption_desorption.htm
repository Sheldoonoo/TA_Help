Humidity Mark Adsorption/Desorption

Use the Analyze/Humidity/Mark Adsorption and Analyze/Humidity/Mark Desorption functions to manually specify the beginning of each humidity isotherm on the curve. This is useful when the automatic determination of rising and falling weight does not coincide with programmed humidity levels, or when the sample is conditioned (e.g., by drying or saturation) prior to the start of the experiment.

 

Each marked point will be labeled on the curve as "Adsorption" or "Desorption" curve. For convenience, marked points can be applied while the selected X and Y-axis are other than Relative Humidity and Weight.

 

It is not necessary to mark the beginning of the first isotherm of an Adsorption/Desorption or Desorption/Adsorption cycle. Simply mark the first point of the second isotherm and the program will automatically assume that any points prior to the first marked point are of the opposite sorption type. If multiple Adsorption/Desorption cycles occur in the data, the beginning of each isotherm other than the first must be explicitly marked on the curve. All required points should be marked prior to performing the desired humidity analysis.
