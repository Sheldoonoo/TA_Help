Entering Post Test Options

 

Select the Procedure Page, then click the Post Test button to access the Post Test Parameters window, which is used to define functions that will occur at the end of a run. Method-end conditions are functions that can be automatically executed at the end of a run to increase turn-around time between experiments:

  1. Select the radio button next to the desired option for the TGA Furnace: Open furnace and unload or Closed. Select the option that best suits your needs:

    • Furnace Open and Unload will automatically open the furnace at the end of the method and unload the sample pan on the platform. If an EGA furnace is attached, the furnace will not open until the temperature is below 500°C.

NOTE: The EGA (Evolved Gas Analysis) furnace is designed to allow gases that evolve during sample decomposition within the sample tube to be vented to a spectrometer. The EGA is and optional furnace for the TGA.

    • Furnace Closed leaves the furnace shut at the end of the method. If a standard furnace is attached, Furnace Closed will override the Air Cool option.

CAUTION: During vacuum operation, select Furnace Closed to prevent instrument damage. The vacuum must be released before furnace opens.

This option is grayed when the TGA is operated as an Autosampler (AutoTGA) because the furnace must always open and unload between runs. However, if a run sequence is not currently in progress, you may disable the Autosampler function by accessing User Preferences/ Autosampler Page and unchecking the “Disable autosampler” option.

If you have an Autosampler that is enabled, the Autosampler Stop Options will override the post-test options for the last run in the sequence (or when there is only one run in the sequence).

  1. Check the Air Cool box, if you want to have the furnace open and allow room air to cool the sample for a defined amount of time. This function will open (on) or close (off) the internal solenoid valve connected to the Air Cool connector on the exterior of the instrument.  If you check this option, you must also enter the amount of time for air cooling to last. NOTE:  The standard furnace cannot air cool if the furnace is closed.

  2. Select OK to save settings.
